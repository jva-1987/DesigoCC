Stop an HDB

You need to stop the restored and started HDB to proceed with HDB upgrade.

  1. The HDB status is seen as Outdated.
  2. The History Database is started.
  1. In the SMC tree, select Database Infrastructure > [SQL Server Name] > [History Database name].
  2. Click Stop Automatic Maintenance .
  1. The History Database is stopped. No more maintenance activities are performed on the database. So, make sure that the HDB is always started or else there is a possibility of data loss.
