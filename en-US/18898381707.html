Integrating the Esser Fire System Through Offline Configuration

Scenario

Desigo CC is not yet connected to the Honeywell Esser fire system, and you want to prepare the configuration in offline mode by editing and importing data from an EDE file. For background information, see the reference section.

The offline configuration through an EDE file requires more engineering, but it can address complex requirements that need a non-standard association of Functions.
Performing online discovery is recommended instead if Desigo CC is already connected to the Honeywell Esser fire system, and you do not have to address complex configuration requirements.

Prerequisites

  • The Honeywell Esser extension module is installed and included in the active project.
    The following extension is also selected automatically:
    • DMS Common
  • To be able to work with EDE files, you manually added the following extension:
    • Desigo System (Core)
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
