SAP iPro Configuration

The SAP iPro extension enables Desigo CC to transmit events to the SAP iPro system.

The required configuration includes:

  • Configuration of a NORIS Web Application
  • Configuration of the Event Manager communication parameters
  • Installation and start of the Event Manager service

For instructions, see Configuring and Running the SAP iPro Event Manager.
