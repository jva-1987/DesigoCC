Troubleshooting in Desigo Automation

This section includes information on troubleshooting for Desigo Automation.

Handling Import Error Conditions

Message

Situation

Do the following…

Invalid XML file format

The selected file is not a valid database file from ABT Site.

Create a new XML file and try re-importing it.

Import failed – Unable to create a new data point.
Abort, Continue, Continue All

The import did not recognize one of the objects for import.

If you do not know the exact reason for the missing object type, click Abort to stop the import procedure.

Alternatively, you can choose to Continue and skip the missing object or click Continue All to ignore any further recoverable errors.

In any case, try creating a new XML file and re-importing it. If the problem persists, contact Technical Support.

Multiple networks are present.

The import zip file contains devices from more than one network.

Select the required network to import the data points.

Import not possible. Import data would exceed system limits.

The import function recognized more points than defined on the project. After clicking OK, the import is canceled.

The project is set to Small by default and supports 25,000 data points. Change the project size. NOTE: Only change the project size if your hardware meets the requirements of a larger project.

After the import, system load will be [more than 90%].

The project size of system data points is nearly reached (current system size + the imported size).

Change the project size to import additional data points.

Handling Desigo Automation Driver Configuration Errors

Message

Situation

Do the following…

Invalid Management platform Instance Number

 

Invalid or missing configuration of the Settings expander of the driver’s BACnet tab.

Set the Instance number. See Creating the Desigo BACnet Driver.

The BACnet driver is not fully configured until you add a network connection. Saving without a network connection will allow you to perform XML data (*.s1x)imports, but the driver will not be communicating on the BACnet network.

Invalid or missing configuration of the driver in the BT BACnet Stack Configexpander of the driver’s BACnet tab.

Create the required ports and select a network adapter. See Creating the Desigo BACnet Driver.

The driver cannot de deleted while monitoring the following network: network_xyz.

You are trying to delete a driver which is currently associated to a network.

Delete the corresponding network first and then try deleting the driver again.
NOTE: Deleting a network results in the entire data structure being removed.
