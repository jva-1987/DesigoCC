Configuring Report Parameters

You can customize your report definition by performing the following activities:

  • Adding elements such as tables, plots, keywords, and so on.
  • Configuring the layout of the report by setting orientation, page size, margins, or headers and footers.
  • Formatting the report elements for an enhanced appearance by applying color, adjusting positions of the elements, and so on.
