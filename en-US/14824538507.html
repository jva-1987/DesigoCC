Multiscan Symbols

When integrating the Sensitron Multiscan extension, the following symbols are available in graphics for detectors, relays, and panels objects:

Detector Symbols

DYN_3D_Gas_Detector_Multiscan_None__001_150

  • Default symbol for style 3D_EU in gas detection functions.
  • Symbol background changes color according to the event category color.
  • Also used as default symbol for Multiscan gas detectors object models with style 3D_EU when functions are not associated to the imported detector points.

 

DYN_3D_Gas_Detector_Multiscan_None__002_150

  • Secondary symbol for style 3D_EU in gas detection functions.
  • Symbol background changes color according to the event category color.
  • Also used as secondary symbol for Multiscan gas detectors object models with style 3D_EU when functions are not associated to the imported detector points.

 

DYN_3D_Gas_DetectorCO_Multiscan_None_001_150

  • Default symbol for style 3D in CO gas detection function.
  • Symbol background changes color according to the event category color.
  • Symbol provides the CO concentration value read from the detector.

 

DYN_3D_Gas_DetectorCO2_Multiscan_None_001_150

  • Default symbol for style 3D in CO2 gas detection function.
  • Symbol background changes color according to the event category color.
  • Symbol provides the CO2 concentration value read from the detector.

 

DYN_3D_Gas_DetectorO2_Multiscan_None_001_150

  • Default symbol for style 3D in O2 gas detection function.
  • Symbol background changes color according to the event category color.
  • Symbol provides the O2 concentration value read from the detector.

 

DYN_3D_Gas_DetectorTemp_Multiscan_None_001_150

  • Default symbol for style 3D in temperature detection function.
  • Symbol background changes color according to the event category color.
  • Symbol provides the temperature value read from the detector.

 

DYN_3D_Gas_ValueCO_Multiscan_None_001_150

  • Default symbol for style 3D in CO gas detection function.
  • Symbol provides the CO concentration value read from the detector.

 

DYN_3D_Gas_ValueCO2_Multiscan_None_001_150

  • Default symbol for style 2D in CO2 gas detection function.
  • Symbol provides the CO2 concentration value read from the detector.

 

DYN_3D_Gas_ValueO2_Multiscan_None_001_150

  • Default symbol for style 2D in O2 gas detection function.
  • Symbol provides the O2 concentration value read from the detector.

 

DYN_3D_Gas_ValueTemp_Multiscan_None_001_150

  • Default symbol for style 2D in temperature detection function.
  • Symbol provides the temperature value read from the detector.

Relay Symbol

DYN_3D_Single_Output_None_Horizontal_001

  • Default symbol for style 3D_EU in relay object models.
  • Symbol background changes color according to the event category color.
  • Inherited from Fire_Detection_HQ library.

Panel Symbols

DYN_3D_Gas_Panel_Multiscan_Horizontal_001_150

  • Secondary symbol for style 3D_EU in the Multiscan panels object models.
  • Symbol background changes color according to the event category color or Summary Status color propagated from the points at lower levels.

 

DYN_3D_Gas_Panel_Multiscan_Horizontal_002_150

  • Default symbol for style 3D_EU in the Multiscan panels object models.
  • Symbol background changes color according to the event category color or Summary Status color propagated from the points at lower levels.
