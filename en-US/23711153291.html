Defining Values for Heating Curve Graphic

  1. The heating curve configuration is created and selected.
  1. In the Settings expander, select the Function, and in the drop-down list the CharCnf option.
  2. Click Save .
  3. In the Operation tab, enter a value for each of the properties and click Change:
    • Y Max (for example, 45)
    • Y Min (for example, 22)
    • Y Precision (for example, 1)
    • X Max (for example, 20)
    • X Min (for example, -18)
    • X Precision (for example, 1)
  4. Repeat steps 1-3 for each heating curve configuration.

NOTE:
The values for Y precision and X precision determine the decimal points displayed (for example, 1 = 20.1)
