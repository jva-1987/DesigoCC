Lite User Interface Danger Management System

Scenario

You want to define a user who operates with the DMS Lite UI (simplified) user interface.

For background information, see DMS Lite UI - DMS_EN Client Profiles.

Prerequisites

  • One or more users have been configured in a dedicated group (for example DMSLiteUIusers).

 

1 – Configure the Application Rights of the Group

  1. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Security.
  3. Select the [DMSLiteUIusers] group from the Groups list.
  4. Open the Application Rights expander.
  5. Expand the System Management section.
  6. Deselect Toggle Engineering Mode.
  7. Click Save .

 

2 – Configure the Client Profile of the User

  1. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Users.
  3. Click the Users tab.
  4. Select a user of the [DMSLiteUIusers] group from the Users list.
  5. Open the User Settings expander.
  6. In the Client profile drop-down list, select DMS_Lite_UI.
  7. Click Save .
  8. The profile is associated to the user.
  9. Repeat steps 4 to 7 for the other users of the [DMSLiteUIusers] group.

NOTICE

Never apply a DMS Lite UI client profile to all users or to the main server station. This would prevent you from accessing the engineering functions again.
