Zenitel AlphaCom Symbols

When integrating the Zenitel AlphaCom extension, the following symbols are available in graphics for the AlphaCom server and station objects:

Server Symbol

DYN_3D_AlphaCom_Server_None_None_Horizontal_001_150

  • Default symbol for style 3D_EU in the AlphaCom server object models.
  • Symbol background changes color according to the event category color or Summary Status color propagated from the points at lower levels.

Station Symbols

DYN_3D_AlphaCom_Station_None_None_Horizontal_001_150

  • Secondary symbol for style 3D_EU in the AlphaCom station object models.
  • Symbol background changes color according to the event category color or Summary Status color propagated from the points at lower levels.

DYN_3D_AlphaCom_Station_None_None_Horizontal_002_150

  • Default symbol for style 3D_EU in the AlphaCom station object models.
  • Symbol background changes color according to the event category color or Summary Status color propagated from the points at lower levels.

 

The blinking effect on symbols is not supported in this integration.
