[Concept 3] Archive with Custom Storage Periods

Scenario
You want to create custom archive groups and associate data points with these groups.

1 – Create Custom Archive Groups

  1. System Manager is in Engineering mode.
  2. System Browser is in Management View.
  3. Custom Long Term Storage groups with monthly or yearly time slicing intervals are created in SMC.
    NOTE: If a storage is full before the periods ends, the system automatically creates a new storage. In that case you will have more than one storage for that period.
  1. Select Project > Management System > Servers > Main Server > History Database > Archive Groups Custom.
  2. Click the Object Configurator tab.
  3. Click New and select New HDB Custom Archive Group.
  4. In the New object dialog box, enter the name and description.
  5. Click OK.
  6. In System Browser, select the newly created custom archive group.
  7. In the Operation tab, select the Archive property.
    a. In the drop-down list, select the desired storage.
    b. Click Set.
  8. Repeat steps 2 to 7 for all needed custom archive groups.

2 – Assigning Data Points to Archive Groups

  1. The custom archive groups are created and set to Archive.
  1. Perform either of the following:
    • Select Project > Field Networks > [network] > Hardware > [device] > [data point]. The folder path may vary by view or subsystem.
    • Select multiple objects for bulk engineering (See Bulk Engineering Objects).
  2. Click the Object Configurator tab.
  3. To assign datapoints to Activity and Events custom archive group, do the following:
    a. In the Main expander, select the Archive group field.
    b. In the drop-down list, select the desired archive group.
    c. Click Save .
  4. To assign datapoints to Value custom archive group, do the following:
    a. In the Properties expander, select the desired property, for example, Present Value.
    b. Select the VL (data values, see Log Data Values in Logging Data Values or Events) check box or/and the AL (events, see Log Events in Logging Data Values or Events) check box.
    c. Select the Archive group column.
    d. In the drop-down list, select the desired archive group.
    e. Click Save .
  5. Repeat steps 1 to 4 for all needed data points.

3 – Set the Retention Time for Default Archive Groups

  • Define the retention period (refer Default: Deletion After Maximum Retention Period section) for all unassigned data points.

NOTE:
The steps mentioned above are applicable for new data logged into the system.
For data already logged into the system following steps additionally need to be performed:
1. In System Browser, select the newly created custom archive group.
2. In the Operation tab, select the Archive property.
- a. In the drop-down list, select the storage to None.
- b. Click Set.
- c. In the drop-down list, select the desired storage.
- d. Click Set.
3. Repeat these steps for all needed custom archive groups.
