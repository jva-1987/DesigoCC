Change Notification Template State

This section describes procedure to active/inactive notification template. By default the message template is active. When the message template is inactive, it does not get launched when the event is triggered.

Change Notification Template State using Command

  1. The message template which you want to activate is created and configured.
  1. Select Applications > Notification > Notification Templates.
  2. Select the configured message template to change its state.
  3. In the Operation tab, navigate to the State command and click the appropriate state.
    • Click Active to receive the notification when the event is triggered.
    • Click Inactive when you do not want to receive the notification when the event is triggered.
  4. Click Save as.
  1. The message template state is changed.

Change Notification Template State using Reaction

  1. The message template which you want to launch is created and configured.
  1. Select Applications > Logics > Reactions.
  2. The Reaction Editor tab displays.
  3. Open the Output expander and then click the Action expander.
  4. From the System Browser, navigate to Applications > Notification > Notification Templates.
  5. Drag a Message template, for example, M1, to the Action expander.
  7. From the Property drop-down, select state [IsinactiveTemplate] option.
  8. In the Command drop-down, select the appropriate state.
  9. Click Save As and enter a name and description for the reaction and click OK.
  1. The message template state is changed.

Change Notification Template State using Macros

  1. The message template which you want to launch is created and configured.
  1. Select Applications > Logics > Macros.
  2. The Macro tab displays.
  3. From the System Browser, navigate to Applications > Notification > Notification Templates.
  4. Drag a Message template, for example, M1, to the Macro tab.
  5. From the Property drop-down, select state [IsinactiveTemplate] option.
  7. In the Command drop-down, select the appropriate state.
  8. Click Save As and enter a name and description for the macro and click OK.
