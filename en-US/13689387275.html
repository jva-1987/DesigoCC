Modbus Library

The Global Modbus System library is required in your management platform to support Modbus connectivity.

This library includes the objects that contain Modbus related data types, Managed Types, AutoStartModbusDriver, standard object models, and Texts supported by the current version of the management platform. You can find the discipline specific Modbus libraries by selecting BA > Device and Fire > Device libraries. In order to find the name of the Modbus library, see Verifying the Modbus Library.

It also includes the Import Rules for mapping the data type of Modbus objects specified in the CSV file to the protocol specific Modbus type and transformation type associated with it.

For information on naming conventions for names, paths, and other elements, see Common Rules for Names and Paths in the Management System and Naming Rules for Subsystem Elements in Naming Conventions.
