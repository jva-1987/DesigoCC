Text Group Editor

This section provides step-by-step instructions for Text Group Editor tasks. For background information, see the reference section.

Complete the following procedures as needed:

Customizing Text Groups

  1. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. (Example) Select Project > System Settings > Libraries > L1-Headquarter > Global > BACnet > Texts > BACnet Alarm Values.
    NOTE: After L1-Headquarter, the path to the text group you want to customize will probably be different from the path in the example. There are many selections in the Global folder, and there are many possibilities for the discipline folders as well—BA, Fire, Security, and so on. This means that you will need to substitute some folder names in the path to suit your needs.
  3. In the Text Group Editor tab, select Customize .
  4. Click OK.
  1. The BACnet Alarm Values text group is cloned under the allowed customization level and can now be modified.

Adding a Text Group

  1. You have access rights to the library in which you want to add a text group.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L4-Project > [path to Texts or Text Groups node].
  3. Select the Text Group Editor tab.
  4. Complete the fields in the default text group row. Enter the display text in the language-specific field(s)—for example, en-US.
  5. NOTE: You must enter a value in the Value field for each row. Otherwise, you cannot add a new row.
  6. If you need more than one row, click Add new row .
  7. When you have finished adding rows, click Save .
  8. Complete the fields in the Save Object As dialog box. The Name field is mandatory and must be unique in the system.
  9. NOTE: When you save a new Text Group, it is saved to the library at the current Customization Level. If customization has not been enabled for the selected library at the current Customization Level, the Add new group and Save As icons are not available.
  10. NOTE: When you save a new Text Group, you must follow specific syntax rules:
    - The Object name and Name must use the TxG_ prefix.
    - After the prefix, you can add a descriptive label of your choice.
  11. Click OK.

Deleting a Text Group

  1. You have write access to the text group.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L4-Project > [path to Texts or Text Groups node] > [name of group].
  3. Select the Text Group Editor tab.
  4. Click Delete group.
    NOTE: If the Text Group is at a library level above the current Customization Level, the Delete group icon is not available.
  5. Click OK.

Adding a Row to a Text Group

  1. System Manager is in Engineering mode.
  2. You have write access to the text group.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L4-Project > [path to Texts or Text Groups node] > [name of group].
  3. Select the Text Group Editor tab.
  4. Click Add new row .
    NOTE: If you receive an error after trying to add a new row, make sure you update all the values in existing rows before adding a new row.
  5. In the new row, add a Value, a Color, an Icon, and a descriptive text entry for each supported language.
    NOTE: You must enter a value in the Value field before you can save the entry. Colors and icons are optional, but once assigned, you can copy and paste them from one row in the text table to another.
  6. Click Save .

Deleting a Row from a Text Group

  1. You have write access to the text group.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L4-Project > [path to Texts or Text Groups node] > [name of group].
  3. Select the Text Group Editor tab.
  4. Select the row you want to delete.
  5. NOTE: If there is data in the row from customization levels higher than your current customization level, that data remains after you delete the row. For example, your current customization level is Project. You have data in the first row of the text list, and both the HQ and PR check boxes are checked. After you delete the row, only the HQ check box remains checked, which means the data at that level is still preserved.
  6. Click Delete selected row .
  7. Click OK.
  8. Click Save .

Creating a New Text Group from an Existing Text Group

  1. You have write access to the text group.
  2. You want to copy all of the settings from one group for use in another group.
  3. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L4-Project > [path to Texts or Text Groups node] > [name of group].
  3. Select the Text Group Editor tab.
  4. Click Save As .
  5. NOTE: When you use the Save As feature, the text group is saved to the library at the current Customization Level or lower. If customization has not been enabled for the selected library at the current Customization Level, the Add new group and Save As icons are not available.
  6. In the Save Object As dialog box, select the location for the text group. You can only select locations that are bolded.
  7. Complete the remaining fields. The Name field is mandatory and must be unique in the system.
  8. Click OK.
  9. The new Text Group displays in System Browser, and you can now add or delete rows in the new Text Group, or edit an entry in the text list.
