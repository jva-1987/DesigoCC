Edit Panels and Groups in Node Map

In the Edit mode of Node Map, you can modify groups as well as rename panels.

 

Prerequisites

  • You are enabled to configure the Node Map (user Application Rights).

 

Group Panels

  1. In the Node Map, click Edit mode .
  2. Select the panels you want to group. Hold CTRL key pressed when you click the panels to allow for a multiple selection.
  3. Click Group .
  4. In the dialog box that opens, enter the group Description.
  5. The new group node displays. Next to it, a dashed-line frame shows the panels that belong to the group.
  6. Click Edit mode again to quit edit mode.

Ungroup Panels

  1. In the Node Map, click Edit mode .
  2. Select the node of the group you want to remove.
  3. Click Ungroup .
  4. The group node disappears.
  5. Click Edit mode again to quit edit mode.

Remove a Single Panel from a Group

  1. In the Node Map, click Edit mode .
  2. If the group is collapsed, expand it by double-clicking the group node.
  3. Select the panel you want to remove.
  4. Click Ungroup .
  5. The group is updated.
  6. Click Edit mode again to quit edit mode.

Rename a Panel

  1. In the Node Map, click Edit mode .
  2. Select the panel you want to rename and press F2.
  3. Enter the new name.
  4. Click Edit mode again to quit edit mode.
