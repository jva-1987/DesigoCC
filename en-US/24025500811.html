Associate Doors (SiPass Integration)

Scenario: To monitor the door states (SiPass Integration), Desigo CC objects must be manually associated with the BIM object. The association must be entered manually since the naming is generally different between SiPass data and the BIM object.

  1. You imported the BIM data files.
  2. You imported a CSV data.
  1. In the BIM viewer object tree, select …\[Doors] > Object.
  2. In System Browser, select Management View.
  3. Select Project > [Network name] > […] > [Devices] > [Group] > [Device] > [Door].
  4. Drag the door object to the corresponding BIM hierarchy door.
  5. The Add associations dialog box displays.
    NOTE: This settings are not relevant to SiPass.
  6. Click OK.
  7. The associations are created and are displayed with symbol and bold face text.
  8. Click Save .
