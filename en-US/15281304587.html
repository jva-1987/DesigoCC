Configuring Operator Tasks Templates

  1. System Manager is Engineering.
  2. System Browser is in Management View.
  3. You created a new library manually or customized an existing library.
  4. At least one JSON file for configuring templates was created by librarians. For details, see Operator Tasks Template Configuration (JSON) File.
  1. Select Project > System Settings > Libraries > L4-Project or L3-Region or L2-Country >[…] > [library].
  2. Click New , and select New Operator Tasks Template Folder.
  3. In the New Object dialog box, enter a name and description then click OK.
  4. Select Project > System Settings > Libraries > L4-Project or L3-Region or L2-Country > […] > [library] > Operator Tasks > Templates.
  5. Click New , and select New Operator Tasks Template.
  6. In the New Object dialog box, enter a name and description and click OK.
  7. The new tasks template is added to System Browser under the selected library and automatically selected.
  8. To configure the new template, in the General Settings expander of the Tasks tab, specify the following settings:
    • Template name: Enter a meaningful name for the template. This is the name that will appear among the templates that operators can use to create a new task.
    • Task name: Enter a meaningful name for the task. This is the text that by default displays when operators create a task from a template.
    • Task description: Enter a brief description for the task. This text is displayed in a tooltip that by default appears when operators hover over the template in the list. The same text also displays in the Description field when a task is created from template.
      NOTE: To help the operator to set consistent task targets, use this field to specify the intended purpose of this specific template, and the kinds of objects to which it applies. Additionally, if the template’s configuration data also contains scaled values, add the appropriate unit of measurement to the task description. For example: This template is intended for tasks that will handle coil valves. It applies only to system objects relating to heating coil valves whose unit of measurement is %.
  9. In the Configuration file field, specify the configuration:
    a. Click Browse.
    b. In the Open dialog box, select the appropriate JSON file.
    c. Click Open to import the configuration.
  10. The template configuration data displays in the text box below the settings.
    NOTE: The imported configuration is read-only.
  11. Select the Enabled check box to activate the tasks template and make it available among the templates.
  1. Operators in Operating mode will be able to create a new task from this template by clicking Add in the Tasks tab in Application View.
