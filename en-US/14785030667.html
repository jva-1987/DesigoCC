Integrating CEI-ABI

Scenario

You want to configure Desigo CC to operate on devices that use the CEI-ABI protocol. For background information, see CEI-ABI Integration.

Prerequisites

  • The CEI-ABI extension is installed and included in the active project. The following extensions are also selected automatically:
    • Security Domain Libraries
    • SORIS Driver
  • The CEI-ABI adapter is running and connected to the CEI-ABI system. For instructions, see Installing and Starting the CEI-ABI Adapter.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Licensing

This extension is covered by license. Check that the required license is available; otherwise, you may not be able to configure and work with this extension.
