Create, Configure, and Save an Imported Device Type

You can create a new imported device type in an area or within a sector that you have created. Alternatively, you can copy an existing device.

  1. Select Applications > Powermanager > [area] OR [sector].
  2. Click the Area/Sector tab.
  3. Click Create and select Create Device.
  4. A list of device groups display in a tabular format. By default, Power Devices is selected.
  5. Select the Third Party Devices tab.
  6. The list of available third party device types is displayed.
  7. Select the third party device type and configure the parameters for the device by entering information in the expanders in the Area tab.
  8. Click Save .
  9. The Save Object As dialog box displays.
  10. Enter a name and description for the device. Do not use the characters “ / ” or “ \ ” in the device name.
  11. The device is created and configured and is added to the System Browser under the selected area or sector.
  12. Navigate back to Applications > Powermanager.
  13. The System tab displays.
  14. Configure the following expanders in the System tab, if required.
    • Favorites
    • Poll Groups
    • Synchronization
    • Tariff
  15. Click Save .
  1. The third party device configuration is complete.
