Set the XML Configuration File of a Web Links Library

  1. You imported a library that contains the Web Links library block or you have created a new web application library block and a new web application library link.
  2. An XML configuration file for communicating with an external web application is available. Also, you removed any formatting in this file so that it appears in one line.
  3. You are authorized to handle libraries for a specific customization level (L1-Headquarter, L2-Region, L3-Country, or L4-Project).
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L1-Headquarter or L2-Region or L3-Country or L4-Project > [library] > [web links].
  3. Select the Extended Operation tab, and do the following:
    a. Copy the content of the XML configuration file and paste it into the field next to the XMLFile property. This field is limited to 2050 characters and both HTTP and HTTPS protocol are supported.
    b. Click Set.
