Initiating an Incident and Sending Notification from Mobile App

Scenario
You want to initiate an incident and launch a message from the Mobile App.
For background information, see Setting up the Mobile App Client.

Prerequisites

  • You have downloaded the Desigo CC mobile app (Desigo CC published by Siemens AG) from the Google Play Store (for Android) or from the Apple App Store (for iOS). The mobile app is a free downloadable app available for Android and iOS smartphones and tablets.
  • The Desigo CC Mobile App is connected to the Management Platform server. (see Setting up the Mobile App Client )
  • You have obtained the corresponding root certificate file (for example, WebservicesCA.cer) and installed it on your mobile device, if the Web Service application uses a private CA certificate. However, if the Web Service application of the management platform uses a public CA certificate, the mobile device will automatically recognize it and there is no need to install a certificate file.
  • In System Browser, Application View is selected.
