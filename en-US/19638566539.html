Emphasize Lighting Objects and Hide Furniture

Scenario: Highlight all switched-on lighting objects in color and hide the furniture.

 

1st Step: Find and Hide the BIM Object Model for Furniture

Scenario: You want to find the BIM object model for furniture.

  1. Edit is enabled in BIM Viewer.
  2. Desigo CC Objects and BIM objects are associated.
  3. A Lighting filter with a condition >0 is available.
  1. Select a piece of furniture with the mouse.
  2. Press SHIFT + right click.
  3. Select Hide this element and then to filter: Lighting.
  4. In the Filter Name dialog box, select the Lighting filter.
  5. Click Edit.
  6. Displays the associated BIM object. The IFC Name column reference the select furniture object. The BIM object model IFCFURNITURE displays in the IFC Type column. Lighting information is displayed in section Highlight associated objects.
    • Test: Clear Highlight selected object .
    • Test: Select the Lighting check box.
  7. The lighting configured in the filter is highlighted in color and the selected desk is hidden. The select chair remains visible.
  8. Click Save .

 

2. Step: Hide all furniture and highlight switched-on lighting objects

Scenario: After finding the BIM object model for a piece of furniture, all pieces can be hidden belonging to this object.

  1. In the Filter Name dialog box, select the Lighting filter.
  2. Click Edit.
  3. Displays the associated BIM object. The IFC Name column reference the select furniture object. The BIM object model IFCFURNITURE displays in the IFC Type column.
  4. In section Highlight associated objects, select BIM object model.
  5. Click Edit.
  6. The Emphasized objects dialog box displays.
  7. Delete the entry in the IFC Name text field and enter an asterisk (*).
  8. Click OK and then OK to close both dialog boxes.
    • Test: Select the Lighting check box.
  9. All furniture is hidden and switch-on lighting objects are highlighted in color.
  10. Click Save .
