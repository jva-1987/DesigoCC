Connecting the Configurator Tool to the Adapter

  1. In the Siemens.GMS.CS6.Configurator tool page, click Go online.
  2. The configurator tool is now connected to the adapter.
  3. In the Connect to Adapter dialog box, do the following:
    a. In the IP field, enter the IP address of the machine hosting the adapter.
    b. In the Port field, enter the appropriate value (default communication TCP port is 50100).
    c. Click Connect.
    NOTE: A message box informs you if a configuration already exists. Click Yes to override the existing active configuration with the adapter configuration. Then click OK.
  1. If the connection is established successfully, the adapter status is (Connected).
