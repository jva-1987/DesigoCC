Check the Consistency of a Web Application

  1. You have edited the StartSmc.bat file under [installation drive:]\[installation folder]\GMSMainProject\bin and added the /support switch.
  2. You have relaunched SMC.
  1. In the SMC tree, select the Websites > [website] > [web application].
  2. Click Check Consistency .
  3. The system internally checks the website, web application, and project parameters and displays a message asking if you want to open the log file now.
  4. Click Yes.
  1. A log file opens in the default editor. The log file is also saved under
    [installation drive:]/[installation folder]\GMSMainProject\log in the format [web application name]_[DD MM YYYY]_[HHMMSS].
