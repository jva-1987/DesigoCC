Search for Information

  1. Select the Traces tab.
  2. Select Edit > Find.
  3. The Find dialog box opens.
  4. Enter the desired search term in the text field.
  5. Click Next.
  6. The data entry found first is displayed.
  7. Click Next if you want to find more data entries.
