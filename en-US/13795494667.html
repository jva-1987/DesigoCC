Main Window

 

 

 

Description

1

Menu Items
Provides access to the Browser, View, Services, and Help menu commands.

2

Toolbar Buttons
Provides access to various functions: Print, Save As, Previous/Next Navigation of Views, Clear Errors, Re-read Object List, Refresh, Stop, Write Property, Send Who-Is, and Send Who-Has.

3

Find
The Find field allows you to enter key words in order to search the entire tree for the next match of the criteria you entered. Press ENTER or F3 to start the search.

4

View
Allows you to choose different views depending on what is selected in the system tree. For example, if you select a field panel in the system tree, the default view is View Device Properties. However, in the View field you can click the drop-down arrow and select View Object Log to see a point log of all objects contained in that device. If there are any list properties in the right pane, the appropriate view will be available in the drop-down list to select and expand the list.

When browsing an item that is in a list format, for example a Recipient List, the BACnet Object Browser indicates how many items are in that list. Double-click the item to expand and view the list.

5

Filter
Displays the information in the BACnet Object Browser window by entering key words in the Filter field. For example, if you only want to view the object name for any of the object types listed, type object-name in the field, and only the object name is listed. The default is a wildcard symbol of *, which displays all the property information for an object type.

6

System Tree
Displays the hierarchy of the BACnet network. The root is the entire network, followed by the system name and driver number, followed by devices on the network, followed by folders of groups of objects of the same type, and finally, followed by the objects themselves.

7

Data List
Displays data about the item selected in the system tree.

8

Error / Message
Displays errors and messages related to the work you do in the browser.
