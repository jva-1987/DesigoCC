Copying Desigo Insight Log Data to Project

  1. Select [Installation Drive]:\[Installation Folder]\[Project Name].
  2. Create folder DI_LOG in your Desigo CC project. If you back up the project data, Desigo Insight log data is also saved to the backup file and is therefore not lost.
  3. In the Desigo Insight project.zip file, select the data\Log folder.
  4. In Windows Explorer, copy all files in the Log folder (*.xml and *.csv) to the DI_LOG folder for your project Desigo CC.
