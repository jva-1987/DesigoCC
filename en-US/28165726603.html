Alarm Mapping

The alarm categories in Desigo CC are distinct from those in Building X. As a result, it is necessary to provide a mapping between the two alarms systems. Default mappings can be found in the file BxCategoryMap.json in the data folder of your project (for example, \GMSProjects\<project name>\data\BldgXConnector\BxCategoryMap.json). It is possible to change these mappings by modifying the values in this .json file. However, before making any changes, you should consult the README file in this same directory (BxCategoryMapReadme.txt).
