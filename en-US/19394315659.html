Changing the Snapshot Deletion Time

When automatic removal of snapshots is enabled (see Automatically Delete Snapshots After a Retention Period), deletion of expired snapshots takes place at midnight (00h) and midday (12h) by default. This means that, at midnight and noon of each day, the system checks the snapshot images stored on the Desigo CC server and deletes those that are older than the Snapshot Retention Period configured in the properties of the main Video node.

If you want to change the deletion time, you must first customize the library to make visible the Snapshot Deletion Time property. Then you can use this property to set a different time.

Make the Deletion Time Property Visible

  1. You are trained and authorized to configure libraries.
  2. System Manager is in Engineering mode.
  3. System Browser is in Management View.
  1. In System Browser, select Project > System Settings > Libraries > L1 Headquarters > Video > Video > Object Model > Management.
  2. In the Models and Functions tab, click Customize .
    NOTE: If the icon is dimmed, it means the object model has already been customized to your allowed level. Skip to step 4.
  3. Click OK.
  4. The Management object model is added at your allowed customization level.
  5. In System Browser, select the Management object model at your allowed customization level. For example:
    ...> L4 Project > Video > Video > Object Model > Management.
  6. In the Models and Functions tab, open the Properties expander.
  7. In the property list, select the Deletion Time property.
  8. Select the DL3 check box for this property, to make it visible in the Extended Operation tab.
  9. Click Save .
  10. Stop and restart the Desigo CC project for the customization to take effect.

Change the Deletion Time

  1. System Browser is in Application View.
  1. In System Browser, select Applications > Video.
  2. In the Extended Operation tab, locate the Snapshot Deletion Time property.
  3. Enter the time of day when you want expired snapshots to be deleted.
    Values can range from 00 (midnight) to 23 (11 pm) in increments of one hour. Snapshots will be deleted at 12 hour intervals starting from the time you specify here.
  4. Click Set.
