Integrating Riello UPS

Scenario

You want to configure Desigo CC for connecting to the Riello UPS device. For background information, see the reference section.

Prerequisites

  • The Riello UPS extension is installed and included in the active project.
  • The SNMP service is enabled, configured, and running in Windows. For instructions, see Configuring Windows SNMP Service.
  • The configuration template file for the device you want to integrate is available in the SNMPDataTemplate folder. See Riello UPS Configuration (SNMP) File.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Licensing

This extension is covered by license. Check that the required license is available; otherwise, you may not be able to configure and work with this extension.
