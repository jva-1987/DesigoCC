Configuring Log Data

Scenario

You want to synchronize Desigo Insight log data with Desigo CC project data to generate a Desigo Insight archive report.

Prerequisites

  • The Advanced reporting extension is installed and configured.
  • The Desigo Insight Migration extension is installed.
  • The export packet from Desigo Insight is created and available as a data source.
    NOTE: The Desigo Insight Migration extension may only be uninstalled, if:
    - You no longer need the data in Desigo CC
    - You created the data as a PDF file for further archival.
