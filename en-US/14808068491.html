Combined events

Because the Onyx panel also supports combined conditions (for example, Zone Alarm + Part Disabled), the integration into Desigo CC also provides combined event texts to cover these cases although not specified in the list above.

Analog Values 4-20ma Modules

Analog values for 4-20ma Modules are displayed in the Analog Value property in the Operation tab.
