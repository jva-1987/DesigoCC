Create and Activate Maintenance Objects

  1. Select the newly created Maintenance folder.
  2. Select the Object Configurator tab.
  3. Click New and select Maintenance Aggregator.
  4. In the New Object dialog box, enter a name and description for the panel/device, and click OK.
  5. The newly created maintenance object is added to System Browser.
  6. Repeat steps 3 and 4 to create all the required maintenance objects (panels/devices).
  7. By default, maintenance objects are inactive. To activate those objects do the following:
    a. Select each object.
    b. In the Operation tab, next to the [maintenance object] name, click Set.
