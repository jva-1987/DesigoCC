Scheduling Cache Generation

The information related to energy consumption is stored in a cache. The generation of cache happens on a daily basis and the new consumption details of all the meters are appended to the existing consumption details. Perform the following steps to generate cache.

You must first create and configure a link.

  1. System Manager is in Operating mode.
  2. You have configured the Configuration Page - Advanced Reporting.
  1. In System Browser, select Application View.
  2. Select Applications > Advanced Reporting.
  3. Click the Application Viewer tab, and select the Job Create Cache report from the menu options.
  4. The Job Create Cache report displays.
  5. Click Save As .
  6. In the Save Object As dialog box, enter a destination.
  7. In the Links folder, the saved link displays.
  8. Select the saved link.
  9. Click Set Credentials .
  10. In the Web Viewer dialog box, you must specify the credentials of the user. This user has the access to all the objects and properties.
  11. In the Extended Operation pane, specify the following values for the P2 and P3 properties:
    P2 – pdf
    P3 - all
  1. The link is created. You can now associate it with a reaction.

Once the link is created, you can now associate the link with a reaction.

  1. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Logics > Reactions.
  3. Click the Reaction Editor tab.
  4. Expand the Output expander and then expand the Action expander.
  5. Select the link to be associated with the reaction.
  6. Drag the link to the Action expander.
  7. Select the Execute option in the Command column.
  8. In the Section drop-down list, select caching.
  9. Open the Triggers expander and then the Time and Organization Mode expander.
  10. To specify the date and time for the generation of the data cache, enter values into the following fields of the Time and Organization expander:
    From date: Any
    To date: Any
    Set Start Time: The time when you want to start the cache generation (For example, 01.00 AM)
    Effective days: Occurs all days
  11. Click Save As .
  12. The reaction is now configured with details of the link to be executed.
