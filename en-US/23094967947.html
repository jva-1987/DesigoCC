Historized Offsets Dialog Box

The Historized Offsets dialog box allows you to manually historize the offset value.

Historized Offsets Dialog Box

 

Description

Valid From

Date and time from when the offset of a managed meter is applicable.

Offset

Updated offset value.

Cumulative Offset

Sum of the offset values.

Save

Saves the new offset value along with the data and time from when it is applicable.

Edit

Allows you to edit the Offset and Valid From fields.

Delete

Deletes an existing offset entry.
