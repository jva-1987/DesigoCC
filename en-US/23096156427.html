Load the Configuration from (CSV) File

  1. On the Notifier AM configurator tool page, select System on the right.
  2. In the Control panel CSV file items section, browse the file system for AdditionalSW\CEI_NotifierAM_Configurator, and select the CEIConfiguration.csv file.
  3. Click Load.
  2. The System tree on the left displays the Notifier AM configuration loaded from the CSV file.
