Adding the Schindler Subsystem

  1. Project extensions for other subsystems or profiles can be added to the project after the fact using extension modules, provided the corresponding extension module is available or installed after the fact.
  1. Start the System Management Console .
  2. Click Stop Project .
  3. Click Add to project .
  4. Select the Schindler Extension Module and click OK.
  5. The extension module is added to the project.
  6. Click Start Project .
  1. The project starts running.
