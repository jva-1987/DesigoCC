Upgrading Project With Powermanager Modbus EM

  1. Open SMC.
  2. Stop project.
  3. Click Add .
  4. Select Project Extensions dialog box displays.
  5. Select Powermanager Modbus Adaptor.
  6. Click OK.
  7. Save project.
  8. Start project.
