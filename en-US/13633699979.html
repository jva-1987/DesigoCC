Associating a Graphic to a System Browser View

You can associate a graphic to a System Browser view, so that it displays whenever the operator selects the root node of the view. You can do this for any System Browser view.

  1. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Graphics > [graphic].
  3. The graphic opens in Graphics Editor.
  4. In System Browser, select the Manual navigation check box.
  5. From the drop-down list, select the view (for example, Application View) that you want to associate with the graphic.
  6. Select the root node of the view, that is, the top item of the tree structure, and drag-and-drop it to the graphic.
  7. The icon displays on the graphic.
  8. (Optional) If you do not want the icon to display on the graphic:
    a. Select the icon on the graphic.
    b.     In the Symbol Instance Properties view, under Layout, clear the Visible check box.
  9. Click Save .
  1. The graphic displays anytime the root node is clicked when in Operating mode.
