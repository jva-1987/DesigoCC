Assigning Software Account to PowerQuality

  1. You have created a Software Account. Refer New Software Account User.
  2. You have added PowerQuality and IEC61850 extension modules to project.
  3. System Manager is in Engineering mode.
  4. System Browser is in Application View.
  1. Select Application View > Powermanager.
  2. Navigate to System tab, and Software Account Configuration expander.
  3. Select Software account from drop down.
  4. Click Save .
  1. Software Account is assigned to PowerQuality.
