Configure the Trend View Definition

  1. You have created a trend view definition and it is present in the Trends folder in the System Browser. (see Create a Trend View Definition in Additional Trend View Definition Procedures)
  • Configure the trend view definition by performing the following activities:
    • Define Chart Properties
    • Define the Axis Properties
    • Position the legend
    • Define the Line Properties
    • Display Markers
    • Display Measured Values
    • Display Quality Attributes
    • Position the Y-Axis
    • Edit background or line colors
    • Show or Hide Columns
  1. The trend view definition is configured as per the specified settings.
