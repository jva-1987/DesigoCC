User Variables

This section describes procedures for creating, updating, and deleting a User Variable.

Create a User Variable

  1. Select Applications > Notification > Notification Templates.
  2. The list of notification templates displays.
  3. Select the required Message Template > Content Editor.
  4. Open the Text Content expander.
  5. Open the User Variables expander.
  6. Click Add.
  7. The User Variables dialog box displays.
  8. Enter the Name and Description of the user variable.
  9. Select the Data Type of the user variable from the drop-down list.
    NOTE: The Default value of the user variable depends on the selected data type. For example, if the selected data type is Time, then the user is prompted for a default time value along with the display format. Please refer to the Data Types table below for more information on data types.
  10. Select or enter the corresponding Default value.
  11. Select Global check box to share the user variable and make it available for use in other message templates.
    NOTE: If the check box is not selected, it means the user variable is private to the current message template. A user variable can also be made global at a later point in time.
  12. Click Save.
  1. The user variable is saved.

Update a User Variable

  1. At least one message template with user variables is configured.
  1. Select Applications > Notification > Notification Templates.
  2. The Notification Templates list displays.
  3. Select the message template that contains the user variables to be updated.
  4. Select the Content Editor tab.
  5. Open the Text Content expander.
  6. Open the User Variables expander.
  7. Select the variable to be updated.
  8. The details for the selected variable displays.
  9. Click Edit.
  10. Update the required fields.
    NOTE 1: When editing a user variable marked as Global, any applied changes will affect all other message templates that contain the updated user variable.
    NOTE 2: For more information on the fields, refer to the Creating a User Variable.
  11. Click Save.
  1. The user variable is saved with the updated details.

Delete a User Variable

  1. Select Applications > Notification > Notification Templates.
  2. The Notification Templates list displays.
  3. Select the message template from which the user variable needs to be deleted.
  4. Select the user variable to be deleted.
  5. Click Delete.
  6. A confirmation message displays.
  7. Click OK.
  1. The user variable is deleted.
    NOTE 1: If a user variable that is not used in any message template is deleted, the corresponding user variable is deleted from the system.
    NOTE 2: If a user variable that is marked as Global is deleted from a message template, then the scope of the user variable is changed from global to template-specific, which means that the user variable is removed from the current message template, but remains as a template-specific user variable in all other message templates that use it.
