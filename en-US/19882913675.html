Roger RACS5 Integration

The Roger RACS5 extension allows integrating the Roger RACS5 system into Desigo CC.

To integrate the Roger RACS5 adapter, a SORIS driver and a SORIS network (monitored by one SORIS driver) must be created and configured.
Once the adapter object is created, it starts automatically as a Windows service using the HTTP or HTTPS protocol.
You can also start the adapter manually:

  • Using the HTTP protocol (unsecure): double-click the executable Siemens.Gms.RogerAdapter.exe available in the file system with the installation.
  • Using the HTTPS protocol (secure): enter –secure:[thumbprint] by command line.

For each adapter created, it is necessary to configure the URL IP address of the computer from which the network adapter is run.

For instructions, see Integrating Roger RACS5.

Dependencies

The Roger RACS5 extension is configured to depend on the following extension, which will be automatically added, if it is not already installed in the system, when the Roger RACS5 installation feature is selected:

  • SORIS Driver (see SORIS)
  • Security Domain Libraries
