Deleting Scripts or Folders

You can delete individual scripts from the project, or you can delete an entire subfolder with all the scripts it contains.

Deleting a scripts folder also deletes all the scripts inside it.

 

  1. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Logics > Scripts > [script] or [folder].
  3. In the Operation tab, do one of the following:
    • If you selected an individual script, click Stop to end the script.
    • If you selected a folder, click Stop All to end all the scripts inside it.
  4. In the Script Editor toolbar the command to delete scrips becomes available.
  5. Click Delete .
  6. A confirmation message displays.
  7. Click Yes.
  1. The script or folder is removed from System Browser.
