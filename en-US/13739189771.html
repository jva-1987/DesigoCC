Configuring a New Multimedia Collection

Scenario

You want to create a new multimedia collection to integrate media files into Desigo CC for a specific purpose. When a collection is selected in Operating mode, the Default tab displays a welcome page with a button for each media file configured to be played. For background information, see the reference section.

Prerequisites

  • The Multimedia Collection extension is installed and included in the active project.
  • System Manager is in Engineering mode.
