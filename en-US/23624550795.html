Insufficient HDB User Rights

You can execute either single or multiple functions without having related user rights.

Execute a Single Function

  1. The System Management Console is running under your user name.
  1. Select the function you want to execute.
  2. The Enter New Credentials dialog box is displayed.
  3. Log in with your username and password and the associated rights.
  4. Click OK.
  1. The function is executed.

 

Execute Multiple Functions

  1. The System Management Console is running under your user name.
  1. In the SMC tree, select Database Infrastructure > [SQL server name].
  2. Select the SQL Server expander.
  3. Log in with your username and password and the associated rights.
  4. Select the function you want to execute.

NOTICE

Data Security

You now have the same rights as the person logged into the SQL Server. Execute only those actions you are tasked with. Close the System Management Console. The user rights are deactivated.
