Workspace

The Building X Connector workspace appears in two locations: SMC and Management Station Workspace.

SMC Workspace

Field

Description

Agent Status

Provides the status for the Building Connect Agent.

Control Port

One of two local communication ports for communicating with the Building Connect Agent.
NOTE: Do not modify this port unless it conflicts with your installation. If it does conflict, you can edit this field.

Application Port

One of two local communication ports for communicating with the Building Connect Agent.
NOTE: Do not modify this port unless it conflicts with your installation.

Serial Number

A unique identifier for the Desigo CC server.

Activation Key

A unique key needed to onboard Desigo CC to Building X.

Configured URL

Displays the current configured URL in Building X.

Name

The name of the Flex Client link that displays in Building X.

Description

The description for the Flex Client link that displays in Building X.

URL

The URL used to access the Flex Client from Building X.
NOTE: If there are additional URLs available, they can be accessed by selecting the drop-down arrow.

Management Station Workspace

Field

Description

Software User

The software user account name created for the Building X Connector. This user account is used to determine the objects and events sent to Building X. All commanding done from Building X applications will be executed using the user account’s privileges.

Building X Data State

Displays one of five states:

  • Not Available: Coincides with Operational State ‘Not Ready’ (see Properties and States topic)
  • Download Pending: Waiting configuration download from Building X
  • Download in Progress: Downloading configuration from Building X
  • Download Failed: The Building X configuration is not aligned with the management station configuration (discovery or upload required) Unable to download the configuration from BX.
  • OK: Normal

Discovery State

Displays one of five states:

  • No Data: Discovery has not yet been executed.
  • Discovery in Progress: In progress
  • Upload in Progress: Upload to BX in progress
  • Changes Detected: The Building X configuration is not aligned with the management station configuration (upload required)
  • No Changes: The Building X configuration is aligned with the management station configuration

Building X Preview

List of devices and points that are going to be uploaded to Building X for verification. The total devices and points are displayed in the heading above. Verify that the totals are accurate before you upload.

Export

Creates a bundle of Desigo CC device and point data from the local management station for use during the discovery process.

Discover

Discovers devices and points to be uploaded to BX and displays them in the Building X Preview expander.

Upload

Uploads the discovered devices and points to Building X.

NOTE: Points are uploaded to Building X as deactivated, which means that values are not sent from Desigo CC. Use the Building X Data Setup application to activate/deactivate points.
