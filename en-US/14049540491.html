Edit Central Functions

  1. The Room management editor is enabled in your user group rights.
  2. Desigo Automation data is imported.
  3. Desigo automation stations are online.
  4. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Room management > Central Functions Editor.
  3. Select the Room Management tab.
  4. The Central Functions Editor opens.
  5. Below Logic > Central Functions the corresponding object hierarchy.
    • (Alternative) On a PXC4 automation station, select Logical View and then the hierarchy level Central Functions.
  6. Drag the selected object to the Group Master Structure dialog box.
  7. (Optional) Assignments must be within the same driver in the event of multiple networks and drivers. Select the appropriate driver and click OK.
  8. Group masters and group members are displayed in the Group Master Structure and Unassigned Members dialog boxes.
  9. (Optional) Clear display deletes the broup master structure from the view. No changes are written to the automation station. Previous changes are lost.
  10. Change or remove the group assignment.

Assigning Group Members

NOTICE

Loss of Data during Download with ABT Site:

Edited group assignments are reset to the state of the last engineering data when downloading from ABT Site. Changes made by the management platform are lost and can no longer be restored.

  1. Present project data must first be imported back to ABT Site.
  2. Only then can the engineering data be re-downloaded with ABT Site.
  3. Use the export and import functions in the Central Functions Editor to avoid potential loss of data.
  1. In the Group category drop-down list, select the appropriate group category.
  2. (Optional) Use Filter master to limit the view in the Group Master Structure dialog box.
  3. Select the group master in the Group Master Structure dialog box.
  4. (Optional) Use Filter members to limit the view in the Unassigned Members dialog box. Using this filter also limits the view in the Group Master Structure dialog box.
  5. Select one or more group members in the Unassigned Members dialog box and click or drag it to the group master.
  6. The group member is assigned to the group master and highlighted in green.
    NOTE: They must be located on the same device if using a local group master and local group members.
  7. (Optional) Use Undo to correct a wrong assignment. You can also Refresh to reset the view to the state at the first drag and drop.
  8. Click Save .
  9. The defined configuration is saved on the Desigo automation station.
  10. Displays a success or error message.
  11. Click OK.
  12. Click Summary log.
  1. All edited data is displayed.

Use the filter function to reduce the amount of unassigned group members for example 114 for the group member of room number 114.

 

Removing Group Member Assignments

  1. In the Group category drop-down list, select the appropriate group category.
  2. Select one or more group members in the Group Master Structure dialog box and click .
  3. The group member is no longer assigned to the group master and highlighted in green.
  4. (Optional) Use Undo to correct a wrong assignment.
  5. Click Save .
  6. The edited configuration is saved on the Desigo automation station.
  7. Displays a success or error message.
  8. Click OK.
  9. Click Summary log.
  10. All edited data is displayed.

Unassigned members are automatically set to Out of Service.

 

Export Central Room Configuration

The export function can save the displayed view and import it at a later date. We recommend using it as part of the handover to the customer.

  1. In the Group category drop-down list, select the appropriate group category.
  2. Click Export .
  3. The Export Configuration dialog box opens.
  4. Apply the default name or change as needed.
  5. Click Save.
  6. The data is saved in the project [Installation Drive]:\[Installation Folder]\[Project Name]\devices\CFE_Configuration\[File name]. Do not change the project path to the files since the import function accesses this path.
  7. Click OK. The information dialog closes.

The data must be saved for each category. As part of the project handover to the customer, backup the data to the node Central Functions for each category.

 

Import Central Room Configuration

The saved room configuration can be imported in the event of a wrong configuration; this restores a previous state. Unneeded room configurations can be re-deleted.

  1. Click Import .
  2. The Import Configuration dialog opens.
  3. Select the corresponding file.
    • (Optional) Select Delete to delete unneeded room configurations.
  4. Click Open.
  5. The data is imported to the room management editor.
  6. Click OK.
  7. The information dialog closes.
  8. Differences to the existing configuration are highlighted in color.
  9. The data is not yet changed on the automation station.
  10. Click Save .
  11. The edited configuration is saved on the Desigo automation station.
  12. Displays a success or error message.
  13. Click OK.
