Handle SPC Intrusion Events in Event List

The following procedure presents the typical steps for handling an intrusion event (red color). You can suspend/resume the procedure at any step.

  1. In Event List, select the event (one at a time).
  2. Click Acknowledge .
  3. This informs the local panels that alarm handling has started from Desigo CC.
  4. If Desigo CC presents you with an operating procedure to follow, complete the required steps.
  5. Ensure that the event has terminated.
  6. Click Reset to reset the event.
  7. In Assisted Treatment only, click Close to close the event.
  8. Click the event button.
  1. The event is cleared from Event List.

Alarm handling differs depending on the event category:

  • Danger: same steps as Life Safety, except for that some specific events that may not require Reset.
  • Fault: Reset is generally not required.
  • Exclusion: no Acknowledge or Reset is required.
  • Anomaly or Information: no Acknowledge or Reset is required, except for a few specific event types.

In case of a Life Safety event, the Acknowledge command automatically silences the alarming devices.
