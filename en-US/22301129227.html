Creating Global Objects Folder Mapping

  1. You have read the Global Objects Folder Mapping Reference topic.
  2. System Manager is in Engineering mode.
  3. Your subsystem supports global objects.
  1. In System Browser, select Logical View.
  2. Select the Views tab.
  3. Select the object hierarchy where you want to create the global object folder.
  4. Click Add Aggregator and select Aggregator.
  5. In the Create New Object dialog box, type the following:
    • Names: GlobalObjects
    • Description: Global Objects
  6. Click OK.
  7. Click Save .
  8. In System Browser, select Management View.
  9. Select Desigo CC > Field Networks > [network name].
  10. Select the BACnet tab.
  11. Expand Global Objects Folder Mapping.
  12. In System Browser, select Logical View.
  13. Select Logical View > [network name] > [Global Objects folder].
  14. Drag the root node for global objects to the text box in the Global Objects Folder Mapping expander.
  15. Click Save .

NOTE:
Ensure that there are no child nodes under the node configured in the Global Objects Folder field of the Global Objects Folder Mapping expander.
