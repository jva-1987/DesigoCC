Configure the Standard Report

  1. You have selected the Standard report type for configuration.
  2. The Report Definition configuration page displays.
  3. In the Source expander, select the Create Maximum, Minimum and Average Values check box.
  4. Click New to add a new device.
  5. Select a device from the Device drop-down list for which the report needs to be generated.
    NOTE: You can configure multiple devices and generate a Standard report.
  6. Enter values into the following columns:
    a. Select the required measurement point from the drop-down list under the Measurement Point column.
    b. Unit
    c. Factor
    d. Converted Unit
  7. (Optional)Select the Sum check box to display the summation of the values of the selected measurement point over the defined duration.
  8. Click Save .
    NOTE: If the selected measurement point is not archived, a warning message will be displayed. You can save the report definition with warning. However, if the measurement points are not archived, the generated reports will not have any values.
  9. The Save Object As dialog box displays.
  10. Enter a name and description and click OK.
  1. The Standard report definition is configured.
