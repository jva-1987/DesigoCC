IEC 61850 Devices

Scenario

You want to perform the actions related to IEC 61850 Devices.

For the background information, see reference section.

 

Overview

Update a Device

  1. Select Applications > Powermanager > [area] OR [sector].
  2. Select the device that you want to update from the system tree.
  3. Click Update .
  1. The configurations are reimported based on the IP address specified in the communication section.

Update is enabled on load and it is disabled when Save is enabled.

 

Edit or Delete a Device

  1. Select Applications > Powermanager > [area] OR [sector].
  2. Select the device that you want to edit or delete in the powermanager tree.
  3. Do one of the following:
    • Perform the required configuration changes and click Save to save the changes.
    • Click Delete to delete the device.
      NOTE: You cannot delete a physical device if one or more of its measurement points is used as a source measurement point in any other physical or logical device.

  4. It is not recommended to change the IP address after device creation.
    Instead delete the created device and create the device again.

  5. The configuration changes are saved.

 

Deleting Device Type

  1. You are in Engineering mode.
  2. You have created a device type.
  3. You have deleted all IEC61850 Generic devices under device type.
  1. Navigate to Application View > Application > Powermanager.
  2. Got to IEC61850 Configuration tab.
  3. Navigate to Delete Device Type and select Device Type Name you want to delete.
  4. Click Delete Device Type.
  5. The Device Type is deleted.

 

Editing Generic IEC 61850 Device Type

  1. You are in Engineering mode.
  2. You have created a device type.
  3. In System Browser, select Application View.
  1. Navigate to Application view > Powermanager.
  2. Select IEC61850 Configuration tab.
  3. Navigate to Select File expander, click browse and select IID File.
  4. Navigate to Existing Device Type expander and select the required device type.
  5. Click Edit.
  6. Do the required changes for Objects, Measurement points, Favorites and Trend expanders under Device Type Configuration.
  7. Click Next.
    NOTE: Library, Device name, and Device Description cannot be edited.
  8. Click Update.
  9. Click Ok.
  10. Device type is edited successfully.
    NOTE: During Generic IEC 61850 device type edit, changes made under Measurement points, Favorites, and Custom Trends will not apply to existing IEC 61850 configured devices.

 

Deleting Generic IEC 61850 Devices

  1. You are in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Powermanager > Device Type > Device.
  3. Select the device you want to delete.
  4. Click Delete .
  5. The Generic IEC 61850 device is deleted.
