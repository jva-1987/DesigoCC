Access the OPC UA Web Client

  1. System Manager is in Engineering mode.
  2. System Browser is in Management View.
  3. You created the SORIS driver, network, and adapter manually or using the network wizard.
  1. Select Project > Field Networks > [SORIS network] > [OPC UA Client adapter].
  2. The Adapter Configuration tab becomes available.
    NOTE: If the Adapter Configuration tab does not display, right-click the adapter node, and select Send to the Primary Pane.
  3. Select the Adapter Configuration tab.
  1. The OPC UA Client Adapter web client displays.
