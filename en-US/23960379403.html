Viewing a Report Definition in the Excel format with a Template

  1. Click Create and view Excel .
  2. An Excel file is created and stored at the following temporary location on your machine [Drive]:\Users\[UserID]\AppData\Local\Temp\temp\GMS.
    A dialog box displays asking you to save a copy of this file on your machine, open the file from the temporary location, or cancel the procedure to open or save the file.
  3. Open the Excel file by clicking Open in the dialog box.
    NOTE: If you are opening the file from the temporary location, you must first ensure that you save a copy of this file at a different location on your machine
    and proceed with the further steps on the saved copy.
  4. Open the worksheet with the table information in the saved copy of the Excel file and select a row with data.
  5. From the Insert menu, select the PivotTable option.
  6. The Create PivotTable dialog box displays.
  7. Select the Select a table or range option and specify the table details.
  8. Select the New Worksheet option in the Choose where you want the PivotTable report to be placed section and click OK.
  9. A new worksheet is added to the Excel document with the PivotTable options.
  10. From the PivotTable Field List, select the columns to be added to the PivotTable.
  11. The selected columns are added to the worksheet.
  12. (Optional) Re-organize the columns as per your requirement in the Drag fields between areas below section in the PivotTable Field List.
  13. Any changes you make in this section, reflect in the worksheet.
  14. Select any row from the PivotTable, right-click, and then select PivotTable Options.
  15. The PivotTable Options dialog box displays.
  16. Select the Data tab.
  17. In the PivotTable Data section, clear the Save source data with file check box and select the Refresh data when opening the file check box.
  18. Click OK.
  19. Delete all the sheets in the Excel document, except the sheet having the PivotTable configuration.
  20. Save the Excel document.
  21. The saved Excel document is a template that contains the PivotTable.
  22. Select the report definition you created.
  23. From the Extended Operation tab, specify the path of the template in the Excel Output Template field and click Set.
    By default, the Excel Output Template field does not display in the Extended Operation tab.
    In order to display this field, you must select the DL3 display level check box for the Template Path property in the report definition Object Model.
    NOTE: The template path is the path of the Server machine. A Process Monitor User must have access to the folder in which the template file is present for the PivotTable to display.
  24. Run the report.
  25. The generated report displays in the Run mode.
  26. Click Create and view Excel .
    NOTE: Ensure that Microsoft Excel 2007 or later is installed, for Create and view Excel to be enabled.
  27. Click Open.
  1. The Excel document displays. The first worksheet displays information on the PivotTable and the details of other tables in the Report display in the other sheets.
