Additional Identity Provider Procedures

Complete the procedures in this section as needed:

Delete an Identity Provider Configuration

  1. Any OpenID user mappings with this configuration have been removed in the Users tab. If not, the configuration cannot be deleted.
  1. Navigate to the Identity Providers tab.
  2. From the Identity Provider Configurations section, select the configuration you want to delete.
  3. From the toolbar, click Delete .

 

Modify an Identity Provider Configuration

  1. From the Identity Provider tab, in the Identity Provider Configurations list, do one of the following:
    • Click on the configuration you want to modify.
    • Click on the Filter and in the Name dialog box type the name of the configuration.
  2. To change the Identity Provider Configuration name, do the following:
    a. From the toolbar, click Edit.
    b. Enter the text for the new name.
    c. Click OK.
  3. The Identity Provider Configuration name is updated.
