Kabona WDC Symbols

When integrating the Kabona WDC extension, the symbols in the Symbol List below are available in graphics and associated to the Desigo CC functions.

NOTE: Many of the symbols have parameter properties that you can set to adjust the graphic as necessary to represent your system.
As illustrated in the example below, a set of Substitutions can be configured to set:

  • The input/output signal represented in the graphic symbol:
    Drag the corresponding node from System Browser into the Object Reference field.
  • Graphical options, typically based on a numeric value:
    Change the numeric value to adjust the symbol as necessary.
    Typical range is from 0 to 4. Note that:
    - In Pump e Fan symbols, you can modify the symbol Style.
    - In digital objects, you can set Unit of measure and Precision.

In the Graphic Editor, to use this symbol in your graphic, drag it from the Library Browser (see note 1 below), and set the required Substitutions (see note 2) in the Symbol Instance Properties.

Notes

 

1)





To display the Library Browser, do the following:
a. In Graphics Editor, select the View tab.
b. Select Library Browser

To search for the appropriate symbol:
a. Select the BA_Device_KabonaWDC_150_HQ1 library from the drop-down list.
b. Use the Type to Filter box to filter based on the symbol name.

2)

A substitution with an asterisk (*) means that this substitution corresponds with the object reference and drag-and-drop is not allowed in that field.
In this case, drag the point corresponding to this substitution into the Object Reference field.
