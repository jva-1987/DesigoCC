Verify and Backup the Existing Installation

Before you proceed with the upgrade you should verify the existing management station installation, verify the integrity of the existing Server project, HDB, remove the licensing dongle and so on.

 

 

Complete the following steps in the same order as listed.

  • Log on with Administrative Rights
  • Verify the Integrity of the Existing Server Project
  • Verify the Integrity of the History Database
  • Verify the Integrity of the Installed Extensions
  • Manual Project Backup from the Management Platform
  • Manual Backup of HDB Data from the Management Platform
  • Exiting the Installed Client Application
  • Stop the Server Project and Close SMC
  • Remove the Licensing Dongle
