Setting CS6 Guarto Configuration Through Online Engineering

Once the adapter is connected to the CS6 Guarto intrusion panel, you can override online data with new configuration data. For background information, see CS6 Guarto Integration.

 

  1. After installation, the application Siemens.GMS.CS6.Configurator is available in AdditionalSW\CEI CMSDL CS6 Configurator locally or over the network.
  2. The CS6 adapter is running and connected to the CS6 intrusion panels. For instructions, see Installing and Starting the CS6 Guarto Adapter.
  3. The NK82xx configuration INI file is available.
  1. Start the configurator tool.
    • Double-click the application Siemens.GMS.CS6.Configurator.
  2. The Siemens.GMS.CS6.Configurator tool page is displayed.
  3. Load NK82xx configuration.
    • For instructions, see Loading NK82xx Configuration.
  4. Connect the configurator tool to the adapter.
    • For instructions, see Connecting the Configurator Tool to the Adapter.
  5. Load the current online data, in the From Adapter section, click Load.

    NOTE: If the CS6 configuration does not exist yet, load the CS6 configuration from file. For instructions, see Loading CS6 Configuration.
  6. (Optional) If any adjustments are required, do one or more of the following:
    • Adjusting CS6 Configuration
    • Adjusting the Object Function Mapping
  7. Override online data. In the To Adapter section, from the Protocol drop-down list, select one of the following settings:
    • CMSDL
    • CEI
  8. Click Download.
  9. The online data is overridden with the current configuration.
  11. Disconnect the configurator tool from the adapter.
    • In the Siemens.GMS.CS6.Configurator tool page, click Go offline.
  1. The CS6 configurator tool is disconnected from the adapter.
