Link Parameters for Pharma Reports

Link Parameters for Pharma Reports

Properties of Links

Parameter

Label

Value

P1

Report Template

Name of the report template.

P2

Format

Output format of the report may be PDF, HTML, XLSX, DOCX.
This is a required value.

P3

Data Range

Specify Time Range for the report data.
For acceptable values, see P3 Link Parameters for Pharma Reports.
This is a required value.

P4

Interval

Time interval
none – no interval
D – day
W – week
1M – Month
1Y – Year
This is a required value for MKT. It is not used in other reports.

P5

Enable Limits Trend Chart

Specify how horizontal limits should display in the report.
0 – no limit lines displayed
1 – display range (shaded)
2  - display lines (only)

P6

High Limit

Upper limit line value.
This is a required value.

P7

Low Limit

Lower limit line value.
This is a required value.

P8

Alternative Title

Optional title displayed at the top of the report.

P9

Display Elements

Optional Charts and Data Grids can be chosen.

P10

Connect Missing Value and Axis (Semi-Colon for Separator)

Specify what to display in charts when data is unknown.
true – show horizontal line connecting last good data points.
false – do not display any line to make it more obvious that data is missing.

For Trend Axis
For each series set if it uses Axis 1 (left) or Axis 2 (right)
For example:
True;1;1;2;2;2;1;1;1;1;1
Series 1,2,6,7,8,9 & 10 for Axis 1
Series 3,4,5, for Axis 2

For Trend Multi-Axis
You can combine several series in the same axis.
For example:
True;1;1;1;4;4;6;7;7;7;10
True for connected value
In axis of series 1, add series 2 and 3
In axis of series 4, add series 5
In axis 6, series 6
In axis of series 7, add series 8 and 9
In axis 10, series 10

P11

Alternative Purpose

Optional purpose and text description displayed near the top of the report.

P12

Display Elements in table for Audit Trail and Alarm summary

Specify how display elements appear in the table:

0 – time line
1 – object and time line
