Create Calibrator Objects with Dimensions

  1. Select Applications > Calibrators.
  2. The Calibrators displays.
  3. In the toolbar, click New .
  4. An empty row for entering the details of the dimensions object is added to the Dimensions table.
  5. Click a cell entry in the empty row and enter values for the following:
    • Dimension Name
    • Dimension Description
    • Current Value
    • Unit: You must select a value from the drop-down list.
  6. Click Save and enter a name for the calibrator object.
  7. The calibrator object is created and is selected by default and becomes available below the Calibrators node in System Browser along with the specified dimensions.
  8. The newly-added current value entry also displays in the Dimension History section along with the date and time in the Valid From field that indicates the date/time when the value is added. The value in the Valid To field is not applicable in this case, as this is the current entry.
