Troubleshooting OPC Data Alignment

Error/Issue

Situation

Solution

The number of Items found in the OPC DA Server tab does not correspond to the Server Items Number in the Operation tab.

A misalignment has occurred between project data and OPC server data (see OPC Items Data Alignment).

Stop Desigo CC OPC DA server. See Stopping the OPC DA Server.

Result: The first time any third-party OPC clients connect to the Desigo CC OPC DA server, it starts again (see Start the OPC DA Server). Data appears aligned, that is the number of Items found will correspond to the Server Items Number.
