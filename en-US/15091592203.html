Room Booking Integration

The Room Booking extension is the common library required to allow integrating automated room booking services in Desigo CC.

Library

The Room Booking extension installs the Desigo CC library BA_Room_RoomBooking_Any_HQ_1, which is available at Project > System Settings > Libraries > L1-Headquarter > BA > Room.

For more details, see Room Booking Graphic Template, Room Booking Object Models and Room Booking Symbols.

This library also provides a set of text groups to cover the different status information and the need of localizable texts for this integration.

Dependencies

The Room Booking extension is configured to depend on the following extensions, which will be automatically added, if they are not already installed in the system, when the Room Booking installation feature is selected:

  • SORIS Driver (see SORIS)
  • Scripts
