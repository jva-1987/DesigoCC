Integrating CS6 Guarto

Scenario

You want to configure Desigo CC to operate on the CS6 Guarto intrusion panels. For background information, see CS6 Guarto Integration.

Prerequisites

  • The CS6 Guarto extension is installed and included in the active project. The following extensions are also selected automatically:
    • Security Domain Libraries
    • SORIS Driver
  • The CS6 Guarto adapter is running and connected to the CS6 Guarto intrusion panels. For instructions, see Installing and Starting the CS6 Guarto Adapter.
  • The CS6 Guarto configuration is set through online engineering, offline engineering, or from scratch.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Licensing

This extension is covered by license. Check that the required license is available; otherwise, you may not be able to configure and work with this extension.
