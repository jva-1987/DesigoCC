Install the Adapter Software

You can install the adapter software service on any networked computer that can connect to hotel management applications and to Desigo CC. Typically, it is installed it on the Desigo CC server station.

  1. Run Oracle_FIAS.msi.
  2. Follow the wizard instructions to complete the installation.
  3. When the installation is completed, locate the FIAS.config file in the file system in C:\Program Files (x86)\Siemens\OracleFIAS.
  4. Open this file with a text editor and modify the following parameters:
    • Host Address (Default is 192.169.7.220)
    • TCP_Port (Default is 4001)
    • Sanity_Check (Default is 240000)
  5. Save the changes and close the file.

After the upgrade of this extension, the adapter must be manually upgraded by running the adapter MSI file, and then selecting the upgrade option.
