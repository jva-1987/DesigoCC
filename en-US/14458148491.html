KNX Compatibility and System Limits

Compatibility

  • ETS tool versions for KNXPROJ files:
    • ETS 4
    • ETS 5.1
    • ETS 5.7
    • ETS 6
  • Desigo CC V4.0 or higher

Size Limits

  • In SORIS, see System Limits.

Configuration Limits

  • Maximum number of KNX/IP routers or interfaces per KNX adapter tested: 5
  • For large configurations with more than 5 KNX adapters, please:
    • Consider splitting the ETS project configurations into smaller parts through different KNX/IP routers, associating each KNX/IP router to an individual KNX adapter to obtain better performance.
    • Create separate folders for all the required KNX adapters. Also, each KNX adapter must have unique name and its own port.
    • Contact Customer Support for further evaluation and approval.
