Discover New Locations

  1. You created one or more locations and now you want to refresh the System Browser tree to display these locations.
  1. In System Browser, select Project > Field Networks > [SORIS network] > [Meteoblue adapter].
  2. In the Extended Operation tab, next to the URL property, click Discover.
  1. The previous configured parameters are retrieved from the configuration file. Any new location objects (that is, days, 24 hours for each of the first 7 days, and 8 hours for subsequent days from 8 to 14) are added to System Browser.
