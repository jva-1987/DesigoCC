Checking the Stations Associated to a Monitor Group

Perform this procedure to determine which stations use a particular monitor group in their video views. For background information see Monitors and Monitor Groups Reference.

  1. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Video > Monitor Groups > [monitor group].
  3. In the Monitor Control tab, Associated stations for Operating lists the stations that use this monitor group for their normal surveillance video view. Associated stations for Events lists the stations that use this monitor group for the video view that displays during alarm-handling.

NOTE: Multiple stations can be associated to the same monitor group. In this case, their video views will display the same set of images. However, each station can have only one operating monitor group and one events monitor group.
