[Example] Automate Scripts with a Macro

In this example, we will create a macro that automatically invokes a set of scripts one after the other. For background information about macros, see Macros.

  1. System Manager is in Engineering mode.
  2. The scripts that you want to automate are enabled for the execution.
  1. In System Browser, select Applications > Logics > Macros.
  2. The Macro tab displays.
  3. Select the Manual navigation check box.
  4. Select Applications > Logics > Scripts.
  5. Drag the scripts from System Browser into the empty area inside the Macro tab.
  6. For each script, a new instruction row is created.
  7. For each instruction, configure the other fields in the row: set Property to Status and Command to Start.
  8. (Optional) In the Initial Delay field, set the delay before the command is sent out.
  9. Click Save As .
  10. In the Save Object As dialog box, select the main Macros folder or any subfolder under it as the destination where you want to save the new macro.
    a. In the Name field, enter Automatic Scripts Run by Macro.
    b. Click OK.
  1. When this macro is invoked, the scripts are now automated and will be started.
