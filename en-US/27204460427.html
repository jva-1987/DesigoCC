Editing Display Rules for Graphics

The Flex Client desktop app allows you to load graphics on specific monitors by triggering a specific graphical element in a graphic. For example, buildings’ graphics can always be visible on the main monitor, while floors are displayed on a second monitor, and so on.

To allow such behavior you must set up communication rules between the source and target windows by defining criteria and specifying the source and target windows.

Prerequisites:

  • The Flex desktop app displays on the screen with all the required multiples windows already displayed on the different screens. For more information about configuring a default window layout, see Setting a Default Window Layout for Flex Client.
  • You already configured one or multiple graphics and their graphic elements for navigation.

Open the Rule Wizard

  • In Flex desktop app, select  and then choose Advanced > Edit display rules.
  1. The Display rules wizard displays on the screen.

Add a New Rule

  1. The Display rules wizard displays on the screen.
  1. Select Add rule.
  2. A new row appears in the Display rules table. Below the table, a step-by-step process guides you through the configuration of the rule. For each rule,  indicates that its configuration is to be completed;  indicates that the rule has been successfully configured. Under Define Criteria, you can set up the rule criteria.
  3. Select the Object field.
  4. The Select object dialog box displays.
  5. From the views drop-down list, select the appropriate view.
  6. Select the node that corresponds to the object in a graphic for which the display rule will apply.
    NOTE: Alternatively, you can select  and search for objects in the System tree.
  7. Confirm your choice with Select.
  8. The Select object dialog box closes. The object location path for the selected object displays in the Object field.
  9. From the Type of criterion drop-down list, select the filter you want to apply to the selected object, among:
    • Object function
    • Object model
    • Object type
    • Object discipline
    • Object instance
    • Object managed type
  10. The Result field displays the matching result for the selected object based on the applied filter.
  11. Select  Next.
  12. Under Source window, choose between:
    • Any window (default)
    • Specific window (choose among one of the available windows)
  13. Select  Next.
  14. Under Target window, choose among one of the available windows.
  15. Select  Next.
  16. Under Save display rule, confirm changes with Save.
  17. Only objects matching the applied filter criterion will be sent from the source window to the target window.
  18. Optionally, save the changes to the rules configuration with the default window layout.
    • In the Layout panel of the Flex desktop app, select  and then choose  Save layout.
      NOTE: Changes will be lost with  Reset layout, if not previously saved with  Save layout.

Modify Rules

  1. The Display rules wizard displays on the screen.
  1. In the Display rules table, select the rule you want to change.
  2. Under Define Criteria, the settings are available for editing.
  3. Repeat steps 2 – 13 in Add a New Display Rule.

Enable or Disable Rules

  1. The Display rules wizard displays on the screen.
  2. By default all the rules are enabled and you want to disable one or more rules. Or one or more rules were previously disabled and you now want to enable them again.
  1. Do one or more of the following:
    • To enable a rule, in the Display rules table, select the check box  next to the rule you want to activate.
    • To disable a rule, in the Display rules table, deselect the check box  next to the rule you want to deactivate.
    • To enable all the rules, in the Display rules table header, select the check box .
    • To disable all the rules, in the Display rules table header, deselect the check box .
  2. Optionally, save the changes to the rules configuration with the default window layout.
    • In the Layout panel of the Flex desktop app, select  and then choose  Save layout.
      NOTE: Changes will be lost with  Reset layout, if not previously saved with  Save layout.

Delete Rules

  1. The Display rules wizard displays on the screen.
  1. In the Display rules table, for the rule you want to remove, select .
  2. Confirm deletion with Delete.
  3. Optionally, save the changes to the rules configuration with the default window layout.
    • In the Layout panel of the Flex desktop app, select  and then choose  Save layout.
      NOTE: Changes will be lost with  Reset layout, if not previously saved with  Save layout.

Close the Rule Wizard

  • To exit the wizard, select one of the following:
    • 
    • Close

Recover Broken Rules

When working with the configuration of displays rules, accidentally closing any of the separated windows will result in broken rules: the  icon indicates that the rules configuration is to be completed.

To restore the previous configuration, you must reset the multi-monitor layout as follows:

  • In the Layout panel of the Flex desktop app, select  and then choose  Reset layout.
