Toolbar Buttons

Icon

Name

Description

Save As

Launches a File dialog box that allows you to select a filename to save data from the data list as a comma separated value (.CSV) file.

Print

Allows you to print the data displayed in the data list.

Previous

Allows you to navigate back through the views you have selected.

Next

Allows you to navigate forward through the views you have selected.

Clear Errors

Clears all error messages from the bottom pane of the BACnet Object Browser window.

Re-read Object List

Re-reads the object list for the selected device and then refreshes the tree.

Refresh

Scans for any device updates and then refreshes the tree.

Stop

Cancels all pending requests to read data.

Write Property

Allows you to view and change the current value.

Import Points

Allows you to selectively import objects (points) from third-party devices into the System Manager database.

Send Who-Is

Broadcast a Who-Is request and listen for I-Am responses on the BACnet network.

Send Who-Has

Allows you to broadcast a Who-Has to view which devices has the object name you are searching for. This is an optional service and it may not be supported on all devices.
