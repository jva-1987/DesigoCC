Initiate an Incident

  • Initiate an incident from either of the following:
    NOTE: Whenever an incident is initiated manually, an event of type Status is generated.
    • Incident template or Notification Template or Easy Buttons
    • Command (refer Initiate an Incident using Command section)
      Alternatively you can also launch a message from a recipient, recipient group, graphics, and command.
