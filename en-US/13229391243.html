Updating the Version Information of a Library Block

After making some changes to a library block, you can update its versioning details.

 

  1. You are trained and authorized to work with system libraries at the target customization level of the library that contains this block.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L4-Project or L3-Country or L2-Region > [folder] > […] > [library] > [block].
  3. The Library Configurator tab displays the version information of the selected library block.
  4. Update one or more of the following fields as needed:
    • Version
    • Revision
    • Date
    • Notes
  5. Click Save .
