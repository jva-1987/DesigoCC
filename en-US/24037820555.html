Check and Adjust the Imported Graphics

  1. In System Browser, select Application View.
  2. Select Applications > Graphics.
  3. Open and check the newly creates graphics and the imported symbols.
    NOTE: You can change the graphic description from the Object Configuration tab.
  4. If necessary, you can delete the imported graphics and start the import process from scratch, or just change some settings and re-import.

NOTE: When re-run the Importer Tool, any previous resizing of the Select layouts/layers and Select blocks/attributes windows is retained, subject to a minimum required dimension.
