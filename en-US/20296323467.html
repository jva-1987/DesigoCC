Configuring and Running the SAP iPro Event Manager

Scenario

The communication between SAP iPro and Desigo CC is managed by a dedicated service, the SAP iPro Event Manager, which you must install and run on a networked PC that can connect to SAP iPro and to Desigo CC. Typically, you can install it on the Desigo CC server station. For background information, see the reference section.

Prerequisites

  • The SAP iPro extension is installed and included in the active project. The following extensions are also selected automatically:
    • Script
  • The NORIS user has Read/Write scope rights on SAP iPro System node in Desigo CC: System1.ManagementView:ManagementView.ManagementSystem.Servers.Server.SAPiPro_System.
  • The NORIS user has at least Read scope rights on event sources that should be monitored
  • The Windows Service user is able to read/write to the installation folder C:\Program Files (x86)\Siemens\SAPiPro-NORIS Event Manager.
