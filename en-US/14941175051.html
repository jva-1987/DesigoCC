Browsing Existing Macros in the System

  1. In System Browser, select Application View.
  2. Select Applications > Logics > Macros.
  3. The macros configured in the system display in the Macros folder, and may be further organized into subfolders under it.
  5. To examine a particular macro, select it in System Browser.
  1. Properties and commands for the selected macro display in the Operation tab.
