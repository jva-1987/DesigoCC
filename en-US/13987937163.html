User Group Administration

Depending on your project-specific requirements, you can create different types of user administration. For security reasons, adopt a restrictive approach when granting permissions to user groups. Make sure that users can only access those applications they actually need to operate.

In the Additional User Group Administraton Procedures section, there are procedures for adjusting user group settings.
