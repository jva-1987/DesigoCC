Creating a Software User Account

Open the Desigo CC System Manager application

To configure Desigo CC for Building X connectivity, you need to define a user account of the type Software account. This user account will be used to determine the objects and events sent to Building X. All commanding done from Building X applications will be executed using the user account’s privileges.

  1. System Manager is in Engineering mode.
  2. System Browser is in Management View.
  1. Select Project > System Settings > Users.
  2. Select the Users tab.
  3. Click New .
  4. The New User dialog box displays.
  5. Do the following:
    a. For User type, select Software account.
    b. In the User name field, enter the name of a new user that will be used by Building X (for example, Building X User).
    c. Click OK.
  6. Click Save .

For additional information about user accounts, see User and User Group Administration.
