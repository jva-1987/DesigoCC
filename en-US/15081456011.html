Advantronic AD300 Commands

The following describes the commands available for the Advantronic AD300 integration.

Property

Commands

Event Commands

NOTE: To have the event commands to work in Event List, you must configure the related command macros. For instructions, see Configuring Advantronic AD300 Event Commands Macros.

Commands

  • Sirens On (to activate the sirens)
  • Sirens Off (to deactivate the sirens)
  • Evacuation (to activate the evacuation relay)
