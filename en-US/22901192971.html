Check the Result

  1. Set System Manager to Operating mode.
  1. In System Browser, select Applications > Multimedia Collection > [my collection].
  2. To refresh the page, click another tab and then click the Default tab again.
  3. The welcome page refreshes and displays the content of the multimedia collection.
  4. Click the button that corresponds to the media file you want to play.
  1. The Default tab refreshes and the media file relevant to the selected button is played.
