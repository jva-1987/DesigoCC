[Example] Configuring an Event Settings Rule to Automatically Open Event List

Scenario: You want to configure the system so that, if a Life Safety event occurs when the Occupancy Status of the building is Occupied, Event List opens automatically, with the event already selected. You want this behavior to apply throughout the system, to all user groups and stations. For background information see Event Handling Rules.

  1. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Client Settings > Event Settings.
  3. The Event Settings tab displays.
  4. Open the General Settings expander, and from the On new event drop-down list, select Start Fast Treatment.
  5. From the Filters expander, open the Time and Organization Modes expander.
  6. In the Organization Mode column, click None and, from the drop-down list, select the Occupied check box.
  7. Open the Events expander and click New.
  8. In the Category column, click All and, from the drop-down list, select Life Safety.
  9. Click Save As .
  10. In the Save Object As dialog box, select the main Event Settings folder as the destination where you want to save this event setting rule.
    a. In the Name field, enter AutoOpenEventList.
    b. Click OK.
