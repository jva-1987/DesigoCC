Scalance Events

The following is the list of events available in Desigo CC for the Scalance integration:

  • Device Missing Configuration
  • Device Not Reachable
  • Device System Fault
  • Device Interface/Port x Down1)
  • Device Interface/Port x Spanning Tree Active1)

 

1)

These events are not enabled by default. You can enable each of them manually for a specific Scalance device or for all the devices. See Enabling the Scalance Events.
