Generic Link Parameters

This section provides information on specifying parameter values for reports that are scheduled for automatic execution.

Properties of Links

Parameter

Label

Value

P1

Report Template

Template of the report that is to be executed through the link. The value in this field is pre-filled and you do not need to set any value in this field.

P2

Format

Specify the output format of the report (for example, PDF, XLSX, DOCX).

P3

Data range

Specify the data range for the report. The values specified for this field depend on the type of report to be executed through the link. For more information see,
P3 Link Parameters for Load Profile Report
P3 Link Parameters for Max Power Report
P3 Link Parameters for Consumption Report

P4

Interval

Specify the aggregation interval for the report. The values specified for this field depend on the type of report to be executed through the link. For more information see,
P4 Link Parameters for Load Profile Report
P4 Link Parameters for Max Power Report
P4 Link Parameters for Consumption Report

P5

MediaGroup

Specify the index value of the media group from the Media Group text group.

P6

Media (Optional)

Specify the index value of the media from the Media text group.

P7

Graph Type (Optional)

Specify the type of graph to be displayed in the report. This property can have either of the following values:
-1 – Default
0 – Line Sum
1 – Bar
2 – Line Compare

P8

Alternative Title (Optional)

Specify the text that is to be displayed as the title of the report in the report output file.

P9

Consumption from

Specify the meter for which the consumption is to be calculated. You can specify either of the following values:
0 – Main Meter
1 – Source Meter

Values for Date range and Interval parameters to be specified for Load Profile, Max Power, and Consumption Reports to be executed through the link.
