Upgrade Roadmap

The upgrade roadmap provides you with high-level tasks that are required to upgrade the Management Station along with all the installed prerequisites, language packs and extensions with the post-installation steps, if any.

In Version 7.0, upgrade is supported only for Desigo CC Version 5.0 and above. No interim upgrades are supported. For example, upgrade from lower build number to higher build number of Desigo CC Version 7.0 is not supported.
