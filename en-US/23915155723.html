Create a New Display Rule

  1. The librarian has configured a web application library.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Client Settings > Web Applications.
  3. The Rule Editor tab displays.
  4. Open the Output expander and do the following:
    a. From the Application drop-down list, select the value that corresponds to the external web application to display.
    b. (Optional) Modify any application parameters.
  5. Open the Triggers expander and define the conditions to which the display rule applies:
    a. Click New.
    b. In System Browser, select the Manual navigation check box to retain the Rule Editor tab.
    c. Drag the desired target object from System Browser into the Path field.
    NOTE: In Operating mode, this will cause the rule to display the configured external web application in the Application Viewer tab when the associated object is selected in System Browser.
    d. Configure the other fields in the row to filter by Managed Type, Discipline/Subdiscipline, Type/Subtype, and Function. This will cause the display rule to only be triggered when the target object matches those criteria.
  6. (Optional) – To override the default URL, complete the following:
    NOTE: If enabled, passwords are automatically used on the URL call at the handover point. This may result in a potential cybersecurity risk especially on untrusted clients or those not maintained. This functionality is intended for experienced users and should only be used in secure environments.
    • Application URL – Enter the address of the override URL.
    • Username – Enter the username, if any, for the override URL
    • Password – Enter the password for the given username.
    • Reconfirm – Re-enter the password.
  7. Click Save As .
  8. In the Save Object As dialog box, select the main Web Applications folder or any subfolder under it as the destination where you want to save the new rule.
    a. Enter a name and description.
    b. Click OK.
  1. In the General Settings expander, the Application Name field displays the name of the rule. The new display rule object is added to System Browser.
