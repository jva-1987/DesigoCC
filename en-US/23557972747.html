Select the Time Range using the Time Range Scrollbar

  1. You are in an active Trend View.
  1. In the Trend View, navigate to the left or right slider (dark grey area) of the time range slider.
  2. The shape of the mouse pointer changes and the tooltip displays.
  3. Drag the Time Range slider to the left or right until you have reached the desired time range.
  4. The time range change continuously displays.
  5. The Trend View displays the selected time range.
  6. Navigate to the Time Range slider (light grey area).
  7. Drag it to the desired time/data range.
  1. The time range is displayed with the corresponding data period in the Trend View.

NOTE:
Data is compressed for display purposes only if you select a large time range or very large number of measured values. All data is displayed for smaller time ranges.
