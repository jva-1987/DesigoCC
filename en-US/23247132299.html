(Optional) Configure error messages for RDF302 room thermostat

Error messages can be customized for each RDF room thermostat. The customized texts are displayed in the Fault information property when an error occurs.

  1. Select Project > Field networks > [RDF302 Network]> [Interfacename]> [RDF302 room thermostat].
  2. In the Advanced operation tab, select property Fault text 1.
    a. Select Change.
    b.. In the Value text box, enter the new error message.
    c. Select Send.
  3. Select property Fault Text 2 – 4 for further customizations.
  1. The selected setting is applied and the text is updated.
