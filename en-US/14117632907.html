Adding a Ventilation Unit

The following properties can be configured for various ventilation units (heating coils, cooling coils, humidifier):

  • Hydraulics type
  • Pipe length
  1. From the Home tab, in the Modes group, conduct one of the following actions:
    • Click Design. Previously rotated objects appear in the default direction.
    • (Optional) Click Test. In this case, the fan is not displayed to scale.
  2. In the System Browser, select Logical View.
  3. Select Logical > [Network name] \...\ [Plant] >
    • [PreHcl (preheater)]
    • [ReHcl (reheater)]
    • [Ccl (cooling coils)]
    • [Hum (humidifier)]
  4. Drag the object to the graphics page.
  1. The added object displays at the maximum scale. This ensures that the next object does not overlap.

Determining Hydraulics

Various types of hydraulics can be defined for heating coils and cooling coils.

  1. The heating coil or cooling coil symbol is added to the graphic.
  1. In the View tab, select Properties.
  2. The Symbol Properties dialog box is enabled.
  3. In the Mode group, click Test.
  4. The symbol displays as per the imported data points.
  5. Select Symbol Properties > Substitutions.
  6. In the Circuit Type field, enter a number between 0 and 2 (see Table Hydraulic Types for Heating and Cooling Coils).
  7. Click ENTER or change to another entry field to activate the change.

Resizing the Mixing Position

The mixing position for hydraulics can be modified for the heating coils and cooling coils.

  1. The heating coil or cooling coil symbol is added to the graphic.
  1. In the View tab, select Properties.
  2. The Symbol Properties dialog box is enabled.
  3. In the Mode group, click Test.
  4. Select Symbol Properties > Substitutions.
  5. In the Mixing Position text field, enter a number from ‒48 to +96 (see Table Mixing Position of Hydraulics).
    NOTE: Enter an offset in grid 12. The three-way valve is no longer displayed as a mixing pipe once grid 12 is no longer used.
  6. Click ENTER or change to another entry field to activate the change.

Modifying Pipe Length

The length of the pipes can be modified for heating coils, cooling coils, and humidifiers.

  1. The heating coil, cooling coil, or humidifier symbol is added to the graphic.
  1. In the View tab, select Properties.
  2. The Symbol Properties dialog box is enabled.
  3. In the Mode group, click Test.
  4. Select Symbol Properties > Substitutions.
  5. In the Size field, enter a number between 30 and 200 (see Table Modifying Pipe Length).
  6. Click ENTER or change to another entry field to activate the change.
