OPC UA Events

The following is the list of the events available in Desigo CC for the OPC UA Client connectivity:

  • OPC UA Server Connection Status Fault
  • OPC UA Alarm Event
  • Any custom event triggered by the alarms custom configuration. For instructions, see Setting OPC UA Client Advanced Configuration and Customizing Alarm Classes.
