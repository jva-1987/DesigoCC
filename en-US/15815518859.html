Toolbar

Select the toolbar icons as required.

System Toolbar

You can perform the following functions at the system level.

Toolbar Icon

Function

Allows you to create a new area.

Allows you save the system configuration. It is enabled only if any changes are made to system configuration.

Area/Sector Toolbar

You can perform the following functions at the area/sector level.

Toolbar Icon

Function

Allows you create a new sector or a new device.

Allows you save any edits made to the selected area/sector. It is enabled only if any changes are made to the area.

Allows you save any configuration changes made to the selected area/sector as a new area/sector (You can create a copy of a existing area/sector by using this option. Save the existing area/sector under a new name, without making any configuration changes, to create a copy).

Allows you to delete the selected area/sector.

Device Toolbar

You can perform the following functions at the device level.

Toolbar Icon

Function

Allows you save any edits made to the selected device. It is enabled only if any changes are made to the device.

Allows you save any configuration changes made to the selected device as a new device (You can create a copy of the existing device by using this option. Save the existing device under a new name, without making any configuration changes, to copy the device).

Allows you move the device to a different area/sector.

Allows you to delete the selected device.

(Applicable only in case of SEM3 and PAC1200 devices) Allows you to update the configurations of the selected device.
