Assign the Domain User as a Member of the IIS_IUSRS Group

  1. You are on the third-party machine.
  2. A domain user is already created and you want to assign the domain user as a member of the IIS_IUSRS group.
  3. The user is the same user to which the project is shared.
  1. From Windows Start menu, select Computer.
  2. Right-click Computer and select Manage.
  3. In the Computer Management tree, select System Tools > Local Users and Groups > Groups.
  4. Right-click IIS_IUSRS group, and click Add to Group.
  5. The IIS_IURS Properties window displays.
  6. Click Add.
  7. The Select Users, Computers, Service Accounts, or Groups window displays.
  8. Provide the domain user's name in the Enter the object names to select … field.
  9. Click OK.
  10. The domain user is added as a member of the IIS_IURS group.
  11. In the IIS_IURS Properties window, click Apply and OK.

For more information, see http://technet.microsoft.com/en-in/library/cc739265(v=ws.10).aspx
