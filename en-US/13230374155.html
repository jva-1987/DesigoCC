Creating a Customized Events Library

Do this procedure to create a new Events library by cloning the structure of an existing one (for example, L1-Headquarter > Global > Events). The customized library will be placed at your allowed customization level.

For background information, see Events Library.

 

  1. You are trained and authorized to work with system libraries at the target customization level. For example, L4-Project.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L1-Headquarter > Global > Events.
  3. In the Library Configurator tab, click Customize the entire library to a lower level .
  4. Click OK.
  1. The structure of the selected library is cloned under the allowed customization level (for example, L4-Project > Global > Events).
