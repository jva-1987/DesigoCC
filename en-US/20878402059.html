Configuring Printers

You can configure the interoperability with the printers available for the Windows operating system of the Desigo CC server. Printers can be used to print data from system applications or for the journaling printouts feature. For background information and troubleshooting, see Printers. See also Journaling.

Printer errors

Due to operating system limitations, in some printers one or multiple fault errors (offline, faulty, not reachable, door open, paper jammed, paper not available, or toner not available) might not be always managed, or might be delayed.

Prerequisites

  • The required printers are available to the Windows operating system of the Desigo CC server. You cannot use the same printer for both journaling and system applications.
  • Printer drivers have been updated to the latest versions.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Set Shared Printers

To allow Desigo CC server to use a printer shared from a Desigo CC client station do the following:

  1. Configure the printer in Windows for both the client station that will share it over the network, and the server that will use it.
  2. To have permission to log on as service and user access to the shared printer, in SMC, configure a service account for the Desigo CC server. For more details, in System Settings Procedures, see Configure the Service Account.
    NOTE: The default (SYSTEM) user account cannot be used for this configuration.
  3. Configure the shared printer for the Desigo CC server. For instructions, see Configure a Printer for System Applications, below.

 

Configure a Printer for System Applications

Perform this procedure to configure a printer that can be used to print data from Desigo CC applications (for example, Event List, Reports, and so on).

  1. Select Project > Management System > Servers > Main Server.
  2. The System Management tab displays.
  3. Click New , and select New Server Printer.
  4. In the New Object dialog box, enter a name and description.
  5. Click OK.
  6. The new printer object is object is automatically selected in System Browser.
  7. In the System Management tab, select the Printer name. The drop-down list contains all non-virtual printers available for the Windows operating system of Desigo CC server.
    NOTE: If you are doing this configuration from a station that is not the Desigo CC server station (for example, from a client or FEP station), when you select the printer name the Server parameters dialog box displays. Enter Domain\Username and Password, and then click OK.
  8. If necessary, change the Printer type (Page printer or Line printer).
  9. Set the Journaling printer value to No.
    NOTE: The journaling printer must be dedicated to that function and cannot be used to also print from system applications such as Reports, Event List, and so on.
  10. (Optional) Enter any additional information in the Notes field.
  11. Click Save .

 

Configure a Printer for Journaling

Perform this procedure to set up a dedicated printer that will be used for journaling printouts.

  1. Select Project > Management System > Servers > Main Server.
  2. The System Management tab displays.
  3. Click New and select New Server Printer.
  4. In the New Object dialog box, enter a name and description.
  5. Click OK.
  6. The new printer object is automatically selected in System Browser.
  7. In the System Management tab, select the Printer name. The drop-down list contains all non-virtual printers available for the Windows operating system of Desigo CC server.
    NOTE: If you do this configuration from a station that is not the Desigo CC server station (for example, from a client or FEP station), when you select the printer name, the Server parameters dialog box displays:
    a. Enter username and password of the Windows user that has the rights for the computer where the printer is configured.
    b. Click OK.
  8. If necessary, change the Printer type (Page printer or Line printer).
  9. Set the Journaling printer value to Yes.
    NOTE: The journaling printer must be dedicated to that function and cannot be used to also print from system applications such as Reports, Event List, and so on.
  10. (Optional) Enter any additional information in the Notes field.
  11. Click Save .

 

Special Settings for Journaling Printer in Windows 7

Only if the Desigo CC server is running on Windows 7, for the journaling printer to work properly, you must set its properties in the Windows operating system as instructed here.

  1. Click the Windows Start button, and select Devices and Printers.
  2. In the Devices and Printers dialog box, right-click the printer that you configured as a journaling printer, and select Printer properties.
  3. In the Printer Properties dialog box, click the Security tab.
  4. Click Add.
  5. Enter the object SYSTEM.
  6. Click OK.
  7. SYSTEM appears among the Groups or user names.
  8. Select SYSTEM.
  9. In the Permission for SYSTEM, allow the following options:
    • Print
    • Manage this printer
    • Manage documents
  10. Click OK.

 

Delete a Printer

Do this procedure to remove printer object that is no longer required.

  1. Select Project > Management System > Servers > Main Server > [printer].
  2. The System Management tab displays.
  3. Click Delete .
  4. A confirmation message box displays.
  5. Click Yes.
  1. The printer object is removed from System Browser. Desigo CC will no longer be able to print using that printer. If the printer was configured for journaling, the journaling printouts will cease.
