Link the Graphic to the Data Point

Scenario: The plant operator tableau to be displayed if the plant is selected in System Browser.

  1. In System Browser, select logical view.
  2. Select Logical > [network name] >...> [MyAhu].
  3. Drag the object to line 00_Linked Datapoint.
    NOTE: You can designate the display sequence if multiple graphics are assigned to the same data point.
  4. Click Save .
