Disabling Unidentified Events for Out of Scan

Unidentified events occur when Desigo CC receives a message with no corresponding data point. They can only be cleared from Event List by stopping the project and using SMC.

Such unidentified events may be generated from SPC points that are placed out of scan. Accordingly, before taking an SPC point out of scan, it is recommended to disable unidentified events as described here.

  1. Desigo CC is configured to connect to an SPC intrusion system.
  2. System Manager is in Engineering mode.
  3. System Browser is in Management View.
  1. In System Browser, select Project > Field Networks > [SPC network] > [SPC panel].
  2. Select the SPC tab.
  3. Open the Unidentified Event expander.
  4. The Enable Unidentified Event check box is selected by default. This means that such events are visible in the Desigo CC event list.
  5. Clear the Enable Unidentified Event check box.
  6. Click Save .
  7. If the panel is connected (the configuration file is imported and the driver was started in full mode), you must stop and start its driver again to apply the new setting. See Stopping and Starting the SPC Driver.
