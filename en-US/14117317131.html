Adding a Pump

In Desigo CC, there is a default pump for all geographical directions, standards, and technical versions. After adding the graphic, the default pump is configured as per project requirements.

  1. In the Mode group, conduct one of the following actions:
    • Click Design . Previously rotated objects appear in the default direction.
    • (Optional) Click Test . In this case, the pump does not display in its true size.
  2. In System Browser, select logical view.
  3. Select Logical > [Network name] \...\ [Plant] >
    • [PreHcl (preheater)] > [Pu (Pump)]
    • [PreHcl (reheater)] > [Pu (Pump)]
    • [Ccl (cooling coils)] > [Pu (Pump)]
    • [Hum (humidifier)] > [Pu (Pump)]
  4. Drag the object to the graphics page.
  1. The added object displays at the maximum scale. This ensures that the next object does not overlap .

NOTE:
Drag-and-drop only works if a function is assigned to the corresponding object after data import.
You can check the object data on each symbol.

 

Rotating

  1. The fan symbol is added to the graphic.
  1. In the Mode group, click Test.
  2. The pump symbol displays as per the important data points and default direction .
  3. Select the pump symbol.
  4. Left-click and hold, then right-click until the pump symbol displays in the desired direction.
  5. Release the left mouse button.
  1. The pump symbol is displayed in the desired direction.

Changing the Default

  1. The fan symbol is added to the graphic.
  1. In the View tab, select Properties.
  2. The Symbol Properties dialog box is enabled.
  3. In the Mode group, click Test.
  4. The pump symbol displays as per the important data points and the selected direction .
  5. Select the pump symbol.
  6. Select Symbol Properties > Substitutions.
  7. In the Style field, enter a number between 0 and 2 (see Table Define Pump Style).
  8. Click ENTER or change to another entry field to activate the change.
