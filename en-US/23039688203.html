Associate the Video API Service to the Account

  1. In Windows, on the Desigo CC server computer, start the Services tool:
    • Click Start > Control Panel > Administrative Tools > Services
      or
    • From Start > Run... launch services.msc.
  2. In the service list, locate the Siemens Video API Service.
  3. Right-click the Siemens Video API Service and select Properties.
  4. In the Properties dialog box, select the Log On tab.
  5. Under Log On as: select This account and enter the username and password of the Windows user account whose credentials were configured in the Windows Credential Manager.
    NOTE: If you use an Administrator Windows account, you must use the same account to run the Siemens Video API Service.
  6. Click OK and close the Services tool.
