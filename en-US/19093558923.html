Configuring Virtual Objects

You can create virtual objects manually through the Object Configurator, or configure virtual objects in bulk through the import of a configuration file in Desigo CC.
For background information, see the reference section.

Prerequisites

  • System Manager is in Engineering mode.
  • System Browser is in Application View.

Create a Virtual Object

  1. Select Applications > Logics > Virtual Objects.
  2. In the Object Configurator tab, click New and select the type of virtual object that you want to add.
  3. In the New Object dialog box, enter a name and description, and click OK.
  1. The new virtual object is added to System Browser and automatically selected.

Import Virtual Objects

  1. Prepare the CSV configuration file for the virtual objects.
    • To be able to import virtual objects data in bulk, you must first manually create the configuration file in CSV (comma separated values) format, using Microsoft Excel or a text editor. For more details, see CSV File for Virtual Objects.
      NOTE: Use a comma (,) as separator of the values contained in the CSV file.
  2. Select Applications > Logics > Virtual Objects.
  3. In the Import tab, click Browse.
  4. The Open dialog box displays.
  5. Navigate to and select the configuration file (CSV).
  6. Click OK.
  7. Transfer the virtual objects folders to import from the Source Items list to the Items to Import and click .
  8. (Optional) If there are some pre-existing virtual objects in the system for Application View, but you do not want to retain them, select the Delete unselected items from the Views.
  9. Click Import and click Yes to confirm.
  10. The Import tab displays the Import in progress page. When the import completes successfully, the Import dialog box displays its outcome.
  11. Click OK to close the Import dialog box.
  12. New imported virtual objects are added to System Browser. Any pre-existing configuration is updated consequently. In particular, if you set the option Delete unselected items from the Views, any items not present in the configuration file are removed from System Browser.
  13. (Optional) Click Import Log to save the details of the import operation to a TXT file.
