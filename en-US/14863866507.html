Configuring SecuriFire Modbus Commands Macros

Scenario

You want the acknowledgment command to work also from Event List for zone events. For this purpose, you have to create a specific macro for the management station. For background information, see SecuriFire Modbus Commands.

Prerequisites

  • Desigo CC is configured to integrate the SecuriFire system. For instructions, see Integrating SecuriFire Modbus System.
  • System Manager is in Engineering Mode.
