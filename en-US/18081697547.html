Modifying the Kabona WDC Connection Parameters

If necessary, you can modify the connection parameters that you initially defined in the Configuration node.
You do that in the Kabona WDC System<n> node, which corresponds to the affected Kabona WDC system <n>, with <n> being the sequential system number, starting from 1.

  1. Select Project > Field Networks > [SORIS network] > [Kabona WDC Adapter] > [System<n>]
  2. In the Operation tab, modify the following connection parameters as necessary.
    • URL
    • User
    • Password
    • Polling rate (in milliseconds; values below 500 are not recommended)
      For each field, enter the value and click SET.
  3. In Update Connection Details, click Update.
  1. The new configuration data is acquired and System Browser refreshes the Kabona WDC system objects.
