User Wizard

This wizard provides a quick procedure that simplifies the configuration workflow for user management.

The wizard presents the following initial menu:

    • Create new users
    • Create new user group
    • Create new station group
    • Configure LDAP Connectivity - NOTE: When you select this option, the LDAP tab displays. For this Domain Group workflow, proceed to New LDAP Domain Group
    • Check Logged-In Users

See also the reference section
