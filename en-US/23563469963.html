Create and Configure Recipients

  1. Select Applications > Notification > Recipients.
  2. The Recipients Editor tab displays.
  3. Perform the following steps to create and configure recipients:
    • Select an existing device type or create and configure a new device type.
    • Create recipient users or devices.
  4. The recipients are created and configured.
