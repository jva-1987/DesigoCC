S7 Plus

S7 PLUS components support communication with S7-1200 and S7-1500 devices. These devices use the S7 protocol and data is transported via TCP/IP. This protocol is used by Simatic controllers for communication.

S7 PLUS Integration

Desigo CC supports the integration of S7 PLUS devices. It enables Desigo CC to read data from an S7 PLUS device. Desigo CC supports S7 PLUS over the Ethernet. Integrated S7 PLUS devices can be used in all common applications such as Event handling, Reporting, Online Trend.

The integration is enabled by the software component of the management station called Simatic_S7. S7 PLUS is covered by a license.

S7 PLUS component in Simatic_S7 supports integration of S7 PLUS devices by providing access to the devices and supervising fault events.

The S7 PLUS driver provides the following functionalities:

  • Sending data to the PLC
  • Polling data from the PLC
  • Receiving unsolicited data from the PLC via subscriptions
  • Time synchronization of the management system and S7 PLC
  • Safe connection via S7DOS / PC Identifier
  • Optimized data access

If you install the S7 PLUS component, two libraries GMS_S7 PLUS_HQ_1 and BA_Device_S7 PLUS_HQ_1 are available. GMS_S7 PLUS_HQ_1 provides native object models of S7 PLUS data points without any function mappings. The BA_Device_S7 PLUS_HQ_1 library provides object models with function mappings and symbols related to building automation.

NOTE:
The S7 Communication Layer is a prerequisite for Simatic S7. In order to install the S7 Communication Layer, the Windows username should not have a space character in it. If it has a space character and the Installer shows an error message, you must install the S7 Communication Layer manually by navigating to the following location, …\InstallFiles\EM\Simatic_S7\Prerequisites.
