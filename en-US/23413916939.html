(Optional) Configure the Device Type

Perform this step if you do not want to configure a device with your own settings.

  1. Click the Device Type Configuration tab.
  2. In the Device Type drop-down list, select the device type that you want to configure.
  3. The Use Default Settings check box is selected by default.
  4. The default settings for the device type display in the Units & Factors, Measurement Points, Favorites, and Trend expanders.
  5. Uncheck the Use Default Settings checkbox and configure the settings for the device type.
  6. Click Save .
  1. These settings will be applied to all devices of the selected device type that are created in the system.
