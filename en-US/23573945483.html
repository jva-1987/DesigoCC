Add the Extensions from the Distribution Partner Project

While configuring the distribution Partner project in the Manual mode, you must add the extensions that are configured in the Partner projects.

NOTE 1:
When configuring the extensions for the distribution partner project, make sure to add only those extensions that are configured in the selected distribution partner project. Otherwise, you will not be able to work with those extensions when working in distribution.
NOTE 2:
Removing the extensions of the distribution partner entry using Extensions also removes the corresponding profiles from the Profiles expander. It also removes them from the profiles folder of the Originator project, when no other distribution partner project has this extension configured.

  1. The extensions configured in the distribution partner project are installed on the Local System. Otherwise, these extensions are not listed in the Select Project Extensions dialog box when you click Extensions. Consequently, you cannot add them in the distribution partner project.
  2. At least one project is available under the Projects folder and you have enabled distribution for this project, and added the distribution partner project.
  1. In the SMC tree, select Projects > [project].
  2. Depending on whether the project is started or stopped, do one of the following:
    • Click Edit .
    • Click Change Distribution Info .
  3. Open the Distribution Participants expander.
  4. It lists all the entries for the distribution Partner projects. It also displays a read-only entry for the current project (Originator) displays.
  5. Select an entry for the distribution partner project.
  6. Click Extensions.
  7. The Select Project Extensions dialog box displays a list of all the extensions suites along with all the extensions that are installed on the current server.
  8. In the Select Project Extensions dialog box, open the extension suites and select only those extensions that are configured in the distribution Partner project. Click OK.
  9. The extensions are configured and listed in the Extensions column drop-down list.
  10. The corresponding profiles are added to the Profiles expander.
  11. Click Save Project .
