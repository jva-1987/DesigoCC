Create an IEC 61850 device

  1. In the System Browser, select ApplicationView > Powermanager > [Area].
  2. Select Area tab.
  3. Click Create > Create Device.
  4. Navigate to Create expander > PQ Devices, and select IEC 61850 protocol.
  5. Select the required device and variant.
  6. Navigate to Create > Communication, and enter the IP Address.
  7. Select address profile in the Measurement Interval.
    NOTE: Measurement Interval dropdown is not applicable for Q100 devices with firmware version number V1.XX.
  8. Click Save.
  9. Save Object as popup window appears.
  10. Enter the Name and Description.
    NOTE: The network name is created with 3 characters (IEC) by default, and the device name must be created with 17 characters to ensure that the total length of the network and device names does not exceed 20 characters.
  11. Click OK.
  1. Device created successfully. Configure the created device.
