Configuring Icons in a Library

You can configure the icons block of a library. For background information, see Icons Library.

 

Prerequisites

  • You are trained and authorized to handle libraries at the target customization level of the library whose icons block you want to modify.
  • The library you are configuring includes the icons library block. If not, see Adding Blocks to a Library.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Import Icons into a Library Block

You can import some image files into the icons block of a library.

  1. In System Browser, select Project > System Settings > Libraries > L4-Project or L3-Region or L2-Country > […] > [library] > [icons block].
  2. Select the Icons tab.
  3. The Icons expander displays any icons already imported into the block. You can drag the percentage slider to view them in a larger or smaller size.
  4. Open the Import expander.
  5. Click Browse.
  6. In the Browse for Folder dialog box, do the following:
    a. Navigate to the folder containing the icon files you want to import.
    NOTE: Supported file formats are BMP, JPG, PNG, or XAML.
    b. Click OK.
  7. The Import expander displays all the icons (of the supported formats) found in the selected folder. You can drag the percentage slider to view them in a larger or smaller size.
  8. Select the icons you want to import by clicking them. You can also use the Select All or Clear All buttons.
  9. Click Import.
  1. The system starts importing the icons. Check the progress bar at the bottom of the screen. Note that once the import starts, you cannot cancel the operation.
  2. When the import is complete, the imported icons display in the Icons expander.
  3. The imported icon files are stored at the following path: [Installation Drive]:\[Installation Folder]\[Project Name]\libraries\[library folder]\[library]\Icons.

If you update an existing icons library, icons that have the same name are not overwritten but the new version appears with the same name followed by _[icon version].

Delete Icons in a Library Block

You can remove some unwanted icons from the icons block of a library or clear all the icons from the icons library block to import a new set of icons.

  1. In System Browser, select Project > System Settings > Libraries > L4-Project or L3-Region or L2-Country > […] > [library] > [icons block].
  2. Select the Icons tab.
  3. The Icons expander displays the icons currently imported into the block. You can drag the percentage slider to view them in a larger or smaller size.
  4. Select the icons you want to remove.
    You can select the icons one-by-one or hold SHIFT and click to select a subset of icons. To unselect an icon in a selection, press CTRL.
  5. Click Delete.
  6. The selected icons are removed from the Icons expander.
    NOTE: Clicking Refresh results in undoing the delete command.
  7. Click Save .
