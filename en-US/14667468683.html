D3 Visualization Workspace

The D3 Visualization workspace uses the Application Viewer pane to display dashboard charts and web data.

Time Series Chart

The time series chart allows you to dynamically view updated COV information for data points. You can pause, start, and stop the dynamic updates using the tools in the upper-right corner of the pane.

Trend Series Chart Workspace

Icon

Name

Description

Pause

Allows you to continue to generate COVs in the background, however, the chart is not updated.

Start

Allows you to return the chart to updating dynamically. COV’s are generated and the chart is updated accordingly.

Refresh

Allows you to update the chart with the COVs that have been generating during a pause operation.
