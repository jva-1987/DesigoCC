Operating the RDF302 room thermostat

 

Prerequisites:

  • System Manager is in Operation mode.
  • In System Browser, Logical View is selected.

 

Step:

  • In the System Browser or floor plan view, select the room to operate. The graphical view is based on the applied FNC configuration.

 

Changing Operating Mode

  1. Select Room operating mode .
  2. In the Room operating mode drop-down list, select option:
    - Comfort
    - Economy
    - Pre-Comfort
    - Protection
  3. The value is immediately written to the device.

 

Changing a Setpoint

  1. Select Setpoint for Comfort .
  2. Enter the new setpoint between 5…40°C or use the up or down arrows.
  3. The value is immediately written to the device.

 

Manually Enter the Fan Command Value

  1. Select Fan Command Value .
  2. Enter the new value or use the up or down arrows.
  1. The value is immediately written to the device.
