Browse Incidents and Notifications

You can browse for incidents and notifications from either of the following:

 

Browse Incidents Tab

  1. System Manager is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Notification.
  3. Click the Browse Incidents tab.
  4. A list of all the initiated incidents displays under the Incidents expander.
  5. To search the required incidents, click Filter .
  6. The Advance Search Options dialog box displays.
  7. In Status drop-down list, select the required check box, and click Search.
  8. Select the incident to view the associated notifications.
  9. A list of all the associated notifications displays.
  10. To search the required notification, click Filter .
  11. The Advance Search Options dialog box displays.
  12. In Status drop-down list, select the required check box, and click Search.
  13. Select the notification to view its details.
  1. The notification details display.

 

Browse Notifications Tab

  1. System Manager is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Notification.
  3. Click the Browse Notifications tab.
  4. A list of all the launched notifications displays.
  5. To search the required notification, click Filter .
  6. The Advance Search Options dialog box displays.
  7. In the Status drop-down list, select the required check box, and click Search.
  8. Select the notification to view its details.
  1. The notification details display.
