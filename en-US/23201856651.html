Configure the S-Bus Object Structure

The S-Bus adapter can act as an interface to the S-Bus based on a configuration that corresponds to the S-Bus structure. You can set this configuration using the S-Bus configuration tool. See Configuring the S-Bus Object Structure and Alarm Control.
