Onboarding the Management Station to Building X

  1. In SMC, select Projects, then select the project you want to connect from the list, and then click the Building X Connector tab.
  2. NOTE: Check the Control and Application ports under Communication Settings. If you are running more than one project on the same machine, Desigo CC will attempt to verify that the chosen ports are not in use. Each project with an active Building X Connector must have exclusive access to a unique set of ports.
  3. From the Toolbar, click Start Agent .
  4. In the Communication Settings expander, the Agent Status displays Running, device activation required.
  5. From the Toolbar, click Generate Activation Key.
  6. In the Device Activation expander, copy the Activation Key by clicking Copy Key.
  7. Log into the Devices application on Building X.
  8. From the menu, click Sites, and then do one of the following:
    • If the Desigo CC site you want to connect to already exists, select it.
    • If the site does not exist, click Add and complete the fields. Click the Add button to save the new site, and then select it.
  9. Click Add to add a new device to the site.
  10. Paste the Activation Key in the Activation Key field, and then click Validate.
  11. When the key activation succeeds, complete the Custom name and Custom description fields, and then click Add.
  12. In the Assign distribution dialog box, select Desigo CC/Cerberus DMS, and then click Save.
  13. Click the Applications tab, and then select the Install check box.
  14. Click Synchronize.
  15. In SMC, go to the Building X Connector tab and verify that the Agent Status displays Running, Connected.
  1. If the Agent Status displays Running, Connected, Desigo CC has been successfully onboarded. For any other status, see Troubleshooting the Building X Connector in the reference section.
