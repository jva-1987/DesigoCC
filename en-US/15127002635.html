Uninstall Prerequisites including WinCC OA 3.19

The uninstallation program does not uninstall the prerequisites. However, if you uninstall using the custom Installation mode, it displays a list of prerequisites used by Desigo CC when you run the Uninstall Wizard. You can copy them for further reference.

 

 

Uninstall Prerequisites Including WinCC OA 3.19

If the Desigo CC prerequisites are no longer used by another application, you can uninstall them separately from Control Panel > Programs and Features.

NOTE:
For upgrading Desigo CC on the same server from V4.2 to V6.0, you do not need to uninstall any prerequisite of Desigo CC as Installer automatically uninstalls and upgrades all the required prerequisites, before installing the required version.

  1. In the Program and Features window, select one by one, each of the following prerequisite and right-click and select Uninstall.
    • Automation License Manager

  2. Automation License Manager (ALM) is no longer supported in Desigo CC

    You must manually uninstall ALM if installed on your system. If not uninstalled then some features may not work correctly and the WinCC OA console along with some other panels will not work.

    • License Management Utility (LMU)
    • Microsoft Application Request Routing (ARR)
    • Visual Studio 2019 Redistributables_x64
    • Visual Studio 2019 Redistributables_x86
    • WinCC OA 3.19
    • Microsoft SQL Server 2022 - Express Edition
    • OPC Core Components
  3. Click Yes.
  4. The uninstallation program completes without displaying any verification message.

 

Uninstall Microsoft SQL Server or Uninstall 3 SQL Prerequisites

  1. Select Control Panel > Programs and Features,
    - Microsoft SQL Server 2014 - Express Edition     (in V4.2)
    - Microsoft SQL Server 2022 - Express Edition (in V5.0, V5.1, V6.0, V7.0 and V8.0)
  2. Click Uninstall.
    NOTE: If you skipped the SQL installation, then during installation, you must uninstall the following three components:
    - Microsoft SQL Server 2012 - Native Client
    - Microsoft SQL Server 2016 - System CLR Types
    - Microsoft SQL Server 2016 - Management Objects
  1. The uninstallation program completes without displaying any verification message.

 

Uninstall WinCC OA

Perform the following procedure to completely uninstall the prerequisite WinCC OA.

NOTE:
You do not need to uninstall WinCC OA for upgrading Desigo CC from V4.2 to V7.0, because, while installing the current version, the Installer automatically uninstalls any previous version of WinCC OA, before installing the required version WinCC OA 3.19.

  1. In the Program and Features window, select:
    • the Desigo CC Base Platform 3.10 for V 1.1.
      In V 4.0, select WinCC OA 3.16
      In V 5.0, select WinCC OA 3.17
      In V 6.0, select WinCC OA 3.18
      In V 7.0, select WinCC OA 3.19
      In V 8.0, select WinCC OA 3.19
  2. Right-click and select Uninstall.
  3. Click Yes.
  4. The uninstallation program completes.
  5. Delete the C:\[company name]\WinCC_OA folder, if present.
  6. (Optional) Restart the computer.
