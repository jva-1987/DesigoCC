Open the Scopes Application

You can select an existing Scope definition to modify, or begin configuring a new one from scratch.

  • Do one of the following:
    • To create a new Scope definition, select Project > System Settings > Scopes or one of its subfolders.
    • To modify an existing Scope definition, select Project > System Settings > Scopes > […] > [scope definition].
  • The Scopes tab displays.
