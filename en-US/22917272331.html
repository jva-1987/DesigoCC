Add the Scripts Extension to a Project

Perform this procedure to check whether the Scripts extension is present in a project, and to add it if needed.

NOTE: Once you have added an extension to a project you cannot remove it. Only add the necessary extensions to a project.

  1. You are logged into Windows with admin rights.
  2. The Scripts extension is installed.
  1. Exit Desigo CC.
  2. Launch the System Management Console (SMC).
  3. In the SMC tree, select Projects > [project].
  4. In the Project Settings tab, open the Extension Information expander and look for Scripts in the list of the project extension modules.
  5. If Scripts is included in the list and its status is Updated, then the project already includes the necessary extension, skip to the end of this procedure. Otherwise, continue with the steps below.
  6. If the Project status in the Server Project Information expander is Started, stop the project as follows:
    a. Click Stop .
    b. Click OK.
  7. The status bar at the bottom of the screen displays Project is Stopping. Once finished, the Project status changes to Stopped.
  8. If Scripts is listed in the Extension Information expander but its status is Outdated, click Upgrade .
  9. The extension changes are applied to the project. This may take several minutes. Check the progress bar at the bottom of the screen. Once finished, Upgrade successful displays.
  10. If Scripts is not listed in the Extension Information expander, add it to the project as follows:
    a. Click (+) Add to project.
    b. In the Select Project Extensions dialog box, expand Standard Modules.
    c. Select the Scripts check box.
    d. Click OK.
    e. Click Save Project .
  1. The extension is added. Once finished, extensions added successfully displays.
