Set Node Description

By default, for each OPC UA server, OPC UA adapter nodes to be imported into the Desigo CC project will have the OPC UA display name as Desigo CC nodes description. If required, you can specify that they will use the OPC UA description instead. For nodes that do not have an OPC UA description text, the OPC UA display name will be used in any case.

  1. In the OPC UA Client Configuration section, click Server Settings.
  2. By default, the OPC UA display name is for the Desigo CC node description .
  4. Under Node Description, select Description.
