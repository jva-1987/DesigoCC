Import Notifications

This section contains procedures for importing notifications.

Import a Notification Template Hierarchy

  1. The location of the exported notification template hierarchy ZIP file is known.
  1. Select Applications > Notification > Notification Templates.
  2. Click Import .
  3. The Select file dialog box displays.
  4. Browse to the exported .zip file location.
  5. Select the .zip file to be imported.
  6. Click Open.
  7. The notification template hierarchy is imported.

Import a Message Template

  1. A message template is exported and the location of the exported ZIP file is known.
  1. Select Applications > Notification > Notification Templates.
  2. Select the message template to be imported.
  3. Click Import .
  4. The Select file dialog box displays.
  5. Browse to the location of the exported ZIP file.
  6. Select the ZIP file, to be imported.
  7. Click Open.
  1. The message template is imported.
