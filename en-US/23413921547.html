Create, Configure, and Save a Device

You can create a new device in an area or within a sector that you have created. Alternatively, you can copy an existing device.

  1. Select Applications > Powermanager > [area] OR [sector].
  2. Click the Area/Sector tab.
  3. Click Create and select Create Device.
  4. A list of device groups displays in a tabular format.
  5. Select a desired device type for example PAC4200.
  6. Configure the parameters for the device by entering information in the expanders in the Device Configuration. To create devices using gateway communication, enable the Gateway Communication checkbox.
    NOTE: It is recommended to connect 8 to 12 devices to one Gateway Communication (one IP address) for better performance.
  7. Click Save .
  8. The Save Object As dialog box displays.
  9. Enter a name and description for the device. Do not use the characters “ / ” or “ \ ” in the device name.
  10. The devices are created and configured and are added to the System Browser.
