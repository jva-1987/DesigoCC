[Example] Creating a New Organization Mode Based on an Existing One

Scenario: The Open and Close organization mode is being used to reflect the status of Area1 of the facility. You now want to create a new organization mode (Open and Close 2) that can be used to similarly reflect the status of another area, Area2.

  1. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Organization Modes > Open and Close.
  3. The Object Configurator tab displays.
  4. Click Save As .
  5. In the Save Object As dialog box, select Organization Modes as the destination where you want to save the new organization mode:
    a. In the Name field, enter Open and Close 2.
    b. Click OK.
  1. The new organization mode is added to the Organization Modes folder. It also becomes available in all the areas of the system that use organization modes as inputs or outputs.
