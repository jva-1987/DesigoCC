Creating and Configuring a Unit Conversion Table

  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > [folder].
  3. Create a unit conversion folder by clicking New on the toolbar and selecting New Unit Conversion Folder from the menu options.
    NOTE: You can add only one unit conversion folder below a libraries folder.
  4. Select the folder created in step 3 and create a unit conversion table by clicking Add New Table and selecting New Unit Conversion Table from the menu options.
  5. Select the Unit Conversion tab.
  6. The Unit Conversion application displays with an empty unit conversion table.
  7. Select the desired text group using any of the following procedures:
    a. Select the text group from the Text Group drop-down list.
    b. Drag a text group from System Browser to the Unit Conversion tab.
  8. The selected text group displays in the Text Group drop-down list and the From Unit and To Unit fields in the first row of the unit conversion table are populated with the text entries of the selected text group.
  9. Select the source and destination units in the From Unit and To Unit fields respectively. For example, if you want to convert Watts to Kilowatts, select Watts in the From Unit field and Kilowatts in the To Unit field.
  10. In the Factor field, enter the conversion factor. For example, if you are converting Watts to Kilowatts, enter 0.001 in the Factor field as 1 Watt equals to 0.001 Kilowatts.
  11. In the Offset field, specify the value for the offset.
  12. Click Save .
  1. The unit conversion table is created and displays below the unit conversion folder.
