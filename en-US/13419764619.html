Executing a Report Step in Assisted Treatment

  1. You are in the Assisted Treatment window, and the operating procedure includes a report step that you must manually execute. For background information about the Desigo CC reporting feature, see the reference section.
  2. From the Steps checklist, select the [report step].
  3. The preconfigured report displays in the Default tab. A new report is generated each time that you select this step.
  4. If the report contains form controls (for example, editable fields or drop-down lists), specify the necessary information and click Save User Input .
  5. Click Send to Output .
  1. The report is routed to a file, email, or printer, depending on its configured output destination.
  2. When you generate an alarm printout, a report, or an event-handling form using a virtual printer (for example, a PDF printer), the output files are saved in the following location: [Installation Drive]:\[Installation Folder]\[Project Name]\data\Reporting\Reports.
  3. When you complete an assisted procedure of an event, any attachment (such as, alarm printout, report, or event-handling form) is saved in the following location: [Installation Drive]:\[Installation Folder]\[Project Name]\shared\attachments. The Activity Log report includes a link to these attachments.
