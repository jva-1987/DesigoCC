Adding Sensors

Different sensor symbols are used in Desigo CC for ventilation ducts and pipes in order for the sensor symbols to fit precisely on the drawing grid. The associated sensor symbol is added if you drag a data point to the graphics page.

    • [TOa (outside air sensor)]
    • [TR (room sensor)]

  6. Drag the object to the graphics page.
  1. The added object displays at the maximum scale.

 

Rotating

  1. The sensor symbol is added to the graphic.
  1. In the Mode group, click Test .
  2. Select the sensor symbol.
  3. Left-click and hold, then right-click until the sensor symbol displays in the desired direction.
  4. Release the left mouse button.
  5. The sensor symbol is displayed in the desired direction.

Configuring Display and Symbol

As an option, you can configure the frame, display precision, and unit for the analog display.

  1. In the Display Box field, enter a number from 0 through 1.
    • 0 = No frame displays.
    • 1 = Frame display.
  2. In the Precision field, enter a number from 0 through 5.
    • 0 = No decimal points display.
    • 1-5 = Displays number of decimal points as per entry from 1 through 5.
  3. In the SensorFontBold field, enter a number from 0 through 1.
    • 0 = Text is displayed as normal.
    • 1 = Text is displayed as bold face.
  4. In the SensorFontSize field, enter a number from 10 through 14 (default is 12).
  5. In the SensorName field, enter a letter from A through Z (default is T). The text display as a symbol.
  6. In the Units field, enter the unit. The unit is displayed by value.
    NOTE: A manually entered value overwrites the imported unit.
  7. (Applies only to Wall sensor) In the WallColor field, enter 0 or 1.
    • 0 = Wall background as per side background.
    • 1 = Wall background is red.
