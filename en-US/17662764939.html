SNMP Agent Troubleshooting

Error/Issue

Situation

Do the following...

The SNMP Configurator tool is displayed, but grayed out.

When you start the SNMP Configurator and try to establish the connection, and WSI is unreachable.

Wait until the connection is reestablished.

See your system administrator.

Misaligned Views

When the connection with the SNMP web service interface is up and running, but the configuration in the Desigo CC project is modified.

Disconnect and reconnect to the SNMP web service interface to retrieve the latest configuration data.

SNMP Manager request is timed out

A response dos not arrive for GetBulk and GetSubtree requests.

Use the MIB browser to adjust the following parameters appropriately:

  • Max Repetitions
  • Timeout

Views removed from Desigo CC are still available in the MIB browser.

Views removed from Desigo CC are not automatically removed also from the MIB browser.

Manually remove those instances from the MIB browser, as follows:

  1. Stop the SNMP Agent service.
  2. Navigate to C:\Program Files\Siemens\SNMPAgent\Data\SNMP_Agent folder.
  3. Delete the MIB files that correspond to the views deleted in Desigo CC.
  4. Restart the SNMP Agent service.
