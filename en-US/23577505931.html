Configure User Device Types and Delivering Methods

  1. Select Applications > Notification > Recipients.
  2. The Recipients Editor tab displays.
  3. Open the User Device Types expander.
  4. The list of default user device types display.
  5. Select a user device type from the list to work with an existing user device type. To add a new device, perform the following steps:
    a. Click Add: A new row with a user device type is appended to the list.
    b. Update the name and description of the newly-added user device type.
  6. In the Delivering Methods section, click Add and specify the delivery method.
  7. Click Save .
