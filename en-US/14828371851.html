Configuring Fibrolaser III Event Commands Macros

Scenario

You want commands for zone events to also work from Event List. For this purpose, you must configure specific macros in Desigo CC. For background information, see Fibrolaser III Events in Modbus Integration.

Prerequisites

  • Desigo CC is configured to integrate the Fibrolaser III heat detection system using the Modbus TCP protocol. See Integrating Fibrolaser III Using Modbus TCP.
  • System Manager is in Engineering Mode.
