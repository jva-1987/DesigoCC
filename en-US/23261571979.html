Analyze the Trend Data

  • Specify the time range for which you want to display the data displayed in the trend view from the following:
    • Select the time range from the time range scrollbar
    • Select the time range from a list of predefined time ranges
    • Select an absolute time range
    • Select a relative time range from the current date
    • Select a relative time range from the start date
    • Select a relative time range from the stop date
  1. The trend data displays as per the specified time range.
