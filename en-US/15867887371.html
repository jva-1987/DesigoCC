Customizing the Fire Brigade Map Symbols

Scenario

In the fire brigade maps, to represent the fire objects, you can use any of the symbol style available in the library, or you can use one or more customized symbol in a new style as described in this section.

Prerequisites

  • You are authorized and trained to customize the system libraries.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

NOTICE

System malfunction

Incorrect modifications to system libraries will cause the product to malfunction.
Do not modify system libraries unless you have enough knowledge and experience to change them without destroying or corrupting crucial data.

 

Customize the Fire Brigade Map Symbols

  1. You know the name of the running Desigo CC project and the location of the project files on the disk.
  1. Check if any of the predefined symbols in the Fire Brigade Maps library can suit your needs:
    a. Select Project > System Settings > Libraries > L1 Headquarters > Fire > Device > Fire Brigade Map > Symbols.
    b. Select the Graphics tab.
    c. Choose one of symbols and note down its name.
    d. Proceed to step 6.
  2. (Optional) If none of the predefined symbols can be used, you can customize the library and create a new symbol. You can skip this step if you have already customized the library for a new graphic template (see Customizing the Fire Brigade Map Template):
    a. Select Project > System Settings > Libraries > L1 Headquarters > Fire > Device > Fire Brigade Maps.
    b. Select the Library Configurator tab.
    c. Select Customize the entire library to a lower level .
    d. Click OK.
  3. The structure of the Fire Brigade Maps library is duplicated under the allowed customization level. For example, […] > L4-Project > Fire > Device > Fire Brigade Maps.
  4. Select again Project > System Settings > Libraries > L1 Headquarters > Fire > Device > Fire Brigade Maps > Symbols.
  5. Choose one of symbols, select Customize and click OK.
  6. The Graphic Editor opens.
  7. Modify the symbol as necessary and save it. Note down the symbol name.
    NOTE: For background information about graphic symbols, see the Graphic Editor section.
  8. Customize the function you intend to use for the fire brigade maps:
    a. Select Project > System Settings > Libraries > L1 Headquarters > Fire > Detection > Functions > [function].
    b. Select Customize and click OK.
    c. The function is duplicated under the allowed customization level. For example, […] > L4-Project > Fire > Detection > Functions > [function].
    d.     Select the new function and then select the Models and Functions tab.
    e. Close the Main, Properties and Function Mapper expanders to better display the Symbols expander. Scroll the list of symbols as necessary to have a free drop area.
  9. Associate the symbol to the customized function. You need to operate with two windows on the screen: Desigo CC and Windows Explorer.
    a. From Windows Explorer, open the running project folder <project file location>\libraries\Fire_Device_Fire_Brigade_Maps_Project_1\Symbols.
    b.     Drag the <symbol name>.CCS file into the Symbols expander.
    c. Right-click the symbol in the Symbols expander and select Style > New Style, then enter for example FBM.
    NOTE: You need to create the style only once. If the style is available in the right-click menu, just select it.
    d. Click Save .
  10. Repeat step 6 and 7 for more functions as necessary.
  1. The new graphic symbol style is available for you to use in the fire brigade maps.
