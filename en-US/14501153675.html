NK823x Status LEDs

The front panel of the NK823x Ethernet Port houses 9 labeled LEDs.

Each Ethernet connector includes 2 LEDs.

Front Panel LEDs

 

Color

Description

Power

Green

Power (hardware controlled)

Vital function

Green

Software vitality:

  • Blinking (1 flash): core software running.

Tamper

Red/Green

Tamper:

  • Red: tamper alarm (hardware controlled).
  • Green: tamper disabled (from management station).

Download

Red

Network diagnostics:

  • OFF: Status OK.
  • Blinking (3 flashes): FTP channel open (default IP addresses - see NK823x Internal Jumpers and DIP Switches).
  • ON: Critical/hardware fault.

Diagnostics

Yellow

Internal diagnostics:

  • OFF: Status OK.
  • Blinking (fast): Booting operating system after restart.
  • Blinking (1 flash): Missing or insufficient license.
  • Blinking (2 flashes): Trouble with I2C bus to I/O modules; DF8000 configuration mismatch.
  • Blinking (4 flashes): Trouble with the serial/network interface.
  • Blinking (5 flashes): Trouble with application modules (some application modules cannot start).
  • ON: Critical/hardware fault.

COM 1

Red/Green

COM status:

  • Red: RX.
  • Green: TX.
