Run the Template Information Report

NOTE: Overloading a data range in a query may result in slower load times. The number of values logged should not exceed 100 thousand.

  1. You have configured an InfoCenter report template that is triggered by an object in System Browser.
  2. You have created a web rule link for the report.
  3. System Manager is in Operating mode.
  1. In System Browser, select Management View.
  2. Navigate to the system object on which the InfoCenter report is configured.
  3. In the Application Viewer tab, select the report you want to run.
  4. The Parameter dialog box displays.
  5. Set the fields to your preferences.
  6. Click Execute to generate the report.
