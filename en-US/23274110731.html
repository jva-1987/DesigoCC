Configure the FS20 BACnet Fire Network

Create the BACnet Network

  1. You configured a BACnet driver.
  1. In System Browser, select Project > Field Networks.
  2. In the Object Configurator tab, click New , and select New BACnet Network.
  3. In the New Object dialog box, enter a name and description.
  4. Click OK.
  5. The newly created network is automatically selected in System Browser.
  6. Select the BACnet tab and open the Network Settings expander.
  7. In the Monitored by driver drop-down list, select the driver.
  8. Click Save .

Configure the Block Commands

In the Block Commands Behavior expander, you can enable broadcasting of the alarm-handling commands across the fire network.

  1. Open the Block Commands Behavior expander.
  2. Select the check boxes to enable the following commands:
    • Block acknowledge
    • Block reset
    • Block silence/unsilence
  3. Click Save .
  1. Macros for broadcasting the commands that you enable here will be automatically created under Applications > Logics > Macros > Block Command Macros.
    NOTE 1:     During event handling, the block commands apply to the entire fire network. In System Manager, the block commands only apply to the selected panel.
    NOTE 2: If the block command macros are subject to validation, make sure that the BACnet network is set with the same validation profile as the block command macros or higher.

Configure the T1/T2 Countdown Display

In the Fire System Options tab, in the Alarm Handling Timers expander, you can enable the alarm-handling countdown display on the user interface when a relevant event is presented.

  1. Select Project > Field Networks > [BACnet network].
  2. Select the Fire System Options tab.
  3. Open the Alarm Handling Timers expander.
  4. Select the Enable T1/T2 Timers check box to enable the countdown display during alarm handling.
  5. Click Save .

NOTE: Make sure the management stations and the fire panels are set to the same local time zone. This is required for the countdown timers to work properly.

Configure the Import of Map Numbers

In the Fire System Options tab, in the Import Map Numbers expander, you can enable the import of map numbers and set the objects whose description will include the map number.

  1. Select Project > Field Networks > [BACnet network].
  2. Select the Fire System Options tab.
  3. Open the Import Map Numbers expander.
  4. Select the Enable import of map numbers check box to enable the import.
  5. In the Add the map number to list, select one or more check boxes to add the corresponding map number to the description of the objects when importing the panel configuration. You can select area, section, zone, and detector objects.
  6. Click Save .

In the BACnet tab, the Auto Discovery Progress and the Global Objects Folder Mapping expanders are not used.
