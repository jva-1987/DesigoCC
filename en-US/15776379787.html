InfoCenter

For procedures or workflows, see the step-by-step section.

The InfoCenter feature allows you to generate reports based on existing report templates in your InfoCenter database. You can run these reports to display the details of historical data from Desigo CC . In order to work with the InfoCenter extension module, you must first configure the Advanced Reporting feature, and then configure the InfoCenter module. Once you run a report with the desired parameters, you can then schedule it to run automatically based on a trigger time and date.

Desigo CC displays data in the same manner as InfoCenter, with one exception. If a requested time interval does not have any data, that time interval does not display in the Desigo CC scatter charts, etc. Charts only display the start and end of data in the report.
