Configuring Scripts Libraries

Desigo CC scripts can be packaged together into a library. This allows those scripts to be transported with all the versioning benefits of libraries.
You can also perform operations such as hashing, signing, or object association on the scripts included in a library. For background information, see Scripts Libraries.

Prerequisites

  • You are authorized to handle libraries at the customization level of the scripts library.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
