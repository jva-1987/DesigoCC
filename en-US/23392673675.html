Create and Configure a CSV File

  1. You have knowledge about the format and content of the CSV file.
  2. The properties of the object model to which you want to assign the address are specified in the [Property] field in the [POINTS] section of the CSV. For more information, see N to 1 Mapping in Format of the CSV Data from Librarian Configuration Workspace topic.
  • Create the CSV file at any location on your hard drive and thereafter modify the information present in the [DEVICES] and [POINTS] sections as per your device configuration.
  1. Save the CSV file.
