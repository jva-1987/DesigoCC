Browsing Device Information from a Web Browser

  1. A device connection object is created below the IEC61850 network.
  1. In the System Browser, navigate to Application View > Documents.
  2. In the Document Settings section, select the Web URL option and enter the IP address of the connected device whose details you want to view.
  3. Click Preview. The details of the device display.
  4. Save the device details by clicking Save Document As .
  5. The device details are saved as a document node below the Documents folder.
  6. Navigate to the Management View and select the device connection object from below the IEC61850 network.
  7. In order to view the details of the device, navigate back to the Application View, select the Documents folder, and thereafter select the saved document.
  8. Drag the document to the Related Items expander in the Object Configurator tab and click Save .
  10. The document displays in the Related Items tab.
  12. Select the document from the Related Items tab.
  1. The details of the document display in the Secondary pane.

NOTE:
You can view the device information in a web browser only if the connected device supports web services.
