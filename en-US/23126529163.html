Apply Filters

  1. In System Browser, select Application View.
  2. Select Applications > Log Viewer.
  3. Log Viewer displays with the combined data from the Activity Log and Event Log.
  4. Apply the following filters according to the type of columns:
    • Non-Date Time Columns
    • Date Time Columns
  5. Save the log view definition by clicking Save .
  1. The log view displays data according to the specified filters.
