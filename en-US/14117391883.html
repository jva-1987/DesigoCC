Drawing Pipes

  1. A graphics page is open.
  1. In the View menu, select Library.
  2. The Library Browser opens.
  3. In the Mode group, click Test .
  4. Select the Library folder.
  5. In the Filter text box, enter the following filter criterion: *2D*pipe*.
  6. The pipes are displayed.
  7. In the corresponding color, select the straight pipe symbol, angle, T-piece or crossing and drag it to the graphics page.

NOTE:
The pipe color cannot be changed. The pipe symbol must be deleted and replaced with a new one if the color is wrong.

 

Repositioning a Straight Pipe Piece

  1. Select the pipe symbol using the cursor.
  2. Press the left mouse button to move the pipe symbol to the corresponding X/Y position. The pipe marking is not visible while positioning.
  3. Release the left mouse button at the desired position.
  1. The pipe piece is repositioned.

 

Extending Straight Pipe Piece

  1. Select the pipe symbol using the cursor.
  2. Position the mouse pointer over the left or right marking until the pointer's form changes.
  3. Left-click and drag the pipe end to the corresponding length. The pipe marking is not visible while resizing.
  4. Release the left mouse button at the desired position.
  1. The length of the pipe piece changes.

 

Rotating a Straight Pipe Piece

  1. Select the pipe symbol using the cursor.
  2. Position the mouse pointer over the green highlighting until the pointer's form changes.
  3. Left-click and hold to turn the pipe symbol to the desired angle. The pipe marking is not visible while positioning.
  4. Release the left button as soon as you have reached the desired angle.
  1. The pipe symbol is positioned at the appropriate angle.

NOTE:
The angle of the desired symbol can be entered at the following location: Symbol Instance Properties > Layout > Angle.

 

Rotating an Angle or T-Piece

  1. In the Mode group, click Test .
  2. Select the angle symbol or T-piece symbol.
  3. Left-click and hold, then right-click until the pipe symbol displays in the desired direction.

NOTE:
The angle of the desired symbol can be entered at the following location: Symbol Instance Properties > Layout > Angle.
