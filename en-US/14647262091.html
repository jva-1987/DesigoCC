Integrating CS11 Systems in Cerloop

Scenario

You want to configure one or more CS11 fire systems in a Cerloop network, integrated into Desigo CC via NK823x on a BACnet network.
For background information, see the reference section.

Prerequisites

  • The Legacy systems via NK8000 extension is installed and included in the active project.
  • You have completed the following workflows:
    • Connecting to the NK823x Unit
    • Configuring the NK823x Unit
  • You have defined the unique BACnet address (Device ID) for all CS11/CK11 units.
    See BACnet Networks for NK823x DMS Applications
  • From the AlgoWorks tool, you have exported the configuration files.
    The configuration files are text files (.txt) that contain the complete data structure of the panels.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
