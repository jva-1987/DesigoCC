Security Access for the Graphics Editor

If you have access rights for the Graphics Editor application, you can create, modify, and delete graphics, symbols, graphic templates, and graphic folders.

Graphics Editor: Security

Access

If Enabled

Show

You can access the Graphics Editor.

Create

You can create new and save existing graphics and graphic folders.

Delete

You can delete graphics and graphic folders.

Save

You can save any changes to a graphic.

 

NOTE:
If you have no access rights for the Graphics Editor, you can still open a graphic, but only in Read-Only mode. No changes can be made.
