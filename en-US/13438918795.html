Licensing Troubleshooting

The following table may help you to resolve possible issues with Desigo CC licenses.

Problem

Cause

Resolution

The Desigo CC server station is isolated from the license pool for a long interval of time (approximately 10 or 15 minutes). Consequently, the Desigo CC server starts releasing the license options. When the connection is reestablished, you may encounter a short-term in which:

  • The Desigo CC server starts and makes a double check out of the license options
  • The Courtesy Mode might occur because a under licenses issue is detected temporarily.

Network disconnection.

When the connection is reestablished, this issue is automatically solved after the next license check carried out by the Desigo CC server.
