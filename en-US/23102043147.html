Run the OTM Event Manager

  • From the event manager installation folder, you can run one of the following commands:
    • Siemens.GMS.OTM_NORIS_EventManager.exe, to run the event manager in console mode.
    • Install_Service.bat, to install and run the service.
      NOTE: Use Uninstall_service.bat to stop and remove the event manager from the Windows service list.
