Hide Walls and Doors with the Same Filter

Scenario: All walls and doors are hidden with just one filter. Use the existing Walls filter as the starting point. The BIM object model for doors in unknown and must be first found.

 

1st Step: Find and Hide the BIM Object Model for a Door

Scenario: You want to find the BIM object model for a door to hide.

  1. Edit is enabled in BIM Viewer.
  2. A defined Walls filter is available.
  1. Select a door with the mouse.
  2. Press SHIFT + right click.
  3. Select Hide this element and then to filter: Walls.
  4. In the Filter Name dialog box, select the Walls filter.
  5. Click Edit.
  6. Displays the associated BIM object. The IFC Name column references the selected door object below the wall object. The BIM object model IFCDOOR displays in the IFC Type column.
  7. Enter a name for the walls and doors in the Name text field.
  8. Click OK.
  9. Test: Select the Walls and doors check box.
  10. The highlighted doors as well as the walls are hidden.
  11. Click Save .

 

2. Step: Hide all Doors

Scenario: All doors belonging to this BIM object model are hidden after finding a door.

  1. Click Edit.
  2. Displays the associated BIM object. The IFC Name column references the door object. The BIM object model IFCDOOR displays in the IFC Type column.
  3. In section Exempt from BIM model, select the IFCDOOR BIM object model.
  4. Click Edit.
  5. The Hidden objects dialog box displays.
  6. Delete the entry in the IFC Name text field and enter an asterisk (*).
  7. Click OK and then OK to close both dialog boxes.
    • Test: Select the Walls and doors check box.
  8. All walls and doors are hidden. Other objects such as stairs, radiators, furniture, etc. are still.
  9. Click Save .
