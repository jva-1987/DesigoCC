Print the Trend View

  1. You must define the Print in color setting in advance if you want to print in color.
  1. Select the time range to print using the time range scrollbar. The time range displayed on the screen is printed.
  2. Click Print .
  3. Select or clear the Fit to page check box. Select the corresponding option in the toolbar if Fit to page is not selected.
  4. Define print properties for:
    • Margins (top, bottom, left and right).
    • Printer.
    • Orientation (portrait or landscape).
    • Paper size.
  5. Click Print and Close to print, or click Close.
  1. The trend data is printed in the form of a chart or a table as displayed in the trend view. You can also print the trend data from Reports.
    For more information, see Printing Trend Data from Reports.
