Managing the SNMP Libraries

This section provides instructions for importing, exporting or upgrading SNMP libraries.

Prerequisites

  • You are trained and authorized to work with system libraries.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Export an SNMP Library

You can export an SNMP library to use in other projects.

  1. In System Browser, select Project > System Settings > Libraries.
  2. The Library Configurator tab displays.
  3. In the list of the installed libraries, select the SNMP library to export.
  4. Click Export.
  5. In the Export Libraries dialog box, select the target directory and click Save.
  1. The SNMP library is exported in a GMS file (a file with extension .gms), which is saved in the chosen directory of the file system.

 

Import an SNMP Library

An exported SNMP library can be imported from a GMS file (a file with extension .gms) in other projects where you want to re-use any SNMP devices contained in it.

  1. A GMS file containing the configuration of an SNMP library is available in the file system.
  1. In System Browser, select Project > System Settings > Libraries.
  2. The Library Configurator tab displays.
  3. Click Import.
  4. In the Import Libraries dialog box, select the GMS file that corresponds to the SNMP library you want to import.
  5. Click Open.
  1. When the import is complete, the SNMP library appears under its specific library level and folder in System Browser.

 

Upgrade an SNMP Library

When a new library revision (only the minor version changes) is released for an SNMP device, you can upgrade the SNMP library in the existing projects.

  1. Export the library from the project where you have created the new revision.
  2. The library is exported to a GMS file (a file with extension .gms).
  3. Copy the GMS file (a file with extension .gms) to the Desigo CC computer where the SNMP library must be upgraded.
  4. In System Browser, select Project > System Settings > Libraries.
  5. The Library Configurator tab displays.
  6. Click Upgrade.
  7. In the Browse for Folder dialog box, select the folder containing the GMS file that corresponds with the SNMP library revision you want to upgrade.
  8. Click OK.
  9. When the Upgrade Libraries dialog box displays, click Yes to upgrade the SNMP library.
