Print Trend Data from Reports

  1. You have selected an existing Report Definition in System Browser > Reports or you have created a Report Definition.
  2. One or more local printers are configured in the system.
  1. Click the Home tab.
  2. Do one of the following:
    • From the Insert group box, click the Plot group box, and select a Trends plot and drag it onto the selected Report Definition.
      NOTE: The cursor changes and indicates the selected plot name.
    • Right-click the Report Definition and select the required type of plot from the Insert Plot option.
    • Drag the desired Trend View Definition from System Browser on the Report Definition. The Name filter of the inserted plot is set to the dropped Trend View.
  3. The plot placeholder is added to the Report Definition. By default, the plot is left-aligned.
    NOTE: In Run mode, this plot image is replaced by the actual data retrieved from the Report service.
  4. Click the Settings tab.
  5. From the Report Output group box, click Dialog Launcher .
  6. The Report Output Definition dialog box displays.
  7. Select PDF in the Report format list.
  8. Select Printer in the Destination types list and specify the other printer details.
  9. Click Add.
  10. The selected format and destination are added to the Output Definition list.
  11. Click OK.
  1. The configured Report Output Definitions display in the Report Output group box and the report is printed in the PDF format on automatic execution.
