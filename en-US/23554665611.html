Checking the System Information for Custom Graphics

  1. You have exported the System Information Report from SICLIMAT X.
  1. Open the System Information Report file SystemInformationReport.xml with an application that can read XML files.
  2. The APICs (Automatic PICtures) section shows a list of standard and custom graphics. System graphics are standard Siclimat X graphics that are automatically mapped to Desigo CC graphics. Custom graphics are customer-specific Siclimat X graphics that must be mapped to Desigo CC functions before you migrate the graphics.
  4. If custom graphics exist, map the custom graphics to Desigo CC functions.
