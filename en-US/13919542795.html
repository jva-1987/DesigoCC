Room operation up to V5.1

Each room is represented as a graphic and displays the components depending on the application used:

  • HVAC application
  • Lighting
  • Blinds

Prerequisites:

  • The system browser is in Operating mode.

 

Selecting Room

Views to the Rooms

View in System Browser

Floor Plan View

  1. In the System Browser or Floor Plan View, select the room to operate.
  2. The room graphic opens and displays the most important values in the Operation tab.
    NOTE: Do not make changes in the dialog boxes.
  3. Perform one the following procedures as needed.

 

Changing Operating Mode

  1. The room is selected and the room graphic displays.
  2. The present priority property allows you to write a command (see Note).
  1. Select Values > Room operating mode.
  2. Click the Operation tab and select the Value property.
    • Click Manual.
    • In the Value drop-down list, select the Operating mode.
    • Click Send.
  1. The message Manual successful is displayed.
  2. In the Extended Operation tab, the Present Priority and Priority Matrix changes to 8 or 13.

NOTE:
Successful operation only possible if you can write at a higher priority than is displayed on the status bar for the Priority Matrix. You can, for example, overwrite at the present priority 13 at 8, but not 7.
The display arrow indicates an active priority of 1 to 8. At priority 8, you can change the operating mode, for example, Protection to Pre-Comfort.

 

Changing Setpoints

  1. The room is selected and the room graphic displays.
  2. You can write a command at the Priority Matrix (see NOTE).
  1. Select Settings > Setpoint heating comfort or Setpoint cooling comfort.
  2. In the Operation tab, select the Value property.
    • Click Manual.
    • Complete the Value field.
    • Click Send.
  1. The message Command successful appears.
  2. In the Extended Operation tab, the Priority Matrix changes to 13.

NOTE:
Successful operation only possible if you can write at a higher priority than is displayed on the status bar for the Priority Matrix. You can, for example, overwrite present priority 13 with priority 8.

 

Switching Lights On and Off

  1. The room is selected and the room graphic displays.
  1. Select the object lighting and object value .
  2. In the Operation tab, select the Value property.
    • Click Manual.
    • Complete the Value field or use a sliding controller.
    • Click Send.
  1. The message Command successful appears.
  2. The light objects displays in yellow.
  3. In the Extended Operation tab, the Priority Matrix changes to 13.

NOTE:
Successful operation only possible if you can write at a higher priority than is displayed on the status bar for the Priority Matrix. You can, for example, overwrite present priority 13 with priority 8.

 

Releasing Manual Operation

  1. The room is selected and the room graphic displays.
  1. Select one or more settings to enable manual operation:
    • Values > Room operating mode
    • Settings > Setpoint cooling comfort
    • Settings > Setpoint heating comfort
    • Settings > Setpoint shift
  2. In the Operation tab, select the Value property.
    • Click Auto.
  3. In the Extended Operation tab, the Priority Matrix changes to 13.
    • Click Send.
  1. The message Auto successful is displayed.
  2. The room operates in automatic mode if not an other manual operating state is active.

 

Enabling Manual Operating at a Higher Priority

A trained user can overwrite the present operation at priority 13 using a higher priority 8.

  1. The room is selected and the room graphic displays.
  1. Select one or more settings to enable manual operation:
    • Values > Room operating mode
    • Settings > Setpoint cooling comfort
    • Settings > Setpoint heating comfort
    • Settings > Setpoint shift
  2. In the Extended Operation tab, select the Priority matrix property.
    • Click Command.
    • In the Priority drop-down list, select the desired priority.
    • Click Send.
  1. The message Command successful is displayed.

 

Positioning Blinds

  1. The room is selected and the room graphic displays.
  1. Select the Blinds object and Value object .
  2. Click the Operation tab.
  3. The properties of the blind displays.

Positioning Height

  1. Select the Height property and click Manual.
  2. Complete the Value field or use a sliding controller.
  3. Click Send.
  1. The message Manual successful is displayed.
  2. The blind object displays in a dynamic status.
  3. Present priority and Priority Matrix are set to 13.

Positioning Angle

  1. Select the Angle value property and click Command.
  2. Complete the Value field or use a sliding controller.
  3. Click Send.
  1. The message Manual successful is displayed.
  2. The blind object displays in a dynamic status.
  3. Present priority and Priority Matrix are set to 13.

NOTE:
Successful operation only possible if you can write at a higher priority than is displayed on the status bar for the Priority Matrix. You can, for example, overwrite present priority 13 with priority 8.
