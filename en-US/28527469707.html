Defining the Dataset

  1. In System Browser, select Application View > Powermanager.
  2. Select the Bulk Virtual Device Creation tab.
  3. Navigate to the logical view or user defined view with the System Browser node hierarchy and drag-and-drop the parent node to be included in the dataset in the Parent Node field.
  4. Identify a child node within the parent node and drag-and-drop the same child node to the Hierarchy Depth field. The selected child node denotes the end point of the node hierarchy for the dataset. Depending on the level of the selected child node the hierarchy depth is automatically populated.
  1. The dataset is defined. You can now proceed to filtering the unwanted nodes from the dataset.
