Root Certificate Information Expander (.pem)

When you click Create Certificate > Create Root Certificate (.pem), the Root Certificate Information expander displays. It allows you to create a .pem root certificate.

Root Certificate Information Expander

 

Description

Certificate file name

Type the file name of the root certificate. The certificate and the key file name should not contain blanks for special characters (/,\,?,<, >,*,|,").
NOTE: The certificate file and key file names cannot be the same.

Key file name

Type the key file name.

Key file password

Type the password required to secure the root key file. This password is required for creating host certificates.

Confirm password

Re-enter the password for confirmation.

Path

Browse for the location to store the certificate on the disk.

Expiration

Set the validity period. Once a certificate's validity period is over, a new certificate must be requested by the subject of the now-expired certificate. By default, the certificate expires after 3650 days.

Subject Identifier Information

Provide the subject's identifier information:
— Subject name
— Department
— Organization
— City/district
— State/province
— Country code (only two characters)
