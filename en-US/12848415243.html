Integrating Calibrator Objects into the Management Platform

Scenario

You want to integrate calibrator objects into the management platform and associate these objects with their dimensions to aggregator nodes in views.
For background information, see the reference section.

Prerequisites

  • You have installed the extension Virtual Devices and Building Data and added it into a project using the SMC.
  • You have assigned application rights to calibrators.
  • System Manager is in Engineering mode.
  • System Browser is in Application View.
