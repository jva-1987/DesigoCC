Display Desigo Insight Log Data

  1. The System Browser is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Log Viewer.
  3. The Application Viewer tab opens.
    NOTE: If a file is displayed as Invalid in the Validation column, it is manually changed after the fact and the checksum is thus false.
  4. Select the export file from the applicable section List of log files or List of audit files that matches the desired date range.
  5. The Parameter dialog box opens.
  6. Define:
    • In the drop-down list Time range for report content, selection option None (default setting for Desigo Insight Migration).
    • In the From date field, you can limit the start date of the desired file.
    • In the To date field, you can limit the end date of the desired file.
    • In the Number of maximum rows field, the number of report lines to be created or select option No value (= all lines).
  7. Click Execute.
  8. The report is generated and is displayed as a preview.
  9. (Optional) Click Back to xml list, if you want to execute another report.
  10. Click Export report and select the output format.
  11. Click Execute.
