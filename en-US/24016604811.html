Create Project from existing Template

  1. Select Start > All Programs > [Supplier name] > Desigo CC >Tools > Desigo CC SMC.
  2. The System Management Console displays.
  3. In the System Management Console tree, select Projects.
  4. Management tab displays.
  5. In the Management toolbar, click Restore Template Project .
  6. Open the Project and System Information expander and provide the details as follows:
    a. Enter the project name.
    b. In the Project path field, click Browse to select target folder. The default project path is [Installation Drive]:\[Installation Folder].
    c. (Optional) Type the port numbers for Pmon, Data, Event, HDB Reader and CCom ports. The default port numbers display as the config file present at
    [Installation Drive]:\[Installation Folder]\GMSMainProject\_DefaultProject\config.
  7. In the History Database Connection expander, select the HDB to link to the new project.
  8. (Optional for some extension modules):
    a. Click Next .
    b. Engineer the settings by the selected extension module.
  9. Click Save .
  10. Click OK to create the project.
  11. If confirmed, the data entered during project creation, including the HDB selected, is validated and saved.
  12. The new project is created under Projects in the System Management Console tree.
  13. A project folder structure is created with subfolders and files at the specified path.
  14. A Pmon service is created in the Windows services using the System user in the following format: GMS_WCCILpmon_[Project Name]. This service provides support for runtime systems.
  15. Click Activate and then Yes.
  16. Click Start Project .
  17. Launch Desigo CC.
    NOTE:     Ask your supervisor for name and password to log on.
