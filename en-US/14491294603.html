Starting or Stopping the SORIS Driver

The SORIS driver must be running (in Started state) to connect to and import the configuration files. You may also need to stop and restart the driver for certain settings, such as changes to alarm tables, to take effect.

  1. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Depending on where the driver was created, select one of the following:
    - Project > Management System > Servers > Main Server > Drivers > [SORIS driver]
    - Project > Management System > FEPs > [FEP station] > [drivers folder] > [SORIS driver]
  3. In the Extended Operation tab, the Manager Status property indicates whether the driver is Stopped or Started.
  4. If the driver is Stopped, click Start to start the driver with connection to the field.
  1. The Manager Status changes to Started. The SORIS driver is fully operational.
