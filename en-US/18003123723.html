Assigning Graphics / Symbols to an Object Model or Function

You can associate graphics or symbols to models and functions. For background information, see Graphic Templates and Symbols Expanders.

Assign Graphics to an Object Model or Function

  1. In System Browser, select Project > System Settings > Libraries > [...] > [object model or function].
  2. In the Models & Functions tab, open the Graphic Templates expander.
  3. Select the Manual Navigation check box.
  4. In System Browser, navigate to and select the required Graphic Templates library.
  5. Right-click the selected Graphic Templates library and select Send to the Secondary pane.
  6. The Secondary pane opens and displays available graphic templates.
  7. In the Secondary pane, select the desired graphic template and drag-and-drop it to the Graphic Templates expander.
  8. Repeat the preceding step for all graphic templates you want to assign to this object model or function.
  9. Define the default graphic template. Right-click the desired graphic template and select Set as Default.
  10. Click Save .
  1. The graphic templates are assigned to the object model or function.
  2. The default graphic template is defined.

During graphics engineering the graphic template designated as the default graphic template is dragged and dropped to a data point where the graphic is placed.

 

Assign Symbols to an Object Model or Function

  1. In System Browser, select Project > System Settings > Libraries > [...] > [object model or function].
  2. In the Models & Functions tab, open the Symbols expander.
  3. Select the Manual Navigation check box.
  4. In System Browser, select the symbol library you want to use. For example,
    Project > System Settings > Libraries > L1-Headquarter > BA > [subsystem] > Symbols.
  5. Right-click the selected symbol library and select the Send to the Secondary pane.
  6. The Secondary pane opens and displays available symbols.
  7. In the secondary pane, select the desired symbol and drag-and-drop it to the Symbols expander (left-click and hold).
  8. Repeat the preceding step for all symbols you want to assign to this object model or function.
  9. To define the library style, right-click the desired symbol and select Style: > 2D, 2D+ or 3D.
  10. To define the default symbol for each library style, right-click the desired symbol and select Set as Default.
  11. Click Save .
  1. The symbols are assigned to the object model or function.
  2. The library style is defined for each symbol.
  3. The default symbol for each library style is defined.

During graphics engineering the symbol designated as the default symbol is dragged-and-dropped to a data point where the graphic is placed.

 

Delete an Assigned Graphic Template

  1. In System Browser, select Project > System Settings > Libraries > [...] > [object model or function].
  2. Select Project > System Settings > Libraries > [object model or function].
  3. In the Models & Functions tab, open the Graphic Templates expander.
  4. Select the graphic template for deletion.
  5. Right-click the graphic template and select Delete.
  6. Click Save .
  1. The graphic template is deleted.

 

Delete Assigned Symbols

  1. In System Browser, select Project > System Settings > Libraries > [...] > [object model or function].
  2. In the Models & Functions tab, open the Symbols expander.
  3. Select the symbol for deletion.
  4. Right-click the symbol and select Delete.
  5. Click Save .
  1. The symbol is deleted.
