Customizing a Nested HQ Symbol

Scenario

You want to create a Symbol for a smoke detector that displays a specific background color for a specific event/information condition. The HQ Library already has a Symbol for this situation; however, the color for the background is gray. You want to create an additional customized Project detector Symbol to display the purple background for the Information event condition. All other runtime behavior as for the original Symbol as well as the event lamp can remain the same.

Prerequisites

  • Function must be known
