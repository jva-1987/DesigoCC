Create Software Option for Tile

  1. Select folder Project > Settings > Libraries > [L2-Region, L3-Country or L4-Project] > Common > Common > Software options.
  2. Select the Library Configurator tab.
  3. Select Add a new object .
  4. Select from the GMSRoomOverview_RoomToAdd list.
  5. In the New Object dialog box, enter the name and description.
  6. Click OK.
  1. The new object is created in the Software options folder.
