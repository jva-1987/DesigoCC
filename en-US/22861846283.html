Finishing Steps After SPC Import

This procedure is part of the workflow for integrating an SPC intrusion system.

 

Prerequisites

  • You have imported the configuration file for an SPC panel.

 

Finishing Steps

  1. When the intrusion system is not set, changes of state of intrusion zones can create unnecessary data traffic. If the zone state filter cannot be applied at panel level (see Configure Protocol Settings), make sure to filter unnecessary notifications at the management platform level (see Setting the Zone State Filter).
  2. To prevent generation of unidentified events when an SPC point is taken out of scan, such events need to be disabled. See Disabling Unidentified Events for Out of Scan.
  3. When finished with all the imports, make sure the Transaction Mode is set back to Logging.
  4. If the SPC driver was running in Configuration Mode for the import, stop it and restart it in full mode for normal operation. See Stopping and Starting the SPC Driver.
