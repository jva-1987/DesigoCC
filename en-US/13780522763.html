D3 Visualization

The D3 Visualization extension module is a web client application that allows you to display and view visual representations of data displayed in a dashboard or a web page. The dashboard and web pages display in the Application Viewer pane.

You can re-size and zoom in/out of a dashboard and maintain the aspect ratio.

For related procedures and workflows, see the step-by-step section.
