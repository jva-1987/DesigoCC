Viewing Trend log Report data with Pivot table

  1. You can view the Trend log report data with Pivot table in new excel format.
  2. Make sure that the Report output is generated in the new excel format and a new worksheet is added to the excel document with Pivot table options.
  • You need to manually update the pre-defined pivot settings.
  1. For detailed information to view Reports data with pivot table, refer View a Report Definition in the Excel format with a Template in Reports section.
