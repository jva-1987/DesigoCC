Adding Text to the Graphics Page

  1. Click Text .
  2. Move the cursor to the corresponding position on the graphics page. The cursor changes to .
  3. Left-click and drag the cursor to the desired size .
  4. Release the left mouse button.
  5. The text field size is defined.
  6. Enter project-specific text:
    • In the Text field
      or
    • Select Text Properties > Text > Text.
  7. Format the text as needed .

NOTE:
Text entered manually cannot be translated for multilingual projects.

 

Adding Multilingual Texts

  1. A multilingual Desigo CC installation and project is available.
  2. The text group is created with texts specific to the project and translated into the applicable languages.
  1. Click Text.
  2. Move the cursor to the corresponding position on the graphics page. The cursor changes to .
  3. Left-click and drag the cursor to the desired size .
  4. Release the left mouse button.
  5. The text field size is defined.
  6. Select Text Properties > Text > Text Type and the Translated Text option.
  7. In System Browser, select Management View.
  8. Select Project > System Settings > Libraries >
    • To use project-specific texts, select Project > Common > Texts.
    • To use standardized texts, select BA (HQ) or Global.
  9. Select the text group (with the corresponding text content) and drag it to the dialog Text Properties > Text > Text.
  10. The object reference is added to the Text field, for example, TxG_ProjectTextGroup.2.
    NOTE:
    - The prefix TxG_ is added to each reference to a text group.
    - The suffix (for example, .2) refers to the second value entry from the selected text group table (see Table Suffix for Text from the Text Group).
  11. On the graphics page, double-click Text and select the Project-specific text from the displayed list.
  12. Format the text as needed .
  1. The text is added to the graphic and is displayed in Online mode as per the language in the user profile.

Creating a Project-Specific Text Group for Multiple Languages

For multilingual projects, you must create an entry in a project-specific text group for each graphic. Create a unique project-specific text group for each discipline and/or building structure. You should limit the entries per text group to 20 to maintain an overview.

  1. A multilingual Desigo CC installation and project is available.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > Project > General >Texts.
  3. An empty text group is created.
  4. Enter the data in the first row:
    • Value = 1
    • En-US (default language) = Enter text 1 for en-US or leave blank.
    • Language 2: Enter text 2 for language 2
    • Language 3: Enter text 3 for language 3
  5. Click Save .
  6. Enter the data for the new text group:
    • Object name: The object name is used for referencing. Special characters and spaces are not permitted.
    • Name: Text group name in System Browser. Special characters and spaces are not permitted.
    • Description: Descriptive name for the text group in System Browser.
  7. Click OK.
  8. Click to create additional project-specific entries.
  9. Click Save .
  1. The project-specific texts are created and translated.

NOTE:
For graphic text referencing, the suffix TxG_ precedes the object name (for example, TxG_ProjectTextGroup.2).
