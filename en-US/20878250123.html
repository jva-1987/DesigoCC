Monitoring the Hard Drive of a Station

Scenario

You can configure Desigo CC to automatically monitor the hard drive of a server, installed client, or FEP station, so that it generates an alarm when that station’s hard drive is detected to be faulty. You can also (only for the server station) configure the disk capacity monitoring, so that an alarm is generated when the free disk space on the Desigo CC server station is detected to be insufficient. For background information, see the reference section.

Prerequisites

  • The station whose hard drive you want to monitor is configured in System Manager. See Creating Stations in System Manager.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
