(Optional) Test the Symbol Instance

  1. From the Library Browser, drag-and-drop the Symbol to a graphic.
  2. With the Symbol instance selected on the graphic, from System Browser, drag a data point node into the Symbol Instance Properties > Symbol Instance > Object Reference field.
  3. From the Home tab, in the Modes group, click the Runtime icon.
  1. You should observe the color and height of the bar chart change as the data point’s value changes.
