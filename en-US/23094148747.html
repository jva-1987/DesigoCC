Configure the NK823x Internal Inputs

The I/O tree of the NK823x can have a maximum of 3 inputs (Input 1, Input 2, and Input 3). You can individually configure the inputs as normally open or normally closed.

You can also optionally allocate these three inputs to external power supply supervision, in which case they are replaced by a single External Power Supply Supervision node.

Set NO/NC Logic of Internal Inputs

  1. In System Browser, select Project > Field Networks > [BACnet Network] > [NK823x] > I/O > [input].
  2. Select the NK8000 tab.
  3. In the Configuration expander, select one of the following:
    - Normally open (default)
    - Normally closed.
  4. Click Save .

 

Use Internal Inputs for External Power Supply

You can allocate the 3 inputs for supervision of an external power supply.

  1. In System Browser, select Project > Field Networks > [BACnet Network] > [NK823x] > I/O.
  2. Select the NK8000 tab.
  3. Click New and select New External Power Supply.
  4. Click OK to confirm.
  5. The 3 inputs are replaced by External Power Supply Supervision, which is automatically selected.
  6. In the Configuration expander, select Normally open or Normally closed.
  7. Click Save .

NOTE: To undo this setting, delete the External Power Supply Supervision node and manually add the inputs back again.
