Browser Menu

Field/Button

Description

Save As

Launches a File dialog box that allows you to select a file name to save data from the data list as a CSV file.

Print

Allows you to print the data displayed in the data list.

Clear Errors

Allows you to clear the user and error messages in the bottom pane of the application.

Exit

Allows you to close the application.
