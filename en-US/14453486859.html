Creating a Migration Package in Desigo Insight

Scenario

You want to create a migration report and migration package for Desigo CC migration. For background information, see the Desigo Insight Data Acquisition section.

  • Creating the Migration/Export Packet
  • System Report

Prerequisites

  • The project is active in the project utility.
  • Projects, created in versions prior to 4.0, must first be updated to version 4.0 and then updated to version 6.0 SP3. Additional information is available in Installation and Configuration of Desigo Insight (CM110591).
  • You have a configuration dongle license for Desigo Insight.
  • A backup of the Desigo Insight project is available and imported.
