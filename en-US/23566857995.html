Export a Log View Definition

  1. In System Browser, select Application View.
  2. Select Applications > Log Viewer > [log view definition].
  3. Click Export .
  4. The Browse for Folder dialog box displays.
  5. Browse for the desired location and click OK.
  1. A confirmation message displays. The log view definition is exported and saved.
