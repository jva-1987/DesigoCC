Additional Procedures for Virtual Devices and Building Data

This section provides additional procedures for working with calibrator objects and dimensions.

Add Dimensions to an Existing Calibrator Object

  1. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Calibrators.
  3. Select the calibrator object to which you want to add new dimensions.
  4. The dimensions of the calibrator object display in the Dimensions table.
  5. Click New.
  6. An empty row for entering the details of the dimensions object is added to the Dimensions table.
  7. Click a cell entry in the empty row and enter values for the following:
    • Dimension Name
    • Dimension Description
    • Unit: You must select a value from the drop-down list.
  8. Click Save .
  1. The dimensions are added to the calibrator object.

 

Modify the Unit for a Dimension

  1. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Calibrators > [calibrator].
  3. The dimensions of the object display in the Dimensions table.
  4. Double-click the unit field of the dimension.
  5. Modify the unit for the dimension.
  6. Click Save

 

Add/Modify/Delete the Dimension History

  1. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Calibrators > [calibrator].
  3. The details of the dimensions for the selected calibrator object displays in the Dimensions section.
  4. Select the dimension whose history is to be modified.
  5. The history for the selected dimension displays in the Dimension History section.
  6. Perform the following steps to add, modify, or delete a history entry.
    • To add a new entry to the history, click Add. This is applicable, if you want to add a new dimension record to the history. In the new record, specify the values for Value and Valid From fields. If the value for the Valid To field is not specified, this entry will be treated as the current entry and value of this entry will be treated as current value. If the value for the Valid To field is not specified for the previous entry, the Valid From of the new entry will be set as the Valid To of the previous entry.
    • To modify a dimension entry in the history, double-click the entry to make it editable and modify the values for Value, Valid From, and Valid To fields.
    • To delete an entry, select the entry in the dimension history and click Delete.

 

Modify the Current Value of a Dimension from the Extended Operations Tab

  1. You have customized the calibrators library according to your customization level. For details, see the Libraries Help.
  2. In the Models & Functions tab, the display level of the Current Value property of the Calibrator Dimension object model is set from Reserved to Generic Display And Software.
  1. In System Browser, select Application View.
  2. Select Applications > Calibrators > [calibrator object].
  3. Select the dimension of the calibrator object.
  4. The current value of the dimension displays in the Extended Operation tab.
  5. Modify the current value and click Execute.
  1. The updated value reflects in the Dimension History section.

 

Delete Calibrator Objects

  1. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Calibrators > [calibrator object].
  3. Select the Calibrators tab.
  4. The details of the calibrator object display.
  5. Click Delete .
  1. The calibrator object is deleted along with any dimension objects that are associated with it.

 

Delete Dimension Objects

  1. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Calibrators > [calibrator object].
  3. The dimensions of the object display in the Dimensions section.
  4. Do one of the following:
    • Select the dimension to be deleted and click Delete.
    • In System Browser, select the dimension below the respective calibrator object and click Delete .
    • To remove a record in the dimensions history, select the record in the Dimensions History section and click Delete .
  5. Click Save .
