Install the Haystack Adapter

  1. Run Haystack_Adapter.msi.
  2. Follow the wizard instructions to complete the installation.
  3. Open the installation folder and run the batch file StartAdapter_Configuration.bat.
  4. When running for the first time in console mode, the adapter creates the Adapter.config file and opens it in a new window with Notepad.
    NOTE: in all .config files, blank lines and lines starting with # are comments.
  5. In the Notepad window, configure the Adapter.config file by specifying the default Haystack tags to use for Site, Equipment, and Point entities when imported in the Desigo CC Name and Description in System Browser. Any missing tag will be replaced by the id identifier (default tag):
    Sites: <Haystack Tag for Name>, <Haystack Tag for Description>
    Equipments: <Haystack Tag for Name>, <Haystack Tag for Description>
    Points: <Haystack Tag for Name>, <Haystack Tag for Description>
    NOTE:     Using id for the name is always recommended.

    For example:
  7. Select File > Save and then File > Exit.
    NOTE: To change this setting in future, run again StartAdapter_Configuration.bat.
  8. In the adapter console window, press Enter to continue.
  9. When running for the first time in console mode, the adapter starts the HaystackAdapterConfigurator tool.
