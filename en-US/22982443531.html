SEM3 devices do not communicate with Powermanager.

The Explicit Congestion Notification (ECN) setting on the Windows Server 2012 R2/Windows Server 2016 is enabled. Disable the ECN setting to work with SEM3 devices.

To disable the ECN setting, perform the following:

- Open Command Prompt and run the below command to check the status of the ECN setting:
netsh interface tcp show global

- If ECN capability is set to enabled, then run the below command in Command Prompt to disable the ECN setting:
netsh int tcp set global ecncapability=disabled

When restoring a project from an earlier version without unlinking the Notification DB.

  1. Navigate to SMC, and select the project.
  2. Stop the project.
  3. Navigate to folder path C:\Siemens\SENTRON\Powermanager\ <ProjectName> \mns\Mns_ProjectSpecific.config in the local machine.

The system drive storage is getting filled up by the logs from Microsoft IIS for every web request and response, leading to insufficient storage in the future.

Typically, IIS logs all success and error messages for any web request-responses. If the IIS logging feature is enabled in Windows, all IIS messages will be logged, eventually filling up the system drive over time.

To avoid system drive storage consumption by IIS logs, it is recommended to delete the IIS logs manually and disable the IIS logging feature for the Powermanager website.

  • To delete the logs, navigate to the path %SystemDrive%\inetpub\logs\LogFiles and delete all the logs inside the folders named W3SVCx (For example: W3SVC1, W3SVC2, etc).
    NOTE: Some of the logs may not be deleted.

To disable the logging functionality, follow the steps listed below:

  1. Open Internet Information Services (IIS) Manager.
  2. On the Connections pane, navigate to Sites > PowermanagerWebsite.
  3. On the IIS expander, double click the Logging.
  4. The Logging page opens.
  5. On the Actions pane, click Disable.
  1. The logging is successfully disabled.

If the Internet Information Services (IIS) Manager is not available on your system, enable the IIS Management Console from Windows features.

To enable IIS on your system, follow the steps listed below:

  1. Open Turn Windows features on or off using the search bar.
  2. Navigate to Internet Information Services > Web Management tools.
  3. Enable IIS Management Console checkbox.
  4. Click OK.
  1. Internet Information Services (IIS) Manager is now available on your system.
