Test the Rotating Graphics Display

  1. You have configured the Rotating Graphics display profile .LDL and assigned a designated user.
  1. Login to Desigo CC using the designated user account for the rotating graphic display.
  2. In System Browser, navigate to and select the graphic or object node designated for the rotating display.
  3. The rotating graphic display is initiated.
  4. If you have set the rotating graphic display node as the Favorite in the previous optional step, the rotating graphic display will start automatically upon logging in.
