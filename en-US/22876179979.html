Set Windows Security Options

Sometimes the following error has been found in the log file:

WCCOAopc (1), xxxxxx, IMPL, SEVERE, 54, Unexpected state, OPCServer, initialize, Could not start OPCEnum on remote server: xxx.xxx.xxx.xxx

In this case, to allow the WinCC OA Opc driver to connect to the remote OPCEnum, the following additional DCOM configuration must be performed on the remote machine:

  1. In the Windows search box on the taskbar, enter Control Panel.
  2. Select Administrative Tools.
  3. Select Local Security Policy.
  4. Navigate to Security Settings > Local Policies > Security Options.
  5. In the right pane, right-click Network access: Let Everyone permissions apply to anonymous users, and select Properties.
  6. Choose the Enabled option.
  7. Click Apply.
  8. Click OK.
