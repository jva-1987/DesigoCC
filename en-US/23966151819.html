Time Filter

  1. Select a table in the report to which you want to apply the time filter.
  2. Do one of the following:
    • Click Dialog Launcher in the Time Filter group box.
    • Right-click the selected table and select Time Filter from Filters.
    • Double-click inside the Time Filter group box.
  3. The Time Filter dialog box displays.
  4. For Events, Activities, Orphan Events and Orphan Activities table only, select the column on which you want to apply the time filter from the Select Column drop-down list.
  5. Select the appropriate values for Date time specification: Exact, Custom, or Relative.
  6. From the Trend Interval drop-down list, select the interval in which the trend data has to be displayed.
    NOTE: The Trend Interval drop-down list is only available for Trends table.
  7. Click OK.
    NOTE: By default, the Unlimited option is selected.
  1. The Time filter is added in the Time Filter group box.
