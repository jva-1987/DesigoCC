Creating a Graphic

  1. You are in Design or Test mode and have a canvas or graphic displayed in the Graphics Editor.
  2. NOTE: Any graphic property changes made during the editing session are maintained for each graphic you create until you end your current session by exiting Desigo CC.
  1. To create a new graphic, do one of the following:
    • From the ribbon, click File, and then select New Graphic.
    • If the Quick Access Toolbar is visible, select New Graphic .
    • A blank canvas displays in the Graphics Editor with default properties.
  2. Click the canvas, and from the Property Viewer adjust the following Graphic Properties as needed:
    • Layout – Adjust the Width and Height of the graphic. Default is set to: Width 970 and Height 665.
    • Colors – Set a background color. The default value is white or #FFFFFFFF.
    • Graphic – Set the logical scale factor, width, height, and units for the graphic, as well as how the graphic automatically increases and decreases in order to display the full content. You can also enter a limit on the number of status and command lines allowed. For posterity, the version used when you create the graphic, and once saved, the date of creation are also displayed.
    • Workspace – Configure the properties associated with any grids, guidelines, or axis points on your graphic.
  3. Any other graphics created during this editing session will retain the values as defined above. When you exit the Graphics Editor, the property values go back to the defaults.
  4. Configure and create your graphic by doing any or all of the following:
    • Add a background graphic
    • Create depths and layers
    • Add primitive elements and Symbols
  5. From the click Save As .
