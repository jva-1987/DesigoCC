Modifying the Client Profile

  1. Select Project > Management station > Server.
  2. Click the System Management tab.
  3. In the Client profile drop-down list, select the desired profile.
  4. Click Save .
  5. Restart Desigo CC to make the profile active.

NOTE:
Each user can have his/her own client profile assigned (see Configuring User > Defining User Properties).
