Services Menu

Field/Button

Description

Send Who-Is

Allows you to send a Who-Is request to a range of devices or as a broadcast on the BACnet network. Any device that responds is added to the Who-Is list. You can select those devices to add to the Browser tree list.

Send Who-Has

Sends a broadcast request out on the BACnet network to view which devices have the object name you are searching for. This is an optional service and may not be supported on all devices.

Write Property

Allows you to view the current value, and change the value and priority information if necessary.

Device Communication Control

Allows you to select device communication control commands you want sent to BACnet devices. Also allows you to disable or stop communication to selected field panels for a period of time if you are running certain types of tests.

Re-Initialize Device

Re-initializes the selected device to the default values of the device.

Life Safety Operation

Performs the life safety operation when you have a Life Safety Point or Zone selected. Possible operations include Silence, Silence Audible, Silence Visual, Reset, Reset Alarm, and Reset Fault.

Create Object

Allows you to create a new object in a BACnet device. If the device does not support this, you will receive an error message.

Delete Object

Allows you to delete an object from a BACnet device. If the device does not support this, you will receive an error message.

Send Text Message

Allows you to send a text message to a device. You can also broadcast an unconfirmed message to a local subnet, a remote subnet, or globally.

Private Transfer

Allows you to transfer a private message to a BACnet device.

Add/Remove List Element

Allows you to add list elements to or remove them from a BACnet device.

BBMD Configuration

Allows you to send the Broadcast Distribution table to one or more BACnet devices.

Import Points

Allows you to selectively import objects (points) from third-party devices.
