Link the Symbol to the Function

  1. In the customized OPC library, under Functions, select the function you want to link the symbol to (for example, Automatic Fire Detector).
  2. Click the Model & Functions tab.
  3. Open the Symbol expander.
  4. In Windows Explorer of the PC, open the symbol folder of the customized OPC library, for example:
    GMSProjects\[Project]\libraries\Fire_Device_OPC_Project_1
  5. Drag the CCS file of the symbol (for example, DYN_3D_Fire_OPC_Detector_Ceiling_None_Horizontal_001.ccs) to the Symbol expander.
  6. The symbol displays in the Symbol expander.
  7. Right-click the symbol, select Style and select a graphic style (for example, 3D).
  8. Right-click the symbol and select Set as Default.
  9. Click Save .
  1. The new symbol is associated to all data points which use the function.
