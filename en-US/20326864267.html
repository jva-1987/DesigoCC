Integrating the ESMI Fire System Through Online Discovery

Scenario

Desigo CC is connected to the Esmi fire detection panel, and you want to run the online discovery. For background information, see the reference section.

Performing an offline configuration is recommended instead if you want to import data into Desigo CC through an EDE file to populate the logical view.

Prerequisites

  • The ESMI extension module is installed and included in the active project.
    The following extension is also selected automatically:
    • DMS Common
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
