Modify OPC DA Server DCOM Settings

Regardless of the installed update of the operating system, on the computer where Desigo CC is installed, do the following:

  1. In the Windows search box on the taskbar, enter DCOMCNFG.
  2. The DCOM configuration process is initiated.
  3. In the Component Services window, navigate to Console Root > Component Services > Computers > My Computer > DCOM Config.
  4. In the list of objects in the right window pane, right-click Siemens.OPC.Server.DA and select Properties.
  5. In the OPC DA Server Properties dialog box, select the General tab.
  6. From the Default Authentication Level drop-down list, select Packet Integrity.
  7. Click Apply.
  8. Click OK.
  • It is suggested to repeat this configuration on the remote computers for all the third-party OPC servers, unless specified otherwise in the third-party servers documentation.
