Integrating SIPORT

This workflow describes how to configure Desigo CC for connecting to a SIPORT access control system via Modbus. For background information see the SIPORT reference section.

 

Prerequisites

  • The SiPort Modbusextension module is installed and included in the active project.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
