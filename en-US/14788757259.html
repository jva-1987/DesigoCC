Retrieving the Configuration for a Meteoblue Adapter

Once you configured the settings of a Meteoblue adapter, and added new locations to be monitored in a project that is already running, System Browser must be refreshed to reflect the configuration updates in a project that is already running.

  1. Stop the adapter: in the Meteoblue adapter executable window, click Close .
  2. Stop the Desigo CC project.
  3. Restart the adapter and the Desigo CC project.
  4. Select Project > Field Networks > [SORIS network] > [Meteoblue adapter].
  5. In the Extended Operation tab, the Online property indicates Connected while the Discovery Status property indicates Idle.
  6. Next to the URL property, click Discover.
  1. The previous configured parameters and locations are retrieved from the configuration file. Any new related objects are added to System Browser.
