Link Parameters for AIC Pharma Reports

Properties of Links

Parameter

Label

Value

P1

Report Template

Name of the report template.

P2

Format

Output format of the report may be PDF, HTML, XLSX, DOCX.
This is a required value.

P3

Data Range

Specify Time Range1) for the report data.
This is a required value.

P4

Interval

Time interval
5 Min
10 Min
15 Min
30 Min
One Hour
Two Hours
Six Hours

This is a required value.

P5

Enable Limits Trend Chart

Specify how horizontal limits should display in the report.
0 – no limit lines displayed
1 – display range (shaded)
2 – display lines (only)

P6

High Limit

Upper limit line value.
This is a required value.

P7

Low Limit

Lower limit line value.
This is a required value.

P8

Alternative Title

Optional title displayed at the top of the report.

P9

Alternative Purpose

Optional purpose and text description displayed near the top of the report.

P10

Number of Elements

Specify the number of elements considered for the Report, so that the page layout is properly handled.

This is a required value for Trend Aggregated Table.

1)

For acceptable values, see P3 Link Parameters for Pharma Reports.
