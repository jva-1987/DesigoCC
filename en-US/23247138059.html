(Optional) Additional configurations on the RDF302 room thermostat

The following values can be transmitted to the RDF302 room thermostat via Modbus:

  • Present value of the outside air temperature
  • Window state
  • Electric heating state

 

  1. Select Project > Field networks > [RDF302 Network] > [Interfacename] > [RDF302 room thermostat].
  2. In the Advanced operation tab, select property Outside air temperature.
    a. Select Change.
    b.. In the Value field, select the desired outdoor temperature.
    c. Select Send.
  3. In the Advanced operation tab, select property Window state.
    a. Select Change.
    b. From the Value drop-down list box , select Open or Closed.
    c. Select Send.
  4. In the Advanced operation tab, select property Enable electric heating.
    a. Select Change.
    b. From the Value drop-down list box , select True or False.
    c. Select Send.
  1. The selected setting is applied and RDF302 room thermostat updated.
