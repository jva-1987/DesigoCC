Associating Data Points and Symbols to a Graphic

Scenario

You have imported an AutoCAD drawing and you want to add data points and symbols to the graphic. There are two methods you can use to add data points and symbols to a graphic:

  • Drag a data point from System Browser onto the graphic, and then select the symbol from the object model, or
  • From the Library Browser, select a symbol, and then associate it with an appropriate data point.

For background information, see Symbol Basics.

Prerequisites

  • System Manager is in Engineering mode.
  • System Browser is in Management View.
