Integrating ATS 4500

Scenario

You want to configure Desigo CC to operate on the ATS 4500 system. For background information, see ATS 4500 Integration.

Prerequisites

  • The ATS 4500 extension is installed and included in the active project. The following extensions are also selected automatically:
    • Security Domain Libraries
    • SORIS Driver
  • The ATS 4500 adapter is running and connected to the ATS 4500 system. For instructions, see Installing and Starting the ATS 4500 Adapter.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Licensing

This extension is covered by license. Check that the required license is available; otherwise, you may not be able to configure and work with this extension.
