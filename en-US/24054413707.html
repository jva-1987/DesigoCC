Setting Internet Explorer as the Default for the SSA Wizard

The web client SSA Wizard or ABT-SSA Wizard or Web Interface is based on the Chrome web controls. This permits the correct display of Desigo automation station graphics. The manufacturer provides the security updates for the control. Web controls can be switched off in the event of a threat to cyber security by selecting the check box Use the standard IE browser for web access. The SSA wizard or ABT-SSA wizard or Web Interface then applies IE web control. Regular Windows updates eliminate threats to cyber security.

  1. Select Project > Field Networks > [Network name].
  2. Select the Desigo Automation and Control tab.
  3. Open the Other Settings expander.
  4. Select the check box Use standard IE browser for web access.
  1. Internet Explorer is now the standard browser.
