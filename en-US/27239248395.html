Installing the Desigo CC Connector in Power BI

In order for Power BI to recognize the data that comes from Desigo CC, you must install a connector within Power BI.

The latest version of the Desigo CC installation package includes this connector software.

Before installing the connector, make sure you have the latest version of Microsoft Power BI Desktop installed on your PC.

Then follow these steps:

  1. In the Desigo installation package, navigate to the file ..\InstallerDVD\InstallFiles\EM\POWER_BI\AddSW
  2. Now you must copy the installer file, “DesigoCC_Enh.mez,” into your local [Documents]\Power BI Desktop\Custom Connectors folder. If this folder does not exist, create it manually.
  3. Start Power BI Desktop and navigate to File > Options and Settings > Options > Security > Data Extensions. Select “Allow any extension to load without validation or warning.”

Restart PowerBI Desktop. The Desigo CC extension should now be enabled within Power BI

Note: When you start the connector, be sure the connector version displayed within PowerBI matches your current version of Desigo CC.

Configure Security and User Permissions

To create and modify Power BI reports, you must configure the correct permissions for the current user in the System Manager.

Prerequisites

  • System Manager is in Engineering Mode
  • System Browser is in Management View

Step-by-Step

  1. Select Project -> System Settings -> Security
  2. Select the security group that will be configuring and producing Power BI reports (for example, Default Admin). Next to the list of security groups, you will see a checklist of applications.
  3. Scroll down to the Power BI entry. Make sure that both Show and Configure are checked or the user will be unable to create and save reports.
