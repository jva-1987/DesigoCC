GAMMA Symbols

For the GAMMA building control systems, the following symbols are available in graphics with this integration:

DYN_3D_Detector_Presence_None_Central_001

  • Default symbol for the following presence detectors: UP 258E22, UP 258D12, UP 258D31, UP 258D41, UP 258D51, and UP 258D61.
  • Symbol background changes color according to the Summary Status color propagated from the point itself, or the points at lower levels.

DYN_3D_RoomSensor_Temperature_Analog_Central_014

  • Default symbol for the following flush mount room thermostats: RDF600KN and RD600KNS.
  • Symbol background changes color according to the Summary Status color propagated from the point itself, or the points at lower levels.

DYN_3D_RoomSensor_Temperature_Analog_Central_081

  • Default symbol for the RDF800KN touchscreen room thermostats.

DYN_3D_RoomSensor_Temperature_Analog_Central_070

DYN_3D_RoomSensor_Temperature_Analog_Central_075
