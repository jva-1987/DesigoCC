Drawing a Line Element

The line element allows you to draw a line segment between two points---a start and end handle--on the canvas.

Select the topic related to your task:

Convert a Line Element

  1. You are in Design or Test mode and you have line element on the active canvas you want to modify into one of the three path element options: Arc, Cubic or Quadratic Bezier. Or, if you want to split the line.
  1. Select the line element you want to modify.
  2. The line element is active on the canvas.
  3. Right-click the mid-segment icon of the line element.
  4. The menu that displays on right-click displays. The Line option is checked by default.
  5. Select from one of the following:
    • Arc
    • Cube Bezier
    • Quadratic Bezier
    • Split
  1. The element is updated to a path and in the Properties view, the Path Properties.

 

Draw a Line Element

  1. You are in Design or Test mode and have a graphic open and want to draw a line segment.
  1. On the Home tab, from the Elements group, click Line .
  2. The mouse pointer changes to crosshair when you move the mouse over the canvas.
  3. On the canvas, click and drag the crosshairs cursor to draw a line.
  4. Release the mouse button when you are done.
  1. A line segment with a mid-segment handle is created.
