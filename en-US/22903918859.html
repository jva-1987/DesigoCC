Start the OPC DA Server

The Desigo CC OPC DA server starts when at least one third-party OPC client connects for the first time. Or you manually launch its executable file (Siemens.Gms.OPCServer.exe).

The Desigo CC OPC DA server name exposed to any third-party OPC clients is Siemens.OPC.Server.DA.1.

  1. You retrieved the latest OPC DA server project data. See Retrieving the OPC DA Server Configuration Data.
  1. Start a third-party OPC client.
  2. Connect to the Desigo CC OPC DA server.
    NOTE: The way a third-party OPC client can connect to the Desigo CC OPC DA server depends on the OPC client (for example, by selecting the Desigo CC OPC DA server, and clicking a specific icon or button).
  1. If the connection is successful, the Desigo CC OPC DA server starts and its status becomes Running.

 

To troubleshoot the Desigo CC OPC DA server, see Troubleshooting OPC Server DA Status.
