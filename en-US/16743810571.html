Notification Licenses

The License Management platform allows you to activate a particular Notification license package. The Notification license model is categorized into following packages:

  • RenoPlus (sbt_gms_mns_opt_RenoPlus): Enables Reno plus features.
  • MNS Connector (sbt_gms_mns_add_500_recipients): Allows to add 500 recipients.
  • MNS Connector (sbt_gms_mns_add_20000_recipients): Allows to add 20000 recipients.

The list of licenses that are activated and that can be activated, can be viewed from Project > Management System > Servers > Main Server > Licenses along with their token information.

Licenses can be viewed in both Operating and Engineering modes, but activate them in only Engineering mode.
