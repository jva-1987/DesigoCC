Setting the Informational Text

You can define technical information or instructions for operators or intervention forces relevant to the off-normal and normal conditions of system objects. The respective configured texts will display in the Informational Text column of Event List.

  1. In System Browser, select the wanted view and then the wanted system object. For example, Project > Field Networks > [network] > Hardware > [device] > [data point].
  2. In the Object Configurator tab, open the Main expander.
  3. In the Informational Text area, do one of the following:
    • To define an instruction, select Text, and then enter the instruction. For example, Call service technician at 123456.
    • To define a technical text, select Reference and then select a text group from the Text group drop-down list and a reference text from the Referenced text drop-down list.
  4. Click Save .

 

If you upgrade V4.0, V4.1, or V4.2 projects to V5.0, only the old event texts for the off-normal conditions are migrated and merged into the informational texts.
