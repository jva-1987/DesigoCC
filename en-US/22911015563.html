Edit Advanced Data Settings

  1. Open the configuration file ([AdvancedPointConfiguration].csv) with Excel.
  2. Edit the following columns as required:
    • [CustomName]
    • [CustomDescription]
    • [FunctionName]
    • [AlarmKind]
      NOTE: Use D for discrete alarms. Use C for continuous alarms.
    • [AlarmClass]
      NOTE: Use the dollar sign ($) as value separator.
    • [AlarmValue]
      NOTE: Use the dollar sign ($) as value separator.
      For more details about the alarm condition configuration, see the tables below for reference.
  3. Save and close the file.
