Powermanager Migration Tab

Powermanager allows you to migrate projects of powermanager V3.6 HF1 to powermanager V6.0. This process includes migration of configurations and data of the projects. Configurations of the project including all device, report, and system level configurations are part of the migration operation. Archived data over required period for selected devices is also a part of this operation. You can perform this operation by using the migration tool. The migration operation consists of two key workflows:

  • Export from powermanager V3.6 HF1: Allows you to select the configurations and the data to be migrated from powermanager V3.6 HF1 to powermanager V6.0.
    NOTE: You should have powermanager V3.6 HF1 to perform the export. Upgrade to powermanager V3.6 HF1, if you are using an older version of powermanager.
  • Import to powermanager V6.0: Allows you to import the exported files from powermanager V3.6 HF1 to powermanager V6.0.
    NOTE: You should have powermanager V6.0 to perform the import. Upgrade to powermanager V6.0 if you are using an older version of powermanager.

 

For more information on how to perform powermanager project’s migration, see Powermanager Migration Help in More Engineering Resources > Others.
