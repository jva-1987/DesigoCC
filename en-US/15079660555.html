KNX Events

The following is the list of the events available in Desigo CC for the KNX integration:

  • System Connection Fault
  • System Connection Partially Fault
  • System Status Request In Progress
  • Device Connection Fault

A partial system connection is sufficient for KNX to align. In this case, both Partially Fault and In Progress events will be generated simultaneously.
