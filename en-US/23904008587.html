Map a Journaling Printer to a Template

  1. At least one server printer is configured as a journaling printer at Management Station > Management System > Servers > Main Server.
  1. Select Project > System Settings > Journaling.
  2. The Journaling Printers tab displays.
  3. Select an existing entry in the list, or click Add to insert a new one.
  4. Select a printer from the Printer drop-down list.
    NOTE: Only server printers configured as journaling printers display in this list.
  5. (Optional) From the Template drop-down list, select a template.
    NOTE: If no template is specified (by selecting None) in the Template drop-down list, the Default Template is used for printing.
  6. Click Save .
  7. The template is mapped to the journaling printer.
