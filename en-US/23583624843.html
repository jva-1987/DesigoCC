View Report Output

You can view a report by selecting a building, floor, or managed meter for which you want to generate the report. However, you must ensure that you have performed all the required steps to configure the energy reports.

  1. In System Browser, select the view with the hierarchical structure of the building with floors, and managed meters installed on the floors.
  2. Select the building, floor, or managed meter for which you want to generate the report.
  3. Depending on the specified configuration in the display rule, the reports corresponding to the selected object display In the Application Viewer tab.
  4. Select the report that you want to execute.
  5. The Parameter dialog box displays. You can filter the data displayed in the report by specifying the required parameters. The filters in the Parameter dialog box will vary according to the type of report selected. Available filters are:
    a. Select and View the Required Report Media Group
    b. Media
    c. Time Range for report content
    d. From Date
    e. To Date
    f. Interval
    g. Graphic Type
    h. Alternative title to the default
    i. Consumption from
  6. The report displays the data as per the specified configurations. For information on the different energy reports, see Overview of Energy Reporting.
  7. You can export the report output to a Word (DOCX), PDF (PDF), or Excel (XLSX) format by clicking Export Report that displays below the Application Viewer tab and select the required format option from the Export Report dialog box.
