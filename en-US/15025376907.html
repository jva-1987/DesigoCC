Configuring Video API for Hanwha Samsung SSM

Scenario

You want to configure Desigo CC for working with Hanwha Samsung SSM video management on Desigo CC stations. For background information, see the reference section. This task is performed as part of the integration workflow, for details see Integrating Video Surveillance with Hanwha Samsung SSM Provider.

Prerequisites

  • The VMS server is installed.
  • HanwhaSamsungProvider.exe was installed, and Siemens Video API Service is available among the Windows services.
  • The Hanwha Samsung SSM and Video Management extensions are installed and included in the active project in the System Management Console (SMC).
