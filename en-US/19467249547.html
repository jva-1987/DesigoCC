Additional Functions

The Room grouping window is open.

 

Remove all Room Grouping

  1. In the room grouping window, select the group to remove.
  2. Click Remove.
  3. Click Save .
  1. Grouping and any connections to the rooms are deleted.

 

Remove Rooms from Room Grouping

  1. Select a grouping for editing.
  2. Click Edit.
  3. The Edit Group dialog box opens.
  4. Select the rooms to remove.
  5. Click Remove.
  6. Click OK.
  7. Click Save .
  1. The room or rooms are removed from this group.

 

Ungroup Association

  1. Select the desired grouping with an association.
  2. Click Edit.
  3. The Edit Group dialog box opens.
  4. Click Remove.
  5. Click Save .
  1. The association to the grouping is removed.

 

Export Room Groupings

Scenario: Backup the existing configuration before undertaking major changes in the Room grouping window.

  1. Select a grouping.
  2. Click Export.
  3. Select the location and enter a file name.
  4. Click Save.
  1. The current configuration from the Room grouping window is saved.

 

Import room groupings

  1. Select a grouping.
  2. Click Import.
  3. Select the save location and file name.
  4. Click Open.
  5. The saved configuration is important and can be viewed in the Room grouping window.
  6. Click Save .
