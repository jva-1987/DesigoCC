Multimedia Collection Additional Procedures

This section provides additional procedures relevant to the integration of the Multimedia Collection extension module in Desigo CC.

Changing the Multimedia Collection Default Background Image

For the background of a multimedia collection, you can use an image different from the default one.

  1. System Manager is in Operating mode.
  2. System Browser is in Application View.
  3. The required file (JPG) is available to you.
  1. Copy the image file (JPG) to the appropriate multimedia collection folder either the default location [Installation Drive]:\[Installation Folder] > GMSMainProject > Help > Multimedia or a subfolder under it.
  2. In System Browser, select Applications > Multimedia Collection > [multimedia collection].
  3. In the Extended Operation tab, next to the BackgroudImage property enter the name of the image contained in the multimedia subfolder (for example, mybackgroundimage.jpg) and click Set.
    NOTE: To reset this setting, delete the name of the image and click Set.
  4. Click another tab and then click the Default tab again.
  1. The welcome page refreshes and displays the new background image.

 

Creating an HTML Page to Show a Media File

  1. In the directory [Installation Drive]:\[Installation Folder]\GMSMainProject\Help\Mutimedia locate and open the Default.html file.
  2. Use this file as a template to create your own HTML page as follows:
    a. Replace the link in the HTML page with the wanted link. For example, the link of a video-sharing website like YouTube.
    b. Save this file with a different name in the appropriate multimedia collection subfolder.

 

Replacing a Media File in a Multimedia Collection

You can replace media files in an existing multimedia collection.

  1. System Manager is in Engineering mode.
  2. System Browser is in Management View.
  1. In System Browser, select the multimedia collection where you want to replace a file: Applications > Multimedia Collection > [multimedia collection].
  2. Select the Extended Operation tab.
  3. The Configuration Subfolder property indicates the media files path for this collection, relative to the default folder. If the property is blank, media files are retrieved from the default folder [Installation Drive]:\[Installation Folder]\GMSMainProject\Help\Mutimedia.
  4. The Textgroup property indicates the Object name of the text group defining the button labels and icons for this collection.
  5. Copy the replacement media file (MP3, MP4, or HTML page) to folder used by this multimedia collection, either the default folder [Installation Drive]:\[Installation Folder]\GMSMainProject\Help\Mutimedia or a subfolder under it.
  6. Select the text group used by this multimedia collection. For example, Project > System Settings > Libraries > L4-Project > Global > Multimedia Collection > Texts > [TxG_MyTextGroup].
  7. In the Text Group Editor tab, locate the text group entry to modify.
  8. In the en-US language column, modify the text with the exact name of the file replaced in the multimedia collection directory. This is the text that will display on the specific button that will play the corresponding replaced media file in Desigo CC.
  9. Click Save .

 

Deleting a Multimedia Collection

  1. System Browser is in Application View.
  1. Select Applications > Multimedia Collection > [multimedia collection].
  2. In the Object Configurator tab, click Delete .
  3. A confirmation message displays.
  4. Click Yes.
  1. The multimedia collection is removed from System Browser.
