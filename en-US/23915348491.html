Program a Fixed Link Report to Execute During the Night

You will configure a reaction that will execute a report during night time hours.

  1. System Manager is in Engineering mode.
  2. You set the user’s execution credentials for an external web application fixed link.
  1. Select Applications > Logics > Reactions.
  2. Select the Reaction Editor tab.
  3. In System Browser, select the Manual navigation check box.
  4. In the Output expander, open the Action expander.
  5. Select Applications > Links > [fixed link].
  6. Drag the external web application fixed link into the Scope/Target fields of the Action expander.
  7. In the Action expander, select the newly-created instruction row and set the following values:
    • Property = Application Name [AppName]
    • Command = Execute [WebAppExecute], FileExtension = HTML[1]
  8. In the Triggers expander, open the Time and Organization Mode expander.
  9. Click the first row.
  10. In the Time column, clear the Begin of day and End of day check boxes, and select the Set End Time check box.
  11. In the Set Start Time field, enter 12:00:00 AM.
  12. In the Set End Time field, enter 07:00:00 AM.
  13. The Time field is set to From 12:00 to 7:00 AM.
  14. In the Effective Days field, from the drop-down list, select Recurrence = Occurs all days.
  15. The Effective Days field, is set to Occurs all days.
  16. In the General Settings expander, enter DownloadWebAppReportNighttime in the Notes field.
  17. Click Save As .
  18. In the Save Object As dialog box, select the main Reactions folder or any subfolder under it as the destination where you want to save this reaction.
  19. In the Name field, enter DownloadWebAppReportNighttime.
  20. Click OK.
