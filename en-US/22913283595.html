Create Outputs

  1. A new group title with the name Outputs is created in the next available row (see Create a Group Title).
  1. In System Browser, select logical view.
  2. Select Logical > [network name] >...> [MyAhu].[Setpoint].
  3. Drag the output object to Row 08 Reference.
  4. In Row 08 Type, enter the value 1 for multistate object.
  5. Scroll to the lower part of the Properties dialog box.
  6. In Row 08 Command, enter the value WritePrio08.
    NOTE: The value is based on the applicable Subsystem object model.
  7. In Row 08 CommandRelease, enter the value ReleasePrio08.
    Note: The value is based on the applicable Subsystem object model.
  8. In Row 08 DisplayRelease, enter the value 1.
  9. In Row 08 Parameter, enter the value Value.
  10. Repeat Steps 2 to 9 for all outputs in this group.
