Integrating the Esser Fire System Through Online Discovery

Scenario

Desigo CC is connected to the Honeywell Esser fire system, and you want to run the online discovery. For background information, see the reference section.

Performing an offline configuration is recommended instead if you want to import data into Desigo CC through an EDE file that addresses complex and non-standard configuration requirements.

Prerequisites

  • The Honeywell Esser extension module is installed and included in the active project.
    The following extension is also selected automatically:
    • DMS Common
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
