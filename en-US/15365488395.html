Limiting Motion Detection Events

By default, motion detection events are enabled for all cameras, in both Desigo CC and the video management system (VMS). If this results in excessive motion alarm traffic, you can partially or entirely exclude motion detection events as follows.

 

Disable Motion Events for Specific Cameras in the VMS

In the Milestone / Siveillance VMS, you can enable or disable motion detection for specific cameras in the Motion tab of the Channel Device. See Configure Motion Detection.

 

Disable all Motion Events in the Desigo CC Video Alarm Table

To exclude motion detection events from all cameras, you can disable the corresponding event in the video alarm table. To do this:

  1. Customize the Video library structure and the Alarm Table Generic Events to the allowed customization level.
  2. Modify the customized alarm table as follows: for the Event Type = Motion Detected, set Skip Alarm Generation to True.
  3. Customize the Management object model to the allowed customization level.
  4. Apply the modified alarm table to the customized Management object model.

For detailed instructions see Customizing Video Alarm Tables.
