Configure the Events Filter of the Journaling Definition

  1. In the Journaling Configurator tab, open the Filters expander.
  2. In the Events expander, you can set the types of alarms that will be printed:
  3. Click New to create a new row with all the fields set to All. This means all types of events originating from all system objects will be printed.
  4. To restrict the objects whose events are printed, drag a Scope from Project > System Settings > Scopes into the Scope field.
  5. To restrict what events are printed based on Category, Type/Subtype, Discipline/Subdiscipline use the drop-down lists and the = or ≠ operators.
    NOTE: An event will be printed if it matches all the criteria in a row.
  6. Repeat the above steps to add additional rows.
