Helvar Lighting Symbols

When integrating the Helvar Lighting extension, the following symbols are available in graphics and associated to the Desigo CC functions:

DYN_2D_Helvar_Channel_None_Horizontal_001

  • Default Symbol for Helvar Lighting Channel

DYN_2D_Helvar_Channel_None_Horizontal_002

  • Secondary Symbol for Helvar Lighting Channel (with control)

DYN_2D_Helvar_ControlLight_Command_None_None_Horizontal_001

  • Symbol for Helvar Lighting Control Light Command

DYN_2D_Helvar_Device_None_None_Horizontal_001

  • Symbol for Helvar Lighting Device

DYN_2D_Helvar_Scene_Command_None_Central_001

  • Symbol for Helvar Lighting Scene Command

DYN_2D_Helvar_Sensor_None_Central_001

  • Symbol for Helvar Lighting Sensor

DYN_2D_Helvar_Sensor_None_Central_002

  • Symbol for Helvar Lighting Sensor with Occupancy Animation (green colored when occupied)

 

The blinking effect on symbols is not supported in this integration.
