Enabling Control Station in Danger Management

You can enable a Desigo CC station or an identified Flex Client to control Fire panels.

 

By default, the Control Station check box is cleared. This means that a Desigo CC station or an identified Flex Client cannot manage fire commands, and the operator can view (but not issue) fire commands. When this setting is not set, to properly configure a Desigo CC station or an identified Flex Client, remember to disable the Command Groups within the relevant scope (for example, Fire).
If the Control Station check box is selected, a Desigo CC station or an identified Flex Client can manage fire commands, and the operator can issue fire commands. To properly configure the control station, you must also create a fire scope, and enable the Command Groups.

 

  1. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select one of the following:
    • Project > Management System > Servers > Main Server
    • Project > Management System > Clients > [client station] or [web server] or [identified Flex Client]
    • Project > Management System > FEPs > [FEP station]
  3. The System Management tab displays.
  4. Select the Control station check box.
  5. Click Save .
  1. The selected Desigo CC station or identified Flex Client can manage Fire commands, and operators can issue commands to the Fire systems.
