Deleting Reactions and Folders

  1. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Logics > Reactions > [reaction or folder].
  3. The Reaction Editor tab displays.
    NOTE: Deleting a reactions folder also deletes all the reactions inside it.
  4. Click Delete .
  5. A confirmation message displays.
  6. Click Yes.
  1. The reaction or folder is removed from System Browser.
