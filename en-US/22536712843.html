Creating a BACnet Schedule

Scenario

You need to create a weekly occupancy schedule in a field panel that controls when the lights turn on and off for a school building.
You also want to create a holiday calendar that is used as an exception to the schedule, so that the lights will stay off during the holiday break, when the building will be vacant.
For background information, see BACnet Schedules.

Prerequisites

  • System Browser is in Application View.
  • You have configured Schedule application rights.
