Configure BACnet Network for NK823x

  1. In System Browser select Project > Field Networks.
  2. In the Object Configurator tab, click New , and select New BACnet Network.
  3. In the New Object dialog box, enter a name and description.
  4. Click OK.
  5. The newly created network is automatically selected in System Browser.
  6. Click the BACnet tab and open the Network Settings expander.
  7. In the Monitored by driver drop-down list, select the driver you configured above.
  8. Click Save .

NOTICE

Do not use the Block Commands Behavior, to avoid conflicts with the fire system settings.
The block command options should be disabled.

In the BACnet tab, the Auto Discovery Progress, Block Commands Behavior, and Global Object Folder Mapping expanders are not used.
You can ignore the Fire System Options tab.
