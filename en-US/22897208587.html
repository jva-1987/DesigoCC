Create a New Location

Method 1: Create a new location by providing Its name only

  1. You know only the exact name of the location you want to create.
  1. Select Project > Field Networks > [SORIS network] > [Meteoblue adapter] > Configuration > New Location.
  2. Select the Operation tab.
  3. Next to the Name property, do the following:
    a. Enter the location name, and click Write.
    b. Click Get.
  4. A query is sent to the Meteoblue website. If the Meteoblue weather forecast service is connected, the command is executed and, for the provided location name, the values relevant to the best matches are retrieved for any other location properties (Latitude, Longitude, Altitude, Country, Region/District, and Timezone).
  5. Next to the Location Point property, click Create.
  1. The new location is created for the provided location name. The user interface refreshes and all the properties are reset to blank or to any default value.

Method 2: Create a new location by providing its name and coordinates

  1. You know the exact name, latitude, longitude, and optionally altitude of the location you want to create.
  1. Select Project > Field Networks > [SORIS network] > [Meteoblue adapter] > Configuration > New Location.
  2. Select the Operation tab.
  3. Next to the Name property, enter the location name and click Write.
  4. The Name property updates and displays the location name.
  5. Next to the Latitude property, enter the location latitude value and click Write.
  6. The Latitude property updates and displays the latitude.
  7. Next to the Longitude property, enter the location longitude value and click Write.
  8. The Longitude property updates and displays the longitude.
  9. (Optional) Next to the Altitude property, enter the location altitude value and click Write.
  10. The Altitude property updates and displays the altitude.
  11. Next to the Location Point property, click Create.
  1. The new location is created for the provided location name and coordinates. The user interface refreshes and all the properties are reset to blank or to any default value.

Method 3: Create a new location by providing the approximate location coordinates

  1. You provided the name only of the location but the resulting match does not correspond to the location you want to create. Consequently, you need to adjust the results to get the wanted location.
  1. In System Browser, select Project > Field Networks > [SORIS network] > [Meteoblue adapter] > Configuration > New Location.
  2. Select the Operation tab.
  3. Next to the Name property:
    a. Enter the location name and click Write: The Name property updates and displays the location name.
    b. Click Get: A query is sent to the Meteoblue website and the values relevant to the best matches are retrieved for any other location properties (Latitude, Longitude, Altitude, Country, Region/District, and Timezone).
  4. Next to the Latitude property, enter the wanted latitude value and click Write.
  5. The Latitude property updates and displays the new value.
  6. Next to the Longitude property, enter the longitude value and click Write.
  7. The Longitude property updates and displays the new value.
  8. Next to the Location Point property, click Refresh.
  1. A query is sent to the Meteoblue website, and for the provided location name, the values of the best matches based on the new latitude and longitude values are retrieved for any other location properties (Latitude, Longitude, Altitude, Country, Region/District, and Timezone).
