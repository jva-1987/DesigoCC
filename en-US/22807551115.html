Configuring DALI Report Settings

  1. In System Browser, select Application View.
  2. Select Application View > Reports.
  3. The Reports tab displays.
  4. In the Home tab, do the following:
    a. Click the Table group box and select GAMMA KNX table.
    NOTE: Alternatively, you can right-click the report definition area, and select Insert Table > GAMMA KNX table.
    b. Select the check box Show in Related Items.
  1. The DALI report definition object is available in System Browser, while its related item is available in the Related Items tab.
