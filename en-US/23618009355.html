[Concept 2] Simple Archive with Two Storages

Scenario
You want to create a separate Long Term Storage for a particular data type (for example, Value Log data), when a large number of data of this type is expected. In this case, it is recommended to set the time slice interval to one month.

  1. System Manager is in Engineering mode.
  2. System Browser is in Management View.
  3. Two storages are created in SMC Value Log (1) and Activity-, Event- and Incident Log (2).
  1. Select Project > Management System > Servers > Main Server > History Database > Archive Groups.
    a. Select Activity Log and continue with step 2 to 4.
    b. Select Event Log and continue with step 2 to 4.
    c. Select Incident Log and continue with step 2 to 4.
  2. In the Extended Operation tab, select the Archive property.
  3. In the Archive drop-down list, select storage (2).
  4. Click Set.
  5. Select Project > Management System > Servers > Main Server > History Database > Archive Groups > Value Log.
  6. In the Extended Operation tab, select the Archive property.
  7. In the Archive drop-down list, select storage (1).
  8. Click Set.

NOTE:
In the Main and Properties expander, the Archive group must be empty.
