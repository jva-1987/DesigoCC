Migrating CS11 AlgoRex to FS20

This section provides general reference information about the CS11-FS20 migration. For related procedures, see the step-by-step section.

Log Files

After migrating, you can analyze two log files.

Analysis Log

<drive>:\Users\<username>\AppData\Local\Temp\<SiB-X filename>_parsing_mm-dd-yyyy_nn.txt

This log provides a report of the object mapping between CS11 and FS20. The objects are presented in a structured list that you can handle as a CSV file. Refer to the heading line (just after the Data parsing phase title) to interpret the data fields.

For example:
