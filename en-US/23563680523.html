Adding Powermanager Modbus EM to Existing Installation

  1. Go to Desktop, right-click the Update icon and select Run as administrator.
  2. Welcome to the Desigo CC Install Wizard dialog box displays.
  3. Click Next.
  4. Setup Type Selection dialog box displays.
  5. Click Modify adjacent to Feature Selection.
  6. Feature Selection dialog box displays.
  7. Click Add EM.
  8. Browse for Folder dialog box displays.
  9. Select Powermanager Modbus Adaptor.
  10. Click OK.
  11. Feature Selection dialog box displays.
  12. Click OK.
  13. Setup Type Selection dialog box displays.
  14. Click Next.
  15. Upgrade Selection dialog box displays.
  16. Click Next.
  17. Post Installation Steps Selection dialog box displays.
  18. Click Next multiple times until Ready to Install the Program dialog box appear.
  19. Click Install.
