Link Dimensions with an Aggregator Node

  1. You have created views with aggregator nodes. For information on creating views with aggregator nodes, see Creating and Building a User-Defined View.
  2. Navigate to the view with the aggregator nodes and select the aggregator node to which you want to link the calibrator object.
  1. Select Applications > Calibrators > [calibrator object].
  2. Drag the dimension from the calibrator object to the respective node in the Views tab.
  1. The dimension is successfully linked to the aggregator node.
