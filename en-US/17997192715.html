Poll Groups

This section provides information on creating and configuring Poll Groups. For background information, see the reference section.

Create Poll Groups

  1. Select Project > Management System > Servers > Main Server > Poll Groups.
  2. The Poll Groups application displays.
  3. Click Add New Poll Group .
  4. An empty row with fields to add the poll group name and poll interval displays.
  5. Specify a name for the poll group in the Poll Group Name field and its poll interval in the Poll Interval (ms) field.
  6. Click Save .
  1. The poll group is added to Poll Groups in System Browser.

 

Delete Poll Groups

  1. Select Project > Management System > Servers > Main Server > Poll Groups.
  2. Perform either of the following steps to delete the poll group.
    • Select the poll group from the Poll Group tab and click Delete .
      NOTE: Multi-select can be used here.
    • Select the poll group from below the Poll Groups node and click Delete .
  3. Click Yes.
  1. The poll group is deleted.

NOTE:
Deleting poll groups from the Poll Groups application can cause inconsistencies if the poll groups are already assigned to the points.

 

Modify Poll Groups

You can only modify the poll interval for a poll group.

  1. Select Project > Management System > Servers > Main Server > Poll Groups.
  2. Perform either of the following steps to modify the poll interval for a poll group.
    • Select the poll group from the Poll Group tab in the Primary pane and thereafter specify a new value for the poll interval.
    • Select the poll group from the list of poll groups below the Poll Groups node in the System Browser. Specify the new poll interval either from the Poll Group Information expander or from the Extended Operation pane.
  3. Click Save .
  1. The poll interval for the selected poll group is modified.
