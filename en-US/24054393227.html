Log On for the Setup and Service Assistant V6

Scenario: You want to enter the Desigo Automation and control profile data for the Setup and Service Assistant to connect to the Desigo automation station. It can be defined across the network for each user group so that a subsequent log on is not required.

Note:
One password can be defined for each user group per Desigo automation station. The password must be entered when connecting to the Setup and Service Assistant if passwords differ on the network. In the event of multiple passwords, we recommend using the password that is used on most Desigo automation stations.

  1. Desigo automation stations all have the same user name (see NOTE).
  2. The Desigo room automation station need not be online.
  1. Select Project > Field Networks > [Network name].
  2. Click the Desigo Automation tab.
  3. In the Profile expander, select a user group.
  4. Enter a user name and password for the Desigo Automation user profile.
    NOTE: Use the StandardOperator on validated projects.
  5. Confirm the password.
  6. The password is only checked one time as of this point.
  7. (Optional) Click Copy to Groups to apply the entry to all user groups.
  8. Click Set.
  9. The Desigo CC password is not checked against the Desigo Automation user profile on the Desigo room automation station.
  10. Click Save .
  11. Repeat the Steps 3 through 6 for additional user groups.
