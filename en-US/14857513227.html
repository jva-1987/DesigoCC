Integrating a Haystack Adapter

Scenario

You want to configure Desigo CC to operate on the Haystack system. For background information, see the related reference session.

Prerequisites

  • The Haystack extension is installed and included in the active project. The following extensions are also selected automatically:
    • Scripts
    • SORIS Driver
  • The Haystack adapteris running and connected to the Haystack system. For instructions, see Installing and Configuring the Haystack Adapter.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
