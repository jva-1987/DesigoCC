Populating the Geographical View with KNX Configuration Data

Scenario

In Desigo CC, you want to create a geographical view of KNX configuration data. To do this, you must create a user-defined view, map the hierarchy, and discover the KNX configuration data that will build up the geographical structure. For background information, see the reference section.

Prerequisites

  • System Manager is in Engineering mode.
  • System Browser is in Management View.
  • The JSON configuration file (a file with extension .json) was generated out of a KNX project file (a file with extension .knxproj).
