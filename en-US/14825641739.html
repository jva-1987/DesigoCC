Multiscan Commands

The following describes the commands of the Multiscan Galileo objects.

To have the event commands work in Event List, you must configure the related command macros. For instructions, see Configuring Multiscan Galileo Event Commands Macro.

Commands on Detector Objects

Property

Commands

Status

  • Ack
  • Reset
  • Disable
  • Enable

NOTE: Disable and Enable commands for detectors are only supported starting from Multiscan++ v1.9.7.

 

Commands on Panel Objects

Property

Commands

Buzzer

  • Silence

 

Commands on Relay Objects

No commands are available for relay objects.
