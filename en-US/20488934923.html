Configuring Video Export for March Networks Video Surveillance System

Scenario

You want to configure Desigo CC for working with the March Networks video surveillance system on Desigo CC stations. For background information, see the reference section.
This task is performed as part of the integration workflow, for details see Integrating March Networks Video Surveillance System.

NOTE: Video files are exported to the Export Video folder. Snapshots are saved to the following location: C:\Users\Administrator\Pictures\Snapshot\.

Prerequisites

  • The following extensions are installed and included in the active project:
    • Command VMS by March Networks
    • Video Management
  • The VMS server is installed.
  • The provider (that connects to the Command Enterprise Software (CES)) was installed. See Installing the Provider for March Networks Video Surveillance System.
  • The Siemens Video API Service is available among the Windows services.
