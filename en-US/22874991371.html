Prepare for Siport Configuration

This procedure is part of the workflow for integrating a SiPort Modbus system.

 

Prepare for the SIPORT Configuration

Before you start to configure SIPORT on Desigo CC, complete the following steps:

  1. Create a CSV file that contains all needed SIPORT information to import. For the file specifications see SIPORT Configuration File.
  2. Configure the SIPORT system to support the Modbus interface. See SIPORT System Settings for more information.
