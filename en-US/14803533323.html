Riello UPS Configuration (SNMP) File

To integrate the Riello UPS device into Desigo CC, before you can import the configuration data, you must manually copy the SNMP configuration template file:

Win32_UninterruptiblePowerSupply.Riello.1_0.SNMP

available at the following path:
[Installation Drive]:\[Installation Folder]\GMSMainProject\profiles\SNMPDataTemplate\

to the following directory:
[Installation Drive]:\[Installation Folder]\[Project Name]\devices.

You can then use this file to import the configuration.
