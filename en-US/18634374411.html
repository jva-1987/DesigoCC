Setting CS6 Guarto Configuration Through Offline Engineering

When the adapter is not yet connected to the CS6 Guarto intrusion panel, you can adjust CS6 configuration in offline mode by editing and loading configuration files, and adjusting the settings. For background information, see CS6 Guarto Integration.

 

  1. After installation, the application Siemens.GMS.CS6.Configurator is available in AdditionalSW\CEI CMSDL CS6 Configurator locally or over the network.
  2. The CS6 adapter is running and connected to the CS6 intrusion panels. For instructions, see Installing and Starting the CS6 Guarto Adapter.
  3. The NK82xx configuration INI file is available.
  4. The configuration (CSV) file is available, and contains CS6 configuration data.
  1. Set the protocol type:
    a. In the file system, locate AdditionalSW\AdditionalSW\CEI CMSDL CS6 Adapter\Files.
    b. Open the ConfigurationSettings.xml file with a text editor (for example, Notepad), and specify <ProtocolType> in one of the following ways:
    - 1 (CMSDL)
    - 0 (CEI-ABI)
    c. Save the changes.
  2. Start the configurator tool.
    • Double-click the application Siemens.GMS.CS6.Configurator.
  3. The Siemens.GMS.CS6.Configurator tool page is displayed.
  4. Load NK82xx configuration.
    • For instructions, see Loading NK82xx Configuration.
  5. Load CS6 configuration.
    • For instructions, see Loading CS6 Configuration
  6. (Optional) If any adjustments are required, do one or more of the following:
    • Adjusting CS6 Configuration
    • Adjusting the Object Function Mapping
  7. Save the configuration. Select System on the right, and in the Configuration area, do one of the following:
    • Export the configuration to a CSV file:
      a. In the To File section, click Export.
      b. Click Yes.
      c. In the Browse for Folder dialog box, select AdditionalSW\CEI CMSDL CS6 Adapter\Files as location where you want to save the files CEIProtocolConfiguration.xml, CMSDLProtocolConfiguration.xml, and FieldConfiguration.csv, and click OK.
      NOTE: If you already saved the CSV configuration file at this location, the existing CSV configuration file is overwritten with the new configuration.
    • Override online data:
      a. Connect to the adapter. For instructions, see Connecting the Configurator Tool to the Adapter.
      b. In the To Adapter section, click Download.
