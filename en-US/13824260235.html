Grouping Elements

An element can be grouped with one or more elements on a graphic using the group options in the Selection group.

Select the topic related to your task:

Group Elements

  1. You are in Design or Test mode, you have drawn two or more elements on a canvas, and you want to place them in a group.
  1. From the keyboard, press CTRL.
  2. Select the elements you want to place in a group.
  3. From the Home tab, in the Selection group, click Group .
  1. The elements are placed in a group.

 

Join a Group of Elements

  1. You are in Design or Test mode, you have an existing group of elements, and you would like to add an element to the group.
  1. From the canvas, select the element that you want to add to the group.
  2. From the keyboard, press CTRL.
  3. Select the group that you want the element to join.
  4. From the Home tab, in the Selection group, click Join Group .
  1. The element joins the group.

 

Remove an Element from a Group

  1. You are in Design or Test mode, you have an existing group of elements, and you would like to remove an element from the group.
  1. From the Element view, navigate to the group that contains the element you would like to remove.
  2. Select the element.
  3. From the Home tab, in the Selection group, click Quit Group .
  1. The element is removed from the group.

 

Remove a Group

  1. You are in Design or Test mode, you have an existing group of elements, and you would like to remove the group from the graphic.
  1. From the canvas, select the group you want to remove.
  2. From the Home tab, Edit group, click Delete .
  1. The group, and all the elements contained in the group, is removed.

 

Reposition the Inner Group Elements

  1. You are in Test or Design mode, you have a group of elements, and you would like to reposition an inner element.
  1. From the canvas, select the group of elements containing the inner element you want to reposition.
  2. From the Element Tree view, navigate to the highlighted group.
  3. Expand the group.
  4. Select the element you want to reposition.
  5. The element on the canvas is selected.
  6. From the canvas, move the inner element to its new position.
  7. Click outside of the group of elements.
  1. The inner element is repositioned while remaining part of the group of elements.

 

Resize an Element Group

  1. You are in Test or Design mode, and you would like to resize a group of elements.
  1. From the canvas, select the group of elements you want to resize.
  2. The selection handles display around the group.
  3. Hover over one of the selection handles until the cursor changes to
  4. Click and drag a selection handle on the group of elements until you have the desired size.
  5. Release the selection handle.
  1. The group of elements is resized.

 

Resize a Single Element in a Group

  1. You are in Test or Design mode, you have a group of elements, and you want to resize one of the grouped elements.
  1. From the canvas, select the group of elements containing the element you want to resize.
  2. From the Element Tree view, navigate to the highlighted Group.
  3. Click to expand the group.
  4. The elements associated with the group display under the Group folder.
  5. From the list of elements, select the element to resize.
  6. The element on the canvas is selected.
  7. From the canvas, hover over a selection handle on the element.
  8. When the resize cursor displays, click and drag the selection handle on the element and move the point to resize the element.
  9. Click outside of the group of elements.
  1. The element is resized independently from the other elements in the same group.

 

Select an Element in a Group

  1. You are in Design or Test mode, you have a graphic open, you have the Element Tree open, and you would like to select an element in a group to modify it without ungrouping the elements.
  1. From the canvas, select the group that contains the element you want to modify.
  2. The group is highlighted in the Element Tree.
  3. Expand the group.
  4. Select the element you want to modify.
  1. You can now modify the element independent of the group.

 

Ungroup Elements

  1. You are in Design or Test mode, you have two or more elements grouped on a canvas, and you want to remove them from a group.
  1. From the canvas, select the group of elements you want to ungroup.
  2. From the Home tab, in the Selection group, click Ungroup .
  1. The elements are removed from the group.
