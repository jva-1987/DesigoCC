Filtering the Dataset

  1. Dataset is defined in Management View.
  1. In the First Level Node Filter section, configure the first level node filter by selecting either of the following filter options:
    • All nodes under parent node: Drag-and-drop the parent node. All the nodes under this parent node is included in the dataset.
    • By matching token till: Add the token in the field and all the nodes matching to the specified token are included in the dataset.
    • By specific node: Select the nodes from the dataset present in Management View. Drag-and-drop the selected nodes in this field.
  2. All the selected nodes are excluded.
  3. (Optional) In the Exclude Filter section, drag-and-drop the nodes to be excluded from the dataset that is derived after applying the filter options All nodes under parent node or By matching token till.
    NOTE: It is not possible to use this Exclude Filter option with By specific node field.
  1. The dataset is filtered and you can now proceed to creating object models and device instances.
