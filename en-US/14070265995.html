Upgrading Projects with the Node Map

Symbols in Node Maps after Upgrading

After a project upgrade, the Node Map updates the symbols used to represent the nodes to the latest library settings. Note that this is not done in the Desigo CC graphics, which will keep the previous settings and customizations. To resolve possible inconsistencies, you can update the symbols in the graphics (refer to the Graphic Editor) or modify the Node Map symbols as described in Customizing the Node Map.
