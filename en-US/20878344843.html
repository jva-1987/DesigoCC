Monitoring the Battery of a UPS

Scenario

You can configure Desigo CC to monitor the battery status of a connected uninterruptible supply (UPS) using the WMI (Windows Management Instrumentation) service. For background information, see the reference section.

Prerequisites

  • Desigo CC is connected to an uninterruptible power supply (UPS) that supports the WMI Windows Management Instrumentation (WMI) protocol.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
