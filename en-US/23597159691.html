Determine the Language Packs and Supported Languages

Verify the Languages Folder

The Languages folder is located in …\InstallFiles and contains the languages supported by WinCC OA 3.19, as a language pack folder. Each language pack contains the EM and GMS subfolders that, in turn, contain the relevant folders structure.

During installation, use the Language Packs dialog box to select the required language packs to be installed on the selected setup type.

The Installer then copies the selected language packs from the installation media to a pre-defined location on the target machine. You can configure these languages in a project so that the Desigo CC Server and all clients types can adapt its user interface according to the selected language pack.

NOTE: For UL and ULC applications, verify that the language selected in Desigo CC is the same as the language selected on the Fire Alarm panels connected to it.

Desigo CC software is delivered in English and can be extended with three additional languages. Following software language packages are available.
