Troubleshooting AIC Pharma Reports

The following table is designed to help you resolve issues while running reports.

Problem

Cause

Resolution

You receive an exception error that mentions text groups and colors while running a Trend chart report.

There are more trend series in the report than entries in the associated text group.

Customize the AIC List Series text group, and then add new rows with colors to match the number of trend series in the report.

  • To customize the text group, see Customizing the AIC List Series Text Group.
  • To add rows to the text group, in Text Group Editor, see Adding a Row to a Text Group.
