Integrating Aguilera AE/SA Cx

Scenario

You want to configure Desigo CC for connecting to the Aguilera AE/SA Cx fire panel. For background information, see the reference section.

Prerequisites

  • The Aguilera AESA Cx extension is installed and included in the active project.
  • You prepared the configuration (CSV) file for the Aguilera fire panel that you want to integrate. See Aguilera AE/SA Cx Configuration (CSV) File.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
