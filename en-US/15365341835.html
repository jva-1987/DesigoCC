Connecting the Video System

When connected, Desigo CC can display video images and provide video functionality, and System Browser is aligned with Video Configurator. Manual connection to video is typically required for new projects, for restored projects if not aligned, or if the video system was manually disconnected for maintenance.

The installed video devices must be configured and running properly in the VMS (for Milestone or Siveillance, see Configuring Video Devices in the VMS) before connecting or aligning it to Desigo CC.

  1. A Video Disconnected or Video Disconnected Not Aligned event is active
  2. You have a correct configuration in the Video Configurator tab. If not, a refresh command may be needed.
  1. In System Browser, select Application View.
  2. Select the Video folder.
  3. Select the Extended Operation tab.
  4. Next to the Video System property, click Connect.
  1. System Manager is aligned to Video Configurator, if necessary, then the value of the Video System property changes to Connected.
    If the command fails, see Video Troubleshooting.

NOTE: The first time you Connect the video system, a refresh of the video configuration is automatically performed, and all the video objects are acquired into System Browser and sorted into their groups. On subsequent Connect commands, a manual Refresh and/or Sort command may be needed to update the configuration in System Browser.
