Configure the BACnet Server Connection

  1. You have read the BACnet Server Network Considerations topic to understand your configuration options.
  1. Select Project > Management System > Servers > Main Server > BACnet Server.
  2. Click the BACnet Server tab.
  3. In the BACnet Server Connection Configuration section, click Add, and then select IPv4.
    NOTE: If you want the BACnet server to connect to additional physical networks, you can add more IPv4 connections (but you do not need additional virtual networks).
  4. From the Adapter list, select the adapter you want to use for this configuration.
  5. Enter a UDP Port number based on your network configuration.
  6. From the IP Address list, select the address you want to use for the physical network.
  7. In the BACnet Server Settings section, click Add, and then select Virtual.
    NOTE: The network number must be unique on your BACnet system. The port ID must be unique in this list.
  8. Click Save .
  9. Click Yes to restart the BACnet Server.
  10. The BACnet Server restarts, and the status in the Extended Operation tab changes to Running.
