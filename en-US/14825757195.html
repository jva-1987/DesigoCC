Integrating Notifier ID3000

Scenario

You want to configure Desigo CC for connecting to the NOTIFIER® ID3000 fire panel. For background information, see the reference section.

Prerequisites

  • The Notifier ID3000 extension is installed and included in the active project.
  • You prepared the configuration (CSV) file for the NOTIFIER® ID3000 fire panel you want to integrate. See Notifier ID3000 Configuration (CSV) File.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
