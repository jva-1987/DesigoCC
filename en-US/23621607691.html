Post Installation Error Codes and Troubleshooting

The following table describes the error codes you may encounter when working with post installation steps.

For error codes, see the respective error logs present at the following location:

SMC: [Installation drive:]\[Installation folder]\GMSMainProject\Log

Error_Code

Description

Corrective Measures

ERROR_SMC_SystemConfigurationFailed

Failure in system configuration steps

General failure while performing system configuration.
For details, see smc.log

ERROR_SMC_InvalidConfigurationFile

ERROR_Web_Service_Interface_InvalidConfigurationFile

ERROR_Flex_Client_InvalidConfigurationFile

ERROR_D3_Visualization_InvalidConfigurationFile

ERROR_Advanced_Reporting_InvalidConfigurationFile

Failure in finding the AutomaticProjectConfiguration.json file

Add the AutomaticProjectConfiguration.json file manually at following location:
...GMSProjects\GMSMainProject\PostInstallation\EM\SMC\AutoProjectStartup\AutomaticProjectConfiguration.json and retry the post installation.
For information, see Verify and Enable a Post-Installation Step for Execution > Post-Installation Step of Project Setup (SMC) > Considerations for Post-Installation Step Project Setup.

ERROR_SMC_WebsiteCreationSkipped

ERROR_Web_Service_Interface_WebsiteCreationSkipped

ERROR_Flex_Client_WebsiteCreationSkipped

ERROR_D3_Visualization_WebsiteCreationSkipped

ERROR_Advanced_Reporting_WebsiteCreationSkipped

Skipped the website creation steps

Create the website manually.
For information on creating a website, see 2 – Create a Website in Setting up the Windows App Client.

ERROR_SMC_HDBLinkingFailed

Failure in linking the HDB with the project

Link HDB with the project manually
For information on linking HDB to the project, see 5 – Link an HDB to a Server Project in Restoring a Project or Creating a Project from a Template

ERROR_SMC_ProjectActivationFailed

Failure in project activation

Activate the project manually
For information on activating the project, see 6 – Activate the Project in Restoring a Project or Creating a Project from a Template

ERROR_SMC_ProjectStartFailed

Failure in starting the project

Start the project manually
For information on starting the project, see 7 – Start the Project in Restoring a Project or Creating a Project from a Template

ERROR_SMC_SystemAccValidationFailed

Failure in validation for PMON user. Project will not be configured (restored or upgraded)

Invalid password provided. Retry with a valid password manually

ERROR_SMC_ProjectUpgradeFailed

Failure in project upgrade

Upgrade the project manually
For information on upgrading the project, see 2 – (Optional) Upgrade a Server Project in Restoring a Project or Creating a Project from a Template

ERROR_SMC_ProjectRestoreFailed

Failure in project restore

Restore the project manually
For information on restoring the project, see 1 – Restore a Project or Create a Project from a Template in Restoring a Project or Creating a Project from a Template

ERROR_SMC_ProjectSetupFailed

Failure in project setup

Project is not available at the specified path for restore.
Ensure that the project is present at the specified path and retry.
For information, see Verify and Enable a Post-Installation Step for Execution > Post-Installation Step of Project Setup (SMC) >Considerations for Post-Installation Step : Project Setup in Obtain and Verify the Software Distribution

ERROR_SMC_HDBStepsSkipped

Skipped HDB steps

As the SQL Server (GMS_HDB_EXPRESS) is not running, HDB steps are skipped.
Ensure that the SQL Server (GMS_HDB_EXPRESS) is running and retry.

ERROR_SMC_HDBConfigurationFailed

Failure in HDB configuration

Configure the HDB manually. For information, see Creating History Infrastructure

ERROR_SMC_PostInstallationSetupFailed

Error_MNS_MNSPostInstallationSetupFailed

Failure in SMC post installation setup

Verify and enable the post installation step. For information, see Verify and Enable a Post-Installation Step for Execution in Obtain and Verify the Software Distribution

ERROR_SMC_WebsiteCreationFailed

Failure in the creation of the website

Create the website manually
For information on creating a website, see 2 – Create a Website in Setting up the Windows App Client

ERROR_SMC_WebUserValidationFailed

Failure in website creation due to failure in validation of user password

Invalid password provided. Retry with a valid password

ERROR_SMC_WebUserValidationFailed

Failure in website creation as the user does not exist in the domain

Use a valid user for website and web applications.

ERROR_SMC_WebsiteSetupFailed

Failure in the setup of the website

General failure in website setup.
For details, see smc.log

ERROR_GMS_LicenseActivationFailed

Failure in license activation

Ensure that all the considerations for the License Activation Post-Installation Step are met and retry post installation. For information, see Verify and Enable a Post-Installation Step for Execution > Post-Installation Step of License Activation (GMS) in Obtain and Verify the Software Distribution

ERROR_GMS_LicenseNotFound

License not found at the specific path

Ensure that the license file (.lic) file or ActivationID.txt/EntitlementID.txt file containing the IDs is available at the configured license path: [Installation Drive:\GMS\Licenses] and retry post installation.
For information, see Verify and Enable a Post-Installation Step for Execution > Post-Installation Step of License Activation (GMS) in Obtain and Verify the Software Distribution

ERROR_GMS_LicenseIsInvalid

License is invalid

Ensure that the correct license file (.lic) file or ActivationID.txt/EntitlementID.txt file containing the IDs is available at the configured license path: [Installation Drive:\GMS\Licenses] and retry post installation.
For information, see Verify and Enable a Post-Installation Step for Execution > Post-Installation Step of License Activation (GMS) in Obtain and Verify the Software Distribution

ERROR_SMC_WebApplicationCreationFailed

ERROR_Web_Service_Interface_WebApplicationCreationFailed

ERROR_D3_Visualization_WebApplicationCreationFailed

ERROR_Flex_Client_WebApplicationCreationFailed

ERROR_Advanced_Reporting_WebApplicationCreationFailed

Failure in creation of the web application

Create the web application manually.
For information on creating a web application, see 3 – Create a Web Application in Setting up the Windows App Client

ERROR_SMC_WebSiteUpgradeFailed

ERROR_Web_Service_Interface_WebsiteUpgradeFailed

ERROR_D3_Visualization_WebsiteUpgradeFailed

ERROR_Flex_Client_WebSiteUpgradeFailed

ERROR_Advanced_Reporting_WebsiteUpgradeFailed

Failure in the upgrade of a website

Upgrade the web site manually
For information on upgrading the website, see Upgrade a Website in Website Procedures

ERROR_SMC_WebApplicationUpgradeFailed

ERROR_Web_Service_Interface_WebApplicationUpgradeFailed

ERROR_D3_Visualization_WebApplicationUpgradeFailed

ERROR_Flex_Client _WebApplicationUpgradeFailed

ERROR_Advanced_Reporting_WebApplicationUpgradeFailed

Failure in the upgrade of a web application

Upgrade the web application manually
For information on upgrading a web application, see Upgrade a Web Application in Web Application Procedures

ERROR_SMC_WebApplicationSetupFailed

ERROR_Web_Service_Interface_WebApplicationSetupFailed

ERROR_D3_Visualization_WebApplicationSetupFailed

ERROR_Flex_Client_WebApplicationSetupFailed

ERROR_Advanced_Reporting_WebApplicationSetupFailed

Failure in the setup of the web application

General failure in web application setup.
For details, see smc.log

ERROR_Advanced_Reporting_ApacheTomcatInstallationFailed

Failure in installation of advanced reporting prerequisites

Perform the installation of the advanced reporting prerequisites manually.
For more information, see Prerequisites in Installing and Configuring Advanced Reporting

ERROR_Advanced_Reporting_TemplateSynchronizationFailed

Failure in template synchronization

Deploy the Advanced Reporting templates from the library to Tomcat server manually.
For more information see, 4 – Synchronize Advanced Reporting Templates in Installing and Configuring Advanced Reporting

ERROR_MNS_NDBLinkingFailed

Failure in linking the NDB with the project

Link the NDB to the project manually using SMC.
For information on linking the NDB with the project, see 3 – Link an NDB to the Project in Restoring and Upgrading the Notification Database (NDB)

ERROR_MNS_NDBUpgradeFailed

Failure in NDB upgrade

Ensure that the SQL Server (GMS_HDB_EXPRESS) is running.
Ensure that you have sufficient rights to create the database on the SQL server and retry post installation.
For details, see smc.log.

ERROR_MNS_NDBCreationFailed

Failure in NDB creation

Ensure that the SQL Server (GMS_HDB_EXPRESS) is running.
Ensure that the following paths, DataFilePath and LogFilePath are valid in AutomaticNdbConfiguration.json file.
Ensure that you have sufficient rights to create the database on the SQL server and retry post installation.
For details, see smc.log.

ERROR_Powermanager_DefaultWebAppsConfigurationFailed

Failure in creation of the default web application

Ensure that the Dashboard, Reports and WSI applications are available.
In SMC, select the current project and navigate to Powermanager tab.
Manually select the relevant web applications available in drop down list.
For more information, see Modify a Web Application on a Server in Web Application Procedures.
