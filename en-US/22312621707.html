Add the Network Information

  1. In the network information section, do the following:
    • Enter the Network name and Network description.
    • (Optional) To change the language selection, click the flag icon in the Network description field, select the desired language, enter the desired text, and then click OK.
  2. In the adapter information section, do the following:
    • Enter the Adapter name and Adapter description.
    • (Optional) To change the language selection, click the flag icon in the Network description field, select the desired language, enter the desired text, and then click OK In the Discovery/Import Mapping section, do the following:
    • Enter the Adapter URL, which must include the port number.
      Example: http://136.xxx.xxx.xxx:8080
    • (Optional) If you want to create the adapter later, select I’ll create the adapter later.
  3. Click Next page .
