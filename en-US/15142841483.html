Enlighted Energy Manager Integration

The Enlighted Energy Manager extension allows configuring lighting profiles and behavior, monitor energy consumption, and fixture status.

The adapter is not part of the Desigo CC installation. After installation, the adapter installer package (MSI) is available in [Installation Drive]:\[Installation Folder]\GMSMainProject\AddSW\Enlighted_EnergyManager. Once you install and start this software service, the adapter is ready to be executed and used in Desigo CC. For instructions, see Installing and Starting the Enlighted Adapter.

To integrate the Enlighted adapter, a SORIS driver and a SORIS network (monitored by one SORIS driver) must be created and configured. For instructions, see Integrating Enlighted Energy Manager.

Once the adapter object is created, it starts automatically as a Windows service using the http or https protocol. You can also start the adapter manually:

  • Using the http protocol: double-click the executable Siemens.Gms.EnlightedAdapter.exe available in the file system with the installation.
  • Using the https protocol: enter –secure:[thumbprint] by command line.

For each adapter created, it is necessary to configure the URL IP address of the computer from which the network adapter is run. For instructions, see Integrating Enlighted Energy Manager.

Dependencies

The Enlighted Energy Manager extension is configured to depend on the following extension, which will be automatically added, if they are not already installed in the system, when the Enlighted Energy Manager installation feature is selected:

  • SORIS Driver (see SORIS)
