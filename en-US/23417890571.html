Create a New Software Account User

NOTE:
The software account user does not need a password and the account does not expire. An existing OPC user from 3.0 will not be deleted.

  1. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Users.
  3. Select the Users tab.
  4. Click New .
  5. In the New User dialog box:
    a. From the User type drop-down-list, select Software account.
    b. Select the Distribution option Global or Local.
    c. Enter a user name. The name must be unique.
    NOTE: It is recommended that when you create a new user, the user name should not have a blank space.
    d. Click OK.
  6. Click Save .
  1. The software account user is created but not configured yet.
