Integrating Fibrolaser III Using Modbus TCP

Scenario

You want to configure Desigo CC for connecting to the Fibrolaser III heat detection system using the Modbus TCP protocol. For background information, see the reference section.

Prerequisites

  • The Fibrolaser extension is installed and included in the active project.
  • You prepared the configuration (CSV) file for the heat detection system you want to integrate using the Modbus TCP protocol. See Fibrolaser III Modbus Configuration (CSV) File.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
