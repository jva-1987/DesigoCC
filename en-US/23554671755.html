Editing Migrated Graphics

  1. You have migrated SICLIMAT X graphics to Desigo CC.
  1. In System Browser, select Application View.
  2. Select Applications > Graphics.
  3. Double-click the migrated graphic to open it in the Graphics Editor.
  4. The migrated graphic contains different layers with elements, such as basic shapes, Desigo CC symbols, page navigation buttons.
  5. Switch the layers on/off to see the graphic elements and edit them.
