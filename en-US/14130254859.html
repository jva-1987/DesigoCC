OPC DA Server Connectivity

The Desigo CC OPC DA server allows enterprise systems and other management stations to access any integrated subsystem data by using third-party OPC clients.

Desigo CC supports OPC Data Access (OPC DA) specifications which consists in OPC Classic specifications that enable the exchange of data (including values, time, and quality information) between multi-vendor devices and control applications.

In OPC DA, data sources are represented within the server by OPC items. Third-party OPC clients can use an OPC DA server to obtain data from or send commands to the automation subsystems.

Reference Documents

The OPC Foundation publishes a number of other documents that are necessary and/or useful for understanding the underlying principles used in the OPC DA server for Desigo CC, including:

 

  • OPC Common 1.00 (1998-10-27)
  • OPC DA 2.05 Specification (2002-06-28)
  • OPC DA 3.00 Specification (2003-03-04)

The access to these documents may be regulated by privileges (account registration or being a member of the OPC Foundation). For more information, see https://www.opcfoundation.org.
