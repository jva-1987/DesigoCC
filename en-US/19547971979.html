Setting up Maintenance Reminders

You can set up maintenance reminders by setting up a specific operator task.

  1. System Browser is in Application View.
  1. In System Browser, select Applications > Operator Tasks.
  2. The Tasks tab displays.
  3. Click Add and, from the drop-down list, select Maintenance Aggregator.
  4. The new task is added to the list.
  5. Open the General Settings expander.
    For full details about these fields, see Operator Tasks Workspace > Tasks General Settings.
  6. Modify the default Task name and Description.
    The task name you enter must be unique.
  7. Next to Start date, specify when you want the task to begin after the Start command is clicked:
    • Immediately at task start: The task will start without any delay.
    • At: The task will start at the specified date and time.
    • In: The task will start after the specified delay has elapsed.
  8. Next to Due date, specify when you want the task to end:
    • At: The task will end at the specified date and time.
    • In: The task will end after the specified duration has elapsed.
  9. Open the Targets expander.
    For full details about these fields, see Operator Tasks Workspace > Tasks Targets.
  10. From System Browser, drag panels/devices of type Maintenance aggregator to the Target field.
  11. Click Save.
  12. Click Start to initiate the task.
    For full details about this action, see Handling Operator Tasks > Start an Operator Task.
  13. Once the maintenance is completed, the revert command is automatically sent when the due date is reached.
    (Note that the Revert option is read-only, and set to Automatic.
    For full details about this option, see Operator Tasks Workspace > Revert Settings and Command Result Icons.)
    As result, in the Operation tab, the Maintenance property is set to Required.
    Also, an event is generated in Event List when the tasks status is Ready to be closed.
    For reference, see Operator Tasks Workspace > Operator Tasks Reminders in Event List.
  14. Click Close to conclude the task.
    For full details about this action, see Handling Operator Tasks > Close an Operator Task.
  15. Do one of the following:
    • If a new service period is planned (some panels might have a fixed duration/maintenance schedule), the maintenance reminder must be repeated.
      Duplicate the task. See Setting Up Operator Tasks > Duplicate an Operator Task.
    • If a service period is not planned (some panels might require one-time maintenance) or postponed, to clear the event from Event List, do the following:
      a. In the Targets expander, select a panel/device.
      b. In the Operation tab, next to the Maintenance property, click Reset.
      c. Repeat the previous substeps for each panel/device listed in the Targets expander.
