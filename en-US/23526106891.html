Activate the Project

The very first project that you create, restore or create from template is activated automatically. Use the following procedure to activate all the subsequent projects that you create/restore.

  1. (Required only for projects on Client/FEP) You have provided Shared project path when you create a project on Client/FEP.
  1. Click Activate .
  2. A confirmation message displays.
  3. Click Yes.
  1. The selected project becomes active.
