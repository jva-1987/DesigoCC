Configure Report Output as a File

  1. Select Applications > Reports.
  2. Click the Settings tab.
  3. From the Report Output group box, click Dialog Launcher .
  4. The Report Output Definition dialog box displays.
  5. Select File as the destination type in the Destination types list.
  6. Click Configure Folders.
  7. The Report Output Folders Configuration dialog box displays.
  8. In the Folder Alias field, type a name for the Report Output folder.
  9. Click Browse to select a destination folder.
  10. The selected destination path displays in the Folder Path field.
  11. (Optional) In the Folder Description field, type the folder description.
  12. Click New.
  13. The output folder is added to the List of Folders for the Report Output section.
  14. Click Close.
  15. The configured output folders display in the File drop-down list of the Report Output Definition dialog box.
  16. Select the required report format (PDF, XLS, CSV, or XML) in the Report format list.
  17. Select File in the Destination types list.
  18. From the File drop-down list, select the destination folder where you want to save the file.
  19. The File drop-down list displays all the report output folders that you have configured.
  20. Select Enter custom file name to add the file name. The default option is Use report name as file name.
  21. Do one of the following:
    • Select the Append date/time to file name check box to add the date and time to the file name when saved.
      NOTE: The Create new/overwrite existing file and Append data options become unavailable when you select the
      Append date/time to file name check box.
    • Select the Append data option button to append data in the same folder but creating new document with incremental number.
    • Select Create new/overwrite existing file to create a new file or overwrite the existing file with the same file name.
  22. Click New.
  23. The selected format, destination, and file name are added to the Output Definition list.
  24. Click OK.
  1. Configured Report Output Definitions display in the Report Output group box.
