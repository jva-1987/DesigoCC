DALI Symbols

For the DALI building lighting systems, the following symbols are available in graphics with this integration.

DYN_2D_DALI_ECG_WhiteTunable_Central_001

  • Color temperature symbol for DALI ECG lamps.

DYN_2D_DALI_ECG_WhiteTunable_Central_002

  • Color temperature symbol internally used as part of the DALI ECG lamp graphic template.

DYN_2D_KNX_Light Simple_None_Central_005

  • Default symbol for simple light.
  • Symbol background changes color according to the Summary Status color propagated from the point itself, or the points at lower levels. Also, the symbol blinks depending on the status of the Switching property.

DYN_3D_DALI_Gateway_None_Central_001

  • Default symbol for the DALI gateway.
  • Symbol background changes color according to the Summary Status color propagated from the point itself, or the points at lower levels.

DYN_3D_DALI_ECG_None_Central_002

  • Default symbol for the ECG lamp with the default property.
  • Symbol background changes color according to the Summary Status color propagated from the point itself, or the points at lower levels. Also, the symbol blinks depending on the status of the Switching property.

DYN_3D_DALI_ECG_None_Central_003

  • Secondary symbol for the DALI ECG lamp with default property. It can also be used for the emergency light.
  • Symbol background changes color according to the Summary Status color propagated from the point itself, or the points at lower levels. Also, the symbol blinks depending on the status of the Switching property.

DYN_3D_DALI_Dualsensor_None_Central_001

  • Default symbol for the presence detector.
  • Symbol background changes color according to the Summary Status color propagated from the point itself, or the points at lower levels.

DYN_3D_DALI_PushButton_None_Central_001

  • Symbol background changes color according to the Summary Status color propagated from the point itself, or the points at lower levels.

DYN_3D_DALI_Group_None_Central_002
