Integrating SCAME S81-HS

Scenario

You want to configure Desigo CC for connecting to the SCAME S81-HS unit. For background information, see the reference section.

Prerequisites

  • The SCAME S81-HS extension is installed and included in the active project.
  • You prepared the configuration file for the device you want to integrate. See SCAME S81-HS Configuration (CSV) File.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Licensing

This extension is covered by license. Check that the required license is available; otherwise, you may not be able to configure and work with this extension.
