Server

The Server is the main component of Desigo CC . It is a computer where the core system services run and the project data and software, that monitor and and control the facility including runtime data, reside.

The Desigo CC Server installation always includes an Installed Client and a System Management Console (SMC).
Projects are managed using the SMC. The Desigo CC project contains all engineering and operation information created for a system. Field system information is described by Desigo CC objects inside the project.

On the Server is a root folder called GMSMainProject that contains software information. Individual project folders maintain information about respective projects.

Another root folder called GMSDatabases, contains information about history databases and the backup files.

It has interface connections to the field either directly or by using a Front-End Processor (FEP) and provides a centralized database and other services to the connected clients. The Server can support a number of clients.

For small jobs or sites, you can use just one computer that acts as project Server, database host, and client. For medium to large installations, you would use one computer for Server activities only and one or more computers for client activities.

The Server when hosts a web server (IIS) acts as a local web server (IIS) facilitating you to work locally with Windows App Clients.
