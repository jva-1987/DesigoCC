Create User Groups by Copying Settings from Existing Users

  1. You are in the User Management Wizard step 2 – User Group creation.
  1. Select Create one or more user groups by copying settings from existing ones.
  2. Click Add.
  3. The Copy user group dialog box opens.
  4. In Copy from, select the user group you want to copy the settings from.
  5. (Optional) Enter the Group name if you want to customize it.
  6. Select the Configured users you want to associate to the new user group.
  7. Select to add a single user or to add all the configured users to the new user group.
  8. (Optional) In the Group membership column, select the configured user you want remove, then select. Select to remove all the users.
  9. Click OK.
  10. The previously configured user group is added to the table of user groups that will be created.
  11. (Optional) Select Add and repeat all the steps for each additional user group you want create.
  12. (Optional) Select a user of the table and select Remove to delete it.
    NOTE: You cannot edit a user from the table. If you need to correct any parameter, you have to remove it and create it again.
