Object Configuration

Object Configurator enables you to define the characteristics and behavior of objects in Desigo CC. You can select an individual object in System Browser and then use the Object Configurator to modify only that particular object. For background information, see the reference section.

NOTE: It is also possible to an object model in the libraries and then use the Models & Functions tab to modify it, so that the changes apply to all the objects that are based on that object model.

Any changes may require alarms to be rechecked.

 

Prerequisites

  • System Manager is in Engineering mode.
  • System Browser is in Management View.
