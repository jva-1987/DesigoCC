Reconfiguring FS20

This section provides general reference information about the FS20 Reconfiguration. For related procedures, see the step-by-step section.

Log Files

After migrating, you can analyze two log files.

Analysis Log

<drive>:\Users\<username>\AppData\Local\Temp\FS20_migration_parsing_mm-dd-yyyy_nn.txt

This log provides a report of the object mapping between the old FS20 and the new one. The objects are presented in a structured list that you can handle as a CSV file. Refer to the heading line (just after the Data parsing phase title) to interpret the data fields.

For example:

Migration Analysis File Log

 

Date: 10/19/2019 15:56:52

Operation completed in : 0 min 24 sec

------------------
