Import Instances Expander

The Import Instances expander allows you to import the System Browser data represented as device instances in the JSON file to the Application View > Powermanager root folder or any specified folder within the Powermanager root folder.

  • Target location - Specify the location in the Powermanager node where you want to create the device instances. If no location is specified, the instances are created in the Powermanager root folder.
  • Select File - Browse and specify the JSON file with details of device instances to be added to the Powermanager folder or any of it's specified sub-folders.
  • Import - Imports the device instances in the JSON file to the Powermanager root folder or any of it's sub-folders.
