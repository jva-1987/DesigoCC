Accessing the Graphics Editor

You can switch between the Graphics Viewer and the Graphics Editor in Operating or Engineering mode.

To Access the Graphics Editor in Engineering Mode

  1. The Graphics Viewer displays.
  2. System Manager is in Engineering mode.
  1. Click Edit .
  2. The Graphics Editor displays in the Graphics tab.
  3. To return to the Graphics Viewer, do one of the following:
    • Click Edit .
    • Click Engineering.
  1. The Graphics Viewer displays.
  2. System Manager switches to Operating mode.

 

To Access the Graphics Editor in Operating Mode

  1. The Graphics Viewer displays.
  2. System Manager is in Operating mode.
  1. Click Edit .
  2. The Graphics Editor displays.
  3. To switch to Engineering mode in the Graphics Editor, click Operating.
  4. The Graphics Editor displays in the Graphics tab.
  5. To return to the Graphics Viewer from Engineering mode, click the Edit .
  1. The Graphics Viewer displays.

 

To Switch from Graphics Editor Engineering Mode to the Graphics Viewer Engineering Mode

You are in the Graphics Editor in Engineering mode.

  1. System Manager is in Engineering mode.
  • Click Edit .
  1. The Graphics Viewer displays.

 

To Switch Graphics Editor from Operating to Engineering Mode

  1. The Graphics Editor displays.
  2. System Manager is in Operating mode.
  • Click Operating.
  1. System Manager switches to Engineering mode.
  2. The Graphics Editor displays.
