Setting Up a KIOSK Station in Building Automation Profiles

  1. You have a running Building Automation installation.
    NOTE: The following procedure applies only to Building Automation EN and Building Automation NA Client Profiles.
  2. You want to configure a client station to become a KIOSK station.
  1. Using the separate SMC (System Management Console) application, create a project starting from Building Automation template project.
  2. Log on using an administration user account with configuration rights.
  3. Create the KIOSK graphic.
    You can configure a single graphic, possibly including multiple layers with dynamic cycling effects, or create multiple graphics and include navigational links that allow users a limited interaction with the system.
    Make sure that the only links users can select on the KIOSK graphics are other KIOSK graphics, and provide links back to a starting (home) page to prevent users from getting lost.
  4. Create a user-defined view (the KIOSK view) with all the graphics you want to display in the KIOSK.
  5. Associate the top node of the KIOSK view with the KIOSK home graphic. This automatically occurs if you drop the top node of the KIOSK view on the KIOSK home graphic.
  6. Create a new KIOSK user group with read-only permissions, and then a new user, and associate the Building Automation Client Profile to the user.
    It is recommended that you only allow read access to the KIOSK user group.
  7. On the KIOSK client, log off and then log on again with the new user credentials.
  8. Navigate to the top node of the KIOSK view.
  9. Click Favorite location in the Navigation bar to bookmark the KIOSK home as favorite location, so that it always displays as the initial location.
  10. Log off.
  11. Log on again using an administrator user account with configuration capability.
  12. Navigate to the KIOSK user and associate the KIOSK Client Profile to the KIOSK user.
  13. Save the KIOSK user and log off.
  14. Log on as KIOSK user and test the KIOSK client.
  15. Adjust the configuration as necessary.
    • If closed mode is required, go to the station configuration, and select the Closed mode check box. Make sure to associate the KIOSK Client Profile to the closed-mode user (GMSDefaultUser).
      NOTE: To exit from an installed client in KIOSK mode, unlock the closed mode and use Task Manager to stop the Application Framework.
      To exit from the Web Client in KIOSK mode, press ALT-F4 to close the Web Client KIOSK view.
