Event Treatment with the Common Event Object

The Common Event application supports intrinsic reporting of a data point, for example, to Fault, Alarm or Out of Service. The messages are displayed in a table with the applicable state. It does not matter whether the data point was imported to the system or must be read directly by the automation station. The Common Event object must however be in the application for Rooms or Plants.
