Integrating RDF302 Room Thermostat

 

Scenario

You want to configure Desigo CC to connect to the RDF302 room thermostat via Modbus. For background information, see Building Automation Modbus Building Automation Modbus section

Requirements

  • The Building Automation Modbus extension is installed and included in the active project.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
