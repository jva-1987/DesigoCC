Schindler Elevators

Scenario

You want to configure Desigo CC to connect to Schindler elevators. The workflow guides you through the required steps. Background information is available in the Reference section.

Prerequisites:

  • The required Schindler extension is installed.
  • An HDB is available for linking in the History Databases tab of the Databases expander.
  • The following applies to all procedures performed in Desigo CC:
  • System Manager is in Engineering mode.
  • In System Browser, Management View is selected.

To speed configuration, make sure the Desigo CC Transaction Mode is set to Simple or the Automatic Switch of Transaction Mode is set to True.
The automatic switch setting is recommended. Without it, to re-enable the logs, you need to set the Transaction Mode to Logging at the end of the configuration procedure.
