Disabling License Expiration Notification

You can disable the capability to generate an event in Event List when the Desigo CC license subscription is about to expire.

  1. In System Browser, select Management View.
  2. Select Project > Management System > Servers > Main Server > Licenses.
  3. Select the Extended Operation tab.
  4. Next to Subscription counter threshold property, enter 0 and click Change.
