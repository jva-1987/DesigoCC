Restoring the Easy Button tab

The Easy Buttons tab is not displayed if the user deletes the Windows folder icon in the Easy Buttons editor screen. Perform the following steps to restore the Easy Buttons tab.

  1. System Manager is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Graphics > Easy Buttons.
  3. Click Edit .
  4. The Easy Buttons editor tab displays.
  5. Select Applications > Notification.
  6. Drag the Notification node on the Easy Buttons expander.
  7. The Windows folder icon displays.
  8. Right-click the Windows folder icon and select Visible.
  9. Click Save .
  10. Restart Desigo CC .
