Configure BACnet Server Devices

  1. Select Project > Management System > Servers > Main Server > BACnet Server.
  2. In the BACnet Server tab, click Create new device .
  3. In the New object dialog box, enter a name (must be unique on the BACnet Network) and description.
  4. Click OK.
  5. The virtual device appears in System Browser but is not running.
  6. Click the BACnet Server tab.
  7. In the Device Settings expander, enter a device instance number that is unique on the BACnet network.
  8. (Optional) If you want to allow commanding and alarm acknowledgment from BACnet, clear the Device is read only check box.
  9. In System Browser, select Manual navigation, and then navigate to the location of the objects you want to add.
  10. Drag the desired objects to the Device Point Mapping section.
    NOTE: If the objects you drag are validated or life safety points, you must consent to displaying them by clicking Yes when the following dialog box appears.
  11. Click Save .
  12. Repeat Steps 2 – 10 for each device you add.
