Configure the Adapter Secure Key

By default, any requests from the Meteoblue adapter to the Meteoblue weather forecast service are sent with no encryption. To prevent an unauthorized use of the API key the adapter object created in Desigo CC can be configured with a secure key to secure the communication between the Meteoblue adapter and the Meteoblue weather forecast service.

  1. Select Project > Field Networks > [SORIS network] > [Meteoblue adapter] > Configuration > Adapter Settings.
  2. In the Operation tab, the Secure Status property indicates Disabled.
  3. Next to the Secure Key property, enter the secure key code in the field and click Change.
  1. The Secure Status property becomes Enabled.
