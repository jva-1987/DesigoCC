Assigning a Function to an Object

If a function cannot be assigned automatically to an object during import, you must assign it individually using Object Configurator. This is normally the case when you integrate a third-party product.

  1. In System Browser, select the object. For example,
    Project > Management System > Servers > Hardware > Main Server > [data point].
    NOTE:     The folder path may vary by subsystem.
  2. In the Object Configurator tab, select the Main expander.
  3. In the Function list, select the Function to assign (for example, for an analog input for temperature measurement = SensorAirTemperature). Check box All means:
    Cleared: Displays only the Function assigned to the Object Model in the Function Mapper.
    Selected: Displays all Functions in the system.
  4. The data point in the system behaves according to the selected Function (for example, you selected a supply air sensor by drag-and-drop in the graphic).
  5. Click Save .
  1. The Operation tab now displays only the properties according to the selected Function configuration.
