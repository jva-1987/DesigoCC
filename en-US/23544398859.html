Generated Report Cannot be Opened with a Link, Macro, or Reaction

Problem: Generated report cannot be opened with a Link, Macro, or Reaction

Cause 1: The Credentials of Link is not set or is invalid.

Solution 1: Navigate to the Link and set credentials in Application Viewer with an account that has full system access rights.

Cause 2: The file format property of the link (P2 Value) is not the same as set in Execute expander of link or command column of the macro or reaction.

Solution 2: Ensure that the file format is the same everywhere.
