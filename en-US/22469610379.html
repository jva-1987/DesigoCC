Create a Device Depth and Layer

  1. You have created a background graphic with viewport subsections called zones. You want to create a layer and a depth that display, for example, all the Fire Detector devices.
  1. Navigate to the Depths view, and click Add New Depth to add a new depth.
  2. A new depth named Depth 1 displays.
  3. Under the Depth Name column, click the Depth 1 text and type a name for the depth.
    • For this example, Device.
  4. In the Element Tree, click Add New Layer .
  5. The new layer displays in the Element Tree. In this example, Layer(2).
  6. Navigate to the Properties view (Layer Properties), expand the General section, and in the Description field, type a name for the layer.
    • For this example, type Device.
  7. In System Browser, select Management View.
  8. Drag the devices that you want to display on the Devices layer.
  9. Click Save .
