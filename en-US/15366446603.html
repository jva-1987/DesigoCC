Video Scope and Application Rights

The application and scope rights are accessed in Engineering mode as follows:

  1. In System Browser select Management View.
  2. Select Project > System Settings > Security.
  3. In the Security tab, under Groups select the user group for which you want to define rights.
  1. The Scope Rights and Application Rights expanders provide the Video-related settings detailed below

Video Application Rights

The video application rights control the availability of specific features as follows:

Application right

Makes available...

Show Video Display

the video view, which contains the video streams

Show Video Wall

the monitor group display, with icons representing the monitors

Show Video Configurator

the interface to the connected VMS.

Show Video PTZ Control

the PTZ control panel

Show Archive:

the video recording archive

Query Archive

the query functionality of the video recording archive

Camera Snapshot

the overlay icon for taking a snapshot directly from a monitor

Video Scope Rights

To display video streams, users must have Sta. Command Group rights on camera objects

The Snapshot command button in the Extended Operation tab is available only if the Adv. Command Groups flag is set.

NOTE: The Create and Delete options in the scope definition have no effect on the video configuration.
