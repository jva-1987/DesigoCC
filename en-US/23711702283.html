Creating the h,x diagram

  1. In the System Browser, select Logical View.
  2. Select the View tab.
  3. The View dialog pane opens.
  4. Select Logical > [Hierarchy] >[Ventilation plant].
  5. Click Create Element and select Element.
    NOTE: The button must be enabled to create an element.
  6. In the New Object dialog window, enter a name and description.
  7. Click OK.
  1. The new elements is added to the System Browser and the View dialog pane in the appropriate hierarchy.
