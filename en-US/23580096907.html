Select Incident or Message Template

  • Perform the following substeps as required:
    • To select an incident template, select Applications > Notification > Incident Templates and select the required incident template from the list of available templates. For example, Earthquake.
    • To select a message template, select Applications > Notification > Notification Templates > Message Templates. Select the required message template from the list of available templates. For example, Tornado.
  • The enabled command displays with three vertical black squares.
