(Optional) Monitor Server Disk Capacity

Only for the Desigo CC server station, you can additionally configure the system to generate an event when the disk capacity falls below a specified limit.

  1. Select Project > Management System > Servers > Main Server > [hard disk drive].
  2. Disk space values (Disk free space, Disk total space, and Disk used space) display in the Operation tab.
  3. Select the Object Configurator tab.
  4. Open the Properties expander, and select the Status.DiskfreeSpace property.
  5. Open the Alarm Configuration expander.
  6. Select the Valid check box.
  7. Select the Management Station radio button.
  8. From the Alarm kind drop-down list, select Continuous.
  9. Define the Alarm Class as follows:
    a. From the Alarm Class field, select the alarm category.
    b. In the Limiting Value fields, select the relational operator > and then enter the limit value (in MB). The recommended value is 1000 MB.
    c. In the Informational Text area, select Text and enter an instruction for the alarm. For example, Low disk. Contact admin to free up disk space on the station.
  10. Select the Alarm config. activated option.
  11. (Optional) To log entries in the system Activity log for any changes of the disk capacity, in the Properties expander, select the AL check box for the Status.DiskfreeSpace property.
    NOTE: Use this setting with caution, because it might result in a considerable amount of logged data.
  12. Click Save .
  1. When the free space on the disk falls below the values specified in Limiting Value, the configured alarm will occur in the Desigo CC stations.
