Example: Simple Evaluation

Scenario: You want to see the Current Value of a data point.

  1. You have a Text box element selected on the canvas.
  2. In the Evaluation Editor, from the Property drop-down menu, select: Text.
  3. From the Type drop-down menu, select: Simple
  1. From System Browser, drag the data point and drop it over the Expression field.
  2. From the Ribbon, on the Home tab, in the Modes group, click or .
  1. The current runtime value of the data point displays on the canvas.
