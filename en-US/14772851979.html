Adding a Subsystem to the IBase Report

Scenario

You installed and added IBase to a project. By default, the IBase script generates a report that covers only the management platform objects. You now want to expand the report so that it also covers the objects of an integrated subsystem. This requires creating an IBaseConfigurationScript.json file for the subsystem, that will be invoked whenever you run the main IBase script. For background information see IBase Configuration File.

Prerequisites

  • You configured the IBase report functionality in Desigo CC. See Setting Up IBase.
