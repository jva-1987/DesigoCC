Select Table View

Select Table View

  1. Click Stop .
  2. The automatic data update is stopped.
  3. Click table view
  4. The table opens in default view.
    Click the time stamp header to sort the rows by ascending or descending order.
  5. Click to show or hide interpolated values.
  6. Interpolated values are displayed in light-grey.
  7. Click table view again .
  8. The graphical Trend View re-displays.
  9. Click Run to start the automatic data update.
    NOTE: The priority displays in the table if a subsystem supports information on BACnet write priority (1-16).
