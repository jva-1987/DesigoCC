Integrating Notifier Onyx

Scenario

You want to configure Desigo CC for connecting to the NOTIFIER® Onyx fire panel. For background information, see the reference section.

Prerequisites

  • The Notifier Onyx extension is installed and included in the active project.
  • You prepared the configuration (CSV) file for the NOTIFIER® Onyx fire panel you want to integrate. See Notifier Onyx Configuration (CSV) File.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Licensing

This extension is covered by license. Check that the required license is available; otherwise, you may not be able to configure and work with this extension.
