Supported Report Template Controls

The InfoCenter Migration Tool supports the migration of InfoCenter Report Templates with various controls. The default.rptdesign base-template is then used to generate new custom templates in Desigo CC and display them in the output folder you specify with an .rptdesign file extension.

A report template may consist of different control types, however, after executing the report, only the following controls are supported. Templates that contain controls other than the ones listed below do not display and those template have to be created manually:

Charts

  • Bar
  • Column
  • Histogram
  • Scatter

Tables

  • Alarm Issues
  • Alarm Summary
  • MKT
  • Point Value
  • Statistics
  • System Activity
  • System Activity Summary

Other Controls

  • Date
  • Header and Footer
  • Picture
  • Report Data
  • Text
  • Time
