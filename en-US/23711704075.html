Assign data points

The h,x diagram is available in the View dialog pane.

  1. Select the Manual Navigation check box.
  2. The View remains selected until the check box is cleared.
  3. In System Browser, select Management View.
  4. Select the temperature sensor Project > Field Networks > [Network] > Hardware > [Device] > Outside air temperature:
    • Drag-and-drop creates a Softlink from the System Browser Outside air temperature to [h,x diagram] in the Views tab.
    • Create additional Softlinks for:
      a. Supply air temperature
      b. Room air temperature
      c.       (Optional) Extract air temperature
  5. Select the temperature sensor Project > Field Networks > [Network] > Hardware > [Device] > Outside air humidity:
    • Drag-and-drop creates a Softlink from the System Browser Outside air humidity to [h,x diagram] in the Views tab.
    • Create additional Softlinks for:
      a. Supply air temperature
      b. Room humidity
      c.       (Optional) Extract air humidity
  6. Select the temperature sensor Project > Field Networks > [Network] > Hardware > [Device] > Setpoint heating:
    • Drag-and-drop creates a Softlink from the System Browser Setpoint heating to [h,x diagram] in the Views tab.
    • Create additional Softlinks for:
      a. Setpoint cooling
      b. Setpoint humidification
      c. Setpoint dehumidification
      NOTE: The appropriate Replacement value is take if not all setpoint data points are available.
    • (Optional) A Softlink need only be created for the object SpField, if used instead of individual setpoints.
  7. Click Save .
  8. Assign a Function for h,x diagram.
