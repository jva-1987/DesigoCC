New Bulk User Import

Scenario
You want to import a bulk group of users using the CSV template to configure users. The users can be new, existing, or a combination of both and they must be a part of one of the following user security group types listed below:

  • Desigo CC
  • Software Account
  • OpenID

For general information on user administration, see User and User Group Administration.

For reference information on OpenID connections, see Identity Providers. See also the comprehensive example in Example: Auth0 Identity Provider Configuration.

Prerequisites

  • You have created one or more security groups under the folder Project > System Settings > Security (see User Group Administration).
  • The user importing the CSV file has Engineering rights for the Security application
  • For Local OpenID Users: The identity provider configuration for OpenID users is configured.
