Refreshing the Cameras Configuration

After making changes to the camera settings or triggers in the Video Management System (VMS), a manual Refresh command is needed to acquire the configuration changes into Desigo CC.
NOTE: To avoid incorrect data acquisition and possible data loss, make sure to have a correct and complete VMS camera configuration in place before you start the refresh command.

  1. In System Browser, select Application View.
  2. Select the Video folder.
  3. Select the Extended Operation tab.
  4. Next to the VMS Config property, click Refresh:
    • The status changes to Refresh in Progress. The refresh action may take a few seconds or some minutes, depending on the configuration size.
    • When the configuration has been acquired, Refresh successful displays. (If the command fails, see Video Troubleshooting.)
  1. The configuration changes made in the VMS propagate to Video Configurator and from there to System Browser (when the Video System is Connected). In case of video objects added, renamed or removed, a Sort command may be needed to rearrange the objects into their correct groups. See Update Sorting of Video Objects .
