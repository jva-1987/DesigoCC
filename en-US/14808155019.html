Notifier Onyx Symbols

When integrating the Notifier Onyx extension, the following symbols are available in graphics:

DYN_3D_NF_Onyx_Panel_Fire_None_Horizontal_001_150

  • Default symbol for the panel.
  • Symbol background changes color according to the event category color.

DYN_3D_NF_Printer_None_None_Horizontal_001_150

  • Default symbol for the printer.
  • Symbol background changes color according to the event category color.

DYN_3D_NF_Onyx_Detector_Ceiling_None_Horizontal_001_150

  • Default symbol for the automatic detectors.
  • Non-default symbol for automatic and logical zones.
  • Symbol background changes color according to the event category color.
  • displayed bottom right of the symbol means that the related object is disabled.

DYN_3D_NF_Onyx_Detection_Zone_None_Horizontal_001_150

  • Default symbol for the automatic and logical zones.
  • Symbol background changes color according to the event category color.
  • displayed bottom right of the symbol means that the related object is disabled.

DYN_3D_NF_Onyx_Callpoint_Manual_None_Horizontal_001_150

  • Default symbol for the manual detectors.
  • Non-default symbol for manual zones.
  • Symbol background changes color according to the event category color.
  • displayed bottom right of the symbol means that the related object is disabled.

DYN_3D_NF_Onyx_Callpoint_ManualZone_None_Horizontal_001_150

  • Default symbol for the manual zones.
  • Symbol background changes color according to the event category color.
  • displayed bottom right of the symbol means that the related object is disabled.

DYN_3D_NF_Onyx_Control_Generic_None_Horizontal_001_150

  • Default symbol for the modules and 4-20ma modules.
  • Non-default symbol for bell circuits and panel circuits.
  • Symbol background changes color according to the event category color.
  • displayed bottom right of the symbol means that the related object is disabled.

DYN_3D_NF_Onyx_Device_Evacuation_None_Horizontal_001_150
