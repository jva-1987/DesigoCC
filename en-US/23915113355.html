Enable Application Rights for the Desigo Graphics Migration Tab

  1. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System settings > Security and open the Application Rights expander.
  3. Select the Desigo Insight Migration check box.
  1. The Desigo Data Migration and Desigo Graphics Migration tab are visible in the Management View under Project > System Settings > Conversion Tools >.
