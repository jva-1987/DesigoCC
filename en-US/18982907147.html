Configuring Meteoblue Weather Using the Network Wizard

Scenario

To simplify configuration steps, the Network Setup Wizard extension is installed by default, and you want to use this wizard to easily configure a SORIS driver, network, and adapter to integrate the Meteoblue Weather service in Desigo CC. For background information, see the reference section.

Prerequisites

  • The Meteoblue Weather extension is installed and included in the active project. The following dependent extensions are also included automatically:
    • SORIS Driver
    • Weather Forecast
  • The Meteoblue adapter is running and connected to the Meteoblue weather forecast service. For instructions, see Installing and Starting the Meteoblue Weather Adapter.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
