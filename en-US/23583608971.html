Configure Email Settings

Perform this procedure only if you want to schedule the automatic execution of the report through email.

  1. System Manager is in Operating mode.
  2. You have configured SMTP Email device using Notification extension.
  3. You have created recipients in the Recipients Editor tab of the Notification application. See Create Recipient Users or Recipient Devices in Creating and Configuring Recipients.
  1. In System Browser, select Application View.
  2. Select Applications > Links.
  3. Select the link for which you want to configure the email settings.
  4. Click Email Settings .
  5. The Email Settings dialog box displays.
  6. In the Address Book section, select either People or Group from the drop-down list. Alternatively, you can also search for individuals or groups by typing the names in the second drop-down list.
  7. The list of individuals or groups displays.
  8. Select the individuals or groups to whom you want to send the report as an email and click the > arrow to transfer the names to the Recipients section.
  9. Enter a subject for the email.
  10. In the Body field, enter a message you want to appear in the body of the email, or, leave the default text.
  11. Click OK.
  1. The email settings are configured.
