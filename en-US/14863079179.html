SecuriFire Compatibility and System Limits

Compatibility

  • SecuriFire fire detection system family, including:
    • FCP500
    • FCP1000
    • FCP2000
    • FCP3000
  • SecuriFire Modbus interfaces B6-NET2-485 and B8-NET2-485
  • SecuriFire BACnet interface B5-BIM 15-1
  • Desigo CC V3.0 SR3 or higher

Size Limits

Since no specific limits are identified for this integration, refer to Desigo CC system limits defined in the System Description document (A6V10415500).

Other Limitations

  • Due to a limitation in the Desigo CC Modbus integration, the Silence and Reset event commands are available for the panel object only. They cannot be replicated in the Operation tab for the zone elements.
  • When a zone event occurs, the Ack command becomes available in Event List. Note that the event blinks constantly, event after the Ack command is successfully executed.
  • In order to have the Ack command work in Event List, a specific configuration step in the Desigo CC station is required. For instructions, see Configuring SecuriFire Modbus Commands Macros.
