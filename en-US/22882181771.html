(Option 1) Apply Alarm Table to SiPass Object Model

This procedure is part of the workflow for customizing SiPass alarm tables.

 

Prerequisites

  • You configured a customized SiPass alarm table.

 

Apply the Modified Alarm Table to a Customized SiPass Object Model

Use this method to modify the alarm table for all SiPass points of a given type. For example, all access control door sets.

  1. In the Headquarter SiPass library, select the SiPass object model you want to customize. For example:
    • […] > Libraries > L1-Headquarter > Access > Common > Object Models > AC DoorSet
    • […] > Libraries > L1-Headquarter > Access > SiPass integrated > Object Models > SiPass Server
  2. In the Models & Functions tab, click Customize .
    NOTE: If the icon is dimmed it means that the selected SiPass object model is already customized. Skip to step 4.
  3. Click OK.
  4. The customized object model is added to the customized SiPass library.
  5. Select the customized object model. For example [...] > Libraries > L4-Project > Access > Common > Object Model > AC DoorSet.
  6. In the Models & Functions tab, open the Properties expander.
  7. In the list on the left, select the Property of this object model to which you want to associate the modified alarm table. This must be one for which the FS column is checked. For example, Alarm.Reader1State.
    NOTE: Pay attention to replace the correct customized table in the properties that are linked to the corresponding original table. For more information, search for Assigning an Alarm Table to an Object Model.
  8. The Details, Alarm Configuration and other expanders on the right update to show the settings of the selected property.
  9. In the Alarm Configuration expander:
    a. Leave the Alarm Configuration field set to Field system.
    b. From the Alarm table reference drop-down list, select the [customized SiPass alarm table] that you want to use for this customized object model.
  10. Click Save .
  11. Stop and then re-start the SiPass driver to enable the new table. See Stopping and Starting the SiPass Driver.
  1. The modified alarm table will apply to all the SiPass points that are based on this object model.
