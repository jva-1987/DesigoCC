Stop the Server Project and Close SMC

  1. Open SMC.
  2. In the SMC tree, select the Projects node.
  3. In the Projects tree, select the project that you want to stop.
  4. Click the Project Settings tab.
  5. Project Settings displays.
  6. Click Stop .
  7. A confirmation message displays.
  8. Click OK to stop the started project.
  9. (Recommended) Close SMC.
