Adding a Valve

In Desigo CC, there are default valves for all geographic directions, 2 or 3-port valves, and technical versions. After adding the graphic, the default valve is configured as per project requirements.

  1. In the Mode group, conduct one of the following actions:
    • Click Design . Previously rotated objects appear in the default direction.
    • (Optional) Click Test . In this case, the valve does not display in its true size.
  2. In System Browser, select logical view.
  3. Select Logical > [Network name] \...\ [Plant] >
    • [PreHcl (preheater)] > [Vlv (valve)]
    • [ReHcl (reheater)] > [Vlv (valve)]
    • [Ccl (cooling coils)] > [Vlv (valve)]
    • [Hum (humidifier)] > [Vlv (valve)]
  4. Drag the object to the graphics page.
  1. The added object displays at the maximum scale. This ensures that the next object does not overlap.

 

Rotating

  1. The fan symbol is added to the graphic.
  1. In the Mode group, click Test.
  2. The valve symbol displays as per default direction .
  3. Select the valve symbol.
  4. Left-click and hold, then right-click until the valve symbol displays in the desired direction.
  5. Release the left mouse button.
  1. The valve symbol is displayed in the desired direction.

Changing a 2 / 3-Port and Actuator Type

  1. The fan symbol is added to the graphic.
  1. In the View tab, select Properties.
  2. The Symbol Properties dialog box is enabled.
  3. In the Mode group, click Test.
  4. The valve symbol displays as per the valve type and the selected direction .
  5. Select the valve symbol.
  6. Select Symbol Properties > Substitutions.
  7. In the 2 or 3 Port field, enter a number:
    • 2 for 2-port valve .
    • 3 for 3-port valve .
  8. In the Type field, enter a number from 0 through 2 (see Table Valve Types).
  9. Click ENTER or change to another entry field to activate the change.
  10. (Optional) Change valve symbol.
    a. Right-click the symbol and select Symbol Instance > Replace. An object reference must be assigned to change the symbol.
    b. Select the new valve symbol.
