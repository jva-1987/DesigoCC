Configure Zones

 

Due to a known CEI protocol limitation, zone items are not discovered. This means that you must configure the zones manually.

 

  1. Select System on the left.
  2. The Configuration area is displayed on the right.
  3. Expand the branch (panel) you want to configure.
  4. Switch to the Info tab, and create the zone items as described in Manually Set New Zones.
