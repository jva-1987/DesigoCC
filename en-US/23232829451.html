Configure the Graphic Header

  1. The new graphic is selected and displayed in the Graphics Editor.
  1. Make sure the Element Tree view is visible; from the bottom left side of the view, click Show all Elements .
  2. The Element Tree now displays all elements and objects that are placed on the graphic.
  3. Expand Layer (Layer1).
  4. Locate and select Symbol Instance (Header).
  5. The Symbol Instance Properties view displays the substitutions and properties associated with that particular symbol instance.
  6. In System Browser navigate to Project > Field Networks > [RFC1628 network] > [UPS device].
  7. Drag the UPS device object onto the Object Reference field of the Symbol Instance (Header) properties.
  8. (Optional) By default, in Symbol Instance Properties > Substitutions, the ShowName field is set to 0. This means that the description of the UPS properties will display on the graphic header. To show the MIB name, enter 1.
