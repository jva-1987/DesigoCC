Creating Reports

This provides instructions on how to run a report on all Active Events, convert it into a PDF and print it out to give to a service technician, for example for addressing faults.

 

Generate an Active Events Report

Do this procedure to manually generate the report of Active Events.

  1. In System Browser, select Reports.
  2. Locate and select the Report Definition Active Events.
  3. The Report Definition displays.
  4. Click one of the following:
    • Run to run the Report Definition according to your login language.
    • Run As to run the Report Definition according to the selected language. Localized data is retrieved and loaded in the cells of a table/plot in the report.
  5. The report execution status displays in the Report Management section below the report definition. On successful report execution, the generated report displays in Run mode.

 

Create a PDF of a Report

  1. A previously-generated report displays.
  • Click Create and view PDF .
  1. A PDF of the report opens in the PDF viewer.
  1. In the PDF viewer, click Download.
  2. In the Save File dialog box, select the destination.
  3. Click Save.

 

Print Manually Generated Reports

  1. In System Browser, select Reports.
  2. Click Create and View PDF .
  3. The PDF output displays.
  4. Click Print .
  5. The Print dialog box displays.
  6. Select the configured printer and click Print.
