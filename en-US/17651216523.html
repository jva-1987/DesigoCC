NK823x Objects

In System Browser, the representation of NK823x includes a base and an optional set of objects.

The basic set is automatically created when you create an NK823x unit and assign a BACnet Instance number to it (refer to Connecting to the NK823x Unit). These objects represent components that are always present in any NK823x configuration.

You can then add optional objects a necessary to match your actual system (refer to Configuring the NK823x Unit). These components may or may not be present depending on the hardware configuration.
