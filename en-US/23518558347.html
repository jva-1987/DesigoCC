Enable Events for Individual Scalance Devices

Use this method to modify the alarm configuration for individual Scalance devices.

  1. Select the Scalance device for which you want to enable alarm: Project > Field Networks > [SNMP network] > [Scalance device] among the following:
    • Scalance S615
    • Scalance XB-200
    • Scalance XB-208
    • Scalance XC-200
    • Scalance XC-224
    • Scalance XM-400
      NOTE: You can also use CTRL or SHIFT to make a multiple selection, for example of two or more devices to which you want to apply the same modification.
  2. In the Object Configurator tab, close the Main expander and open the Properties expander.
  3. In the list on the left, select the Property for which you want to enable the event. For example:
    • SystemStatus
    • InterfaceOperationalState1
    • InterfaceOperationalState2
    • InterfaceOperationalState3
    • InterfaceOperationalState4
    • InterfaceOperationalState5
    • InterfaceOperationalState6
    • InterfaceOperationalState7
    • InterfaceOperationalState8
    • InterfaceOperationalStateAdditional1
    • nterfaceOperationalStateAdditional2
    • nterfaceOperationalStateAdditional3
    • nterfaceOperationalStateAdditional4
    • SpanningTreeEdgeOperationalState1
    • SpanningTreeEdgeOperationalState2
    • SpanningTreeEdgeOperationalState3
    • SpanningTreeEdgeOperationalState4
    • SpanningTreeEdgeOperationalState5
    • SpanningTreeEdgeOperationalState6
    • SpanningTreeEdgeOperationalState7
    • SpanningTreeEdgeOperationalState8
  4. The Details, Alarm Configuration, and other expanders on the right update to show the settings of the selected property.
  5. In the Alarm Configuration expander:
    a. Set the Valid field so it displays a gray (not blue) checkmark.
    b. Select the Alarm config.activated check box.
  6. Click Save .
  1. The modified alarm configuration will apply to the selected Scalance devices.
