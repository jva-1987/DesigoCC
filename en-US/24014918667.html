Associate Web URL with Web Application

To associate the URL with the dashboard in the management station.

  1. In SMC, your project is started and you have added it to the D3 Visualization Web Application.
  2. In the management station, you have imported the D3 Visualization library.
  1. In SMC, from the Website section, select the D3 Visualization application node and from the Web Application Details section, click Copy URL.
  2. In the management station, from System Browser, select Application View > D3 Visualization Application.
  3. In the Extended Operations tab, navigate to the Main URL property, and right-click in the Value field and select Paste.
  4. Remove the “s” from https and remove the port number from the URL.
  5. At the end of the URL type: /#/dashboard, and click Set.
  6. You have now configured the dashboard web application. In order for the D3 visualization dashboard to display in the management station you must either first create application rules or use any other web viewer.
