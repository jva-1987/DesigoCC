Enlighted Energy Manager Troubleshooting

This section covers several areas of troubleshooting.

Troubleshooting SORIS

In SORIS, see SORIS Troubleshooting.

Invalid Energy Manager API Key

Issue

Situation

Do the following…

The energy manager does not work and an event is generated in Event List to inform you about the issue.

Invalid API Key.

  1. In System Browser, select Project > Field Networks > [SORIS network] > [Enlighted adapter] > [Enlighted manager].
  2. In the Operation tab, in the field next to the API Key property, enter a valid and unique API code identifier for the user and click Save.
  1. If the API key is valid, the energy manager runs successfully and the event cleared from Event List.

Energy Consumption Mismatch

The Aggregated Energy property of the Area object indicates the energy consumption in watt-hour (Wh) for all the fixtures and plugloads connected to the energy manager.
While the energy manager shows the hourly consumption, in Desigo CC consumption data is aggregated at 5-minute intervals, and rounded to the nearest 15-minute interval set.
