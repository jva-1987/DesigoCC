Create a Web Application Library Block

  1. You are authorized to handle libraries for a specific customization level (L1-Headquarter, L2-Region, L3-Country, or L4-Project).
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L2-Region or L3-Country or L4-Project > [folder] > […] > [library].
  3. The Library Configurator tab displays.
  4. Click Add new object and select New Web App Library Block.
  5. NOTE: For each library you can add only one library block of this type, however, you can add multiple web links to the library block.
  6. (Optional) In the Library Configurator tab, for the New library block, modify the following settings:
    • Version
    • Revision
    • Date
    • Notes
  7. Click Save .
  8. (Optional) In the New Object dialog box, modify the description.
  9. Click OK.
  10. The new web app library block is available in System Browser.
