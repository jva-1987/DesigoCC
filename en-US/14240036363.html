Migration from Desigo Insight to the Management Platform

Scenario: You want to migrate data from Desigo Insight to Desigo CC. Perform the steps in Desigo Insight first. Then migrate to Desigo CC. Desigo Insight compiles the following information to support the migration process:

  • The system report (pdf) includes information on system and project configuration. The system report helps you estimate the size of the migration and during the actual migration.
  • The Migration package includes the system report and data, for example, graphics, Trend Views, Trend data, Todo texts for migration to Desigo CC.

NOTE:
Avoid changes to the plant after creating a Desigo Insight project backup. Any change can result in inconsistencies in Desigo CC that require additional post-processing.

 

Reference: For background information, see the Migration of Desigo Insight to the management platform section.

 

Workflow:

  1. Creating a Migration Package in Desigo Insight
  2. Migrate Desigo Insight Graphics
  3. Migrate Desigo Insight Data

 

Prerequisites:

  • A backup of the Desigo Insight project.
