Creating Device

  1. In System Browser, select Application View > Powermanager > Area > Create Device .
  2. Navigate to Imported devices section and select IEC 61850 protocol.
  3. Select the required IEC 61850 device type.
  5. Navigate to Create > Communication and enter the IP Address and IED name.
  6. Navigate to Extended Operation tab and enter the http Server Port Number.
  7. Configure parameters for Units & Factors, Measurement Points, Favorites and Trends expanders.
  8. Click Save .
  9. Enter the Name of the device.
    Note: You cannot have duplicate device name.
  10. Click OK.
  1. The device is created successfully.
