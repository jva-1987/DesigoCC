Creating an object model for 3rd party room devices

This procedure consists of 3 steps basically done with the EdeApplications.exe tool.

  1. Load and read the EDE file.
  2. Create the meta data file, and import into the Desigo CC library to create the object model.
  3. Create a basic configuration for the properties of the object model.

After you have created the object model, you can import the 3rd party room device just like a Siemens room device.
Importing Siemens room devices with an EDE file

Create an object model for 3rd party room devices

  1. You have a valid EDE file of a 3rd party room device.
  2. The library BA_Device_BACnet_RoomThermostat_HQ_1 is installed.
  1. Click Create Meta Data.
  2. The tool creates an xml file with meta data information. The xml file is in the same location as your EDE file.
  4. Open Management view > SystemSettings > Libraries.
  5. Go to HQ > BA > Device > BA_Device_BACnet_RoomThermostat_HQ_1.
  6. Customize the library according to your requirements.
  7. Open the Import Rules of the customized library.
  8. Open Extract from file, and click Browse.
  9. Select the created meta data file.
  11. The new object of the object model is created.
  12. Click Save.
  13. The new object model is visible in your library.
  14. Go back to the EdeApplications.exe.
  15. Click Configure OMs.
