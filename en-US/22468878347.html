Add and Configuring a Device Symbol to a Graphic

You want to drag an existing monitoring device associated with a coverage area onto your graphic.

  1. System Manager is in Operating mode.
  2. You have a project graphic open in the Graphics Editor.
  3. In System Browser, select the appropriate view that contains the monitoring device you are adding to the graphic.
  1. From that view, navigate to a camera or monitoring device, and then drag the device on the graphic.
  2. The camera or monitoring device displays on the graphic.
  3. To select and adjust the coverage area shape, press SHIFT and click the coverage area handles (for example, the Path shape). You can now modify the element size, shape, color, and so on.
  4. To add color to the coverage area, go to the Home tab, select the Brush Editor group, and then select a color.
    NOTE: To revert back to the default state of the symbol instance, right-click and select Revert to Default. All changes to the symbol instance are cleared. Additionally, if you move your cursor over the arrow, it lists the changes made to the elements within the symbol instance.
  5. To associate other objects or devices, such as sensors to the coverage area, do one of the following or do both if needed:
    a. Drag a detector or sensor from the Management View onto the graphic. Repeat as needed until you have all the devices you want monitored in the coverage area.
    b. For existing detectors, sensors or any elements already associated with a data point on the graphic, drag the detector or sensor objects into the coverage area.
  6. To edit elements within the symbol instance, see Modifying Elements in a Symbol Instance .
  7. Once you have added and modified the coverage area by adding objects and adjusting the coverage area size, click Save .
  1. All the objects that are within a coverage area on any layer are now associated to the main object or device. When you select the device in System Browser, Related Items displays the main object.
