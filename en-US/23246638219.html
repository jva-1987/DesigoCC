Configure the Modbus driver for RDF302

Configure the Modbus driver for RDF302

  1. From the System Browser, select Project > Management System > Server > Server > Driver.
  2. Select New and New MODBUS driver.
  3. Enter a description in the New object dialog box.
    NOTE: The description must be different from all other drivers on the servers.
  4. Click OK.
  5. The new driver is added to the System Browser but not yet started.
  6. Select the driver folder and select the Extended operation tab.
  7. Select property Manager status.
  8. The status of a newly created driver is Stopped.
  9. Select Start to activate the driver.
    NOTE: For configuration purposes, you can also select Start Config... to enable the driver without establishing a real connection to the subsystem network.
