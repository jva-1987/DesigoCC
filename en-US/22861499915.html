Create an SPC Network

This procedure is part of the workflow for integrating an SPC intrusion system.

 

You can skip this section if you want to configure panels under an existing SPC network.

  1. Select Project > Field Networks.
  2. Click New , and select SPC Network.
  3. In the New Object dialog box, enter a name and description.
  4. Click OK.
  5. The newly created network is added to System Browser.
