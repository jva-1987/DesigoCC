InfoCenter Migration Tool

This tool allows you to migrate templates from the InfoCenter database to generate BIRT Advanced Reporting templates in Desigo CC. The InfoCenter Migration Tool is delivered with the InfoCenter extension module and is executed from the following path: …\InstallFiles \EM\InfoCenter\AddSW\InfoCenter Report Migration Tool on the distribution media.

When the InfoCenter Migration Tool is executed, you can choose to migrate all templates, or you can select templates as needed.

The ADDSW folder contains the following components:

  • ReadMe.txt – The text file that provides the prerequisites, requirements, and the instructions to follow before preparing to run the InfoCenter Migration Tool.
  • ReportMigrationTool.jar – The executable file to launch the InfoCenter Migration tool. It must be executed from an elevated Admin level command prompt.
  • default.rptdesign – The base template that the migration tool uses to populate and lay out the individual InfoCenter custom templates. This default template stores the information required to generate custom reports.
