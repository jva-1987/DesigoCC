New Domain User

Scenario
You want to configure a user who is part of a domain group.
For general information on user administration, see User and User Group Administration.

Prerequisites

  • You have created one or several user groups under the folder Project > System Settings > Security (see User Group Administration). The domain user group was assigned to at least one of these groups.
  • A supervisory user was created by copying the existing default admin and configuring adequate user rights.
