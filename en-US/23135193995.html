Create User Groups for Running Events Reactions Scripts

  1. Select Project > System Settings > Security.
  2. Click New .
  3. In the New Group dialog box, define the group of users to which the Desigo CC user belongs:
    a. Select the User radio button.
    b. In the Group name field, enter the account name. For example, Dept100.
    c. Click OK.
  4. The newly created user group displays in the Groups list.
  5. Drag one of the Desigo CC users from the Configured Users expander to the User Members list. For example, Dept100User.
  6. Repeat the previous step to add all the required Desigo CC users.
  7. Click Save .
  8. Repeat steps 2 – 6 above to create all the required user groups.
  1. The user groups created here are available for the scopes configuration.
