Deleting an SNMP Configuration

This section provides instructions for deleting a previously configured SNMP device, network, or driver.

Prerequisites

  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Delete an SNMP Device

Do this procedure to stop SNMP monitoring of a device or property.

  1. In System Browser, select Project > Field Networks > [SNMP Network] > [SNMP device] or [SNMP property].
  2. The System Management tab displays.
  3. Click Delete .
  4. A confirmation message box displays.
  5. Click Yes.
  1. The SNMP device or property is removed from System Browser and no longer monitored.

Delete an SNMP Network

  1. Any SNMP devices configured under that network have been deleted. See Delete an SNMP Device, above.
  1. In System Browser, select Project > Field Networks > [SNMP network].
  2. The System Management tab displays.
  3. Click Delete.
  4. A confirmation message box displays.
  5. Click Yes.
  1. The SNMP network is removed from System Browser.

Delete an SNMP Driver

  1. If the SNMP driver is still associated to a network, any SNMP devices configured under that network have been deleted. (otherwise an alarm—Driver not associated—is generated). See Delete an SNMP Device, above.
  1. In System Browser, depending on the SNMP driver you want to delete, select one of the following:
    • Project > Management System > Servers > Main Server > Drivers > [SNMP driver]
    • Project > Management System > FEPs > [FEP station] > [Drivers] > [SNMP driver]
  2. The Object Configurator tab displays.
  3. Click Delete .
  4. A confirmation message box displays.
  5. Click Yes.
  1. The driver is removed from System Browser.
