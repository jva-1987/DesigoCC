Configure the Fire Objects Functions

  1. Expand the first server structure to display the list of fire objects.
  2. In the list, select the first Zone and then choose the Desigo CC Function to associate (Generic, Automatic or Manual).
    Repeat this step for the other zones.
  3. Select the first Detector and then choose the Desigo CC Function to associate (Generic, Automatic or Manual).
    Repeat this step for the other detectors.
  4. Select Apply.
  5. Repeat the steps above to configure the functions of more fire panels, as necessary.
  6. Select Save.
