Changing an Object Classification

The classification of an object includes its discipline, type, subdiscipline and subtype.

  1. In System Browser, select the object you want to modify. For example, Project > Field Networks > [network] > Hardware > [device] > [data point].
  2. Select the Object Configurator tab.
  3. In the Main expander, change the following entries:
    • 1 = Discipline
    • 2 = Subdiscipline
    • 3 = Type
    • 4 = Subtype
  4. Select the Valid check box.
    NOTE: The check box may not be green or blue.
  5. Click Save .
