Create a Notification Class and Recipients

  1. Select Project > Management System > Servers > Main Server > BACnet Server > [device].
  2. Click the BACnet Server tab.
  3. Expand the Notification Classes section.
  4. Click Add, and the select the newly added row.
  5. The Notification Properties section expands and displays default settings.
  6. (Optional) Modify the Instance No. and Priority for the various alarm states.
  7. Expand the Recipients section.
  8. Click Add, and then select the newly added row.
  9. The Recipient Properties section expands and displays default settings.
  10. Accept the defaults, or modify the settings as needed.
  11. Repeat Steps 7–8 to add more recipients to this Notification Class.
  12. Click Save .
