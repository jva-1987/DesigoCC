GMS_Room_System

Property

Description

CommunicationStatus

Diagnostic information relating to the status of the communication between the FIAS adapter and the Fidelio system which can be one of the following states:

  • Normal
  • Fault

SynchRequest

Synchronization request between the FIAS adapter and the Fidelio system to align also the Desigo CC database.

The Synchronization button is provided for this purpose.

AlarmAlarmDPE

Internal property used to link the SORIS AlarmTable to generate FS alarms.
