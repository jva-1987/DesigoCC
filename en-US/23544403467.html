Media Group Selection in the Parameter Dialog Box does not show Expected Values

Problem: Media Group selection in the Parameter dialog box does not show expected values.

Cause: The Default Web Client's Management Station Security Group restricts the access.

Solution: Remove the Default Web Client from the DefaultStations Security Group to grant access. (See No Access to View Reports for more information).
