Mapping System Browser Data with Powermanager Devices using Delimeter Separated Data Mapping

Scenario

You want to map System Browser data with Powermanager devices. This System Browser data is represented in the form of a flat structure and is separated by delimiters such as (') format.
For background information, see the reference section.

Prerequisites

  • You have the software distribution containing the Powermanager extension.
  • System Manager is in Engineering mode.
  • System Browser is in Application View.
  • The System Browser data to be mapped with Powermanager devices is present in the Management view.
