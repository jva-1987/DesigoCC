Checking the Event Schema Mappings

The event schema is applied at the project level, and defines the mapping between alarm classes and categories. For more information, see Event Schemas.

  1. In System Browser, select Project > System Settings.
  2. In the Extended Operation tab, the Event Schema property indicates the currently applied schema. For example, EN.
  3. Select the schema in question under the global events library: Project > System Settings > Libraries > L1-Headquarter > Global > Events > Schemas > EN.
  4. Select the Event Schema tab.
  5. Open the Event Mapping expander.
  1. The mapping table of the alarm classes to the alarm categories displays.
