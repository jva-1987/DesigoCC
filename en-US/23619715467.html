Recipient Groups

Create Recipient Groups

  1. Click Add Group .
  2. Select General and enter the name of the group.
  3. Select Members and click Add.
  4. The Groups Membership dialog box displays.
  5. Select the recipients to be associated with the group and click Move to move the selection to the text box on the left.
    NOTE 1: Add a maximum of 1000 recipients to a recipient group.
    NOTE 2: Click Filter to narrow the search based on the recipient type. Refer to steps 6 and 7 in the Update a Recipient Group section for detailed instructions.
  6. Click OK.
  7. The Members list is updated with the selected recipients.
  8. Select Groups Membership and click Add.
    NOTE 1: Refer a group to another group to build larger structures. However, it is not possible to have recursive groups where groups refer to each other.
    NOTE 2: For detailed instructions on groups membership, see Create/Import Recipient User OR Create Recipient Devices.
  9. Click Save .
  1. The recipient group is created.

Update a Recipient Group

  1. The recipient groups are created.
  1. Select Applications > Notification > Recipients.
  2. The Recipients Editor tab displays.
  3. Open the Recipients expander.
  4. The list of recipients displays.
  5. Select the recipient group to be updated.
  6. The details displays.
  7. Select General and update the Name.
  8. Select Members, click Filter to select the recipient type to be viewed.
  9. (Optional) Enter a keyword in the Search field.
  10. (Optional) To add a recipient to a group, do the following:
    a. Click Add: The Group Membership dialog box displays.
    b. Select the recipient and click Move to move the selection to the text box on the left.
    c. Click OK.
  11. (Optional) To remove a recipient from the group, select the required recipient from the Members list and click Remove.
  12. (Optional) To add the group to other recipient groups, do the following:
    a. Select Groups Membership.
    b. Click Add: The Group Membership dialog box displays.
    c. Select the group to which you want to add this group.
    d. Click Move to move the selection to the text box on the left.
    e. Click OK.
  13. (Optional) To remove the group from another recipient group, select the recipient group and click Remove.
  14. Click Save .
  1. The recipient group is saved with the updated details.

Delete a Recipient Group

  1. The recipient groups are created.
  1. Select Applications > Notification > Recipients.
  2. The Recipients Editor tab displays.
  3. Open the Recipients expander.
  4. The list of recipients displays.
  5. Select the Recipient Group to be deleted.
  6. Click Remove .
  7. A confirmation message displays.
  8. Click Yes.
  1. The recipient group is deleted.
