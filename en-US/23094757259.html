Create Notification Profiles

  1. Select Applications > Energy And Power > Notification Profiles.
  2. The Notification Profiles application displays.
  3. Click the Notification Profile tab.
  4. In the Warning Notification section, select the relevant check box to issue a warning message in the event of a meter rollover, negative meter reading, or meter reading displaying 0.
  5. If you have selected the Meter Roll Over check box, ensure that you specify the warning limit. By default, this field is set to 50% of the maximum meter reading.
  6. Click Save .
