Configuring OPC DA Discovery Settings

Before scanning the OPC network for OPC servers you have to define the computer where the discovery will be performed.

  1. System Manager is in Engineering mode.
  2. An OPC driver has been added, an OPC network has been added, and an OPC driver has been assigned to the OPC network.
  1. In System Browser, select Management View.
  2. Select Project > Field Networks > [OPC network].
  3. In the Discovery tab, open the Discovery Settings expander. For background information, see OPC DA Discovery Settings.
  4. Specify the computer where you want to perform the discovery of OPC servers in one of the following ways:
    • To have the OPC discovery performed on the current computer: click Local.
    • To have the OPC discovery performed on a remote computer, do the following:
      a. Click Remote.
      b.       In the Host name field, enter the remote computer host name. Alternatively, click Browse, in the Browse for computer network dialog box, select the host name of the remote computer from the network, and click OK.
      c. In the User name and Password fields, enter the user credentials.
  5. Click Save .
  6. You can now scan the network for OPC servers and their OPC items. See Discovering OPC Servers and OPC Items.
