Integrating Zenitel AlphaCom

Scenario

You want to configure Desigo CC for connecting to the Zenitel AlphaCom equipment. For background information, see the reference section.

Prerequisites

  • The Zenitel AlphaCom extension is installed and included in the active project.
  • You prepared the configuration file for the device you want to integrate. See Zenitel AlphaCom Configuration (CSV) File.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Licensing

This extension is covered by license. Check that the required license is available; otherwise, you may not be able to configure and work with this extension.
