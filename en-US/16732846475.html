Configure the Macro Viewer for a Station

You can add the Macro Viewer to a station.

 

  1. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Depending on which station you want to modify, select one of the following:
    • Project > Management System > Servers > Main Server
    • Project > Management System > Clients > [client]
    • Project > Management System > FEPs > [FEP]
  3. The System Management tab displays.
  4. From the Client Profile drop-down list, select the one that you want to apply.
    NOTE: The client profiles that include the Macro Viewer have …Macro_BTM_x1… or …Macro_BTM_x3… in their name, with BTM indicating the position (bottom of the screen), and x1 and x3 indicating the number of rows available for the symbols.
  5. Click Save .
  6. The client profile is set on the station.
  7. Upon configuring the Macro Viewer for the first time, on the main station, in the Summary bar, select Menu > Exit to log off, and then start Desigo CC again.
  1. The Macro Viewer is configured. Perform the procedure to Add and Remove Macro Viewer Objects.
