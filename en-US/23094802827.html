Historizing Metering Point and Media Unit for Meter Replacements

  1. You have created a managed meter and it displays in the Managed Meters folder in the System Browser.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Energy and Power > Managed Meters > [managed meter].
  3. The managed meter details display.
  4. Replace the metering point in the Metering Point field with a different metering point.
  5. (Optional) Change the media unit in the Media Unit (input) field.
  6. Click Save .
  1. The metering point and media unit values are historized as per the current date and time.
