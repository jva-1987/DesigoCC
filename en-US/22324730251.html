Discover thingAdapter Objects Manually

  1. The Auto discover feature has not been programmed into the adapter.
  2. You have added a new thingAdapter object to an adapter, which the adapter recognizes, but Desigo CC does not.
  1. In System Browser, select Management View.
  2. Select Project > Field Networks > [name of network] > [name of adapter].
  3. In the Extended Operation tab, the Online property displays Connected, and the Consistency property displays Rediscover Required.
  4. In the Extended Operation tab, go to the URL property, and then click Discover.
  1. System Browser refreshes the thingAdapter object list, the Consistency property displays Up to date, the alarm is removed from the Event List, and the Activity Log records the transaction.
