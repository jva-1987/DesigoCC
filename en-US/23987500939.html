Select Desigo Insight Project Data

  1. Select Project > System settings > Conversion tools > Desigo Data Migration.
  2. Select the Desigo Data Migration tab and open the Select Folder expander.
  3. Click Browse.
  4. Select the Desigo Insight project folder for the exported project.
  5. Click OK.
  1. The selected project is displayed.
