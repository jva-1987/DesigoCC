Show Lighting Objects

Scenario: Highlight all associated lighting objects in color.

 

1st Step: Highlight an Associated Lighting Object

Scenario: You want to highlight an associated lighting object to evaluate the syntax.

  1. Edit is enabled in BIM Viewer.
  2. Desigo CC Objects and BIM objects are associated.
  3. An empty Lighting is available.
  1. Click Toggle objects tree display .
  2. Click Highlight selected object .
  3. In the object tree, select the room and then the lighting object. You can also do this in the System Browser in the Logical View as an alternative.
  4. Right-click and select Add to filter, followed by Lighting.
  5. In the Filter Name dialog box, select the Lighting filter.
  6. Click Edit.
  7. The assigned lighting object is displayed. The Data point name references the technical address Lgt(1). The object model is displayed in the Data point type column. The IFC Name column references the selected lighting object. The BIM object model IFCLIGHTFIXTURE displays in the IFC Type column.
    • Test: Clear Highlight selected object .
    • Test: Select the Lighting check box.
  8. The light fixture configured in the filter is highlighted in color.
  9. Click Save .

 

2. Step: Highlight all Lighting Objects

Scenario: After finding the BIM object model for lighting objects, you can emphasize all lighting objects associated with this BIM object model based on the lighting object.

  1. In the Filter Name dialog box, select the Lighting filter.
  2. Click Edit.
  3. Displays the associated BIM object. The Data point name references the technical address Lgt(1). The object model is displayed in the Data point type column. The IFC Name column references the selected lighting object. The BIM object model IFCLIGHTFIXTURE displays in the IFC Type column.
  4. In section Highlight associated objects, select BIM object model.
  5. Click Edit.
  6. The Emphasized objects dialog box displays.
  7. Delete the entry in the IFC Name text field and enter an asterisk (*).
  8. The filter entry can be customized in the Data point name text field:
    • Lgt(1) = Displays only lighting objects under this data point name;
    • Lgt(2) = Displays only lighting objects under this data point name;
    • Lgt* = Displays all lighting objects.
  9. Click OK and then OK to close both dialog boxes.
    • Test: Select the Lighting check box.
  10. All lighting objects under this data point are highlighted in color.
  11. Click Save .

 

3. Step: Set Condition

Scenario: Display all switched-on lighting objects.

 

  1. In the Filter Name dialog box, select the Lighting filter.
  2. Click Edit.
  3. In section Highlight associated objects, select BIM object model.
  4. Click Edit.
  5. Select text field Condition and enter >0.
  6. Click OK and then OK to close both dialog boxes.
    • Test: Select the Lighting check box.
  7. Highlight all switched-on lighting objects in color.
  8. Click Save .
