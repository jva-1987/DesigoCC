Updating the Summary Bar for Multiple Automation Stations Using a Macro

Scenario: You want to update the alarms on all automation stations with a single command.

  1. In System Browser, select Application View.
  2. Select Applications > Logics > Macros and select the Macro tab.
  3. Click Create and select New Macro.
  4. In the New Object dialog box, enter a name and description.
  5. Click OK.
  6. The new macro appears in the Macro tab.
  7. In System Browser, select Management View.
  8. Select Project > Field Networks > [Network name] > Hardware > [Automation station] of the corresponding automation station.
  9. Drag the automation station to the Macro configuration area.
  10. In the Property column, select the option Not defined from the drop-down list.
  11. In the Command column, select the option SynchronizeAlarmStatus from the drop-down list.
  12. Repeat steps 7 to 10 for all automation stations.
  13. Click Save .
  1. The macro is defined and all automation station are assigned.

 

Executing a Macro

  1. In System Browser, select Application View.
  2. Select Applications > Logics > Macros > [Macro name].
  3. In the Operation tab, select the Activity Status property.
  4. Click Execute.
