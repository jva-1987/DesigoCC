Download Initial NK823x Configuration from Management Station

  1. The TCP/IP network properties of the management station are set to let the computer communicate on the operating subnet and on subnet 192.168.9.x (for example, with IP address 192.168.9.254 and subnet mask 255.255.255.0).
  2. You connected to the NK823x in emergency mode.
  3. The firmware and kernel versions are updated and compatible. See Checking and Updating NK823x Firmware and Kernel.
  1. In System Browser, select Project > Field Networks > [BACnet Network] > [NK823x].
  2. Select the NK8000 tab.
  3. (Optional) Open the Configuration expander, and configure the file transfer mode.
    This is set to Use secure download by default (encrypted data transmission on TCP port 20500). If you have a local and safe connection to NK823x, you may want to deselect the check box and apply a faster FTP passive mode. Proceed as follows:
    a. Deselect the check box. Note the warning message about potential vulnerabilities.
    b. Click Save .
  4. Click Download configuration.
  5. A confirmation message box displays.
  6. Click OK.
  7. A final notification displays.
  8. In the Extended Operation tab, the Emergency Mode is automatically set to Disabled.
  9. Turn off the NK823x unit.
  10. On the NK823x mainboard, set back to OFF the DIP switches 1 and 2.
  11. Replace the cover and turn on the NK823x unit.
  12. From Windows, ping the NK823x configured IP address to check the network connection.
  1. The management station can communicate with the NK823x at the configured IP address.

In case of a distributed, multi-servers system (refer to the SMC documentation), the download can only be performed for NK823x units connected to the local system.
