ESPA Paging System Device Message Order

Communication for the protocol ESPA Paging System device follows a specific order of the message content, which is as shown below:

Record Type

Data Identifier

Call address

1

Number of Transmission

5

Beep coding

3

Priority

6

Display message

2

While configuring your pager device, it must be checked if your pager device is also following the above ordering of message content. In special circumstances, the message ordering can be changed, for that contact the Siemens Service Desk for support.
