Project and HDB Backup Procedures

You can create a backup of the project and history database from either the System Management Console or from the backup commands available in the Operation/Extended Operation tabs of the Contextual pane in the management platform.

The backup of the project data and history database data must be made independently. Backing up the History Database does not back up the project data and vice versa.

The following procedures provide step-by-step instructions for making project and HDB backups. For background information, see the reference section.
