Display Floor Information

Prerequisites:

  • System Manager is in Operation mode.
  • In System Browser, Logical View is selected. BIM graphics can also be selected in the Application View as an alternative.

 

Floor Overview: Alarms

  1. Select Logical > [Subsystem name] > [Building name] > [Floor].
  2. The BIM Viewer tab opens.
  3. Click Show room alarms .
  4. Click Show related alarms in EventList .
  5. The event list is filtered and displayed with the floor alarms.
  6. Acknowledge and reset alarm.
  7. Click Close event list .
  1. Alarms are displayed in the corresponding color for the alarm category color in the floor overview graphic.

 

Floor Overview: Temperature

  2. The BIM Viewer tab opens.
  3. Click Show room temperature .

Display 2D-Floor View

  1. The view is only available if the graphic pages are generated from the BIM data.
  1. In System Browser, select Application View.
  2. Select Applications > Graphic > [graphics page].
  3. The Default tab displays.
  4. Select the appropriate rooms.
