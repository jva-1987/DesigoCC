Manually Creating an OPC Discovery Rules Library

You can configure the rules of the OPC discovery library that will be used when importing OPC Items. To create an additional OPC discovery rules library from scratch, you can construct this library manually.

  1. You are trained and authorized to work with system libraries. Also, you are authorized to handle libraries for the specific customization level (L4-Project, L3-Country, or L2-Region).
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L4-Project or L3-Region or L2-Country > […].
  3. In the Library Configurator tab, click Add new object and select: New OPC Discovery Rules Folder.
  4. The new library folder is added to System Browser.
  5. Select the newly created OPC discovery rules folder.
  6. In the Library Configurator tab, click Add new object and select the library element: New OPC Discovery Rules.
  7. In the New Object dialog box, enter a description and click OK.
  8. The new OPC discovery rules library element is added to the OPC discovery rules folder.
  9. You can now continue with configuring library settings, hierarchy settings, and with creating object mapping rules. See Configuring the Settings for an OPC Discovery Rules Library.
