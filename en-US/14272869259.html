Configuring the Animation Element

The Animation element, available from the Insert group, allows you to set the property evaluations that determine which images (symbol instances) will display based on the resulting state or value range of the animation element’s Symbol Reference property.

  1. On the Home tab, in the Elements group, click Animation .
  2. An animation element is created on the active canvas or graphic.
  3. In the Properties view (SymbolInstance Properties), navigate to the Symbol Instance.
  4. From the Symbol Reference property, click , to display the Library Browser view.
  5. From the Library Browser, navigate to the library and symbol that is to represent the animation element.
  6. Drag the symbol into the Properties View Symbol Instance field.
  7. The symbol is now associated with the animated element, and displays on the canvas.
  8. Repeat Steps 1-3, as needed.
  9. Navigate to the Evaluation Editor view, and from the Type drop-down menu, select Animated.
