Migrating and Upgrading FS20 Sinteso Fire Panels

Upgrading Control Panels from V5.2 to V6.0

  • If no changes occurred in the fire system, then no re-import of the SiB-X metafile is required in Desigo CC.

Upgrading Control Panels from V6.0 to V7.0

  • Re-import of the V7.0 SiB-X metafile is required in Desigo CC.

Upgrade Desigo CC from Previous Versions

When you upgrade Desigo CC from V4.x/V5.0/V6.0 to V7.0, just follow the standard Desigo CC upgrade procedure.

Then, in the Desigo CC project, re-import the SiB-X metafile on each fire system to apply the latest library modifications.

Expanding Control Panels

It is possible to expand the fire panels as follows:

  • FC2020 to FC2040
  • FC2020 to FC2060
  • FC2040 to FC2060

A new import of the SiB-X metafiles is required. Existing object references in graphics, macros, user-defined views, schedules and so on are still valid after the new SiB-X import.

Replacing NK823x as a router for the SafeDlink communication

NK823x used to be applied as a router for the fire networks in DMS8000 installations. In connection to Desigo CC, NK823x routing services can no longer support fire networks.
In case of migration, you can replace NK823x with Scalance S615. For more information, see the application guide Scalance S615 Router for SAFEDLINK Fire Networks (A6V13357580).
