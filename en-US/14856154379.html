Integrating an URVE Room Booking Adapter

Scenario

You want to configure Desigo CC to operate on the URVE Room Booking system. For background information, see the reference section.

Prerequisites

  • The URVE Room Booking extension is installed and included in the active project. The following extensions are also selected automatically:
    • Scripts
    • SORIS Driver
  • The URVE Adapter is running and connected to the URVE Room Booking system. For instructions, see Installing and Configuring the URVE Adapter.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
