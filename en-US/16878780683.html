Making a Document Display as a Related Item of an Object

You can optionally create associations between documents and system objects.

  1. You have appropriate user authorizations to view and configure documents in Desigo CC.
  1. In System Browser, select Application View.
  2. Select Applications > Documents > […] > [document].
  3. The content of the document (file or HTML page) displays and is read-only.
  4. Click Operate Mode to enable the Documents workspace for editing ().
  5. Select the Manual navigation check box.
  6. Navigate to the object that you want to associate with this document. For example, Applications > Graphics > [floor plan graphic].
  7. Drag the object to the Manually assigned field of the Associated Objects expander.
  8. Click Save .
  1. When you next select the object, a link to this document will display in the Related Items tab. Clicking that link opens the document in the Secondary pane (see Viewing a Document from Related Items).
