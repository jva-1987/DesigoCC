Create Virtual Device

  1. In the System Browser, select ApplicationView > Powermanager > [Area] OR [Sector].
  2. Select Area tab.
  3. Click Create > Create Device.
  4. Navigate to Create expander > Virtual Devices and select the device type.
  5. The Use Area Settings check box is selected by default.
  6. The default settings for the device display in the Units & Factors, Measurement Points, Favorites, and Trend expanders.
  7. Uncheck the Use Area Settings checkbox and configure parameters for Units & Factors, Measurement Points, Favorites and Trends expanders.
  8. In Measurement Points expander, configure Source and Properties.
  9. To configure Source, drag and drop or select from drop down list
  10. To configure Properties, select from drop down list.
  11. Click Save.
  12. Save Object as popup window appears.
  13. Enter the Name and Description.
  14. Click OK.
  1. Device created successfully.
