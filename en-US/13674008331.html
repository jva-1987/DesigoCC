Exporting Scripts from the Project to a File

You can export scripts from Application View > Logics > Scripts > to a .js file (for an individual script) or to a .zip file (for an entire script folder). This allows you to transfer those scripts directly into another project, or to import them into a library.

  1. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Logics > Scripts > [script] or [folder].
  3. The Script Editor tab displays.
  4. Click Export .
  5. In the File Open dialog box, select the destination path.
    a. Enter a name for the file to export.
    b. Click OK.
  1. When the export operation is completed, a .js file (for an individual script) or a .zip file (for a script folder) is stored in the specified destination path.
