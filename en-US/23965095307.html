Set Page Size

  1. Click the Layout tab.
  2. The Page Setup group box displays.
  3. Click the Page Size menu.
  4. A submenu displays A3, A4, A5, Letter, and More Sizes.
    NOTE: Selecting the More Sizes option displays the width and height (in cm or inch – location dependent) of the selected page size. These fields are read-only.
  5. Select the page size of your choice.
  1. The page size of the report definition changes accordingly.
