Add Additional Room Value Display

Scenario: There is a large room in the BIM data. But for control technology reasons two or more system data rooms must be added to this room. In the case, additional system data rooms must be manually assigned to the BIM data room.

  1. Click Toggle objects tree display .
  2. The BIM hierarchy structure is displayed in the BIM Viewer tab.
  3. In System Browser, select the Manual navigation check box.
  4. In System Browser, select Logical View.
  5. Select Logical > [Subsystem name] > [Building name] > [Floor] > [Room] > > [Room 118] > {RHavcCoo} HVAC room coordination.
  6. Drag the RHavcCoo object to the corresponding room 117 in the BIM hierarchy tree.
  7. Select Applications > BIM graphics > [building graphic].
  8. Click Show floor list .
  9. In the floor list, click the desired floor.
  10. Two or more room value displays are available in the BIM room.
  11. Highlight the room value display (highlighted in red) and move it to the new position.
  12. The changed position remains if the BIM data is reimported. Editing may be necessary if rooms are merged or walls are moved during an update of BIM model.
  13. Click Save .
