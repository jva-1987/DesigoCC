Create System Users for Running Events Reactions Scripts

  1. Select Project > System Settings > Users.
  2. The Users tab is displayed.
  3. Click New .
  4. In the New User dialog box, specify the user settings as follows:
    a. From the User type drop-down list, select the name of the system. For example, Desigo CC.
    b. In the User name field, enter the name of the Desigo CC user. For example, Dept100User.
    c. Enter password and confirm password.
    d. Click OK.
  5. Click Save .
  6. The new user is available in the Users list.
  7. Activate the user by selecting the name in the list and then selecting the Enabled check box alongside.
  8. Repeat steps 2 to 5 above to create all the required Desigo CC users.
  1. The Desigo CC users created here are available for the user group configuration.
