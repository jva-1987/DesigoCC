Restoring the Default Layout Configuration

You can restore the Graphics Editor dock panel layout to the last saved default layout.

  • On the Options tab, from the Layout group, click Restore Default .
  1. The dock panel layout and views settings are restored to the last saved default configuration.

 

Saving the Layout Configuration

  1. You have adjusted the dock panel layout and the views with the default settings you want displayed when the Graphics Editor opens.
