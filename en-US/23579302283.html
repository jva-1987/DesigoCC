Delete Search Filters on Columns Not Present in the Log View

  1. Log data displays in the Log Viewer or the Detailed Log tab.
  2. You have applied a condition filter to a log view definition.
  1. Click Search Filter .
  2. The Search Filter dialog box displays.
  3. Delete the search filter expression from the Filter Expression field.
  4. Click OK.
  1. The log view refreshes displaying the latest data.
