Mobile App System Requirements and Limits

Compatible Desigo CC Management Platform

  • Supported version of Desigo CC with Web Services extension.
  • (Optional) Push Notification extension. Required for full push notification functionality. See Mobile App Notifications Reference.

Licensing Requirements

  • License for Web Services.
  • A sufficient number of Concurrent Webservice Sessions licenses for the number of apps that will simultaneously connect to the management platform.
  • NOTE: For Desigo CC management platforms where the IIS Web server runs on Windows 10 or Windows 11 , there can only be a maximum of three apps connected at the same time. However, it is recommended to connect no more than two.
  • On configurations where the IIS web server is instead hosted on a computer running Windows Server, the number of apps that can connect simultaneously is limited only by the number of Concurrent Webservice Sessions licenses.

➔ For more information about what is required to set up the management platform to work with the mobile app, see Setup Checklist for Mobile App.

Mobile App: Device Requirements

  • Android V6.0 or higher (only non-rooted, and with Developer Options disabled).
    Notes concerning Android:
    • In some Android devices from low-cost manufacturers, the kernel build is not signed with an official release key; instead, it is signed with a less secure ‘test key’. Such devices will appear to be rooted to the mobile app (even if the device was not rooted); therefore, they cannot be used to run the Desigo CC mobile app.
    • There is a known issue with the Huawei P7 device that may cause fallback local notifications to work unreliably. See Mobile App Notifications do Not Display.
  • iOS V12.x or later (only non-jailbroken)
  • Hardware:
    • CPU: recommended 1.4 GHz, minimum 1 GHz, dual core
    • RAM: recommended 1 GB, minimum 512 MB
    • Display size: greater than 4 inches is recommended, minimum is 3.27 inches.

The mobile app has been carefully tested on various mobile devices. However, it cannot be guaranteed that the app is compatible with all types of mobile device.
Please contact customer support to get the most recent list of tested devices and known issues.
