Additional IEC61850 Power Devices Procedures

The following procedure provides additional information on IEC61850 Power Devices.

Setting of IED names for the IEC61850 meters

In Desigo CC , the device object models within the library use a particular IEC61850 device property called Intelligent Electronic Device (IED). This property can be specified in the device settings (under Ethernet configuration). The IED names within the object model in Desigo CC and the device have to be identical.

Default IED names within the object models are:

  • For SICAM Q100: SICAM_Q100_01
  • For Sicam P85x: SICAM_P85x_01

NOTE:
After a firmware upgrade of the device, if the IED name in the device changes, you must change the IED name in the device to the default names.
