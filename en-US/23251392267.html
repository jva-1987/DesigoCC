Make Scripts Available for Execution

The scripts available for execution in a project are located under the folder Applications > Logics > Scripts. You can add scripts to this folder in the following ways:

  • Create a new script object and write the script directly in the Desigo CC script editor. See Writing Scripts.
  • Import a script provided with a library into the project. See Importing a Library Script into the Project.
  • Import a script from a previously-exported .js file (individual script) or .zip file (scripts folder). See Importing Scripts into the Project from a File.
