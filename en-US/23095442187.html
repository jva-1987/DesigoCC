Create and Configure a CSV File

  1. You have knowledge about the format and content of the CSV file.
  1. Navigate to [Installation Drive]:\[Installation Folder]\GMSMainProject\profiles\IEC61850DataTemplate.
  2. The IEC61850_template_3.1_Instances.csv file is provided by default.
  3. Create a copy of this file at any location on your hard drive and thereafter modify the information present in the [CONNECTIONS] and [DEVICES] sections as per the device configuration.
  4. Save the CSV file.
