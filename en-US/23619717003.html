Recipient Users

This section describes procedures updating and deleting recipient users.

Update a Recipient User

  1. The recipient users are created.
  1. Select Applications > Notification > Recipients.
  2. The Recipients Editor tab displays.
  3. Open the Recipients expander.
  4. The Recipients list displays.
  5. Select the recipient user whose information must be updated.
  6. The details displays.
  7. Update the required fields under General.
  8. To update the devices associated with a user, select the required device, click Edit and do one or more of the following:
    • Edit the required details in the Recipient User Devices dialog box, and click OK.
    • To add new recipient user devices, click Add under the Recipient User Devices tab.
    • To remove a device, select the required device and click Remove.
  9. Under Groups Membership, click Add.
  10. The Group Membership dialog box displays.
  11. Select the group to which you want to add this recipient user as a member and click Move to move the selection to the text box on the left.
  12. Click OK.
  13. To remove the recipient user from a group, select the group and click Remove.
  14. Click Save .
  1. The user is saved with the updated details.

Delete a Recipient User

  1. The recipient users are created.
  1. Select Applications > Notification > Recipients.
  2. The Recipients Editor tab displays.
  3. Open the Recipients expander.
  4. Select the recipient user to be deleted.
  5. Click Remove .
  6. A confirmation message displays.
  7. Click Yes.
  1. The recipient user is deleted.

NOTE 1:
The deleted recipient user cannot be used for new messages and will be removed from all message templates. However, the recipient user will continue to be displayed as a recipient in active and expired messages for an accurate historical view.
