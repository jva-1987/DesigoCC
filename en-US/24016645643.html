Schindler Integration Graphics Engineering

An object selected in System Browser, is displayed graphically with a graphic template, for example, an elevator; graphics engineering is not required in this case. You can, however, place the corresponding objects on a floor plan or on its own graphics page.

Creating a Graphics Structure

Set the hierarchy for saving graphics pages before starting to create graphics.

  1. System Manager is in Engineering mode.
  1. In System Browser, select Application View.
  2. Select Applications > Graphics and the tab Graphics.
  3. Click Create and select New Folder.
  4. The Create new folder dialog box opens.
  5. Enter a name in the Name field.
    NOTE: The system name is derived automatically from the description. Spaces are replaced by _ (underscore).
  6. Click OK.
  7. Create additional folders or subfolders as per project requirements.
  1. The folder structure to save graphics pages is created.

Note:
Graphics pages can be moved afterwards under the Views tab.

Creating Floor Plan Page

  1. Click Add and select New Graphic.
  2. Click Save.
  3. Applications > Graphics is selected.
  4. Select Applications > Graphics > [selected graphics hierarchy].
  5. Enter a unique name in the Name field.
  6. Click Save.
  7. Import an image or draw the building or the building's floor plan.
  8. Select the Start tab.
  9. In the Elements group box:
    a. Click Import AutoCAD files.
    b. Select the file.
    c. Click Open.
  1. The floor plan is imported.

Create Floor Plan Page with Viewport

  1. The Start page is completed and open.
  1. In the Viewports group, select option Create Viewports.
  2. Click Manual Viewport.
  3. Create a viewport that is the same size as the graphics page.
  4. Click Save.
  5. The viewport is saved below the corresponding graphics page and is displayed in System Browser.
  6. Select the Viewport page in System Browser.
  7. The Viewport properties dialog box opens.
  8. In System Browser, select Management View.
  9. Select either:
    • Application View > Applications
    • Logical View > Logical
    • Logical View > User
    • User View > User
    • Own Views
      and in Viewport Properties > Advanced, drag-and-drop the applicable folder to the Link References text field. This must take place for all existing views.
  10. Close the Viewport Properties dialog box.
  1. The nodes for all views are linked to the Viewport. The Start page opens if the user selected one of these nodes.

NOTE:
When changing the Start page, the Start viewport must be re-saved.

Apply Graphic References

A symbol reference on the Start page must be created for each available view in System Browser.

  1. The Start page is created.
  1. Open the Start page in the Graphics Editor.
  2. In System Browser, select Application View.
  3. Drag-and-drop the Applications node to the graphics page.
  4. The folder symbol is added to the graphic and highlighted.
  5. Select Symbol Properties > Substitutions > Layout.
  6. Clear the Visible check box.
    NOTE: The folder symbol is not visibly displayed in Online mode if the check box is cleared.
  7. Repeat Steps 2 through 4 for all views available in System Browser (always the first node for the given view).
    NOTE: The last view is saved for each user when exiting Desigo CC. This view is open on the top node of System Browser when logging on again to Desigo CC.
    The reference to the graphics page must be added one time for each view since no one knows the view from which the user exits Desigo CC.
  8. Click Save.

Adding Graphic Objects

  1. In System Browser, select logical view.
  2. Select Logical > [Hierarchies] >
    • Building
    • Group
    • Elevator
    • Elevator car
    • Escalator
    • Moving walks
  3. Drag the object to the graphics page.
  4. Click Save.
