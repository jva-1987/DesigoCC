Discovering the Adapter Configuration

  1. You created the SORIS driver, network, and adapter manually or using the network wizard.
  2. System Manager is in Engineering mode.
  3. System Browser is in Management View.
  1. Select Project > Field Networks > [SORIS network] > [KNX adapter].
  2. In the Extended Operation tab, the Online property indicates Connected while the Discovery Status property indicates Idle.
  3. Next to the URL property, click Discover.
  1. System Browser displays the KNX configuration in a hierarchical structure of objects.
