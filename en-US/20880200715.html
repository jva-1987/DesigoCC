Bulk Device Creation Tab

You can create multiple powermanager devices at a single time from the Bulk Device Creation tab in the Engineering mode. To perform a bulk import you must create a CSV file with the details of the devices to be created.

A CSV file contains data in which values are represented as text and separated using a separator such as comma (,).The file in the CSV format can be edited in simple text editors, such as Wordpad or , or using advanced data processing tools, like Excel. You can also create the CSV file by using the export option in powerconfig application.

The following are the important columns in the CSV file:

  • Device Name: Name of the device.
  • Device Description - Default Language (English): Displays the name of the device in English.
  • Device Description – Additional Language: Displays the device name in the additional language selected.
  • Device Type: Device type to be created.
  • Area: Area below which you want to create the device.
  • Sector: Sector below which you want to create the device. If you want to create the devices below sub-sectors, then add the corresponding sector name below the sub-sector column.
  • IP-Address: IP address of the device for communication.
  • Gateway Communication: (Applicable only for PAC1600 device family, PAC3100, PAC2200, PAC3200, PAC4200, and SEM3) You can specify either of the following:
    • TRUE: Activate gateway communication if the device is connected using Modbus RTU via gateway.
    • FALSE: Deactivate gateway communication.
  • Unit Address: (Applicable for PAC1600 device family and PAC3100) Denotes Modbus address in the Modbus RTU subnet. When using a PAC4200 as the gateway, configure the Unit Address on the device. Default value is 1.
  • Port Number: (Applicable only for PAC1600 device family, PAC3100, PAC2200, PAC2200CLP, PAC3200, and PAC4200) Port number of the device. Default value is 502.
