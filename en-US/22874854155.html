(Option 2) Apply Alarm Table to SPC Points

This procedure is part of the workflow for customizing SPC alarm tables.

 

Prerequisites

  • You have configured a customized SPC alarm table.

 

Apply the Modified Alarm Table to Individual SPC Points

Use this method, for example, to modify the alarm table only for a particular door, rather than for all doors.

  1. Select the SPC point for which you want to use the modified alarm table. For example Project > Field Network > SPC Network > SPC1 > Detection > Areas > […] > Door 1.
    NOTE: You can also use CTRL or SHIFT to make a multiple selection, for example of two or more doors to which you want to apply the same modified alarm table.
  2. In the Object Configurator tab, open the Properties expander.
  3. In the list on the left, select the Property of this object to which you want to associate the modified alarm table. This must be one for which the FS column is checked. For example, Status.Status.
  4. The Details, Alarm Configuration and Command Configuration expanders on the right update to show the settings of the selected property.
  5. In the Alarm Configuration expander:
    a. Set the Valid field so it displays a gray (not blue) checkmark.
    b. Set the Alarm Configuration field to Field system.
    c. From the Alarm table reference drop-down list, select the customized SPC alarm table that you want to use for this object.
  6. Click Save .
  1. After you stop and restart the SPC driver, the modified alarm table will apply to the selected SPC point.
