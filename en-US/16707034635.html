Easy Buttons Tab

Easy buttons are graphical short cuts that represent different types of Emergency Events for which you can quickly initiate incidents and launch notifications.

Each easy button consists of an easily configurable icon to which you can link an incident template or a notification template using the Graphics.

In Engineering mode, the Easy Buttons tab displays when you select:

  • Applications > Graphics > Easy Buttons
  • Applications > Notification

It allows you to add an incident template or a notification template as an easy button by dragging the incident template or a notification template on the easy buttons editor.

Each easy button symbol represents an easy button.

Once you drag-and-drop the incident template or a notification template, you can replace the existing easy button symbol with a new symbol.

If the folder symbol is deleted, you cannot see the Easy Buttons tab. In that case, you must first restore the Easy Buttons tab (see Restoring the Easy Buttons Tab in the Easy Buttons).

Additionally you can customize the Easy Button background with a required image (see Customizing the Background for the Easy Button in the Easy Buttons).
