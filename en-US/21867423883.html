Additional Building X Connector Procedures

Complete the following procedures as needed:

Accessing Information in the Flex Client

  1. In SMC, you have created a Website with a Web Services Application (WSI) and a Flex Client Application.
  2. The Agent Status for the Building X Connector displays Running, Connected.
  1. In the SMC tree, select Projects, then select the project with the Building X Connector from the list, and then select the Building X Connector tab.
  2. In the Flex Client Tunnel expander, click the drop-down arrow and select the desired URL.
  3. Add text to the Name and Description fields that will be displayed within Building X.
  4. Click Save.
  1. The Current URL is updated to the selected URL. Desigo CC sends the URL to Building X, where users can access the Flex Client.

Disabling Building X Connector

You can stop communication between Desigo CC and Building X by doing the following:

  1. In SMC, select Projects, then select the project with the Building X Connector from the list, and then click the Building X Connector tab.
  2. From the Toolbar, click Stop Agent .
  1. In the Communication Settings expander, the Agent Status displays Stopped.

Reset the Connection to Building X

In some cases, you may wish to reset the configured connection to Building X. Follow these steps:

  1. In SMC, click on the project with the Building X Connector from the list, and then click the Building X Connector tab.
  2. First, stop the agent. On the Building X Connector tab. click Stop Agent on the toolbar.
  3. Next, stop the project. Click on your project within SMC and click Stop Project. The reset button becomes available once both the agent and project are stopped. Click the icon "Reset the connection to Building X." A confirmation warning will appear.
  4. If you are sure you want to continue, select “Yes.” The Desigo CC connection to the Building X device has now been reset. If you have already established a connection with Building X, you may want to log on there to determine any additional steps.
