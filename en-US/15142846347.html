Emergency (min)

  • Set Emergency
    The value you set indicates the interval of time (in minutes) during which the energy manager is considered in emergency and all the lights are turned on at the maximum dimming (100%). Default is 0 (60 minutes).

Property

Commands

Emergency (min)

  • Set Emergency
    The value you set indicates the interval of time (in minutes) during which an area is considered in emergency and all its lights are turned on at the maximum dimming (100%). Default is 0 (60 minutes).
