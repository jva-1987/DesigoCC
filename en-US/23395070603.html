(Optional) Setup Secure Communication for a S7 PLUS Device

  1. The device certificates are available in the Personal store and CA certificates are available in the Trusted Root Certification Authorities store of the local machine in Microsoft Management Console (MMC).
  1. Select Project > Field Networks > [S7 PLUS Network].
  2. Select the S7 Device for which you want to establish a secured connection with Desigo CC .
  3. Select the Allow Secured Communication check box.
  4. The Certificate Information expander displays with the CA Certificate and Device Certificate fields.
  5. Click the Browse button next to the CA Certificate field and select the CA certificate from the Select Certificate dialog box.
  6. Click the Browse button next to the Device Certificate field and select the device certificate from the Select Certificate dialog box.
  1. The communication of the S7 PLUS device and Desigo CC is secured.
