Create the S7 Network and associate it with the Standard S7 Driver

  1. In System Browser, select Management View.
  2. Select Project > Field Networks.
  3. Object Configurator displays.
  4. Click New Object and select New S7 Network.
  5. The Create New Object dialog box displays with Child type pre-defined to New S7 Network.
  6. Do the following:
    a. Enter a name and description.
    b. Click OK.
  7. The New S7 Network node appears in the System Browser.
  8. Select the New S7 Network and thereafter select the S7 tab.
  9. Open the Device Information expander.
  10. From the Monitored by driver drop-down list, select the Standard S7 driver you want associated with the network.
  11. Click Save .
  1. The driver is associated with the network. If you need to modify the driver after the association is saved, you must select the empty space in the Monitored by driver drop-down list and click Save . This disassociates the driver with the network. However, if there are devices present below the network, then you cannot disassociate the driver with the network.

NOTE:
You can delete a S7 network by selecting the network and then clicking Delete . However, you cannot delete the network while an import is in progress.
