Import Data

Selected data is no longer re-imported once imported. Only data that is not in the database is imported, for example, for a change to the data range.

  1. Click Import.
  2. A confirmation message is displayed.
  3. Click Yes.
  4. The Import Overview dialog box displays.
  5. Click OK to close the dialog box.
