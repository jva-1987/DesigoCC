Find Graphic Element in Desigo Insight Based on the Animation Number (AN)

  1. In System Browser, select Application View.
  2. Select Applications > Graphics.
  3. Double-click the graphic to open it in the Graphics Editor.
  4. Click the View tab and then select the Element structure check box.
  5. In the Element structure view, click Show all elements.
  6. All elements are displayed to include the ANs.
  7. Click an element to display it to the right of the graphic.
  8. The AN is displayed in the Image Properties view, in the General expander under Description.
  9. Open Desigo Insight.
  10. Open the Go to Object tool in Graphics Builder under Tools.
  11. Enter the AN in Object AN.
