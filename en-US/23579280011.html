Revert to Saved Search Filters

The Revert to Saved Search Filter function enables you to obtain the data provided by a previously defined search filter that is now being overwritten by a newly applied filter.
For example, assume you apply a result filter on the log view to display all records of type Activity.
The filter is moved to a Search filter in order to save it. You have now applied a new result filter to obtain data having Source Description as Users and have moved this to a search filter.
So, the existing search filter is now replaced with the following new filter ('Record Type' = "Activity" AND 'Source Description' = "USERS').
However, you want to retrieve the previously saved search filter ('Record Type' = "Activity").

  1. The log data displays.
  1. Apply a result filter on any column. For example, Record Type = Activity.
  2. The applied filter displays in the Filters area below the log view and the log view refreshes to display all records with Record Type as Activity.
  3. Move the result filter to a search filter in order to save the filter condition.
  4. The result filter is moved to a search filter and displays in the Filters area. Save is enabled.
  5. Save the information as a log view definition.
  6. The saved log view definition is reloaded and the data matching the filter expression displays.
  7. Apply a new result filter on any column. For example, Source Description is = Users.
  8. The applied filter displays in the Filters area below the log view and the log view refreshes to display all records with Source Description as Users.
  9. Move the result filter to a search filter.
  10. The search filter is now modified and displays as follows in the Filters area. ('Record Type' = "Activity" AND 'Source Description' = "USERS').
    Revert to Saved Search Filters is now activated.
  11. Click Revert to Saved Search Filters .
  1. The view refreshes and displays the data according to the previously defined search filter ('Record Type' = "Activity"). The filter condition is also updated in the Filters area.
