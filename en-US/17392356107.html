Configuring Scripts for Events Reactions

Scenario

You want to configure a Desigo CC script to execute macros in response to events status changes.
For example, you want an emergency light to turn on, flash, or turn off depending on whether alarms are present, ackowledged, or cleared. For background information, see the reference section.

Prerequisites

  • The Scripts and Events Reactions Scripts extensions are installed and included in the active project.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
  • You created macros for executing the commands you plan to use in your script. For instructions, see Macros.
  • You created the required site views (logical, physical, or user-defined). For instructions, see Views.

Licensing

This extension is covered by license. Check that the required license is available; otherwise, you may not be able to configure and work with this extension.
