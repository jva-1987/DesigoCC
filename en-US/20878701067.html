Creating Stations in System Manager

In Desigo CC, the various stations that make up a deployment are represented as objects within System Manager. Specifically:

Station type

Represented by System Manager object...

Notes

How to create

Desigo CC server station

Project > Management System > Servers > Main Server

There is only one Main Server object in a project.

Object automatically provided in System Manager

Desigo CC anonymous Flex clients

Project > Management System > Clients > Default Web Client

There is only one Default Web Client object. Its settings will apply to all anonymous web-based stations.

Desigo CC FEP stations

Project > Management System > FEPs > [fep]

 

You must manually create an object to represent each [fep], [installed client], and [identified flex client] station included in the deployment.

Desigo CC installed client stations

Project > Management System > Clients > [installed client]

 

Desigo CC identified Flex clients

Project > Management System > Clients > [identified flex client]

 

Desigo CC web server station

Project > Management System > Clients > [web server]

This is an installed client station that is operating as a remote IIS web server (for example to host web applications such as Flex Client or Advanced Reports).

You can have only one [web server] station in a project.

The [web server] station object, if required, must be manually created.

Through these objects, you can apply configurations to the stations, such as client profiles, control station, or closed mode. You can also enable or disable logon from a particular station.

Prerequisites

  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Create a Station Object

  1. Select one of the following:
    • Project > Management System > Clients to create an [installed client], [web server], or [identified flex client] station.
    • Project > Management System > FEPs to create a [fep] station.
  2. In the Object Configurator tab, click New and select New Station or New FEP.
  3. In the New Object dialog box, enter a name and description, and click OK.
  4. The new station object is added to System Browser and automatically selected.
  5. Select the Management tab.
  6. Specify the Host name in one of the following ways:
    • Manually enter the computer host name.
    • Search for a computer on the network: Click Browse, in the Browse for computer network dialog box, select a host name, and click OK.
    • Click Current Station to obtain the host name of the current computer.
  7. In the Type field:
    • For an installed client, select Client.
    • For an installed client station that hosts Desigo CC web applications, select Web Server.
      This option is not available if there is already another [web server] station object.
    • For an identified Flex client, select Web Client.
    • For an FEP station, FEP will be set automatically.
  8. Click Save.

Delete a Station Object

  1. In System Browser, select Management View.
  2. Select one of the following:
    • Project > Management System > Clients > [installed client] or [web server] or [identified flex client]
    • Project > Management System > FEPs > [fep]
  3. The System Management tab displays.
  4. Click Delete .
  5. A confirmation message box displays.
  6. Click Yes.
  1. The selected object is removed from System Browser.

NOTE: You cannot delete the Main Server and Default Web Client objects.
