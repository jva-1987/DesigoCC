[Example] Automatically Setting an Organization Mode as the Output of a Schedule

Scenario: You want to configure a repeating weekly schedule that automatically sets the Occupancy Status organization mode to Occupied from 8 am to 5 pm on weekdays, and to Unoccupied at all other times.

  1. In System Browser, select Application View.
  2. Select Applications > Schedules.
  3. The Schedules tab displays.
  4. Click New .
  5. Select New Management Station Schedule. This sets the new schedule to be stored under the Workstation Schedules in System Browser.
  6. In the Schedules workspace, select the Outputs tab.
  7. In System Browser, select the Manual navigation check box.
  8. Select Management View.
  9. Select Project > System Settings > Organization Modes.
  10. From the Organization Modes folder, drag Occupancy Status into the Outputs area.
  11. Select the Schedule Entries tab and, in the expanders under Output values, set the Organization Mode values that correspond to Inactive and Active as follows:
    Active=Occupied and Inactive=Unoccupied.
  12. Now you can configure a weekly schedule where you specify, for each day of the week, what times of day correspond to Inactive, and what times of day correspond to Active. Initially, each day of the week is set entirely to the Inactive value (Unoccupied).
  13. In the weekly calendar view on the right of the Schedule Entries tab, click inside the Monday column. Initially, the entire day is set to Inactive.
  14. Right-click in the Monday column and select Add Weekly Entry. Drag the edges of the entry to adjust its start time to 8am its end time to 5pm.
  15. In the Schedule Entries tab, select a switch to Active alongside the start time (8:00:00 AM) and select a switch back to Inactive alongside the end time (5:00:00 PM). In this way, the range from 8 AM to 5 PM is set as Active.
  16. Repeat steps 11-13 above for the remaining days of the week: Tuesday, Wednesday, Thursday, and Friday.
  17. The schedule that you create in this way will repeat on a weekly basis. For example, the time ranges you set for the Thursday will repeat every Thursday.
  18. Click Save .
  19. The Save Object As dialog box displays.
  20. In the Name field, enter Facility occupied 8-17 on weekdays.
  21. Click OK.
  1. The new management station schedule is added under Workstation Schedules in System Browser. On each day of the week, it will automatically set the Occupancy Status organization mode based on the Active/Inactive times scheduled for that day.
