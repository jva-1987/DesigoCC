Set Macro Rights

To allow the macro execution in the management station and Flex Client, users groups must be properly configured with appropriate scope. For background information, see User Group Administration.

 

  1. System Manager is in Engineering mode.
  2. System Browser is in Management View.
  1. Open the Security application:
    • Select Project > System Settings > Security.
  2. The Security tab displays.
  3. Select the user group:
    • To properly configured scope, in the Groups list, select the appropriate user group. For example, OperatorsGroup.
  4. Assign scope:
    • To assign a scope for the selected user group, from System Browser, drag the appropriate scope (for example, OperatorsRoom) to the Scope field in the Scope Rights expander.
  5. Set scope commanding:
    • To allow the visibility and the correct execution of macros commands, in the Command Groups column, select the Sta. (Standard) check box.
  6. Click Save .
