Apply a CS11 Customized Alarm Table

(Option 1) Apply the Modified Alarm Table to a Customized CS11 Object Model

Use this method to apply the modified alarm table to an object type. For example, the digital zones.

  1. In the Headquarter CS11 device library, select the CS11 object model to customize. For example:
    • […] > Libraries > L1 Headquarter > Fire > Device > CS11 > Object Model > Digital Zone
  2. In the Models & Functions tab, click Customize .
    NOTE: If the icon is dimmed it means that the selected object model is already customized. Skip to step 4.
  3. A confirmation message box displays.
  4. Click OK.
  5. The customized object model is added to the customized library.
  6. Select the customized object model. For example [...] > Libraries > L4-Project > Fire > Device > CS11 > Object Model > Digital Zone.
  7. In the Models & Functions tab, open the Properties expander.
  8. In the list on the left, select the Property to which you want to associate the modified alarm table. This must be one for which the FS column is checked.
    For example, Alarm.OffNormal.
  9. The Details, Alarm Configuration and other expanders on the right update to show the settings of the selected property.
  10. In the Alarm Configuration expander, from the Alarm table reference drop-down list, select the customized alarm table that you want to use for this customized object model.
  11. Click Save .
  12. Stop and then re-start the BACnet driver to enable the new table.
  1. The modified alarm table will apply to all the CS11 points that are based on the selected object model.

 

 

(Option 2) Apply the Modified Alarm Table to Individual CS11 Points

Use this method to modify the alarm table of a specific digital zone, rather than for all digital zones.

  1. Select the CS11 point for which you want to use the modified alarm table. For example Project > Field Network > [Fire BACnet] > [CS11] > [CC11] > [Logical] > […] > [digital zone].
    NOTE: You can also use CTRL or SHIFT to make a multiple selection, for example of two or more digital zones to which you want to apply the same modified alarm table.
  2. In the Object Configurator tab, open the Properties expander.
  3. In the list on the left, select the Property to which you want to associate the modified alarm table. This must be one corresponding to the selected object, for which the FS column is checked. For example, Alarm.OffNormal.
  4. The Details, Alarm Configuration and other expanders on the right update to show the settings of the selected property.
  5. In the Alarm Configuration expander:
    a. Set the Valid field so it displays a gray (not blue) checkmark.
    b. Leave the Alarm Configuration field to Field system.
    c. From the Alarm table reference drop-down list, select the customized alarm table that you want to use for this object.
  6. Click Save .
  7. Stop and then re-start the BACnet driver to enable the new table.
  1. The modified alarm table will apply to the selected CS11 points.
