Reduce Disk Space Volume

Perform the following steps to reduce disk space volume:

  1. From the Windows Start menu, select Control Panel > System.
  2. In the window that opens, select System protection.
  3. In the System Properties window that displays, select the drive, and click Configure.
  4. In the window that displays, do the following:
    • Select Turn off system Protection.
    • Position the Max Usage slider to minimum.
    • Click Delete.
  1. On successful deletion, click OK and then click OK to close the System Properties window.
