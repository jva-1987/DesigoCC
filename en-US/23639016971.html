Troubleshooting IIS

Problem: IIS is corrupted.

Solution: Re-install IIS by performing the following steps:

  1. From Control Panel > Programs and Features, click Turn Windows feature on or off.
  2. In the window that displays do the following:
    • Deselect Internet Information Services to uninstall IIS completely.
    • Deselect Windows Process Activation Service.
    • Click OK.
  3. Restart the computer.
  4. Make sure %windir%\system32\inetsrv\ is now clean.
  5. Reinstall IIS.
  6. Reinstall Windows Process Activation Service.
