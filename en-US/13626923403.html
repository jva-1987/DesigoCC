Setting up the Flex Client

Scenario
You want tooperate Desigo CC using the Flex Client. For this purpose, you have created a web communication channel that requires a website and a Web Service Interface communication project.
For background information, see the reference section.

Prerequisites

  • Ensure that Flex Client runs on the latest versions of Chrome, Firefox and Safari. It is tested on Microsoft Windows, MacOS, iOS and Android operating systems.
  • When using Firefox Web browser, hardware acceleration is disabled for proper visualization of graphics.
  • Flex Client extension is installed.
  • WSI extension is installed and added to the project and the project is started.
  • At least one website is created and available under Websites in the SMC tree.
  • (Applicable only for Server and WSI app on the same system) You can leave the WSI settings as local.
  • (Applicable for remote IIS Server hosted on Client/FEP system) You must configure the WSI instance that you link to the project on Server as secured by configuring the Communication as Secured with a host certificate.
    • Ensure that the root of the host certificate is imported in the Trusted Root Certification Authorities (TCRA) of Windows Certificate Store.
