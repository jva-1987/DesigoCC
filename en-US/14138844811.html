Exporting the NK823x Configuration for Local Download

Scenario

You want to export the NK823x configuration and create a ZIP file that you can use with the NK Config Tool for a local download to the NK823x.

  1. In System Browser, select Management View.
  2. Select Project > Field Networks > [BACnet network] > [NK823x].
  3. In the Operation tab, check the Export configuration destination.
    • If necessary, enter a new path and click Apply.
  4. In Export configuration status, click Export
  1. In a few seconds, the ZIP file is created in the destination folder. The file name is NK823X_<IP_Adress>.ZIP.
