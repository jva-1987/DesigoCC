Troubleshooting SIE-VA

The following describes how to handle common import error conditions.

Message

Situation

Do the following...

“The file does not contain servers data! There are warnings or errors, look at the log file” error loading the CSV import file

The selected CSV import file cannot be loaded by the Desigo CC importer.

Verify that the CSV file uses a comma (,) character as separator. Any other character will result in an invalid file format.

No specific error message displays.

The CSV import file is completed successfully, but no point instances are created under the OPC network.

Re-import the same file. The OPC instances will be created under the OPC network.

The configuration file selected for the import is invalid.

In the Import tab, click Analysis Log and check the pre-import log.

Try again using another configuration file.
