Check Mobile Device internet/Intranet Connection

Before you start the app and attempt to connect to a management platform, you should first check that the mobile device has the necessary network connection (internet or intranet) for communicating with the server.

The Desigo CC management platform can be configured to communicate with the mobile app:

  • Over an intranet (WLAN). In this case, check that your mobile device is connected via Wi-Fi to the correct Wireless LAN network on the site. (See the system administrator for more information about this, including the SSID, password, and any other advanced Wi-Fi configuration settings). The mobile app will only be able to receive data from the Desigo CC server while you remain within range of the site WLAN signal.
    NOTE: With an intranet connection, the mobile device will not be able to access the internet. Therefore, other apps on the device that use the internet, including email and messaging, will not work.
  • Over the internet (using Wi-Fi or 3G/4G). In this case, check that your mobile device has an internet network connection, either via Wi-Fi or using its SIM card (3G/4G mobile data connection). The mobile app will be able to receive data from the Desigo CC server as long as the mobile device is connected to the internet.
