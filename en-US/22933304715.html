(Optional) Stop Hard Drive Monitoring of a Station

Perform this procedure if you want the system to cease monitoring the hard drive of a server, installed client, or FEP station.

  1. Select the hard drive you want to stop monitoring, at one of the following paths:
    • Project > Management System > Servers > Main Server > [hard disk drive]
    • Project > Management System > Clients > [Client station] > [hard disk drive]
    • Project > Management System > FEPs > [FEP station] > [hard disk drive]
  2. The System Management tab displays.
  3. Click Delete .
  4. Click Yes to confirm.
  1. The hard drive object is removed from System Browser. The system will cease monitoring the integrity of the station’s hard drive. In the case of a server station, any disk capacity monitoring will also cease.
