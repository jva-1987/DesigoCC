Enter the User Settings

  1. You are in the User Management Wizard step 4 – User Creation.
  1. Enter the User name. The name must be unique.
  2. In the Full name field, enter an expanded user description. This text displays as the user name in the Summary bar.
  3. In the Comment field, enter an expanded description, for example, office number.
  4. (Optional) Select a different Language or a different Client profile and a Flex client profile to associate them to the user.
  5. If you selected Desigo CC User in the previous step, enter the user Password and re-enter it to confirm.
  6. If you selected Windows User in the previous step, click Browse and then choose the user you configured in the operating system.
