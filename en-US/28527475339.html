Importing Object Models

  1. The JSON file with details of the object model to be imported is present.
  1. Navigate and expand the Import Object Models expander.
  2. In the Library Name field, drag and drop the library to which you want to add the object model.
    NOTE: If the object model is dragged and dropped from the library for which the user does not have access, an error message will be displayed while importing the JSON file.
  3. Browse and select the JSON file for the object model.
  4. The selected file displays in the Select File field.
  5. Click Import.
  1. The object models defined in the JSON file are imported to the selected library.
