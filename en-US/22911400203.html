Import the Data Model

  1. In System Browser, select the newly created OPC UA object model library block.
  2. Select the Import tab.
  3. Click Browse.
  4. In the Open dialog box, navigate to and select the configuration (JSON) file that you prepared.
  5. Click Open.
  6. The complete path of the selected file appears in the field. The Source Items preview displays all available items.
  7. Transfer the DPTs to import from the Source Items list to the Items to Import list .
  8. Click Import.
  9. A confirmation message displays.
  10. Click Yes.
  11. The Import tab displays the Import in progress page. When the import completes successfully, the Import dialog box displays its outcome.
  12. Click OK to close the dialog box.
