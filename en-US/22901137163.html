Clone the Default Multimedia Collection Text Group at your Customization Level

A preconfigured text group defines what buttons for playing media files are available in the default multimedia collection. To be able to modify this text group, you must first clone it at your allowed customization level.

Skip this step if the default multimedia collection text group is already cloned at your allowed customization level.

  1. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select the default headquarters text group at Project > System Settings > Libraries > L1-Headquarter > Global > Multimedia Collection > Texts > Multimedia Collection.
  3. In the Text Group Editor tab, click Customize .
  4. Click OK.
  1. The default text group collection is cloned and selected under the allowed library customization level (for example, L4-Project > Global > Multimedia Collection > Texts > Multimedia Collection). Its details display in the Text Group Editor tab.
