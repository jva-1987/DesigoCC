Web Service Interface Procedures

This section provides additional procedures related to web service interface.

Verify Application Rights for Web Service Interface

Trends and Schedules application rights define permissions to perform create, modify and delete operation on Trends and Schedules services provided by the Web Service Interface. You must complete the following procedure to enable access through Web Service end points.

  1. The Web Service extension is added to the project.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Security.
  3. In the Security tab, select the user group that you want to authorize to work with the Web Service Interface.
  4. In the Application Rights expander, perform the following steps to select the Configure rights that you want to allow for Trends and Schedules:
    • Open the Trends expander, and select the Configure check box.
    • Open the Schedules expander, and select the Configure check box.
  5. Click Save.
  1. The selected Configure rights for Trends and Schedules are enabled for this user group. The user from the selected user group can perform operations on Trends and Schedules services of the Web Service Interface.
