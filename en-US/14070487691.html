Macro Viewer

This section discusses the tasks you can perform from the Macro Viewer. For background information, see the reference section.

 

Start a Command Action from the Macro Viewer

  1. In the Macro Viewer, click the symbols representing the object (macro or pseudo-point) that you want to command.
    NOTE: You can select one symbol at the time.
  2. Click Execute Command .
  3. The symbol changes to indicate the command in progress.
  4. As long as the command is in execution, the icon changes to Stop .
    For time-consuming macros, you can click it to stop/cancel the command execution.

 

Show Properties and Commands of an Object from the Macro Viewer

  1. In the Macro Viewer, double-click a symbol.
  2. The corresponding object is selected in System Browser, and properties and commands display in the Contextual pane.
  3. Select the Extended Operation tab to access the extended set of control commands.
