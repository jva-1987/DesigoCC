Create Room Group

The Room group function merges individual rooms into a group. The groups can be derived from existing connections (for example an air handling unit) specific to the application or can form a purely static group.

Prerequisites

  1. Select Application View.
  2. Select Applications > BIM Graphic > [graphics page].
  3. Select the BIM Viewer tab.
  4. Click Show grouping .
  5. Click Edit .
  6. Displays the BIM room hierarchy.
  7. The room grouping window displays in Edit mode. The window is empty for first-time startup.

Create a backup file using the Additional Functions before changing the configuration.

 

Create a Static Group for a Few Rooms

Scenario: All the rooms occupied by company form a group.

  1. Click Add in the room grouping window.
  2. The Edit Group dialog box opens.
  3. Enter a description in the Group name field.
  4. Click Modify and select a color.
  5. Click OK two times.
  6. A new group is created in the room grouping window.
  7. Select the floor (1) and room (2) on the floor plan.
  8. The room is highlighted in the BIM room hierarchy.
  9. Drag the room from the BIM room hierarchy (3) to the new group (4).
  10. Repeat the process for each room belonging to this group.
  11. Click Save .
  12. The rooms are assigned to the new group.
  13. In the room grouping window, select the new group.
  1. The floor as well as the selected rooms are highlighted in color.

 

Create Static Group for an Entire Floor

Scenario: Company B occupies most of the rooms on a floor. These rooms form a group.

  1. Click Add in the room grouping window.
  2. The Edit Group dialog box opens.
  3. Enter a description in the Group name field.
  4. Click Modify and select a color.
  5. Click OK.
  6. A new group is created in the room grouping window.
  7. Select the floor (1).
  8. The floor is highlighted in the BIM room hierarchy.
  9. Drag the floor from the BIM room hierarchy (2) to the new group (3).
  10. Click Save .
  11. All rooms are assigned to the new group.
  12. In the room grouping window, select the new group.
  13. Click Edit (1) and highlight the rooms (2) that do not belong to this group.
  14. Click Remove (3) and then OK.
  15. In the room grouping window, select the new group.
  1. The floor is depicted as a group. The removed rooms are not highlighted in color.

 

Create a Group for a Specific Application

Scenario: All rooms connected to air handling unit (AHU) A form a group. AHU A is designed for basement and ground floor west.

  1. In System Browser, select Logical View.
  2. Select Logical > [Network name] > [Building name] > [Discipline].
  3. Drag the discipline from System Browser to the room grouping window.
  4. A new group is created in the room grouping window.
  5. The name corresponds to the selected discipline.
  6. Rooms are not yet assigned.
  7. Click Modify and select a color.
  8. Click OK two times.
  9. Select the floor (1) and room (2) on the floor plan.
  10. The room is highlighted in the BIM room hierarchy.
  11. Drag the room from the BIM room hierarchy (3) to the new group (4).
  12. Repeat the process for each room belonging to this group.
  13. Repeat the process for the second floor.
  14. Click Save .
  15. The rooms are assigned to the new group.
  16. The Association check box is selected in the grouping window.
  17. In the room grouping window, select the new group.
  1. The floor as well as the selected rooms are highlighted in color.
