Building X Connector

Scenario

You want to establish a Building X connection and remotely monitor the status of your site using Building X applications such as Operations Manager. For background information on the Building X Connector, see the reference section.

Prerequisites

  • In Building X, a Building X organization and site has been established.
  • The Building X Connector extension is installed on the management station.
  • A project that includes the Building X Connector extension is created and started in SMC.
