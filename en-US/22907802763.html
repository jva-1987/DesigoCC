Add Network and Adapter Information

In Step 2 of 5, specify the following settings:

  1. In the network information section, enter the Network name and Network description.
    NOTE: To change the language selection, click the flag icon in the Network description field, select the desired language, enter the desired text, and then click OK.
  2. In the adapter information section, do the following:
    a. Enter the Adapter name and Adapter description.
    NOTE: To change the language selection, click the flag icon in the Network description field, select the desired language, enter the desired text, and then click OK.
    b. Enter the Adapter URL, which must include the localhost and same port number used for installing the adapter with the following format: http://127.0.0.1:<port number>.
    NOTE 1: It is suggested to use a free port number different from the default one (8080).
    NOTE 2: If you want to create the adapter later, select I’ll create the adapter later.
  3. Click Next page .
  1. The wizard displays Step 3 of 5.
