View the Report as PDF or XLSX

  1. In the generated report, click Export report that displays on top of the report output.
  2. The Export Report dialog box displays.
  3. In the Export Format drop-down list, select the required format for the report.
    NOTE 1: You can also print the report output.
    NOTE 2: The reports can be exported only in PDF and XLSX formats. For EN 50160 report, XLSX format is not supported.
