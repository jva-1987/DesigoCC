Enable the WebDAV Authoring Rules for a Website

  1. You are on the target machine and the Internet Information Services (IIS) Manager is open.
  1. Select Sites > [website] for which you want to enable the WebDAV Authoring Rules and double-click WebDAV Authoring Rules.
  3. Click Enable Web DAV from the Actions pane.
  5. Set the Web DAV Settings from the Actions pane and set the Properties behavior to True.
