Fibrolaser III Compatibility and System Limits

Compatibility

  • Modbus: OTS3 and LHD3 Series controllers
  • OPC: OTS3 and LHD3 Series controllers through S4S “FibroLaserIII OPC Server” v1.1.1
  • Desigo CC V3.0 or higher

Size Limits

Since no specific limits are identified for this integration, refer to Desigo CC system limits defined in the System Description document (A6V10415500).

Other Limitations in Modbus Integration

  • Due to a limitation in the Desigo CC Modbus integration, the event commands (Ack and Reset) that are available on the panel object can be replicated on the panel node only. They cannot be replicated in the Contextual pane on zone elements.
  • When a zone event occurs, the Ack and Reset commands become available in Event List with the following limitations for the current version:
    • The Ack and Reset commands are both available at the same time. They remain available also when the acknowledge command has been successfully executed.
    • The event blinks constantly, also after the Ack command has been successfully executed.
    • Symbols in the graphic never blink. This means that the operator cannot determine acked and unacked conditions from the graphics.
    • In order for the Ack and Reset commands to work properly in Event List, a specific configuration step in the management station is required. For instructions, see Configuring Fibrolaser III Event Commands Macros.
  • Modbus addresses for zones are not defined in Desigo CC import rules. To have the proper Modbus addresses for zones, the driver must be stopped and restarted after any import/re-import of the configuration.
  • The Fibrolaser integration in Desigo CC via Modbus does not support temperature point objects because they are not available in the Modbus Interface. These types of objects are instead available in the OPC integration. For more details, see Fibrolaser III Events in Modbus Integration.

Other Limitations in OPC Integration

  • In the current version, Desigo CC supports only one OPC driver and one OPC network.
  • The maximum number of OPC servers per OPC network is 20.
