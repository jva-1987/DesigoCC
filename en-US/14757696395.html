Configuring STT20 on FCnet

Scenario

You want to configure Desigo CC for connecting to an STT20 fire system on a BACnet network.
For background information, see the reference section.

Prerequisites

  • The CMSI STT20 extension module is installed and included in the active project.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
  • The communication network planning is complete and the following BACnet information is available:
    • BACnet addresses: Device IDs.
    • UDP port number.
  • The SiB-X metafile exported from the STT20 tool is available.
