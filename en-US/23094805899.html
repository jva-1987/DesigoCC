Historizing Path for Meter Movements

  1. System Manager is in Engineering mode.
  2. You have created a managed meter and it displays in the Managed Meters folder in the System Browser.
  1. In System Browser, select Application View.
  2. Select Applications > Energy and Power > Managed Meters.
  3. From the Managed Meters folder, select a managed meter.
  4. The managed meter details display.
  5. In System Browser, select Management View.
  6. Select Project > System Settings > Views.
  7. Create a new user-defined view (for example, Energy View) and add the nodes (for example, Campus, Building, and Floor) below this view. Associate the managed meter to the Floor node. This indicates that the managed meter is associated with the Floor.
  8. Click Save .
  9. The information on the new path is historized as per the default date. The historized entry displays in the Historized Meter Path dialog box.
  10. Navigate to the aggregator node in the Energy View and remove the managed meter. The managed meter is removed from this location.
  11. Click Save .
  12. Select the managed meter and click Apply Historization .
  13. In the Enter Valid From dialog box, specify the date and time when the meter was removed from the view and click OK.
  14. The information is historized. An additional entry with the date and time displays in the Historized Meter Path dialog box and has a strikethrough mark to indicate that the meter is removed from the view.
  15. Repeat Steps 6 through 10 to create another floor (for example, Floor2).
