Updating the Version and Dependencies of a Library

After making some changes to a library, you can update its versioning details or its list of dependencies. The system uses this information to manage the importing and the upgrading of libraries.

 

  1. You are trained and authorized to work with system libraries at the target customization level of this library.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L4-Project or L3-Country or L2-Region > [folder] > […] > [library].
  3. The Library Configurator tab displays the General settings and the List of library blocks for that library.
  4. Modify one or more of the following settings as appropriate:
    • Version
    • Date
    • Notes
    • Languages
      Enter a language and click Add. To remove a language, select it and click Remove. You can add more than one language.
    • Dependencies
      Enter the library name dependency (or select it from the drop-down list) and click Add. Alternatively, drag-and-drop a dependency library to this field. To remove a dependency, select it and click Remove. You can add more than one dependency.
  5. Click Save .
