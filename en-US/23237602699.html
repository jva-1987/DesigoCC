(Optional) Load a Custom Task File

The task files must be located in the GMSMainProject > BIN folder in order for the Consistency Checker tool to load them.

  1. You want to load a customized task file (.DLL) into the Consistency Checker tool and it is not loading when you click Refresh . You can manually add a .DLL file to the tool, by doing the following:
  1. From the File menu, select Consistency Checker.
  2. The Consistency Checker tool displays.
  3. Right-click anywhere in the Tasks Tab > Tasks section, and select Load.
  4. The Open dialog box displays.
  5. Select [Installation Drive]:\[Installation Folder]\GMSMainProject\Bin directory, and select the required task file (.DLL). For example: [YourFileName].dll
  6. The task file is loaded into the Consistency Checker.
  7. If the Task list is empty or has not updated with the new tasks, click Refresh and the task list is updated to display all loaded tasks.
  8. To unload a task file (.DLL) so that those tasks no longer run or display.
    • On the Task tab, right-click anywhere in the Task section, and then select Unload.
