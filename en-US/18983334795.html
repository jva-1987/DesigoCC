Configuring FIAS Adapter Using the Network Wizard

Scenario

To simplify configuration steps, the Network Setup Wizard extension is installed by default, and you want to use this wizard to easily configure a SORIS driver, network, and adapter to integrate hotel management applications that use the FIAS protocol in Desigo CC. For background information, see the reference section.

Prerequisites

  • The FIAS Adapter for PMS Systems extension is installed and included in the active project. The following dependent extensions are also included automatically:
    • Room Booking
    • SORIS Driver
  • The FIAS adapter is running and connected to the hotel management application. For instructions, see Installing and Starting the FIAS Adapter.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
