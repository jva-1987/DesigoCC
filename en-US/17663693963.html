Creating a Bar Graph Symbol

Scenario

You want to create an animated bar-graph Symbol so that you can use it on a graphic or graphic template. The Symbol will support drag-and-drop, so that when an object is dragged onto the drawing canvas the Symbol will appear with the correct object reference.

In this workflow, the following conditions are used to create the Symbol:

  • From -30 to 80 and back, the graph rises from 10% to 100% and fall from 100% to 10%
  • If the value is <=10 the graph color is blue.
  • If the value is >=50 the graph color is red.

For background information, seeCreating Advanced Symbols.
