Install .Net 4.7.2, .Net 6.0.26, and .Net 8.0.1

Confirm that the computer on which you installing Desigo CC has the .Net 4.7.2, .Net 6.0.26, and .Net 8.0.1 compatibility.

You must install .Net 4.7.2, .Net 6.0.26, and .Net 8.0.1 before proceeding with the installation. However, if you have an earlier compatible version of .Net installed, for example, .Net 4.7.1, even though a warning displays, you can still proceed with the installation. However, some Desigo CC features may not work.

NOTE:
Make sure that you have configured IIS before installing .Net 4.7.2, .Net 6.0.26, and .Net 8.0.1. However, if .Net 4.7.2 is already installed on your system, you must configure .Net 4.7.2 ManageEngine in the IIS Manager. Otherwise, you cannot save objects on the Windows App Clients. See Unable to Import/Export Libraries, Save an Object in the Windows App Client.

  1. Do one of the following to install .Net 4.7.2, .Net 6.0.26, and .Net 8.0.1:
    • From the distribution media, select InstallFiles > GMS > Prerequisites > Microsoft .Net > .Net 4.7.2 , .Net 6.0.26 or .Net 8.0.1.
    • Select the InstallFiles > GMS > Prerequisites > Microsoft .Net > .Net 4.7.2 , .Net 6.0.26 or .Net 8.0.1 from your local drive.
      NOTE: Do not install directly from a USB drive as this will dramatically increase the installation time.
  2. Double-click the executable file in the .Net 4.7.2, .Net 6.0.26 or .Net 8.0.1 directory.
  3. Follow the instructions in the Installer Wizard.
  4. Reboot the machine, when prompted.
