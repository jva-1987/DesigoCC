Configuring Windows SNMP Service

The Windows operating system can interact with SNMP by providing data about computer hardware, software status, and statistics. Such information can be used in Desigo CC for monitoring tasks.

NOTE: You can perform the following procedures in many ways. For details, refer to the Windows documentation about installing or configuring the SNMP service.

Prerequisites

  • You are logged in with administrator rights.
  • The computer is connected to the Internet.

Install SNMP Service in Windows

  1. Click Windows Start button, and then click Settings.
  2. In Windows Settings, click App.
  3. In Apps & features, click Optional features.
  4. In Optional features, click Add a feature.
  5. In Add a feature, do the following:
    a. Select Simple Network Management Protocol (SNMP).
    b. Click Install.

Configure and Run the Windows SNMP Service

  1. Click the Windows Start button, and select Windows Administrative Tools > Computer Management.
    • If the User Account Control authorization window displays; click Yes to proceed.
  2. The Computer Management window displays.
  3. Expand the Services and Applications folder and click Services.
  4. The Services list displays.
  5. Locate and right-click the SNMP Service, then select Properties.
  6. The SNMP Service Properties window displays.
  7. Select the Agent tab, and do the following:
    a. Select the Physical check-box.
    b. Click Apply.
  8. Select the Security tab.
  9. In the Accepted community names section, click Add.
  10. In the SNMP Service Configuration dialog box, do the following:
    a. From the Community rights drop-down list, select the appropriate privilege.
    b. In the Community Name field, enter public (lower case).
    c. Click Add.
  11. Select the Accept SNMP packets from any host option.
  12. Click OK.
  13. In the Services list, right-click and Restart the SNMP Service.
    • Make sure that the SNMP Service Status shows Started and the Start-up type is Automatic.
  14. Close the Computer Management window.
