UPS RFC1628 Symbols

When integrating the UPS RFC1628 extension, the following symbols are available in the sample graphic:

 

DYN_2D_RFC1628_Alarming_Header_Horizontal_001_150

  • Symbol internally used as part of the sample graphic.
  • Symbol provides the header of the sample graphic.

 

DYN_2D_RFC1628_Alarming_Node_Horizontal_001_150

  • Default symbol for RFC1628 UPS objects.
  • Symbol used as part of the sample graphic.

 

The blinking effect on symbols is not supported in this integration.
