Extended Import Rules

You must use this function if you want to import more data points than were entered in Application modeling. Using extended import rules, you can import data points as defined under User designation or Object type and that belong to the overall project. Set your own extended import rules or use predefined extended import rules under Headquarters.

  • All I/O data points for large open offices
  • All I/O data points with a user designation
  • All I/O data points

For related procedures and workflows, see the step-by-step section.
