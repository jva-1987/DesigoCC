Configuring the OTM Event Manager

Scenario

The communication between OTM and Desigo CC is managed by a dedicated service, the OTM NORIS Event Manager, which you must install and run on a networked PC that can connect to OTM and to Desigo CC. Typically, you can install it on the Desigo CC server station. For background information, see the reference section.
