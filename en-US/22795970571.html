Update the Physical View to Reflect the Changes in Linked Objects

  1. Select [physical view] from the System Browser drop-down list.
  2. In the tree structure, select the root node of the view.
  3. The view builder workspace displays in the Views tab.
  4. Click Update View .
  5. Click Yes.
  6. When the update is complete, the tree structure is collapsed in the preview area.
  7. Expand the tree structure in the preview area.
  8. Any soft links relevant to new elements linked to the tree structure display in green, while broken links display in red.
  9. To remove broken links, select the soft links in red and click Remove.
