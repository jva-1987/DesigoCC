Delete Reference Plant

Scenario: One or more reference plants are no longer needed to automatically assign the functions.

  1. The conversion tool Desigo Function Checker is opened.
  1. Open the Reference plants expander.
  2. Select the reference plant and right-click Delete objects or Delete all.
  1. The reference plant is deleted from the Reference plants expander.
