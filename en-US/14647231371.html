Deleting CC11 Panels

Remove One or More Imported Panels from the BACnet Network

  1. You configured a BACnet network and imported one or more panels.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > Field Networks > [BACnet network].
  3. Select the Fire System Options tab.
  4. Open the Configured Panels expander.
  5. The list of imported panels displays the description (customer text) of each panel.
  6. In the list of imported panels, do one of the following:
    • Select the panel you want to delete and click Delete. Use multi-select by pressing the CTRL or SHIFT key.
    • Double-click the panel you want to delete to select it in System Browser, then click .
  1. The selected panels are removed from System Browser.
