Align a Web Application with the Server Project

You must then re-align a web application on the remote Web server (IIS) hosted on the Client/FEP station with the modified Server project linked to the web application.

  1. You have modified the Server project parameters (such as CCom port number or the communication mode) linked to a web application.
  1. In the SMC tree, select Websites > [website] > [web application].
  2. Click Edit .
  3. Click Align with Server .
  4. Click Save .
