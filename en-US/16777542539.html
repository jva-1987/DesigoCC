Initiating an Incident from an Operating Procedure Step

  1. An operating procedure template with one or more Notification operating procedure step templates is created and configured.
  2. An event is generated in the Summary bar and the scope of the corresponding operating procedure step matches the scope of the event.
  1. Perform the following steps if the incident is initiated from the Manual or Automatic on first treatment execution mode. When you initiate an incident from the Automatic on creation mode, then the incident template associated with the operating procedure step is initiated. In addition, the notification template associated with the operating procedure step is also launched after the event whose scope matches with the scope of the operating procedure step is generated. Navigate to Step 3 to view the incident or notification templates.
  2. Double-click an event in the Summary bar or in the Event list.
  3. The Assisted Treatment dialog box displays.
  4. (Applicable only if you are initiating the incident from the Automatic on first treatment execution mode) The incident template associated with the operating procedure step is initiated and the notification template associated with the operating procedure step is launched.
  5. To view the initiated incident template and the launched notification template, perform the following steps:
    a. Set System Manager to Operating mode.
    b. In System Browser, select Application View.
    c. Select Applications > Notification > Browse Notifications.
  6. (Applicable only if you are initiating the incident manually) Perform the following steps to initiate an incident manually.
    a. Click the Steps tab.
    b. Select the procedure text to activate the new step notification incident: The associated incident template and message template display in the Default tab.
    c. Click Initiate to initiate an incident template or to launch a notification template under an ad hoc incident: A confirmation message displays.
    d. Click OK. The Browse Incidents and Notifications section displays with details of the initiated incident template or launched notification template.
    e. Select the check box that displays in the Steps tab to complete the operating procedure step.
