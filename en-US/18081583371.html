Integrating Kabona WDC

Scenario

You want to configure Desigo CC to operate on the Kabona WDC system. For background information, see Kabona WDC Integration.

Prerequisites

  • The Kabona WDC extension is installed and included in the active project. The following extensions are also selected automatically:
    • Building Automation Common
    • SORIS Driver
  • The Kabona WDC adapter is running and connected to the Kabona WDC system. For instructions, see Installing and Starting the Kabona WDC Adapter.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Licensing

This extension is covered by license. Check that the required license is available; otherwise, you may not be able to configure and work with this extension.
