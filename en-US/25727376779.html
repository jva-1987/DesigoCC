Advanced Reports Executing for Longer Duration

Problem 1: Advanced report generation process takes longer time than 30 minutes and is corrupted.

Cause: Incorrect parameter values filled in the parameter information section.

Solution: Select correct parameter values so that advanced reports can be generated in less time.
For example: Select the minimum time range for the generation of advanced reports.
