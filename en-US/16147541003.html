Edwards EST3 Symbols

When integrating the Edwards EST3 extension, the following symbols are available in graphics:

DYN_3D_EST3_Panel_None_Horizontal_001

  • Default symbol for non UL EST3 fire panels.
  • Symbol background changes color according to the event category color.

DYN_3D_EST3_Panel_None_Horizontal_002

  • Default symbol for UL EST3 fire panels.
  • Symbol background changes color according to the event category color.

DYN_3D_EST3_Damper_None_Horizontal_001

  • Default symbol for dampers.
  • Symbol background changes color according to the event category color.

DYN_3D_EST3_FireDetector_None_Horizontal_001

  • Default symbol for fire detectors.
  • Symbol background changes color according to the event category color.

DYN_3D_EST3_FireDoor_None_Horizontal_001

  • Default symbol for fire doors.
  • Symbol background changes color according to the event category color.

DYN_3D_EST3_FireZone_None_Horizontal_001

  • Default symbol for fire zones.
  • Symbol background changes color according to the event category color.

DYN_3D_EST3_GasDetector_None_Horizontal_001

  • Default symbol for gas detectors.
  • Symbol background changes color according to the event category color.

DYN_3D_EST3_ManualCallpoint_None_Horizontal_002

  • Default symbol for manual zones.
  • Symbol background changes color according to the event category color.

DYN_3D_EST3_ManualCallpoint_None_Horizontal_001

  • Default symbol for manual detectors.
  • Symbol background changes color according to the event category color.

DYN_3D_EST3_TechnicalInput_None_Horizontal_001

  • Default symbol for technical inputs.
  • Symbol background changes color according to the event category color.

DYN_3D_EST3_TechnicalOutput_None_Horizontal_001

  • Default symbol for technical outputs.
  • Symbol background changes color according to the event category color.

DYN_3D_EST3_TechnicalZone_None_Horizontal_001

  • Default symbol for technical zones.
  • Symbol background changes color according to the event category color.

DYN_3D_EST3_WaterflowDetector_None_Horizontal_001

  • Default symbol for waterflow detectors.
  • Symbol background changes color according to the event category color.

DYN_3D_CMD_EST3_FireDetector_None_Horizontal_001

  • Default command symbol to acknowledge or reset fire detector events.
  • Symbol background changes color according to the event category color.

DYN_3D_CMD_EST3_FireZone_None_Horizontal_001

  • Default command symbol to acknowledge or reset fire zone events.
  • Symbol background changes color according to the event category color.

DYN_3D_CMD_EST3_GasDetector_None_Horizontal_001

  • Default command symbol to acknowledge or reset gas detector events.
  • Symbol background changes color according to the event category color.

DYN_3D_CMD_EST3_ManualCallpoint_None_Horizontal_002

  • Default command symbol to acknowledge or reset manual zone events.
  • Symbol background changes color according to the event category color.

DYN_3D_CMD_EST3_ManualCallpoint_None_Horizontal_001

  • Default command symbol to acknowledge or reset manual detector events.
  • Symbol background changes color according to the event category color.

DYN_3D_CMD_EST3_TechnicalInput_None_Horizontal_001

  • Default command symbol to acknowledge or reset technical input events.
  • Symbol background changes color according to the event category color.

DYN_3D_CMD_EST3_TechnicalZone_None_Horizontal_001

  • Default command symbol to acknowledge or reset technical zone events.
  • Symbol background changes color according to the event category color.

DYN_3D_CMD_EST3_WaterflowDetector_None_Horizontal_001

  • Default command symbol to acknowledge or reset waterflow detector events.
  • Symbol background changes color according to the event category color.

DYN_All_EST3_EventCmd_Central_001

  • Internal symbol to dynamically apply onto the other symbols: displayed on the top right of a symbol it means that the event generated by the related object can be acknowledged or reset.

 

The blinking effect on symbols is not supported in this integration.
