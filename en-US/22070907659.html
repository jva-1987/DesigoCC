Setting a Default Window Layout for Flex Client

The Flex Client desktop app allows you to detach the Events workspace or any additional System workspace into separate windows, and arrange these as required on the screen, or on multiple monitors.

  • You can set up a default window layout that will apply to all Flex client users on the computer.
  • The default layout can optionally be "enforced" so that users are constrained to it, and cannot alter their window arrangement.
  • Otherwise, users can freely rearrange their windows, and revert to the default layout when they select Reset layout.

Set Multi Monitor Application Right

This permission is required to be able to save a default layout. Users with this right can always rearrange their windows, even if there is an enforced default layout saved.

  1. Log on to the Desigo CC thick client with the Default Administrator credentials.
  2. Switch System Manager to Engineering mode, and select System Browser Management View.
  3. Select Project > System Settings > Security.
  4. In the Groups list of the Security tab, select the DefaultAdmins user group.
  5. In the Application Rights expander, select the Multi Monitor permission. This automatically sets the Configuration check box under it.
  6. Click Save .

Save a Default Window Layout

  1. The Flex Client desktop app is installed on a computer.
  1. Start the Flex Client desktop app in the usual way, and log on with the Desigo CC Default Administrator credentials.
    NOTE: To be able to save the layout you must log on as a user with the Multi Monitor application right.
  2. In the horizontal navigation bar, select  to open the Layout panel.
  3. In the Manage windows area, open all the windows that you want to have in the layout:
    • To have the Events workspace in a detached window, select  Separate Events window.
    • To have one or more additional System workspace windows, select  New System window.
  4. Resize and rearrange the windows on the same screen or among different monitors as required.
  5. In the Layout panel, select  and then choose  Save layout.
  6. Set whether you want to Enforce layout to all users on this computer:
    • Check box selected: Users without multi-monitor rights will be constrained to this layout. They will not be able to move the windows in any way.
    • Check box cleared: All users will be able to freely rearrange their windows, and can revert to the default layout with  Reset layout.
  7. Select Save.
  1. When operators start the Flex desktop app on this computer, the configured default layout behavior is applied.

Reset the Multi Monitor Layout

  • In the Layout panel, select  and then choose  Reset layout.
