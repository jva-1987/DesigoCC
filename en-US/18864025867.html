Configuring OAP Data Set

Use the OAP configuration tool (Siemens.GMS.SorisOAPAdapterConfig.exe) to configure the OAP objects in Desigo CC to map the corresponding objects in the connected equipment.
You can then save these settings in the SorisOAPAdapterConfig.xml file in the ..:/Program Files (x86)/Siemens/SORIS OAP Adapter/Config folder, which is used by the adapter.

Prerequisites

  • You have installed the Desigo CC software with the OAP Protocol extension.
  • Start the OAP Configuration Tool
  • Configure the OAP Server
  • (Optional) Configure the aggregator
  • Configure the Location
  • Configure the Message
  • Configure the Dynamic Message
  • Configure the Device
  • Adjust and Modify the Configuration
  • Configure the Adapter Settings
  • Save the Configuration File
