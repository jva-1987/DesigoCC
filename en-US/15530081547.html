GAMMA KNX Compatibility and System Limits

Compatibility

  • Desigo CC V4.1 or higher

Supported Devices

  • GAMMA IP Router:
    • N 146/03 (5WG1146-1AB03)
      NOTE: N 146, 2MW (5WG1146-1AB01) and N 146/02 (5WG1146-1AB02) are discontinued products and not compatible with the Desigo CC KNX integration.
  • GAMMA Thermostats:
    • RDF600KN, RDF600KN/S
    • RDF800KN, RDF800KN/NF, RDD810KN/NF, and RDF880KN/NF
    • RDG100KN, RDG160KN, RDG165KN
    • RDG200KN, RDG260KN
      NOTE: RDG200KN and RDG260KN are supported as aggregated devices up to FW V2.1.0. As of FW V5.1.0 these devices are supported in standard mode.
    • RDG400KN, RDG405KN
  • GAMMA Presence Detectors:
    • UP 258D12
    • UP 258E22
    • UP 258D31
    • UP 258D41
    • UP 258D51
    • UP 258D61
  • GAMMA QMX3 Sensors:
    • QMX3.P30
    • QMX3.P40
    • QMX3.P70
  • GAMMA QMX3 Room Operator Units:
    • QMX3.P02
    • QMX3.P34
    • QMX3.P37
    • QMX3.P74
  • DALI Gateway plus:
    • N 141/03 (5WG1 141-1AB03)
  • DALI Gateway Twin plus:
    • N 141/21 (5WG1 141-1AB21)
    • N 141/31 (5WG1 141-1AB3)
