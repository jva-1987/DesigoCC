Set Up Macros Subfolders

You can store macros directly in the main Macros folder, or you can create a structure of subfolders under it to organize your macros.

  1. Select Applications > Logics > Macros or a subfolder under it.
  2. The Macro tab displays.
  3. Click New .
  4. In the New object dialog box, enter a name and description.
  5. Click OK.
  6. The new folder is added to System Browser.
  7. Repeat the preceding steps for any other folders you want to add.
