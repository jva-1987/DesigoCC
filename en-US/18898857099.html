Honeywell Esser Events

The following is the list of events available in Desigo CC for the Honeywell Esser integration:

  • Acu Exclusion Off
  • Acu Trouble
  • Connection Disconnected
  • Fire point Fire Alarm
  • Fire point Alarm
  • Fire point Fault
  • Fire point Exclusion
  • Fire point Fire Active Information
  • Fire point Fire Tal Information
  • BACnet Gateway Disconnected
  • BACnet Line Fire Alarm
  • BACnet Line Fault
  • BACnet Line Disconnection
  • BACnet Line Switch on Phase Information
  • BACnet Line Detector Exchange Information
  • BACnet Output Fault
  • BACnet Output Disconnection
  • BACnet Output Activation
  • BACnet Output Block
  • BACnet Panel CPU Fault
  • BACnet Panel Cover Information
  • BACnet Panel Fault
  • BACnet UE Trouble

 

No events are available for the Command Panel and Disconnect Fire Point Command objects.
