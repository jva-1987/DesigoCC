Prepare the RDF302 configuration

Before configuring the RDF302 room thermostat for the Desigo CC management platform, perform the following steps:

  • Create a CSV file with the RDF302 information required for import. For file specifications, see RDF302 Configuration File. For each RDF302 room thermostat, one entry is required in sections [ CONNECTIONS] and [DEVICES]. The entries in the [ConnectionName] and [ParentConnectionName] columns must be the same.
