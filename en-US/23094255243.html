Add MK7022 to Cerloop for CS11

If not already done, you need to create the MK7022 gateway in the Cerloop Network.

  1. Select Project > Field Networks > [BACnet Network] > [Cerloop].
  2. Select the Cerloop tab.
  3. Click New and select New MK7022.
  4. In the New Object dialog box, enter a name and description, and the Instance number: the BACnet Device Identifier.
  5. Click OK.
  6. The MK7022 is added to System Browser and automatically selected.
  7. In the Configuration expander of the Cerloop tab, enter the following parameters:
    - Local Address: MK7022 address in the Cerloop. Valid values are 800 to 862 (default: 800).
    - Baud rate: 1200 or 2400 (ISO1745).
  8. Click Save .
