Display Desigo Insight Trend Data

Scenario: Desigo Insight trend data must be checked prior to migration.

 

Requirements:

  • The Desigo Insight migration package is on the computer.

 

Steps:

  1. In Windows Explorer, go to the Desigo Insight migration files.
  2. Select Data.
  3. Double-click file ExportViewer.exe.
  4. The Desigo Insight Export Viewer opens.
  5. Select Trend Data.
  6. Select the appropriate trend.
  7. All data on this trend displays.
  8. Use the Date/Time filter to reduce the data display. The entry format is the same the syntax for column Date/Time, for example: 2018-10-25T10:13:00.
  1. Data prior to and including the entered date is displayed.
