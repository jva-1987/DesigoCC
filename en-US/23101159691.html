Create a Scope Rule for the Network

  1. In System Browser, select Management View.
  2. Select Project > System Settings > Scopes.
  3. In the Scopes tab, click New and select New folder.
  4. Enter a name and description for the folder. For example, Notifier ID3000 Scope.
  5. Click OK.
  6. The newly-created folder is added to System Browser. By default, it is selected.
  7. In the Scopes tab, drag the Modbus network (in this case, Notifier ID3000) that you want to include from System Browser into the Scopes field of the Scope Rules section.
  8. The Scopes field automatically being set based on the linked objects.
  9. Click Save .
  10. Enter a name and description for the new scope. For example, Notifier ID3000 Network Scope.
  11. Click OK.
  1. The new scope rule is added to System Browser under the specified Notifier ID3000 Scope folder.
