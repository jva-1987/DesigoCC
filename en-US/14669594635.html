Importing the Library Template

  1. System Manager is in Engineering mode.
  2. System Browser is in Management View.
  1. In System Browser, select Project > System Settings > Libraries.
  2. In the Library Configurator tab, click Import.
  3. In the Import Libraries dialog box, select Global_D3VisualizationApplication_HQ_1.gms.
  4. Click Open.
  5. When the import is complete, the library is added to System Browser.
