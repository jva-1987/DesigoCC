Exporting an SNMP Device Configuration

  1. You previously configured SNMP monitoring of a device or property, and you now want to export that configuration into an SNMP file for future reuse. This is helpful, for example, when you are adding multiple SNMP devices of the same type.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > Field Networks > [SNMP network] > [SNMP device] or [SNMP property].
  3. The System Management tab displays.
  4. Click Export .
  1. The configuration data is exported to an SNMP file (a file with extension .snmp), which is saved in the [Installation Drive]:\[Installation Folder]\[Project Name]\devices folder. The naming convention for the exported file is:
    [class name].[Vendor And Type].[Version].[Device Type].
