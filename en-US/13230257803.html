Creating a Customized Library

Do this procedure to create a new library by cloning the structure of an existing one at a lower level. The customized library will be placed at your allowed customization level (for example, L4-Project). See Customization Levels.

 

  1. You are trained and authorized to work with system libraries at the target customization level. For example, L4-Project. See Determining the Allowed Customization Level.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L1-Headquarter > Global > [library].
  3. In the Library Configurator tab, click Customize the entire library to a lower level .
    NOTE: The button is unavailable if the customized library already exists at your allowed customization level.
  4. Click OK.
  1. The structure of the selected library is cloned under the allowed customization level (for example, L4-Project > Global > [library]). Only the library structure is recreated here. The blocks in the library are not yet configured.
