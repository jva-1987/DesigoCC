Working with the Graphics Generator

Scenario: You want to create a graphic of your Desigo plant using the Desigo CC Graphics Generator.

 

Reference: For background information, see the Graphics Generator section.

 

Flowchart:

 

 

Prerequisites:

  • Driver and network are available.
  • Data was imported.
  • In the user group, under Application rights, Desigo Graphics Generator is selected.
  • System Manager is in Engineering mode.

 

Steps:

1 - Creating Graphics Structure

Set the hierarchy for saving graphics pages before starting to create graphics. There are two main variants for a folder structure:

  • Building and plant structure
    • Building > Plant
    • Plant > Building
  • Own graphics structure as per project requirements
  1. In System Browser, select Application View.
  2. Select Applications > Graphics and the tab Graphics.
  3. From the Graphics toolbar, click Create New , and click New Folder.
  4. The Create new folder dialog box opens.
  5. Enter a Name for the folder and click OK.
    NOTE: The system name is derived automatically from the description. Spaces are replaced by _ (underscore).
  6. Create additional folders or subfolders as per the project requirements.
  1. The folder structure to save graphics pages is created.

Note:
Graphics pages can be moved afterwards under the Views tab. Do not use Windows Explorer to move graphics pages since various object references are not taken over.

 

2 - Basic Settings for the Graphics Generator

  1. In System Browser, select Management View.
  2. Select Project > System settings > Conversion tools > Desigo Graphics Generator.
  3. The Desigo Graphics Generator tab displays.
  4. Select the library type for the entire project in the Mapping Graphics Filter drop-down list. The selected filter reduces the selection of mapping graphics in the Mapping graphic column. The following options are available:
  • All
  • 2D DIN Color
  • 2D Grey Color
  • 2D
  • 2D+
  • 3D

 

3 - Modify Mapping Graphic

Modify mapping graphics for project-specific requirements prior to generating the graphics pages. This minimizes post processing. The following example demonstrates how to move the extract air fan from position A to position B.

  1. Create new folder structure for project-specific mapping graphics
  2. Open mapping graphic in the Graphics Editor
  3. Editing the Mapping Graphic
  4. Save the mapping graphic under the same name

Creating a Folder Structure

The folder structure in the L4 Project must first be created to save project-specific mapping graphics.

  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > [L1-Headquarters] > BA.
  3. Select Graphics Generator Ventilation (Desigo).
  4. The Library Configurator tab displays.
  5. Click Entire library … and then OK.
  6. The folder structure is created under Project > System settings > Libraries > [L4 Project] > [BA] > Graphics Generator Ventilation (Desigo).
  7. Repeat steps 3 and 4 for all libraries Graphics Generator [plant type] (Desigo).
  1. All folders in L4 Project are created for saving project-specific mapping graphics.

Opening Mapping Graphic

  1. Select Project > System settings > Libraries > [L1-Headquarters] > BA > Graphics Generator [Plant type] (Desigo) > Graphics mapping pages.
  2. The available mapping graphics are displayed in the Graphic tab.
  3. Click Edit .
  4. The Graphics Editor opens.
  5. Click Open .
  6. Select [System] > Libraries > [L1-Headquarters] > BA > Graphics Generator [Plant type] (Desigo) > Graphics mapping pages.
  7. In the Name column, select the mapping graphic and click Open.
  1. The mapping graphic opens. Some symbols with a gray background are deposited on mapping graphics and are labeled as variant symbols. The variant symbols are replaced by the correct symbols during generation.

Editing the Mapping Graphic

  1. Open the mapping graphic for editing in the Graphics Editor.
  1. Select the corresponding symbol, for example: Fan off.
  2. Left-click and drag the symbol to the desired position on the mapping graphic.
  3. Release the left mouse button.
  4. The symbol is now in the new position.
  5. In the Properties dialog box, the object reference stays the same, for example: AN=FanEx.
  6. (Optional) Edit additional symbol.

Saving the Mapping Graphic

  1. Click Save As.
  2. A Save Object As dialog box appears.
  3. Select the folder structure System > Library > L4 Project > BA > [Plant] > Graphic mapping pages.
  4. Enter a name in the Name field. Select one of the variants:
    • Apply the default name. Only the project-specific solution is displayed in the selection list when assigning a mapping graphic.
    • Enter a new name for the mapping graphic. The name of the mapping graphic must start with the prefix MAP_. When assigning a mapping graphic, the selection list displays the project-specific solution, as created by headquarters, the region, and RCs.
  5. Click Save.

 

4 - Extend Variant Symbol

The existing headquarter library needs to be extended by an additional function in the fan.

  1. The additional function is available at the project levels L2 – L4.
  2. The new symbol is available for the function.
  1. In System Browser, select Management View.
  2. Select Project > System settings > Libraries > [Project level] > BA > Graphics Generator Ventilation (Desigo) > Graphics variant symbols.
  3. The Graphics preview opens.
  4. Select a variant symbol, for example, a fan.
  5. Right-click and select Edit.
  1. The graphics editor opens for editing.
  1. Click Insert new level .
  2. In the column Level name, enter the condition, for example, FN=Fan4St.
  3. A new level column, Level 9, is displays.
  4. For the new function, select Level 3.
  5. The base symbol for precise placement is displayed.
  6. In the library browser, select the corresponding symbol.
  7. Drag-and-drop the new symbol over the displayed symbol.
  8. Clear the Level 3 check box and select Level 9.
  9. Click Save As .

 

5 - Generate Graphics Pages

The graphics pages can be created or generated in various forms by project type and size. This can take place one time or in multiple steps in a project phase. The following workflows are therefore examples of possible project engineering scenarios.

  1. The Desigo Graphics Generator tab is open.
  1. In System Browser, select logical view.
  2. (Optional) In the system browser, select the check box Manual Navigation. It prevents the unwanted switch to another view. The check box must be cleared to display a generated graphic.
  3. (Optional) Make the project-specific changes to the mapping graphics. This is recommended when multiple plants require the same change on a project, for example:
    - Ducting in the room
    - Fans, sensors, thermostats at another duct position
  4. Create a ventilation plant.
  5. Create a heating plant.
  6. Create a cooling plant.

NOTE:
In a distributed system, the graphics can only be generated within the same system. The Graphics Generator prevents assignment to another system.

 

6 - Ventilation

  1. The Desigo Graphics Generator tab is open.
  1. In the System Browser, select Logical View.
  2. Select Logical > [Network name] > [Plant hierarchy] > [Plant name].
  3. In the Mapping Graphics Filter drop-down list, select [2D DIN Color].
  4. Drag-and-drop the ventilation plant to the table in the Desigo Graphics Generator tab.
  5. In the Discipline column, select the option Air. The selected filter reduces the selection of mapping graphics in the Mapping graphic column.
  6. Open the drop-down list in the Mapping graphic column.
  7. Moving the mouse displays a preview of the mapping graphics. The mapping graphic with the best match is displayed at the top.
  8. Select the appropriate mapping graphic.
  9. Click the button in the Options column.
  10. The Options dialog box displays. Only the options relevant to this mapping graphic are displayed.
  11. Select the hydraulic circuit for each aggregate and click OK.
  12. In the File name column, take over the suggested file name or change it.
  13. In the Target folder drop-down list, select the save location for the generated graphic.
    NOTE: The folder must be created in the graphics editor.
  14. (Optional) Repeat steps 2 – 11 for additional ventilation plants or add cooling plants and heating plants.
  15. Click Generate graphic and then Yes to start the generation process.
  16. The Generation status column displays: In progress.
  17. The Generation report dialog box displays once the graphics page is generated.
  18. Click OK.
  19. (Optional) Click Log file to display additional details on the created graphic.

 

7 - Heating Plant

  1. The Desigo Graphics Generator tab is open.
  1. In System Browser, select logical view.
  2. Create the boiler.
  3. Create hot water generation.
  4. Create heating group 1.
  5. Create heating group 2.
  6. (Optional) Add cooling plants and ventilation plants.
  7. Click Generate graphic and then Yes to start the generation process.
  8. The Generation status column displays: In progress.
  9. The Generation report dialog box displays once the graphics page is generated.
  10. Click OK.
  11. (Optional) Click Log file to display additional details on the created graphic.

Example for a Heating Plant Configuration

Heating boiler

  1. Select Logical > [Network name] > [Plant hierarchy] > [Plant name] (boiler).
  2. In the Mapping Graphics Filer drop-down list, select [2D DIN Color].
  3. Drag-and-drop the plant to the table in the Desigo Graphics Generator tab.
  4. In the Discipline column, select option Heating.
  5. The selected filter reduces the selection of mapping graphics in the Mapping graphic column.
  6. Open the drop-down list in the Mapping graphic column. Moving the mouse displays a preview of the mapping graphics. The mapping graphics with the best match are displayed at the top.
  7. Select the appropriate mapping graphic.
  8. Click the button in the Options column.
  9. The Options dialog box opens. Only the options relevant to this mapping graphic are displayed.
  10. For the heating group, select:
    a. The boiler position on the generated page.
    b. Click OK.
  11. In the File name column, take over the suggested file name and change it.
    NOTE: Pay attention to capitalization in the file name.

Hot Water Handling

  1. Select Logical > [Network name] > [Plant hierarchy] > [Plant name] (hot water supply).
  2. Drag-and-drop the plant to the table in the Desigo Graphics Generator tab.
  3. In the Discipline column, select option Heating.
  4. The selected filter reduces the selection of mapping graphics in the Mapping graphic column.
  5. Open the drop-down list in the Mapping graphic column. Moving the mouse displays a preview of the mapping graphics. The mapping graphics with the best match are displayed at the top.
  6. Select the appropriate mapping graphic.
  7. Click the button in the Options column.
  8. The Options dialog box opens. Only the options relevant to this mapping graphic are displayed.
  9. For the heating group, select:
    a. the heating group position on the generated page.
    b. Click OK.
  10. In the File name column, enter the same file name as the boiler.
    NOTE: Pay attention to capitalization in the file name.
  11. Select the Add check box. No new graphics page is generated when the check box is selected, but rather is generated as part of an existing graphics page.

Heating Group 1

  1. Select Logical > [Network name] > [Plant hierarchy] > [Plant name] (heating group).
  2. Drag-and-drop the plant to the table in the Desigo Graphics Generator tab.
  3. In the Discipline column, select option Heating.
  4. The selected filter reduces the selection of mapping graphics in the Mapping graphic column.
  5. Open the drop-down list in the Mapping graphic column. Moving the mouse displays a preview of the mapping graphics. The mapping graphics with the best match are displayed at the top.
  6. Select the appropriate mapping graphic.
  7. Click the button in the Options column.
  8. The Options dialog box opens. Only the options relevant to this mapping graphic are displayed.
  9. For the heating group, select:
    a. The position of the heating group on the generated page
    b. The graphic symbol for the consumer
    c. Click OK.
  10. In the File name column, enter the same file name as the boiler.
    NOTE: Pay attention to capitalization in the file name.
  11. Select the Add check box. No new graphics page is generated when the check box is selected, but rather is generated as part of an existing graphics page.

Heating Group 2

  1. Select Logical > [Network name] > [Plant hierarchy] > [Plant name] (heating group).
  2. Drag-and-drop the plant to the table in the Desigo Graphics Generator tab.
  3. In the Discipline column, select option Heating. The selected filter reduces the selection of mapping graphics in the Mapping graphic column.
  4. Open the drop-down list in the Mapping graphic column. Moving the mouse displays a preview of the mapping graphic. The mapping graphic with the best match is displayed at the top.
  5. Select the appropriate mapping graphic.
  6. Click the button in the Options column.
  7. The Options dialog box opens. Only the options relevant to this mapping graphic are displayed.
  8. For the heating group, select:
    a. the position of the heating group on the generated page. The group position number must be unique to the graphics page.
    b. the graphic symbol for the consumer.
    c. Click OK.
  9. In the File name column, enter the same file name as the boiler.
    NOTE: Pay attention to capitalization in the file name.
  10. Select the Add check box. No new graphics page is generated when the check box is selected, but rather is generated as part of an existing graphics page.
