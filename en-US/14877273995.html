Customizing the HQ Multimedia Collection

Scenario

When the Multimedia Collection extension is installed, a default multimedia collection is created under Applications > Multimedia Collection. You want to customize this default collection , for example to add new media and new buttons for playing those files when this multimedia collection is selected in Operating mode, or to remove some of the existing buttons. For background information, see the reference section.

Prerequisites

  • The Multimedia Collection extension is installed and included in the active project.
  • System Manager is in Engineering mode.
