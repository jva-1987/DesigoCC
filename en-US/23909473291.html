Configure Report Output

  1. You have a configured report that has the required elements, filters, and layout. You now want to set a destination to route the report output.
  1. Click the Settings tab and in the Report Output group box click Dialog Launcher .
  2. The Report Output Definition dialog box displays.
  3. Configure the destination (File,Email, or Printer.)
    to which you want to route the report output as well as specify the file format (PDF, XLS, CSV, or XML) in which the report output can be generated.
  1. The report output is configured to either a file, email, or printer.
