Installing Flex Client as Desktop App

You can download and install the Flex desktop app on any computer where you already access Flex web app in the browser. This allows for greater convenience for launching Flex app, and the ability to detach the Events workspace or additional System workspaces into separate windows, either on one monitor or in a multi-monitor configuration.

In addition, if the computer is configured as an identified Flex station, the desktop app lets you save a default window layout or configure a closed mode Flex station.

 

Prerequisites

  • The Flex web app is configured in Desigo CC.
    See Setup Checklist for Flex Client.
  • You are working on a Windows computer that is able to access Flex web app in the browser.

 

Download and Install the Flex Desktop App

  1. Log onto Flex web app in the browser with your account.
  2. In the horizontal navigation bar, select >  About.
  3. In the About page, select Download desktop application.
  4. The Siemens-Gms-FlexClient-Installer.exe is downloaded to the Downloads folder on your local computer.
  5. Run the executable file Siemens-Gms-FlexClient-Installer.exe following the instructions in the setup wizard.
  6. When the installation completes, the Run Flex Client check box is automatically selected.
  7. Click Finish.
  1. The Flex desktop app is automatically started.
  2. In case of connection issues, see Troubleshooting Flex Client Issues.

 

Provide the Endpoint Address

  1. The very first time you start the Flex desktop app, the Endpoint address page displays.
  1. Ensure that the Address corresponds to the correct [Flex client application URL]. This is the same URL that you used to connect from the browser.
  2. Select Save.

You can subsequently change the endpoint address as follows:

  1. Select  Desigo CC menu, and then Advanced > Edit endpoint address.
  2. Enter a valid Flex client application URL.

 

Initial Security Settings

  1. If the [Flex client application URL] was "not trusted" in the browser (see Access Flex Client URL for details), then the desktop app will likewise show a Connection not private dialog box. Here you have two options:
    • (not recommended) Choose Accept and proceed to trust the Flex client application URL even without the appropriate certificate. If you also select the Remember selection and don't ask again check box, this dialog will no longer appear the next time the Flex client application is started.
    • Choose Import certificate to import the necessary certificate into the Trusted Root Certification Authorities store on the Flex computer. Once you do this, the Connection not private dialog box will no longer appear.
  2. If the Flex computer has been configured as an identified client station, a certificate dialog box will appear. You have two options:
    • Select No certificate and Continue. In this case, the Flex desktop app will connect as an anonymous client.
    • Select Certificate, select the host certificate that identifies the Flex station, then Continue. In this case the Flex desktop app will connect as an identified client. (Depending on configuration you may also have to provide a password for the certificate.)
      NOTE: In either case, if you also select the Remember selection and don't ask again check box, the certificates dialog will not appear the next time.

You can subsequently clear these certificate settings as follows:

  • Select  Desigo CC menu, and then Advanced > Clear security settings.

 

Unless you want to set up a closed mode Flex station, to prevent software vulnerabilities, after installing or updating the Flex desktop app to a newer version, in C:\Program Files\Siemens-Gms-FlexClient\dist\ delete the closed mode folder.
