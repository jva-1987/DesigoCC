OZW Graphic Symbols

The following graphic symbols are provided with the installation of the OZW extension to represent OZW objects in graphics.

DYN_2D+_OZW_Device_None_Horizontal_001_150

  • Main symbol for the OZW Device
  • Symbol provides graphic, text, and status icons for fault condition (yellow=faulty; blue=normal) and energy information (orange=active; green=normal)

 

DYN_2D+_OZW_MainDevice_None_Horizontal_001_150

  • Main symbol for the OZW Main Device
  • Symbol provides graphic, text, and status icons for fault condition (yellow=faulty; blue=normal) and energy information (orange=active; green=normal)

 

DYN_2D+_OZW_MainDevice_Info_Horizontal_001_150

  • Secondary symbol for the OZW Main Device
  • Symbol provides status icons for fault condition (yellow=faulty; blue=normal) and energy information (orange=active; green=normal)

 

DYN_All_OZW_Command_AnalogSetpoint_None_Central_001_150

  • Main symbol for numeric functions
  • Symbol provides an increment spinner to set the analog value

 

DYN_2D+_OZW_Sensor_Temperature_None_Horizontal_001_150

  • Secondary symbol for numeric functions
  • Symbol provides temperature unit (T: C=Celsius; F=Fahrenheit) and value

 

DYN_All_OZW_Command_DigitalSetpoint_None_Central_001_150

  • Symbol internally used as part of graphic templates
  • Symbol provides a status display and a selection via a drop-down list

 

DYN_2D_OZW_SwitchPoint_None_Horizontal_001_150

  • Symbol internally used as part of graphic templates
  • Symbol provides multiple fields for a scheduling entry

 

DYN_2D_OZW_Scheduler_Scene_Horizontal_001_150
DYN_2D_OZW_Scheduler_OnOff_Horizontal_001_150
DYN_2D_OZW_Scheduler_Jalouise_Horizontal_001_150
DYN_2D_OZW_Scheduler_HVAC_Horizontal_001_150
DYN_2D_OZW_Scheduler_BW_Horizontal_001_150

  • Symbol internally used as part of graphic templates
  • Symbol provides a scheduling 6-entry block

 

DYN_2D_OZW_AlarmHistor_None_Horizontal_001_150

  • Symbol internally used as part of graphic templates
  • Symbol provides a 10-event block

 

DYN_2D_OZW_AlarmAct_None_Horizontal_001_150

  • Symbol internally used as part of graphic templates
  • Symbol provides a 10-event block and two command lines

 

DYN_All_OZW_Command_DigitalSetpoint_None_Central_002_150

  • Symbol internally used as part of graphic templates
  • Symbol provides a status display and a selection via text field

 

DYN_2D_OZW_CalendarElement_None_Horizontal_001_150

  • Symbol internally used as part of graphic templates
  • Multiple fields for a calendar entry

 

DYN_2D_OZW_Calendar_None_Horizontal_001_150

  • Symbol internally used as part of graphic template
  • Symbol provides a calendar 16-row block and a control heading
