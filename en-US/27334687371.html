Common Error Conditions on Project Restart

The following section describes how to handle the common error conditions if for any reason if the project is stopped and restarted from SMC.

Message/Issue

Situation

Do the following

Error message: The session is expired. The user will be logged off. is displayed.

If for any reason if the project is stopped and restarted from SMC and the Installed Client is running parallel. The Installed Client application is unable to reestablish the user session.

Re-login to Desigo CC as all previous user sessions are closed for security reasons. You can re-login because the licenses for logging in are ready when restarting the system.
