Scheduling a Powermanager Report

Scenario

You want to configure, schedule, and route a report it to an email list or a folder.

For background information, see the reference section.

Prerequisites

  • To configure the report output as an email, you have configured SMTP Email server using Notification extension.
  • You have created recipients in the Recipients Editor tab of the Notification application. See Create Recipient Users or Recipient Devices in Creating and Configuring Recipients.
