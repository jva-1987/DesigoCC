Advanced Import Rules

The following procedures provide information for working with instance attributes of import rules. To work with import rules, ensure that Project > System Settings > Libraries > L1-Headquarter > BA > Device > Simatic S7 > Import Rules is selected.

Creating Advanced Import Rules

  1. Click Customize the entire library to a lower level .
  2. A confirmation message displays.
  3. Click OK.
  1. An empty folder structure is created for the project library under Project > System settings > Libraries > [L4-Project] > BA > Device > Simatic S7 > Import Rules.

 

Exporting Instance Attributes

  1. You have extracted a CSV file for specific CSV import rules, and mapped the instance attributes.
  1. In the Import Rules tab, open the Instance Attributes Mapper expander.
  2. Click Export.
  3. The Export Function Mapping Data dialog box displays.
  4. Enter a name and select a destination folder.
  5. Click OK.
  1. The data is exported.
