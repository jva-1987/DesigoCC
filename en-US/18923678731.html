Integrating an Helvar Lighting Adapter

Scenario

You want to configure Desigo CC to operate on the Helvar Lighting system. For background information, see the reference section.

Prerequisites

  • The Helvar Lighting extension is installed and included in the active project. The following extension is also selected automatically:
    • SORIS Driver
  • The Helvar Adapter is running and connected to the Helvar Lighting system. For instructions, see Installing and Configuring the Helvar Lighting Adapter.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
