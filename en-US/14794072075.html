Working with Site Viewer

In the Site Viewer you can filter the list of panels by name or part of the name, and you can sort it by event severity. To better represent the architecture of your site, with an appropriate permission, you can rename the panels and also group them together in meaningful collections represented by a single graphic symbol. You can also select one or more panels and then operate on System Manager and Event List on the selected set to see status information and start control actions.

 

Filter the Site Viewer by Name

  • In the Search by description field of the Site Viewer, enter any part of the panel description you want to display.
  1. The Site Viewer view is updated and only displays the panels whose description contains the text you entered.

 

Sort the Site Viewer by Alarm Status

  • In the Site Viewer, click Sort by Alarm Status .
  1. The Site Viewer view is updated and sort the panels according to the severity and acknowledge status of pending events.
    NOTE: Click Sort by Alarm Status again to sort the panel alphabetically (default).

 

Show Properties and Commands of a Panel from the Site Viewer

  1. In the Site Viewer, double-click a panel.
  2. The panel properties and commands display in the Contextual pane.
  3. Select the Extended Operation tab to access the extended set of control commands.

 

Display Filtered Event List

  1. Select one or more panels in the Site Viewer.
  2. Click Enable Event List interactions .
  3. The Event List displays, filtered according to the Site Viewer selection.

 

Hide Filtered Event List

  1. Click Disable event list interactions to remove the Site Viewer filter.
  2. In the Summary bar, click Close Event List .

 

Group Panels

  1. You are enabled to configure the Site Viewer (User Application Rights).
  1. In the Site Viewer, click Edit mode .
  2. Select the panels you want to group. Hold CTRL key pressed when you click the panels to allow for a multiple selection.
  3. Click Group .
  4. In the dialog box that opens, enter the group Description.
  5. The new group panel displays. Next to it, a dashed-line frame shows the panels that belong to the group.
  6. Click Edit mode again to quit edit mode.

 

Expand/Collapse a Group

  1. In the Site Viewer, double-click a collapsed group to expand it.
  2. The panels that belong to the group display in the Site Viewer. They are represented by slightly smaller panels that display next to the group panel within a dashed-line frame.
  3. Double-click the group again to collapse it.
  4. The panels that belong to the group disappear from the Site Viewer.

 

Ungroup Panels

  1. You are enabled to configure the Site Viewer (User Application Rights).
  1. In the Site Viewer, click Edit mode .
  2. Select the panel of the group you want to remove.
  3. Click Ungroup .
  4. The group panel disappears.
  5. Click Edit mode again to quit edit mode.

 

Remove a Single Panel from a Group

  1. You are enabled to configure the Site Viewer (User Application Rights).
  1. In the Site Viewer, click Edit mode .
  2. If the group is collapsed, expand it by double-clicking the group panel.
  3. Select the panel you want to remove.
  4. Click Ungroup .
  5. The group is updated.
  6. Click Edit mode again to quit edit mode.

 

Rename a Panel from the Site Viewer

  1. You are enabled to configure the Site Viewer (User Application Rights).
  1. In the Site Viewer, click Edit mode .
  2. Select the panel you want to rename and press F2.
  3. Enter the new name.
  4. Click Edit mode again to quit edit mode.
