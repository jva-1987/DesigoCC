Migrate Desigo Insight Data

Scenario: Existing Desigo Insight data must be migrated to Desigo CC. Trend Views, trend data as well as Todo texts can be migrated.

 

Reference:

For background information, see the Migration of Desigo Insight to the management platform section.

 

Prerequisites:

  • You have application rights to the Desigo Data Migration tab.
  • System Manager is in Engineering mode.
  • In System Browser, Management View is selected.
  • The data points used in Desigo Insight are already imported to Desigo CC.
    NOTE: This channel cannot be automatically assigned if the BACnet name has changed in Desigo CC.
