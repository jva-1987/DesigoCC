Add or Remove Media Files Buttons

Add New Buttons

To add new buttons for new media files that you copied to the multimedia folder, you must add new entries to the cloned text groups, as follows:

  1. Select the default text group cloned at your allowed customization level (for example, L4-Project > Global > Multimedia Collection > Texts > Multimedia Collection).
  2. To add a new text group entry:
    a. Click Add new row .
    b. In the Value field, enter a consecutive number that must not be duplicated within the given text group.
  3. To configure a text group entry:
    a. In the language column en-US, enter the text that corresponds to the exact name of the file in the multimedia collection folder.
    b. Click Icon. From the Icon Picker dialog box, select the appropriate library and icon, and click OK.
  4. The selected text and icon will display on the button for playing that media file in Desigo CC.
    NOTE: The en-US language is mandatory. Additional languages can optionally be configured. In any case, the mapping of media files to buttons is always done based only on the en-US language.
  5. Repeat the previous steps for any other text group entries (media files buttons) required.
  6. Click Save .

 

Remove Buttons

To remove existing media file buttons from the default multimedia collection, you must remove the corresponding text group entries, or edit them so that they no longer match any media file name.

  1. Select the default text group cloned at your customization level (for example, L4-Project > Global > Multimedia Collection > Texts > Multimedia Collection).
  2. Locate the entry corresponding to the button you want to remove.
  3. In the en-US language column, modify the name so that it does not correspond to any file name in the folder where the files are stored. (For example, you can add an underscore (_) to the text of the button name).
