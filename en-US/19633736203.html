Hide Walls

Scenario: The filter hides all walls. The BIM object model for walls is unknown and must first be found.

 

1st Step: The BIM object for a wall is found and hidden

Scenario: You want to find the BIM object model for walls.

  1. Edit is enabled in BIM Viewer.
  2. An empty filter Walls is available.
  1. Select a wall with the mouse.
  2. Press SHIFT + right click.
  3. Select Hide the element and then to filter: Walls.
  4. In the Filter Name dialog box, select the Walls filter.
  5. Click Edit.
  6. Displays the associated BIM object. The IFC Name column references the selected wall object. The BIM object model IFCWALLSTANDARDCASE displays in the IFC Type column.
    • Test: Select the Walls check box.
  7. The wall selected in the filter is hidden.
  8. Click Save .

 

2. Step 2: Hide all Walls

Scenario: After finding a wall for the BIM object model, you can hid all walls belonging to this BIM object model based on all walls.

  1. Click Edit.
  2. Displays the associated BIM object. The IFC Name column references the selected wall object. The BIM object model displays in the IFC Type column.
  3. Select the BIM object model in section Exempt from BIM model.
  4. Click Edit.
  5. The Hidden objects dialog box displays.
  6. Delete the entry in the IFC Name text field and enter an asterisk (*).
  7. Click OK and then OK to close both dialog boxes.
    • Test: Select the Walls check box.
  8. All walls are hidden. Other objects, including doors, stairs, radiators, furniture, etc. are still visible.
  9. Click Save .
