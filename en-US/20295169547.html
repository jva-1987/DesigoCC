DMS Advanced Reporting Troubleshooting

This section provides troubleshooting information about the DMS Advanced Reporting application.

Problem

Situation

Do the following...

In the DMS Advanced Reports, the Reaction Time for Ack and Reset is incorrect. Depending on the involved DMS unit, the value can be missing or inexact.

Currently, the Reaction Time calculation is only possible when Ack and Reset commands are performed from the Event List. If operators use the Contextual pane commands in System Manager, then the reports may be incorrect.

When the Reaction Time is required in the DMS Advanced Reports, instruct operators to always use the Event List commands for handling events.
