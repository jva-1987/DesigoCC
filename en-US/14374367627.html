Handling OPC Discovery Error Conditions

Message

Situation

Do the following...

Error sending the command, try again!

The manager is not responding because it is busy with another operation in progress.

Try again.

Error sending the command, will probably not be reachable.

The cohost manager is not responding because there might be a network disconnection or any other OPC connection issue.

Try again.

The driver has not been selected. Please select a driver in the OPC View!

The OPC driver is not associated to the OPC network.

Configure the OPC driver.

For instructions, see Configuring the OPC Client Connectivity.
