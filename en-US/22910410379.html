Display Data to Aggregate

  1. In the OPC UA Client Configuration section, click Data Settings.
  2. Select the desired root node (Objects or Views).
  3. Do one of the following:
    • If this is the first time you connect to the selected OPC UA third-party server, you must let the adapter know the nodes to be exported in one of the following ways:
      - Manually expand the root node, and navigate the tree to display the nodes that you may want to aggregate.
      - Browse for the tree nodes automatically. For more details, see steps 2 and 3 in Configure Data Settings.
    • If the tree was already browsed at least one time, the adapter knows the nodes preloaded from the configuration file. In this case, you may want to navigate the tree to display additional nodes to aggregate.
