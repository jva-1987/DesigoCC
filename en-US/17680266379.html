Integrating Edwards EST3 Panels Through Online Discovery

Scenario

Desigo CC is connected to an Edwards EST3 fire panel, and you want to run the online discovery. For background information on the Edwards EST3 extension, see the reference section.

Performing an offline configuration is recommended instead if you want to import data into Desigo CC through an EDE file that addresses complex and non-standard configuration requirements.

Prerequisites

  • The Edwards EST3 extension module is installed and included in the active project.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
