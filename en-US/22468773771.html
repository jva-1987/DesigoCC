Add a Data Point and Symbol to a Graphic

  1. From System Browser, drag the required data point onto the canvas.
  2. The default symbol instance associated with the data point displays on the canvas.
  3. (Optional) To replace the associated symbol with another symbol associated with the data point:
    a. Right-click the data point and from the context menu navigate to Symbol Instance > Replace.
    b. From the Symbol Library display, select the symbol you want use for the data point. The selected symbol replaces the symbol on the canvas.
    c. On the canvas, right-click the new symbol instance, and then click Reset Size to adjust to the symbol’s original size.
  1. You have placed a data point with an associated symbol instance on the graphic. Repeat as required.
