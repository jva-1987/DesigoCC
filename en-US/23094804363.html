Historizing Offset for Meter Rollovers

  1. System Manager is in Engineering mode.
  2. You have created a managed meter and it displays in the Managed Meters folder in the System Browser.
  1. In System Browser, select Application View.
  2. Select Applications > Energy and Power > Managed Meters > [managed meter].
  3. The managed meter details display.
  4. Specify the offset value in the Offset field after the meter has rolled over to 0, reached the maximum meter reading, or reset to 0 due to repair.
  5. Click Save .
  1. The information on the new offset value is historized by default to the current date and time.
