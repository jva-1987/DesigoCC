Integrating S-Bus

Scenario

You want to configure Desigo CC to operate on the S-Bus system. For background information, see the S-Bus reference section.

Prerequisites

  • The S-Bus extension is installed and included in the active project. The following extension is also selected automatically:
    • SORIS Driver
  • The S-Bus adapter is running and connected to the S-Bus system. For instructions, see Installing and Starting the S-Bus Adapter.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Licensing

This extension is covered by license. Check that the required license is available; otherwise, you may not be able to configure and work with this extension.
