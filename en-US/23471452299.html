Commend Events

The Commend events are acquired via the OPC UA interface and you can import them individually as A&E objects from the OPC UA Data Settings section of the SORIS Adapter Settings.

You can also assign Management Station alarms to the pre-configuration present in the Object Model; remember that you must enable each configuration at the object or instance level. Below are the details of the pre-configuration provided for MS Alarms.

For both solutions, see the configuration guideline in Commend Intercom.
