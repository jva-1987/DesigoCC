PowerQuality Database

This section provides information on how PowerQuality database is designed and works.
For related procedures to create PowerQuality database, see step-by-step section.

To create PowerQuality database, a PowerQuality license is required.

PowerQuality database stores the data from IEC 61850 devices related to PowerQuality. PowerQuality database works similar to historical database.

PowerQuality database can be created in local or remote system.

The size of database is based on SQL Server used.

  • For SQL Server Express edition: 10 GB
  • For SQL Server Enterprise and Standard edition: 250 GB

PowerQuality database in local system

In this scenario, the system with Powermanager installed hosts the database. The database is located in one of the drive of the system.

PowerQuality database in remote system

In this scenario, Powermanager is installed on one system whereas database is hosted on other system.

PowerQuality database in remote system has two possible scenarios:

  • Database in same domain
  • Database in same workgroup
