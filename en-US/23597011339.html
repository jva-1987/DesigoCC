Verify the Integrity of the History Database

The History Database (HDB) is available on the Server only. You should verify the following for an HDB before backing it up.

  1. Open SMC on the Server.
  2. In the SMC tree, select History Infrastructure > [SQL Server Name].
  3. Select the HDB instance that you want to back up.
  4. Verify that the HDB that you want to backup is Started.
  5. Open the Task Info expander and verify the following:
    • No pending tasks for HDB.
    • All tasks are successfully executed.
