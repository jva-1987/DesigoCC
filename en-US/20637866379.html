Management Station Schedules and Calendars

This section provides procedures for migrating and restoring Management Station Schedules and Calendars from Management Station to Flex Client. For background information, refer to Migration of Management Station Schedules and Calendar to Flex Client.

Migrating Management Station Schedules and Calendars to Flex Client

  1. System Manager is in Engineering mode.
  2. System Browser is in Management View.
  3. Ensure that you have all scope and application rights to perform migration of Schedule and Calendar objects. For more information, refer Schedules in Application Rights.
  1. Navigate to System Browser > Projects > System Settings > Conversion Tools > Management Station Schedules Migration.
  2. In the Schedule Migration, that displays, verify the following column:
    • Dependency: Displays the name of the dependent/referred object, if there is any dependency/reference for any of the Schedule and Calendar object.
  3. Select Migrate and click Yes to start the migration.
  4. After migration is complete, you can see the migration summary dialog box.
  5. To view the log summary, click Open Log.
    • If you click OK the log file summary is not visible again. You can save the log file locally for future reference.
    • Check the log summary file for partially succeeded migrations.
  1. The successfully migrated Management Station Schedules and Calendars to Flex Client are converted into Scenes.
    - In addition, the similar set output values (active schedule/inactive schedule) are merged together and a common Scene is created in Flex Client.

NOTE:
In case of exception set for any Management station Schedule and Calendar, only the options listed below are migrated, rest all exceptions you need to add manually in Flex Client.

- Any Start date - Any End date (Either for all days or only weekends or any weekdays or custom days check box is selected).

- Valid Start date - Valid End date (Either for all days or only weekends or any weekdays or custom days check box is selected).

- Valid Start date - Any End date (Only if all days check box is selected).

- Any Start date - Valid End date (Only if all days check box is selected).

NOTICE

NOTE:
After migrating the Schedule and Calendar objects to Flex Client, you can access the object only through Flex Client.

Restoring of Management Station Schedules and Calendars from Flex Client

  1. System Manager is in Engineering mode.
  2. System Browser is in Management View.
  1. Navigate to System Browser > Projects > System Settings > Conversion Tools > Management Station Schedules Migration.
  2. Select Restore and click Yes to restore all migrated Management Station Schedules and Calendars object.
  1. All Management Station Schedules and Calendars from Flex Client to Management Station are restored successfully.
