Integrating Notification into the Management Platform

Scenario

You want to integrate notification devices, such as ESPA Paging System or GSM Modem SMTP Email with the management platform to send notifications out (for example, during an emergency event such as fire).
For background information, see the reference section. Also, refer to the Video Scenarios on YouTube topic.

Prerequisites

  • The Notification extension is installed/upgraded and configured in the project.
  • You have logged into the Desigo CC client.
  • You have configured a user group and assigned Scope rights to the user group for working with the notification extension. (See Assign Scope to a User Group in Security).
  • You have enabled the application rights for the Notification extension (RENO, RENO Plus) and assigned the application rights to the user group. (See Additional Notification Procedures)
  • System Manager is in Engineering mode.
  • System Browser is in Application View.
