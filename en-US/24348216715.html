Configuring Video Export for HikVision

Scenario

You want to configure Desigo CC for working with the HikVision video surveillance system on Desigo CC stations. For background information, see the reference section.
This task is performed as part of the integration workflow, for details see Integrating Video Surveillance with HikVision Provider.

NOTE: Video files are exported to the Export Video folder. Snapshots are saved to the following location: C:\Users\Administrator\Pictures\Snapshot\.

Prerequisites

  • The following extensions are installed and included in the active project:
    • HikVision
    • Video Management
  • The VMS server is installed.
  • The provider was installed. See Installing the HikVision Provider Software.
  • The Siemens Video API Service is available among the Windows services.
