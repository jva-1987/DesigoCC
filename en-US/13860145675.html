Summary Status

In order to simplify the system display and highlight the most important information, the system sometimes combines properties into a single Summary Status property. The Summary Status property displays the highest priority status that is currently active for an object. For example, if an object has an active Fire Alarm and Fire Fault, the Fire Alarm would be displayed in the Summary Status.
