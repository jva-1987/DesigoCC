Easy Buttons

The following procedures provide additional information on easy buttons. For background information related to easy buttons, see reference section.

 

Prerequisites

  • System Manager is in Engineering mode.
  • System Browser is in Application View.

 

Customize Background for an Easy Button

  1. Select Applications > Notification.
  2. Select the Easy Buttons tab.
  3. Click Page Setup .
  4. A confirmation message displays.
  5. Click Yes.
  6. In the Set Easy Buttons background dialog box, browse to the location of the image, select the file, and click Open.
  7. The background for the Easy Buttons screen is changed to the selected image.
  8. Click Save .
  1. The background is saved.

Delete an Easy Button

  1. One or more easy buttons are configured.
  1. Do one of the following:
    • Select Applications > Graphics > Easy Buttons
    • Select Applications > Notification.
  2. The Easy Buttons tab displays.
  3. Select the easy button to be deleted.
  4. Delete the button by either pressing DEL or clicking Delete in the Edit group box on the Home tab.
  5. Click Save .
  1. The selected easy button is deleted.

Restore the Easy Buttons Tab

  1. Select Applications > Graphics > Easy Buttons.
  2. The Easy Buttons tab displays.
  3. Select Applications > Notification.
  4. Drag the Notification node onto the Easy Buttons tab.
  5. The Windows folder icon displays.
  6. Right-click the Windows folder icon and deselect the Visible option.
    NOTE: If the Visible option is deselected, the Windows folder icon is not displayed in the Easy Buttons tab.
  7. Click Save .
  8. Restart the Desigo CC application.

NOTE: You cannot see the Easy Buttons tab in the Operating mode of Applications > Notification if you delete the Windows folder icon in the Easy Buttons tab. In that you must first restore the Easy Buttons tab.

 

Associate a Graphic with an Easy Button

  1. Ensure that one or more incident templates or notification templates are configured.
  1. Select Applications > Graphics.
  2. The Graphics tab displays.
  3. Click Create New and then select New Graphic .
  4. In the editor that displays, associate a notification template or incident template with a graphic by performing the following steps:
    • Select Notification > Incident Templates. Drag the incident template onto the Untitled section.
    • Select Notification > Notification Templates. Drag the notification template onto the Untitled section.
  5. A symbolic representation of the graphic displays.
  6. Click Save .
  8. In the Save As dialog box, enter a name for the configured graphic in the Name field and click Save .
  9. The graphic is created and must now be associated with an easy button.
  10. Navigate to Applications > Notification and click the Object Configurator tab.
  11. Open the Related Items expander.
  12. From the Graphics node, drag the saved graphic to the Related Items expander.
  13. Click Save .
  1. The graphic is associated with an easy button.
