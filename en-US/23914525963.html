(Option 3) Create a Script to Schedule a Fixed Link Email Attachment

You want to configure a script that will carry out the Execute command on the Application Name: Advanced Reports property based on the provided parameters settings. The report output will then be emailed in the specified format to the specific recipients from the Address Book.

  1. System Manager is in Engineering mode.
  2. You have set the user’s execution credentials for an external web application fixed link. See Related Topics below for more information.
  1. In System Browser, select Application View.
  2. Select Applications > Logics > Scripts.
  3. Click the Script Editor tab.
  4. Click Save As .
  5. In the Save Object As dialog box, select the main Scripts folder or any subfolder under it as the destination for saving the new script.
    a. In the Name field, enter a name, for example, WebAppReportScript.
    b. Click OK.
  6. The new script is added to System Browser and automatically selected. The Script Editor tab displays its code.
  7. In the Editor expander, enter the script code to execute a command. For Script examples, see Related Topics below.
  8. Click Save .
  1. When you start the script, the report is executed as configured.
