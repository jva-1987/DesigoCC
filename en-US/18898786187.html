Honeywell Esser Symbols

When integrating the Honeywell Esser extension, the following symbols are available in graphics.

DYN_3D_Esser_AutomaticDetector_None_Horizontal_001_150

  • Default symbol for automatic detectors.
  • Symbol background changes color according to the event category color.

DYN_3D_Esser_AutomaticZone_None_Horizontal_001_150

  • Default symbol for automatic detector zones.
  • Symbol background changes color according to the event category color.

DYN_3D_Esser_ManualCallPoint_None_Horizontal_001_150

  • Default symbol for manual detectors.
  • Symbol background changes color according to the event category color.

DYN_3D_Esser_ManualZone_None_Horizontal_001_150

  • Default symbol for manual detector zones.
  • Symbol background changes color according to the event category color.

 

The blinking effect on symbols is not supported in this integration.
