Configure the Message Template

  1. Select Applications > Notification > Notification Templates.
  2. Select the message template that you want to edit.
  3. Depending on the template you want to configure, click the Message Editor tab.
  4. Open the General Settings expander and configure the parameters for a message template in the Message Editor.
  5. Specify the values to configure Control Options such as priority or escalation in the Message Editor.
  6. Open the Recipients expander and perform the following steps to add the recipient to the message template:
    a. Select the required recipient from the pre-configured list of recipients.
    b. Click Add.
    c. Select the Mandatory check box to make the associated recipient a mandatory recipient of the message template.
  7. Click Save .
  8. The message settings are saved.
  9. Click the Content Editor tab and perform the following steps:
    a. To configure the content for a text message, specify the values in the Text Content expander.
    b. Configure the audio and other file content for the message in Message Content Editor.
    c. The message template is configured.
