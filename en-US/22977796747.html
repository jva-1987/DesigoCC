Create the Adapter under the Network

You can configure multiple adapters under one SORIS network. Skip this step if you want to use an existing adapter.

  1. Select Project > Field Networks > [SORIS network].
  2. In the Object Configurator tab, click New and select New SORIS Adapter.
  3. In the New Object dialog box, enter a name and description, and click OK.
  4. The newly created adapter is added to System Browser. The adapter starts automatically as a Windows service.
  5. Select the newly created adapter and click the SORIS tab.
  6. In the URL field, enter the IP address of the computer from which the network adapter is run using the following format: https://<IP address>:<port number>.
    NOTE: In the executable window, you can check the correctness of the IP address. The default port number is 8080.
  1. In the Operation tab, the Online property indicates Connected and the property URL displays the IP address configured.
