Validated Schedule Objects Migration

The following section explains validation scenarios in the case of Migration and Restoration of schedule objects (such as Management station scheduler, calendars, and scheduled data points).

Migration Use Case

When the Installed Client Management station scheduler, Management station calendar, and scheduled data points are created with different validation profiles and the Migration process is started with appropriate validation input then after the completion of the Migration process, the following is the output in Flex Client:
-Validation is invoked with the highest validation profile amongst Schedules, Calendars, and the scheduled data points.
-Corresponding Flex Client Management station scheduler, Management station calendar and scheduled output points have the same validation profile set as before starting the migration process. Also, for each validation output a log is created in the History tab indicating the migration process is started.
- After migration, in Flex Client, if the migrated Management station scheduler and Management station calendar are modified (adding a new data point), then in Flex Client a log is created in the History tab indicating the migrated Management station schedule, Calendar, or data point is modified accordingly.
While deleting a validated migrated object, Orphan logs are generated.

Restoration Use Case

Consider a case, where the Management station scheduler, Management station calendar, and scheduled data points of different validation profiles are migrated from Install Client to Flex Client.
Now you want to restore the same Management station schedule, Management station calendar, and scheduled data point back to Install Client.
In this case, the Restoration process is started with appropriate validation input.
- The validation dialog is invoked with the highest validation profile amongst the Schedules, Calendars, and scheduled points, based on the backed-up state of the validation profile for all relevant objects.
- After completion of the Restoration process, the corresponding Install Client Management station scheduler, Management station calendar, and scheduled data points validation profile are restored to the original validation profile that was set before starting the migration process. Also, a log entry is generated indicating the Management station scheduler, Management station calendar, and scheduled data points are restored successfully.
