Integrating the IEC 61850 Devices

Scenario

You want to integrate SICAM IEC 61850 devices with the management platform.

For background information, see the reference section.

 

Prerequisites

  • Powermanager IEC 61850 Extension Module is installed and added in the project. Refer Installing IEC 61850 Adaptor Extension Module.
  • You have license for IEC 61850 and activated it.
  • System Manager is in Engineering mode.
  • In System Browser, select Application View.

For SICAM Q100 devices, ensure the IED name in the devices is set to default “SICAM_Q100_01MEAS”.

For SICAM Q100 devices, firmware version of 2.6 and above are supported.

Device cannot be created if proxy is enabled.

After making any changes to an IEC 61850 device through a browser, update the device in the Powermanager application.
