Field System (FS) Alarms

Field system alarms are generated at the level of the subsystem, panel or automation station. Alarm tables are used to map these field events to Desigo CC alarm classes.

Set up a Field System Alarm

  1. In System Browser, select Project > System Settings > Libraries > [... ] > [object model].
  2. Select the Models & Functions tab.
  3. In the Properties expander, select the property (for example, Alarm, Fault).
  4. In the Alarm Configuration expander, select the Valid check box.
  5. Select the Field system option.
  6. From the Alarm table reference drop-down list, select the alarm table you want to apply.
  7. Repeat steps 3 to 6 for each property as needed.
  8. Click Save .
  1. The field system alarm is configured on the property. The corresponding FS check box is selected in the Properties overview.
