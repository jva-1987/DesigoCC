IBase System Information Properties

When you install IBase, a System Information node is added to System Browser under: Application >Advanced Application > System Information.

When you select System Information, the Extended Operation tab displays a list of properties that you can set to configure the behavior of the IBase report.

Output Filename1)

Location and name of the IBase report generated.
The default path is
[Installation Drive:]\[Installation Folder]\[Project Name]\shared\scripting\IBaseReport
The naming convention of the IBase report is SiteInfo_[CSID]_[System Name].xml.

Signature recalculation Interval

Provide the time interval when the checksum report is to be generated with new data
Values can be 0 to 4.where
0 indicates that each time the checksum will be calculated during each report execution.
1 indicates checksum will be calculated weekly. However checksum data will always be included in the IBase report based on the last executed report.
2 indicates checksum will be calculated in two weeks. However checksum data will always be included in the IBase report based on the last executed report.
3 indicates checksum will be calculated in three weeks. However checksum data will always be included in the IBase report based on the last executed report.
4 indicates checksum will be calculated in 4 weeks. However checksum data will always be included in the IBase report based on the last executed report.
