Configuring the TJM CMMS Event Manager

Scenario

The communication between TJM CMMS and Desigo CC is managed by a dedicated service, the TJM NORIS Event Manager, which you must install and run on a networked PC that can connect to TJM CMMS and to Desigo CC. Typically, you can install it on the Desigo CC server station. For background information, see the reference section.
