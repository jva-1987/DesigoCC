Display Web Applications in the Application Viewer

  1. System Manager is in Operating mode.
  2. A system object was configured to display some associated external web applications when selected.
  1. In System Browser, select the system object whose external web applications you want to display (for example, Field Networks).
  2. The Application Viewer tab displays. It may show the associated external web application or the list of the external web applications available for the selected object.
  3. (Optional) If the Application Viewer tab displays the list of external web applications assigned to the selected object, you may want to navigate through them as follows:
    • Click the name of the desired external web application.
  4. The corresponding external web application displays.
  5. Click Show and select another application.
  1. The corresponding external web application displays.
