Multimedia Collection Integration

The Multimedia Collection extension makes it possible to integrate media files (MP3 for audio files, MP4 for videos, and HTML pages with video) into Desigo CC. When an operator selects a multimedia collection object in System Browser, buttons display for playing back the media files included in that collection.

A default media collection, containing media files about the different disciplines (Mass Notification, Energy and Power Reporting, Room Automation, and the mobile app) is deployed with the Multimedia Collection extension installed.

In addition, new multimedia collections can be created for specific purposes.

When you configure a multimedia collection, you can set the media files that it contains, the buttons available for playing back those files, and the background layout image (JPG).

Multimedia Collection Backup

Engineers\customers use Headquarters promotional videos that are installed with the Multimedia Collection extension, and a backup is not required.

Region\Country librarians or OEM add their own videos customizing the libraries of the Multimedia Collection extension, and a backup is not required.

Media files are not saved with the backup of the project since they are stored in the GMSMainProject directory. Therefore, after installation, if project librarians add project-specific media files to the project multimedia collection, a manual backup is required.
