Setting Up Block Command Macros

Scenario

Block-command macros are macros that broadcast the same command—for example, Acknowledge—to all the devices on a specific BACnet fire network. For background information, see Block Command Macros.

Prerequisites

  • System Manager is in Engineering mode.
  • If block commands macros are used for handling fire events, affected commands were already enabled in the Event Rights expander for the involved operators’ group scope. See Configuring Event Rights.
