Assign Event Rights

  1. In the Scope Rights expander, the Event check box in the Command Groups section must be enabled to activate the event rights.
  1. In the Security tab, select the appropriate user group.
  2. Open the Event Rights expander.
  3. Click the Splitter button to hide the Operation/Related Items pane.
  4. Select the Categories check box for the corresponding alarm category, for example Emergency, to be enabled for this user group:
    • Show
    • Acknowledge
    • Reset
    • Silence / Unsilence
    • Close
  5. Select or clear the required functions. By default, all check boxes are checked when you create a new project or upgrade a 3.x project.
  6. Repeat steps 4 and 5 for each alarm category.

NOTE:
When you change the event rights for a user group, the changes take effect immediately, even if those users are currently logged on and using the system.
