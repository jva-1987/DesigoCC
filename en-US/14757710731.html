Configuring STT20 or STT11 on Cerloop

Scenario

You want to configure Desigo CC for connecting to an STT11 or STT20 fire system on a Cerloop network via NK823x.
For background information, see the reference section.

Prerequisites

  • The following extension modules are installed and included in the active project:
    • CMSI STT20
    • Legacy systems via NK8000
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
