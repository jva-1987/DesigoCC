Create the Modbus Network and associate it with the Modbus Driver

  1. Select Project > Field Networks.
  2. Object Configurator displays.
  3. Click New Object and select New Modbus Network.
  4. The Create New Object dialog box displays with Child type pre-defined to Modbus Network.
  5. Do the following:
    a. Enter a name and description.
    b. Click OK. The Modbus Network node appears in the System Browser.
  6. Select the Modbus Network and thereafter select the Modbus tab.
  7. Open the Network Settings expander.
  8. From the Monitored by driver drop-down list, select the modbus driver you want associated with the network.
  9. Click Save .
  1. The modbus driver is associated with the modbus network.
