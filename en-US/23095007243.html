Managed Meters Toolbar

The Managed Meters toolbar allows you to perform the following operations:

 

Name

Description

New

Create a new managed meter.

Delete

Delete an existing managed meter

Save

Save a managed meter configuration.

Save As

Save the configuration of the selected managed meter as a new managed meter.

Apply Historization

Historizes the metering point, path, media input, and offset information.
