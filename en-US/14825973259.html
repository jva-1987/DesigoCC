Configuring Notifier ID3000 Event Commands Macros

Scenario

You want commands for zone, detector, and module events to also work from Event List. For this purpose, you must create specific macros in Desigo CC. For background information, see Notifier ID3000 Events.

Prerequisites

  • Desigo CC is configured to integrate the Notifier ID3000 control panel. For instructions, see Integrating Notifier ID3000.
  • System Manager is in Engineering Mode.
