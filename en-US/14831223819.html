Fibrolaser III Symbols

When integrating the Fibrolaser III extension, the following symbols are available in graphics:

Zone Symbols

DYN_3D_Fibrolaser_Zone_None_Horizontal_001_150

  • Default symbol for the Fibrolaser zone.
  • Only the line changes color according to the event category color.
  • To increase the width of the displayed line, this symbol can be stretched horizontally directly from the graphic (in the Graphic Editor).

 

DYN_3D_Fibrolaser_Zone_None_Horizontal_002_150

  • Both the line and the background change color according to the event category color.
  • To increase the width of the displayed line, it is mandatory not to stretch the symbol from the graphic, but to modify the LineWidth parameter available when selecting the symbol in the Graphic Editor.

 

DYN_3D_Fibrolaser_Zone_AverageTemperature_Horizontal_001_150

  • Only the line changes color according to the event category color.
  • The symbol also displays the average temperature of the zone.
  • To increase the width of the displayed line, this symbol can be stretched horizontally directly from the graphic (in the Graphic Editor).

 

 

The blinking effect on symbols is not supported in this integration.
