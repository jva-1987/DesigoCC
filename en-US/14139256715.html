Migrating and Upgrading NK8000 Units

Upgrading NK823x Units from V2.x or V3.x

When you upgrade Desigo CC, perform the following procedure for all NK823x units:

  • Checking and Updating NK823x Firmware and Kernel

Migrating NK823x Units from MM8000 Installations

On existing MM8000 installations, the following NK823x versions can be migrated:

  • MP4.50
  • MP4.60
  • MP4.70
  • MP4.80
  • MP4.81

Upon migrating from MM8000 systems, perform the following procedures:

  1. (Recommended) Update the NK823x units to latest version using the appropriate Hotfix (contact DMS8000 customer support).
  2. Configure the NK8000 Network: follow the integration workflows as necessary.

Unsupported features in NK823x for Desigo CC compared to NK823x for DMS8000

  • Local and network-wide NK8000 interactions
  • Secure communication (CEI 79 with encryption)
  • Firewall and routing functionality
    NOTE: In replacing NK823x with Scalance S615 as a router, refer to the application guide Scalance S615 Router for SAFEDLINK Fire Networks (A6V13357580).
  • Logging
  • Network redundancy
  • Web Server
  • Time synchronization from NTP server
  • Modbus gateway functionality
  • SNMP
  • IEC 60870-5-104 gateway functionality
