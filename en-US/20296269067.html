SAP iPro Events

The following is the list of the events available in Desigo CC for the SAP iPro connection.

In Management System > Servers > Main Server > SAP iPro System, in Extended Operation:

  • SAP iPro Connection Status (Normal, Fault): connection between the Event Manager service and SAP iPro.
  • Service Connection Status (Normal, Fault): connection between Desigo CC and the Event Manager service.
