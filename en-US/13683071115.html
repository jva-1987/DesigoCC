Management Station Schedules

You can create daily or weekly schedules for your management stations, and a management station can run multiple calendars or schedules at the same time.
Management station schedules and calendars run only if the management station is running.

Management station schedules have only one binary output (value active or inactive). You can assign several objects to that output.
For each assigned object, you can select the mode of action for the output value.
Additionally, management station schedules can process both BACnet and non-BACnet object types (BACnet schedules process only BACnet object types).

You can also create exceptions to schedules. When a management station exception is ON, it overrides the weekly schedule.
When the exception is OFF, control returns to the weekly schedule.

Values are displayed as per value scaled units (if configured).

This section provides the general reference information on management station schedules.
