Configure Web Rules for InfoCenter

For each InfoCenter report you want to run, you must create an Application Web Rule.

  1. System Manager is in Engineering mode.
  2. The default View is set to Show Description.
  3. You have run the Template Information Report
  1. Select Project > System Settings > Application Link Rules.
  2. The Rule Editor displays.
  3. Select the Rule Editor tab.
  4. Open the Trigger Conditions expander.
  5. In System Browser, from any view, drag any system node onto the Path area.
    There is no restriction on the type of node or object to use.
    TIP: You can create a custom view that lists all the reports. Then, when you configure a Web Link trigger, use an object from the custom view as the trigger object for that report. For information on creating a custom view, search the online help for “Views.”
  6. From the menu that displays, select Add new elements and subtree.
  7. In the Output expander, from the Application drop-down list, select InfoCenter Web Application.
  8. The default parameters display and are blank, except for the Format field, which defaults to html.
  9. In the Report Template field, enter the path to the parent directory where the InfoCenter report templates are saved:
    • Standard reports: type: GMS_InfoCenter\<Name_of_Report.rptdesign>.
      NOTE: The GMS_InfoCenter directory is created with you synchronize the Advanced Report templates.
  10. Using information from the Template Information Report that was generated in the previous procedure, complete the following fields:
    • Template ID
    • Control ID (as many as are present)
    • Additional Information
      NOTE: Depending on the report type, the parameters will vary. See the individual Standard Reports and Custom Reports for more information.
  11. Click Save As .
  12. The Save Object As dialog box displays.
  13. Enter a name that reflects the report and a description for the rule.
  14. Click OK.
  15. The new web rule appears in the Application Link Rules folder.
  16. (Optional) Repeat Steps 1 – 12 to create additional web rules for the report templates stored in the GMS_InfoCenter\<Name_of_Report.rptdesign> folder. If you need to see the parameters associated with each report template, refer to the desired report in the InfoCenter reference section.
