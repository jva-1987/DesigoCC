Linking Web Service Application and Flex Web app to project

  1. You have created a Web Service Application and Flex Client Application.
  1. Open SMC and navigate to your project.
  2. Open Power Monitoring tab.
  3. Select Web Service Application from WSI Web App dropdown.
  4. Select Flex Client Application from Flex Web App dropdown.
  5. Click Save .
  1. Web Service application and flex web app are linked to project.
