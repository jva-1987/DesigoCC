Creating the System Topology

Under system topology, states for servers, drivers, databases, printers, and automation stations are displayed optically on the graphics page.

  1. Click New Graphics .
  2. Draw the network structure using Lines .
  3. Click Save .
  4. Applications > Graphics is selected.
  5. Select Applications > Graphics > [selected graphics hierarchy].
  6. In the Name text field, enter a unique name for the graphics page.
  7. Click Save.
  8. Create the following graphic objects Automation station, Printer, Network, Project Backup and HDB backup, as per the instructions in the sections below.

NOTE:
A Viewport of the graphics page must be created once all objects are created on it.

 

Automation Stations

  1. From the Options tab, in the Symbol options group, from the drop-down list select 2D+.
  2. In System Browser, select Management View.
  3. Select Project > Field Networks > [Network name] > [Automation station].
  4. Drag the object to the graphics page.
  5. The automation station is now placed .
  6. Select Text and enter the Description (for example, AS01 North).
  7. Repeat Steps 3 to 5 for each available automation station.

 

Printer

  1. A printer is created and visible in System Browser.
  2. In System Browser, select Management View.
  1. Select Project > Management Station > Servers > Main Server > [printer name].
  2. Drag the object to the graphics page.
  3. The printer symbol is now placed .
  4. (Optional) Select Text and enter a Description (for example, Trend printer 1).
  1. The printer is created and displays all information supported in Online mode, including offline, paper jam, etc.

 

Network

  1. In System Browser, select Management View.
  2. Select Project > Field Networks > [Network name].
  3. Drag the object to the graphics page.
  4. A generic network symbol is placed .
  5. Select Text and enter the Description (for example, Desigo BACnet network).
  6. Repeat Steps 2 to 4 for each available network.

 

Driver

  1. From the View tab, in the Display Views group, select Library Browser.
  2. Enter the following library filter: *Object_None*.
  3. The generic graphics symbol for text display is displayed as a search result.
  4. Drag-and-drop the object Global_Base_HQ_1\DYN_All_Generic_Display_Object_None_Central_001 to the graphics page .
  5. In System Browser, select Management View.
  6. Select Management > Management Station > Server > Main Server > Drivers > [driver name].
  7. Drag-and-drop the object reference to the generic Graphics symbol.
  8. The driver reference is added to the Object Reference text field.
  9. Select Text and enter the Description (for example, Desigo BACnet network driver).
  10. Repeat Steps 2 to 4 for each available driver.

 

Project Backup

  1. From the View tab, in the Display Views group, select Library Browser.
  2. Enter the following library filter *BackupState*.
  3. The graphics symbol backup status is displayed as a search result.
  4. Drag-and-drop the object Global_Base_HQ_1\DYN_All_Generic_Display_Object_BackupState_Central_001 to the graphics page .
  5. In System Browser, select Management View.
  6. Select Project > Management Station > Server > Main Server.
  7. Select the Extended Operation tab and then the Backup Status property.
  8. In Properties > Symbols Instance, drag-and-drop the object reference to the Object Reference dialog box.
  9. The project backup reference is added to the text field object reference.
  10. Select Text and enter the Description (for example, Project backup).
  11. Repeat Steps 3 through 8 for additional information (for example, Last successful backup date).

 

HDB Backup

  1. In System Browser, select Management View.
  2. Select Project > Servers > Main Server > History Database > Archive Groups > Backup.
  3. Drag the object to the graphics page.
  4. A generic database symbol is placed .
  5. Select Text and enter the Description (for example, Backup, HDB database).

 

Creating a Viewport

  1. The System Topology page is completed and open.
  1. In the Viewports group, click Create Viewports .
  2. Click Manual Viewport .
  3. Create a viewport that is the same size as the graphics page.
  4. Click Save .
  5. The viewport is saved below the corresponding graphics page and is displayed in System Browser.
  6. In System Browser, select Viewport.
  7. The Viewport properties dialog box opens.
  8. In System Browser, select Management View.
  9. Select Project > Field Networks > Own Views and in Properties > Advanced, drag-and-drop the applicable folder to the Link References text field.
  10. Close the Viewport Properties dialog box.
  1. The folders Field network and Management system are linked to the viewport. The graphics page System Topology opens if the user selects one of these nodes.

NOTE:
When changing the graphics page System Topology, the viewport System Topology must be re-saved.
