Confirm Windows Services

To open the Windows Services applet, you need to do the following:

  1. You have started the project.
  1. From the Windows Start menu, select Control Panel > Administrative Tools > Services.
  2. The Windows Services applet opens.
  3. In the Windows Services applet, confirm that the following services are present with a status of started:
    • GMS_WCCILpmon_[Project Name]
    • GMS HDB Service (only on server)
    • GMS Closed Mode Service
    • GmsSmcProjectDataService (only on server)
    • SQL Server ([Instance Name]) and SQL Server Browser (MS SQL Server services for the History Database, if the database is deployed on the server)
    • WWCIILdist.exe (used in Distributed system configurations)
    • BT Licensing Server
    • World Wide Web Publishing Service
    • OPCEnum (belongs to OPC Server)
