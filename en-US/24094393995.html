Dalux FM Troubleshooting

Error/Issue

Situation

Do the following...

Dalux FM Connection Status Fault

The Event Manager service cannot communicate with the Dalux FM system.

  • Check network connections.
  • Check configuration (see Configure Dalux FM Event Manager Parameters).
  • Check Dalux FM status.

Service Connection Status Fault

The Event Manager service cannot communicate with the Desigo CC server.

  • Check network connections.
  • Check service status.
  • Check configuration (see Configure Dalux FM Event Manager Parameters).
