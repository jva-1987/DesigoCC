Associate a Coverage Area Symbol with an Object Model

You have a coverage area symbol that you want to associate to an object model.

  1. In System Browser, select Management View.
  2. Select Project > Field Networks > [device].
  3. In the Object Configuration tab, double-click the text Object Model.
  4. The Models and Functions tab displays.
  5. Click the Models and Function tab, and then scroll down to the Symbols section, so that it is visible on the screen.
  6. In System Browser, select the Manual Navigation check box.
  7. Navigate to the symbol library folder where the symbol you want to associate to the object model is saved.
  8. Right-click the Symbol folder, and from the menu that displays select Send to Secondary Pane.
  9. The Symbol Browser displays in the Secondary pane, and displays the symbols from the folder you selected in System Browser.
  10. Drag the symbol you want to associate to the object model and drop it into the Symbol section of the Models and Functions tab.
  11. The symbol is now associated with the object model.
  12. To make the symbol the default symbol of the object model, right-click the symbol in the Symbols pane, and from the menu available on right-click, select Set as Default.
  13. Click Save .
  1. You have now associated the symbol with the object model and made it its default symbol.
