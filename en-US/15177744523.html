Complete the Upgrade Planning Requirements

Once you have obtained and verified the software distribution, you need to complete the upgrade planning requirements.

 

 

Before going to a job site to install Desigo CC, you must verify the installation environment, gather the necessary information and prepare for the upgrade.

Perform the following planning and preparation steps in the same order as listed.

  1. Log on with Admin Rights to the machine where you are performing the upgrade using custom installation.
  2. Verify the Installation Environment for the following including: (See Verify the Installation Environment)
    • Installed Operating System
    • Microsoft Office Version
    • Virtualization and VM Software
    • PDF Viewer
    • Supported Web Browser
    • Installation Folder
    • AutoCAD Files
    • Hardware Requirements
  3. Verify the existing IIS installation, if any. If no IIS component is installed, you can use the upgrade workflow to install and configure IIS.
    Otherwise, you must manually install and configure IIS on the already installed setup type before proceeding with the upgrade.
  4. Verify the existing Network Settings.
  5. Verify the existing Firewall Settings and Software.
  6. Verify the already configured Virus Scanners.
  7. Install .Net 4.7.2, .Net 6.0.26, and .Net 8.0.1.
    In the absence of .Net 4.7.2 or the earlier compatible version, Installer does not launch and you cannot proceed with the upgrade process.
  8. Verify that no other installation is in progress. Otherwise, the Installer waits till the already running installation is completed.
  9. Verify that the Microsoft Windows is updated to the latest Security Updates (See Verify the Installation Environment > Microsoft Windows Security Updates).
