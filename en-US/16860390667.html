Editing a Message Template Before Launch

The following procedures outline the process to edit a message template before launch.

 

Change the Message Template State Using Command

  1. System Manager is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Notification > Notification Templates.
  3. Select the configured message template to change its state.
  4. In the Operation tab, navigate to the State command and click the appropriate state.
    • Click Active to receive the notification when the event is triggered.
    • Click Inactive when you do not want to receive the notification when the event is triggered.
  1. The message template state is changed.

Specify the Variable Value

  1. One or more message templates are configured.
  2. A variable during message template configuration in Engineering mode is specified.
  3. System Manager is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Notification > Notification Templates.
  3. A list of available notification templates folders and message templates displays.
  4. Select the message template.
  5. Select the Initiate Incident tab.
  6. Open the Selected notifications expander.
  7. Select the message template.
  8. Select Message Content and Control.
  9. Select Show Text Content.
  10. Select the variable from the Variable column and specify the corresponding value in the Value column.

 

Specify the Audio Content

  1. One or more message templates are configured.
  2. System Manager is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Notification > Notification Templates.
  3. A list of available templates displays.
  4. Select the message template.
  5. Select the Initiate Incident tab.
  6. Open the Selected notifications expander.
  7. Select the message template.
  8. Select Message Content and Control.
  9. Select Show audio content.
  10. Click Browse.
  11. The Open dialog box displays.
  12. Select the corresponding location and select the audio file.
    NOTE: Notification supports MP3 and WAV audio formats.
  13. Click Open.
  1. The selected audio file is uploaded.

 

Specify the Other File Content

  1. One or more message templates are configured.
  2. System Manager is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Notification > Notification Templates.
  3. A list of available templates displays.
  4. Select the message template.
  5. Select the Initiate Incident tab.
  6. Open the Selected notifications expander.
  7. Select the message template.
  8. Select Message Content and Control.
  9. Select Show other file content.
     NOTE: Other file content can be videos, audio, images, .pdf file, .WORD file, xls.
  10. Click Browse.
  11. The Open dialog box displays.
  12. Select the corresponding location and select the other file content.
  13. Click Open.
  1. The selected other file content is uploaded.

Add Recipients

  1. One or more configured message templates are configured.
  2. System Manager is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Notification > Notification Templates.
  3. A list of available templates displays.
  4. Select the message template.
  5. Select the Initiate Incident tab.
  6. Open the Selected notifications expander.
  7. Select the message template.
  8. Select Message Content and Control.
  9. Select Show recipients content.
  10. Select the recipients from the available recipients list.

 

Specify Control Options

  1. One or more configured message templates are configured.
  2. System Manager is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Notification > Notification Templates.
  3. A list of available templates displays.
  4. Select the message template.
  5. Select the Initiate Incident tab.
  6. Open the Selected notifications expander.
  7. Select the message template.
  8. Select the Message Content and Control.
  9. Select Show control options content.
  10. Edit the required fields.
