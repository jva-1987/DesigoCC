SystemObject Type

The SystemObject data type provides the result of the getOnlineSystems operation. Such type can be used in filters, in addition to the numeric system ID. For further reference, see the following table.

Property

Type

Description

systemId

Integer

Identifier of the system.

systemName

String

Name of the system.

Example of Use

var systems = getOnlineSystems();

for (var i = 0; i < systems.length; i++) {

    console(systems[i].systemId + " " + systems[i].systemName)

}

var currentSystem = getCurrentSystem();

console(currentSystem.systemName);
