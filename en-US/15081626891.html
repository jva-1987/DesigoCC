Scalance Symbols

When integrating the Scalance extension, the following symbols are available in graphics:

DYN_2D_Scalance_Interface_None_001_150

  • Default symbol for interface data in the graphic template.

 

DYN_2D_Scalance_IP_None_001_150

  • Default symbol for IP data in the graphic template.

 

DYN_2D_Scalance_System_None_001_150

  • Default symbol for system data in the graphic template.

 

DYN_3D_Scalance_Device_None__Horizontal_001_150

  • Default symbol for Scalance device objects.
  • Symbol background changes color according to the Summary Status color propagated from the point itself, or the points at lower levels.

 

The blinking effect on symbols is not supported in this integration.
