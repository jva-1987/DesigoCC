Add BACnet Points to the Trend Viewer

  1. The Trends application is in the Edit mode.
  1. In System Browser, select Management View.
  2. Navigate to Project > Field Networks > [network type] > Hardware > [device].
  4. Perform either of the following steps to add the points to the Trend Viewer:
    • Select the Manual navigation check box and thereafter, select the BACnet points. Drag the selected BACnet points to the Trend Viewer.
    • Select the BACnet points and thereafter, select New Trend from the Related Items pane to open the Trends application in the Secondary pane. Click Edit Trend .
  2. The BACnet points display in the Trend Viewer.
