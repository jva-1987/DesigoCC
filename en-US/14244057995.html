Additional Procedures for SX Open Migration

Delete an OPC Network

  1. System Manager is in Engineering mode.
  2. The Schindler OPC network is on the Field Networks list.
  1. In System Browser, select Management View.
  2. Select Project > Field Networks > [Network name].
  3. Click the OPC tab.
  4. Click Delete Object .
  5. A confirmation message displays.
  6. Click Yes.
  1. The System Manager status bar displays Node deleted successfully.

NOTE:
The network structure is deleted in the Management View only. It still remains in the Logical View and the User View. In those views, you have to delete the structure manually. Select the desired view and then the Views tab. Click Remove to delete the structure tree and use the purge function to delete all unused objects.

Purge Unused Objects

  1. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select the desired view and select the top object.
  3. Click the Views tab.
  4. In Views, select the top object.
  5. Click Purge Aggregators and then Yes.
  6. All objects without a child object are deleted in this view.
  7. Repeat the Steps 2 through 5 for all views.

Stop OPC Driver

  1. Select Project > Management station > Servers > Main Server > Driver [Driver to be deleted].
  2. In the Extended Operation tab, next to the Manager status, click Stop.
  1. A confirmation message appears.
