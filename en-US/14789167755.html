Meteoblue Weather Commands

The following describes the commands of the Meteoblue Weather objects.

 

For details about the SORIS adapter properties and commands, in SORIS, see Properties and Commands of SORIS thingAdapter Objects.

Commands on New Location Objects

 

Property

Commands

Location Point

  • Create
    Creates the new location with the best match data retrieved and any relevant field automatically populated. When data for a new location is created, all the properties are reset to blank or to any default value.
    NOTE: This command retrieves also any settings that cannot be configured manually, that is: Country, Region/District, and Time Zone.
  • Refresh
    Refreshes location data when changing, for example, latitude for a location name.

Name

  • Write
    Saves the name you entered.
  • Get
    Retrieves the other location details after a location name is entered.
    NOTE: This command retrieves also any settings that cannot be configured manually, that is: Country, Region/District, and Time Zone.

Latitude

  • Write
    Saves the latitude value entered.

NOTE: You can set the latitude value manually or you can retrieve it when you enter a name for a location and click Get.

Longitude

  • Write
    Saves the longitude value entered.

NOTE: You can set the longitude value manually or you can retrieve it when you enter a name for a location and click Get.

Altitude

  • Write
    Saves the altitude value entered.
    NOTE: You can set the altitude value manually or you can retrieve it when you enter a name for a location and click Get.

Forecast Days

    Saves the setting selected.
