Display Customer Specific Logo on the Advanced Reports

  1. You have deployed the Advanced Reporting templates to the Tomcat Server using template synchronization. During the synchronization process, the ProductLogo.bmp, SiteLogo.bmp, VendorLogo.bmp, and none.bmp files are also copied to [Installation Drive]:\Program Files\Apache Software Foundation\Tomcat 9.0\webapps\gms-birt\shared\Icons\Site.
  1. Select Project > System Settings > Libraries > L1-Headquarter > Global > Advanced Reporting > Advanced Reporting Design.
  2. Click the Template Configuration tab.
  3. Customize the Advanced Reporting Design folder as per your customization level, for example L4-Project.
  4. The Advanced Reporting library is created as per your customization level, for example L4-Project.
  5. In the Windows Explorer, navigate to ..\[Customer Project]\libraries\ Global_AdvancedReporting_Project_1
  6. Delete all folders except AdvRpt.
  7. Navigate to ..\AdvRpt.
  8. Delete the AdvR_GMS_Admin.zip file.
  9. Extract the contents of the AdvR_GMS_AdvancedReporting.zip using the Extract Here option of the unzip tool and thereafter delete the AdvR_GMS_AdvancedReporting.zip file.
  10. In the folder with the extracted contents, navigate to ..\GMS_AdvancedReporting.
  11. Delete all the files and folders except the shared folder.
  12. Navigate to ..\GMS_AdvancedReporting\shared and delete all the folders except the icons folder.
  13. Navigate to icons\site and delete all files except SiteLogo.bmp.
  14. Edit the SiteLogo.bmp file using MSPaint or any other image editing software.
  15. Add the customer-specific logo to this file by ensuring that you retain the width and height of the existing image in the file.
  16. Navigate back to ..\[Customer Project]\libraries\ Global_AdvancedReporting_Project_1\AdvRpt.
  17. Zip the GMS_AdvancedReporting folder to a .zip format.
  18. Rename the zip file to AdvR_GMS_AdvancedReporting.
  19. Delete the GMS_AdvancedReporting folder.
  20. Synchronize the templates.
  21. On the Advanced Reporting Configuration page, click Advanced.
  22. Next to Layout Setting, click Edit.
  23. The Parameter dialog box displays.
  24. Navigate to the Logo to display in report drop–down list and select SiteLogo from the list.
  25. Click OK.
  1. The Advanced Reports will now start displaying the customer specific logo.
