Import the RDF302 configuration file

Import the RDF302 configuration file

  1. The CSV file for import is available locally or over the network.
  1. Select Project > Field networks > [RDF302 Network].
  2. On the Import tab, select Browse and the configuration file.
  3. The Source list contains the items that you can import.
  4. (Optional) Open expander Hierarchies to link one or more configuration file data structures to the optional views:
    a. In the System Browser, select the optional view (e.g.Logical view or Physical view).
    b. Drag the folder of the optional view that you want to link as a base folder to an available data tree listed in expander hierarchies.
    NOTE: The configuration of the hierarchies cannot be changed after import.
  5. In the Source list, select and then click Import.
  6. The configuration data is available in the System Browser.
  7. The import log can now be displayed with the enabled Import Log button activated.

In case of problems, see RDF302 Troubleshooting for more information.

 

NOTE: Incorrect interpretation of messages
If the device configuration is changed after the first import, the configuration file must be re-imported to resynchronize Desigo CC. Subsystem messages may be misinterpreted If the most recent configuration file is not imported. An unidentified alarm is generated when a message is received without a corresponding data point.
