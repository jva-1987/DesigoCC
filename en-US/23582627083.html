Set the Tomcat Memory to a Higher Limit

The default Tomcat memory of 2500 MB is insufficient to generate cache for a system having more than 500 managed meters configured. In such situations, you must set the Tomcat memory to a higher limit, for example, 6000 MB.

  1. From the Services dialog box, stop the Tomcat service (Apache Tomcat 9.0 Tomcat9).
  2. Navigate to the following path ..\Program Files\Apache Software Foundation\Tomcat 9.0\bin and double-click the Tomcat9w.exe file.
  3. The Apache Tomcat 9.0 Tomcat9 Properties dialog box displays.
  4. Select the Java tab and navigate to the Maximum memory pool field.
  5. Set the memory to a higher value, for example, 6000 MB, and click Apply and OK.
  6. Re-start the Tomcat service.
