NK823x Subsystems Compatibility

The following subsystems can be connected to the NK823x Ethernet Port:

  • CS11 AlgoRex (via CK11 and MK7022/Cerloop)
  • FC700A Synova (via FG700A)
  • STT20 (via MK7022/Cerloop)
  • STT11 (via MK7022/Cerloop)
  • MK7022/Cerloop
  • SI400 Sintony (via serial line)
  • DF8000 (via I2C and/or serial line), see DF8000 Reference

NK823x Onboard I/O Specifications

3x Digital Inputs:

  • Suitable to NO or NC dry contacts or open collector signals
  • Max forward current: 7 mA at 33Vdc
  • Over voltage protection on input lines: 1K V
  • Input voltage limit: 33V

1x Relay Output:

  • NO or NC contact
  • Max 1A at 30 Vdc

Unsupported features in NK823x for Desigo CC compared to NK823x for DMS8000

  • Local and network-wide NK8000 interactions
  • Secure communication (CEI 79 with encryption)
  • Firewall and routing functionality
    NOTE: In replacing NK823x with Scalance S615 as a router, refer to the application guide Scalance S615 Router for SAFEDLINK Fire Networks (A6V13357580).
  • Logging
  • Network redundancy
  • Web Server
  • Time synchronization from NTP server
  • Modbus gateway functionality
  • SNMP
  • IEC 60870-5-104 gateway functionality
