Installing and Starting the OPC UA Client Adapter

Scenario

You want to connect Desigo CC to the OPC UA Client adapter. For background information, see the reference section.

Prerequisites

  • After installation, the OPC_UA_Client.msi installer package is available in [Installation Drive]:\[Installation Folder]\GMSMainProject\AddSW\OPC_UA_Client.

In case you want to upgrade the adapter software (SORIS OPC UA Adapter), you must first uninstall the previous version, and then install the latest version.

In Desigo CC V8, the OpcUaAdapter.Settings.xml file content has been extensively modified. Consequently, if you want to take advantage of the new settings as well as keep the old settings, before uninstalling the previous version of the adapter, do the following:
1. Rename the existing OPC UA adapter settings XML file (OpcUaAdapter.Settings.xml or [adapter service name].Settings.xml).
2. Uninstall the previous version of the adapter software.
3. Install the latest version of the adapter software.
4. Manually copy the content from the renamed file to the new OPC UA adapter settings XML file (OpcUaAdapter.Settings.xml or [adapter service name].Settings.xm) created after you installed the latest version of the adapter.
5. Delete the old OPC UA adapter settings XML files.
