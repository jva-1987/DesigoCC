Apply Color to a Pipe Element

By default a Linear gradient is applied to the Pipe and Pipe Coupling elements to create a 3 dimensional look. Use the Brush Editor gradients stops to apply color.

NOTE: This is a recommendation; there are numerous ways to apply color.

  1. You have selected a Pipe or Pipe Coupling element on the canvas.
  1. From the Brush Editor view, click on the first Gradient Stop, and from the color palette, select a color for the pipe.
    • For example, to represent a hot water pipe, select: Red.
  2. Leave the center gradient stop as-is, to retain the 2-dimensional look.
  3. Click on the last gradient stop and from the color palette select the same color, Red.
    • Or, for example, if the pipe is a mixed (hot and cold )water pipe, select Blue.
