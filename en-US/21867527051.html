Properties and States

You can view Building X Connector properties and states in the Extended Operation tab by selecting Project> Management System > Servers > Main Server > Building X Connector.

Property

States

Operational State

Not Ready: Not operational

Ready: Operational

Configuring: Retrieving current Building X configuration

Registering: Registering Building X application

Stopping: Transitioning to Not Ready

Communication State

Fault: Coincides with Operational State ‘Not Ready’

Disconnected: The Building X Connector is not communicating with the Building Connect Agent

Disconnecting: The Building X Connector is in the process of terminating communication with the Building Connect Agent

Connecting: The Building X Connector is in the process of initiating communication with the Building Connect Agent

Connected: The Building X Connector is communicating with the Building Connect Agent

Building X Data State

Not Available: Coincides with Operational State ‘Not Ready’

Download Pending: Awaiting configuration download from Building X

Download in Progress: Downloading configuration from Building X

Download Failed: The Building X configuration is not aligned with the management station configuration (discovery or upload required)

OK: Normal

Discovery State

No Data: Discovery operation has not been executed

Discovery In Progress: Discovery operation in progress

Changes Detected: Discovered Desigo CC configuration differs from the current Building X configuration (upload required)

Upload In Progress: Discovered configuration being uploaded to Building X

No Changes: Discovered configuration matched Building X configuration

Indications State

Disabled: Alarms and values will not be sent to Building X

Enable: Alarms and values will be sent to Building X
