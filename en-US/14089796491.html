Commanding Properties for Multiple Objects

  1. You want to simultaneously command properties for multiple objects of the same type.
  1. In System Browser, navigate to the locations containing the objects you want to select, and then select them.
  2. Click the Extended Operation tab.
  3. Click the triangular symbol in the lower-right-hand corner on the icon next to the property you want to command.
  4. The table row expands to show multiple instances of the property—one for each of the objects selected.
  5. Do one of the following:
    • In the top row of the property, enter the value for the property, and then click Change.
    • In the top row of the property, click the button that represents the action you want to take, for example, Command, Release, Change, Out of Svc, Enable or Ack All. If the action has additional fields, complete them, and then click Send.
  1. The system displays the status of the command.
