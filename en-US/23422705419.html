Create a Custom Project Library

  1. Create a custom library by navigating to Project > System Settings > Libraries according to your customization levels.
  3. Navigate to the library that you created and click the Library Configurator tab.
  4. Create an object model folder in it by selecting New Object Model Folder menu option in the list that displays on clicking Add new object . Also, create an import rules folder by selecting the New Modbus Custom Import Rules menu option. For information, see Adding Blocks to a Library.
  3. The library folder is created along with the Import Rules and Object Model folders.
