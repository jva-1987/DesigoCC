Event Handling Rules

Event Settings is a feature that lets you configure rules to partially automate the handling of certain events. For instructions, see Configuring Event Handling Rules.

When an event occurs in a Desigo CC station or Flex Client, the operator typically needs to manually perform certain steps (such as, opening Event List, selecting the event, displaying relevant documents or floor plan graphics) to handle the event.

In some situations--such as on unmanned Desigo CC stations or Flex Client, or with unskilled operators--it can be useful to configure some of these steps, so that they happen automatically for certain events. For example, you might want to:

  • Configure Event List so that it opens automatically, with the new incoming event already selected.
  • Automatically invoke assisted or investigative treatment without the operator having to do so manually.
  • When the event is closed, automatically close the Assisted Treatment or Investigative Treatment, or automatically deselect the event so that it can be cleared from Event List.

Event settings rules let you configure these types of actions, and also specify for what types of events (for example, only Life Safety events), in what circumstances (for example, only at night), and for which user groups or station groups the automated rule should apply.
