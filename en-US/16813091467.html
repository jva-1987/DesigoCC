Launching a Message

Perform either of the procedures to launch a message from a command, recipient or recipient groups.

 

Launch a Message from a Command

  1. System Manager is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Notification > Notification Templates.
  3. Select the configured message template to be launched.
    NOTE: The message template may contain system variables, event variables or user variables.
  4. Select the Extended Operation tab.
  5. Open the Launch Message expander.
  6. Select the enabled launch message command and do one of the following:
    • If the selected message template does not contain a user variable, click Launch.
    • If the selected message template contains a user variable, enter variable text in the Variable Value field and click Launch.
  1. The selected message template is launched under the new ad-hoc incident.

 

Launch a Message from Recipients or Recipient Groups

  1. System Manager is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Notification > Recipients.
  3. A recipient list displays in the Recipients expander.
  4. Select a recipient or a recipient group for which an incident is to be initiated.
  5. The recipient details display in the General > System user details > Recipient User Devices and Groups Membership expander.
  6. Select the Related Items tab.
  7. Open the Operating Procedure expander.
  8. Click New Notification Message.
  9. A Secondary pane with the incident details displays.
  10. Open the Incident Details expander. Do one of the following to launch the message:
    • Select Initiate new ad-hoc incident to launch the message under ad hoc incident.
    • Select Launch under existing incident to launch the message under one of the existing configured incidents.
      NOTE: In case of Launch under existing incident, the messages associated with the corresponding incident are also displayed along with the ad hoc message. Select any notification from that list to be launched along with the ad hoc message.
    • Select the required incident template from the Copy from Incident Template drop-down list.
      NOTE:       In case of Copy from Incident Template, the messages associated with the corresponding incident are also displayed along with the ad hoc message. Select any notification from that list to be launched along with the ad hoc message.
  11. Do one of the following:
    • To launch an ad hoc message with the incident, follow step 8 onwards.
      NOTE: Under Message Content and Control, check the Show recipients message content section of the ad hoc message to ensure that the selected recipient or the recipient group is included in the recipients list or not.
    • For associating another notification, follow step 6 onwards.
  12. Select Applications > Notification > Notification Templates.
  13. A list of available notifications displays.
  14. Drag the required notification on the launch group in the Secondary pane.
  15. Open the Message Content expander.
  16. Click the notification to view and edit the details. For more information, refer to Editing a Message Template Before Launch.
  17. Open the Progress expander. Do one the following:
    • If the progress bar has reached 100 percent, follow step 10.
      NOTE: The progress reaches100 percent when all the details in the associated notification template are specified.
    • If the progress bar is less than 100 percent, then click Next.
      NOTE: The progress is less than 100 percent when certain details in the associated notification template are not met.
  18. The Message Details and Message Content display.
  19. Check for the symbol, and enter the required details in their respective fields.
  20. Click Initiate.
  1. The incident is initiated.

 

Launch a Message from a Node in the System Browser

  1. System manager is in Operating mode.
    NOTE: This applicable to Engineering mode also.
  1. In System Browser, select Application View or Management View.
  2. Select any node from System browser.
  3. The selected node options display.
  4. Select the Related Items tab.
  5. Open the Operating Procedure expander.
  6. Click New Notification Message.
  7. A Secondary pane with the incident details displays.
  8. Open the Incident Details expander and select Initiate new ad-hoc incident to launch the message under ad hoc incident.
  9. Check for the symbol, and enter the required details in their respective fields.
  10. Click Initiate.
  1. The incident is initiated.
