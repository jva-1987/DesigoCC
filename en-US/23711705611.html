Assign a Function for h,x diagram

A function must be assigned to the aggregator before the h,x diagram can be displayed in the graphic.

  1. The required data points are assigned to the h,x diagram in the View tab.
  2. Clear the Manual Navigation check box.
  1. In the System Browser, select Logical View.
  2. Select Logical > [Hierarchy] > [Ventilation plant] > [h,x diagram].
  3. Select the Object Configurator tab.
  4. In the Settings expander, select Function, and option FnctMollierChartPX from the drop-down list.
    NOTE: Select All to see all functions.
  5. Click Save .
  6. The ,odify function must be adapted for the extract air sensor is used for the h,x diagram instead of the room sensor.
  1. The h,x diagram is configured (is item 6 is not required) and can be tested.
