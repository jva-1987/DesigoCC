Troubleshooting OPC Server DA Status

Error

Situation

Solution

In the Operation tab, the OPC Server DA Status is No License.

The OPC license is not available.

Contact your System Administrator.

In the Operation tab, the OPC Server DA Status is No Configuration.

The OPC Scope or its configuration is missing.

Check that the OPC scope object is present in System Browser, and is properly configured. If not configured, add the scope rules (see Configure the Scopes for OPC Connectivity).

The OPC DA server is not present in System Browser.

See Configure the OPC DA Server).

In the Operation tab, the OPC Server DA Status is No Configurationandthe following error message displays: The OPC configuration is missing. Verify the Security Scope rights.

No scope is associated to the security OPC user group.

Check the OPC security settings (see Link Scopes to the OPC User Group).

In the Operation tab, the OPC Server DA Status is Failure.

Desigo CC OPC server executable file (Siemens.Gms.OPCServer.exe) is not found.

Contact your System Administrator.

An unexpected error occurs.

In the Operation tab, the OPC Server DA Status is Invalid Configurationandthe following error message displays: The OPC configuration is invalid. Verify the Security Scope rights.

One of the following:

  • In the Scope Rights expander, the Scope field contains [name of the system].
  • The associated scope is invalid.

Modify appropriately the OPC security settings (see Link Scopes to the OPC User Group).
