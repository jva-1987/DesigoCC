Creating a Project Library

  1. Select Project > System Settings > Libraries > L1-Headquarter > BA > HVAC (PX).
  2. Click Customize the entire library to a lower level .
  3. A library file is created specific to the project.
  4. Select Project > System Settings > Libraries > L1-Headquarters > BA > HVAC (PX) > Software Options and then select Hcrv Configuration.
  5. The Object Configurator tab displays.
  6. Click Save As .
  7. Select Project > System Settings > Libraries > L4-Project > BA > HVAC (PX) > Software Options.
  8. Enter a description (for example, Hvcr_floor_heating). By default, when you click the Description field, the same text contained in the Name field displays. You can optionally modify this text.
  9. Click OK.
  10. Repeat steps 4‒7, if you require different heating curve configurations for floor heating or radiators.
