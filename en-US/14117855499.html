Animation of a Coffee Machine

The HQ symbol libraries cover most building automation and control requirements. However, there are always customized objects not covered by the library, including:

  • Burners
  • Heat pumps
  • Heating boiler
  • Refrigeration machines
  • Tank displays
  • and so on

To illustrate how to animate such objects in a Desigo CC graphic, the following is an example of the most important functions of a coffee machine, for example:

  • Operating indicator (without operation)
  • event lamp
  • Operating element (operation in the Extended operation tab)
  • Position display (knob)
  • Level display (graphical)
  • Water temperature

NOTE:
This example shows the details of the animation functionality. Graphical detail functions are only explained where necessary. Additional information on the Graphics Editor, including shading, and so on, is available in the Graphics Editor .

 

Prerequisites:

  • System Manager is in Engineering mode.

 

Creating a Page and Adding an Image

  1. Desigo building automation and control import is done.
  1. In System Browser, select Application View.
  2. Select Applications > Graphics.
  3. The Graphics Editor opens.
  4. Click New Graphics .
  5. Copy an image of a coffee machine on the graphics page.
  6. In the toolbar, click Save .
  7. Applications > Graphics is selected.
  8. Select Applications > Graphics > [selected graphics hierarchy].
  9. In the Name text field, enter a unique name for the graphics page.
  10. Click Save.
  11. Click the View tab and select Properties and Evaluation Editor.

 

Operating Indicator

The operating indicator changes and indicates the present state of a digital input:

  • 0 = Transparent
  • 1 = Green

The digital input on the coffee machine cannot be operated in Online mode (indicator only).

  1. The page for the coffee machine is open.
  2. You require a digital input.
  1. In the Mode group, click Design .
  2. Select the Elements tab, and then select the Rectangle and drag-and-drop it to the graphics page.
  3. On the coffee machine, define the object size of the rectangle to be displayed.
  4. Select the Object.
  5. The Rectangle Properties dialog box opens.
  6. Select Rectangle Properties > Colors.
  7. Select Fill to define the color in Design mode and offline print.
  8. The Brush Editor opens to select colors.
  9. Select Stroke > Stroke Thickness and enter line thickness for the frame (0 = no frame).
  10. Select Advanced > Disable selection. Selecting it prevents operation of the graphic.
  11. All other default settings in the Rectangle Properties dialog box can be taken over for this example.

Animating the Object

  1. The object is highlighted.
  2. The Evaluation Editor opens.
  3. The Rectangle Properties dialog box is open.
  1. Select Colors > Fill.
  2. In the System Browser, select Logical View.
  3. Select Logical > [Network name] \...\ [Plant] > [Digital input].
  4. Drag-and-drop the object to the Expression column of the Evaluation Editor.
  5. In the Rectangle Properties dialog box, the Fill property displays the use in the Evaluation Editor.
  6. Define the properties in the Evaluation Editor:
    a. Select Property and Fill.
    b. Under Type, select Discrete.
    c. Under Type, select Reference.
  7. Enter the data in the first row:
    a. In the Condition column, enter the value 0.
    b. In the Value column, enter the value Leave blank (= Transparent).
  8. Click Insert new condition .
  9. A second row is created for the conditions.
  10. Enter the data in the second row:
    a. In the Condition column, enter the value 1.
    b. In the Value column, enter the value 1.
  11. Click Save .

Testing

  1. In the Mode group, click Test .
  2. Click the View tab and then select Value Simulator.
  3. All data points used in the graphic display.
  4. In Object Selection, select the corresponding data point, and in the Value column, enter either the value 0 or 1.
  5. The content of the rectangle changes the height per the set value.
  6. Click Runtime .
  1. The inserted object cannot be selected or operated in the Online mode.

NOTE:
Select Automatic and then select Run Value Simulator. The simulation is run as per the set option.

 

Event lamp

The operating indicator change indicates the present state of a digital input in the alarm state:

  • No alarm = Transparent or the corresponding background color
  • Alarm = Red flashing
  • (Optional) The Fault, Overridden and Out-of-Service states can be displayed for the same object.
  1. The page for the coffee machine is open.
  2. The operating indicator is created.
  3. You require a digital input.
  1. In the Mode group, click Design.
  2. Select the Elements tab, and then select the Rectangle and drag-and-drop it to the graphics page.
  3. On the coffee machine, define the object size of the rectangle to be displayed.
  4. Select the Object.
  5. The Rectangle Properties dialog box opens.
  6. Select Colors.
  7. Select Fill to define the color for the state Normal.
  8. The Brush Editor opens to select colors.
  9. Select Stroke > Stroke Thickness and enter line thickness for the frame (0 = no frame).
  10. You can use all the default settings in the Rectangle Properties dialog box for this example.

Animating the Object

  1. The object is highlighted.
  2. The Evaluation Editor opens.
  3. The Rectangle Properties dialog box is open.
  1. Select Colors > Fill.
  2. In the System Browser, select Logical View.
  3. Select Logical > [Network name] \...\ [Plant] > [Digital input].
  4. Select the Operation tab, and then the State Flag property.
  5. Drag-and-drop the object from Operation tab to the Expression column of the Evaluation Editor.
  6. The data point reference in the field is: x.x.x.digitalInput;.Status_Flags.
  7. In the Rectangle Properties dialog box, the Fill property displays the use in the Evaluation Editor.
  8. Define the properties in the Evaluation Editor:
    a. Select Property and Fill.
    b. Under Type, select Discrete.
    c. Under Type, select Reference.
  9. Enter the data in the first row:
    a. In the Condition column, enter the value &1.
    b. In the Value column, enter the value 1.
  10. (Optional) Click Insert new condition to display another state.
  11. A second row is created for the conditions.
  12. (Optional) Enter the data in rows 2‒4:
    • In the Condition column, enter the value &2, &4 or &8.
  13. (Optional) In the Value column, enter the value 1.
  14. Click Save.

Display Alarm Blinking

  1. The object is highlighted.
  2. The Evaluation Editor opens.
  3. The Rectangle Properties dialog box is open.
  1. Select Layout > Blinking.
  2. In the System Browser, select Logical View.
  3. Select Logical > [Network name] \...\ [Plant] > [Digital input].
  4. In the Operation tab, select the State Flag property.
  5. Drag-and-drop the object from Operation tab to the Expression column of the Evaluation Editor.
  6. The data point reference in the field is: x.x.x.digitalInput;.Status_Flags
  7. In the Rectangle Properties dialog box, the Blinking property displays the use in the Evaluation Editor.
  8. Define the properties in the Evaluation Editor:
    a. Select Property and Blinking.
    b. Under Type, select Discrete.
    c. Under Type, select Reference.
  9. Enter the data in the first row:
    a. In the Condition column, enter the value 0.
    b. In the Value column, clear the check box.
  10. Click Insert new condition to display another state.
  11. A second row is created for the conditions.
  12. Enter the data in the second row:
    • In the Condition column, enter the value 1.
    • In the Value column, select the check box.
  13. Click Save.

Testing

  1. In the Mode group, click Test.
  2. Click the View tab and then select Value Simulator.
  3. All data points used in the graphic display.
  4. In Object Selection, select the corresponding data point, and in the Value column, enter one of the following values:
    • 1 = Alarm
    • 2 = Fault
    • 4 = Overridden
    • 8 = Out of Service
  1. The content of the right corner changes the color as per set value. In alarm state, the rectangle is blinking.

 

Temperature Display

Displays the water temperature for the coffee machine's tank .

  1. The page for the coffee machine is open.
  2. The operating indicator for the coffee machine is created.
  3. You require an analog input.
  1. In the Mode group, click Design .
  2. Select the Elements tab, Text and then drag-and-drop it to the graphics page.
  3. On the coffee machine, define the object size of the text for display.
  4. Select the Object.
  5. The Text Properties dialog box opens.
  6. Select Rectangle Properties > Text.
  7. Define the properties:
    • Select Text and enter the description Temperature.
    • Select Text Type and the Formatted Value option.
    • Select Units and enter the description °C.
    • Select Precision and enter 1 (the value displays to one decimal point).
  8. Change additional text properties as needed, including size, bold face, color, and so on.

 

Level Indicator

The height of the water column is graphically displayed.

  1. The page for the coffee machine is open.
  2. The temperature display for the coffee machine is created.
  3. You require an analog input.
  1. In the Mode group, click Design .
  2. Select the Elements tab, and then select the Rectangle and drag-and-drop it to the graphics page.
  3. On the coffee machine, define the object size of the rectangle to be displayed.
  4. Select the Object.
  5. The Rectangle Properties dialog box opens.
  6. Select Colors.
  7. Select Fill to define the color in Design mode and offline print.
  8. The Brush Editor opens to select colors.
  9. Select Stroke > Stroke Thickness and enter line thickness for the frame (0 = no frame).
  10. All other default settings in the Rectangle Properties dialog box can be taken over for this example.

Animating the Object

  1. The object is highlighted.
  2. The Evaluation Editor opens.
  3. The Rectangle Properties dialog box is open.
  1. Select Colors > Clip Top.
  2. In the System Browser, select Logical View.
  3. Select Logical > [Network name] \...\ [Plant] > [Digital input].
  4. Drag-and-drop the object to the Expression column of the Evaluation Editor.
  5. In the Rectangle Properties dialog box, the Clip Top property displays the use in the Evaluation Editor.
  6. Define the properties in the Rectangle Properties dialog box:
    • Select Colors > Fill.
  7. The Brush Editor opens to select colors.
    • Select Colors > Stroke and enter Stroke = 0 for the frame.
  8. Define the properties in the Evaluation Editor:
    a. Select Property and the Clip Top option.
    b. Under Type, select the Linear option.
    c. Under Type, select Reference.
  9. Enter the data in the Min row:
    • In the Range column, enter the value 0.
    • In the Value column, enter the value 100%.
  10. Enter the data in the Max row:
    • In the Range column, enter the value 100.
    • In the Value column, enter the value 0%.
  11. Click Save .

Clip Top Functionality

For the clip top function, the maximum level or temperature (100% or 100°C) must be entered at minimum value. This is because the zero point of the display is at the top and the value moves downward (black arrow). The maximum value indicates the end of the display (0% or -30°C).

NOTE:
To display a tank object with a dynamic level indicator, you must draw the tank using a second inanimate background object. If selecting a frame for an animated object, it remains animated in the same ratio as the level indicator.

Testing

  1. In the Mode group, click Test .
  2. Click the View tab and then select Value Simulator.
  3. All data points used in the graphic display.
  4. In Object Selection, select the corresponding data point and in the Value column, enter the value of either 0, 50 or 100.
  1. The content of the rectangle changes the height per the set value.

NOTE:
Select Automatic and then select Run Value Simulator. The simulation is run as per the set option.

 

Position Indicator

The position of the corresponding coffee selection is displayed with a knob .

  1. The page for the coffee machine is open.
  2. The level indicator is created for the coffee machine.
  3. You require a multistate input.
  4. In View, in the Workspace tab, Display Grid and Snap are enabled.
  1. In the Mode group, click Design .
  2. Select the Elements tab, and the Ellipsis and drag-and-drop it to the graphics page.
    NOTE: For even easier object editing, avoid creating it on the image of the coffee machine.
    • Select the size of the circle (Layout > Width and Height = 24).
    • Using the Brush Editor, select the color and radiant of the knob.
  3. Create a second circle for the position indicator and define the properties:
    • Layout > Width and Height = 8
    • Colors > Stroke = White
    • Colors > Fill = Black
  4. Move the Position indicator to the knob .
  5. Highlight both objects and click Selection > Middle Alignment .
  6. The 0‒degree position is defined .
  7. Highlight both objects and click Selection > Group .
  8. In the Group Properties dialog box, define the pivot point for the knob:
    • Layout > Angle center X = 12 (half of the circle size)
    • Layout > Angle center Y = 12 (half of the circle size)
  9. The pivot point is defined .
    NOTE: Object grouping must be rescinded if the point is not defined correctly. As a result all definitions are lost for the grouped object in the Evaluation Editor.
  10. Move the groups knob indicator to the correct position on the coffee machine .
  11. Select the actual size of the knob on the coffee machine, for example Layout > Width and Height = 20.

Animating the Object

  1. The object is highlighted.
  2. The Evaluation Editor opens.
  3. The Group Properties dialog box is open.
  1. Determine the angle for each position for display (here for each of the cups):
  2. Select the knob at the green anchor .
  3. Turn the marking of the knob to the lower left cup .
  4. While rotating, a dotted frame is displayed using an angle line.
  5. In the Group Properties dialog box, in Layout > Angle the angle displays, for example, ‒45 degrees.
  6. Define the corresponding angle (60, 120 and 225 degrees) for all other cups.
  7. Select Group Properties > Layout > Angle.
  8. In the System Browser, select Logical View.
  9. Select Logical > [Network name] \...\ [Plant] > [Multistate input]
  10. Drag-and-drop the object to the Expression column of the Evaluation Editor.
  11. In the Group Properties dialog box, the Angle property displays the use in the Evaluation Editor.
  12. Define the properties in the Evaluation Editor:
    a. Select Properties and Angle.
    b. Under Type, select Discrete.
    c. Under Type, select Reference.
  13. Enter the data in the first row:
    • In the Condition column, enter the value 1.
    • In the Value column, enter the value -45.
  14. Click Insert new condition to display another state.
  15. A second row is created for the conditions.
  16. Enter the data in rows 2 through 4:
    a. In the Condition column, enter the value 2, 3 and 4.
    b. In the Value column, enter the value 60, 120 and 225.
  17. Click Save .

NOTE:
All definitions in the Evaluation Editor are lost for the grouped object if you have to ungroup the object. Prior to ungrouping, document the settings made in the Evaluation Editor.

Testing

  1. In the Mode group, click Test .
  2. Click the View tab and then select Value Simulator.
  3. All data points used in the graphic display.
  4. In Object Selection, select the corresponding data point and, in the column Value column, enter the value 1, 2, 3 or 4.
  1. The knob changes the display position as per the set value.

 

Operating

An operating object is required to switch on and off the coffee machine. Operation in the Online mode occurs in the Extended operation tab.

  1. The page for the coffee machine is open.
  2. The temperature display for the coffee machine is created.
  3. You require a digital output.
  1. In the Mode group, click Design .
  2. Select the Elements tab, and then select the Rectangle and drag-and-drop it to the graphics page.
  3. On the coffee machine, define the object size of the rectangle that will display.
  4. Select the Object.
  5. The Rectangle Properties dialog box opens.
  6. Select Colors.
  7. Select Fill to define the color in Design mode and offline print.
  8. The Brush Editor opens for color selection.
    NOTE: Select a color that comes closest to the object background. Selecting for operation is difficult if you select transparent.
  9. (Optional) Select Stroke > Stroke Thickness and enter the line thickness for the frame (0 = no frame).
  10. All other default settings in the Rectangle Properties dialog box can be taken over for this example.

Animating the Object

  1. The object is highlighted.
  2. The Rectangle Properties dialog box is open.
  1. Select Advanced > Selection reference.
  2. In the System Browser, select Logical View.
  3. Select Logical > [Network name] \...\ [Plant] > [Digital Output]
  4. Drag-and-drop the object to the Selection Reference field in the Rectangle Properties dialog box.

    NOTE: The Disable Selection check box must be cleared.
  5. Click Save .

Testing

  1. In the Mode group, click Runtime .
  2. Select the Operating object on the coffee machine.
  3. A red frame displays around the operating object and the properties display in the Extended Operation tab.
  4. Click Command.
  5. In the Value drop-down list, select On or Off.
  6. Click Send.
  1. The coffee machine is switched on or off.

 

Viewport Coffee Machine

Areas from the existing graphics page can also be defined as zoomed viewport.

  1. System Manager is in Engineering mode.
  2. The Design or Test mode is selected.
  3. The graphics page is completed and open.
  1. In the Viewports group, click Create Viewports .
  2. Click Manual Viewport .
  3. Define the viewport view size .
  4. In the Viewport Properties dialog box, select General > Description, and enter a description for the viewport.
  5. Click Save .
  1. The viewport is saved below the corresponding graphics page and is displayed in System Browser.

NOTE:
Functionality of the viewport can only be opened in the Application View in Online mode.

Testing

  1. The graphics page and viewport are created.
  1. Click Engineering.
  2. In System Browser, select Application View.
  3. Select Applications > Graphics > [graphics page].
  4. Click the Default tab.
  5. The graphics page is displayed in online mode.
  6. Select Applications > Graphics > [graphics page] > [Viewport].
  1. The viewport is displayed.

 

Assigning a User Guide

A document can be assigned to a graphics page (or other objects). In this manner, you can quickly access the user's guide in the event of a fault to the coffee machine.

  1. System Manager is in Engineering mode.
  1. In Windows Explorer copy (outside of Desigo CC) the user guide to the following folder: ... [Installation Drive]:\[Installation Folder]\[Project Name]\Documents.
  2. Switch back to Desigo CC.
  3. In System Browser, select Application View.
  4. Select Applications > Documents.
  5. The Documents application opens.
  6. On the right side, select Settings and then select the File option.
  7. Open the drop-down list and select the corresponding document.
  8. Only those documents are displayed that are located in the ...[Installation Drive]:\[Installation Folder]\[Project Name]\Documents folder.
  9. Select Applications > Graphics > [graphics page].
  10. Drag-and-drop the graphics page object to the Linked Objects dialog box.
  11. Click Save As .
  12. In the Save object as dialog box, enter the name and description.
  13. Click OK.
  1. The document is linked to the graphics page.
  2. In Online mode, the document link is displayed under Related Items under Document.

Testing

  1. The graphics page is created and a document is assigned to the page.
  1. Click Engineering.
  2. In System Browser, select Application View.
  3. Select Applications > Graphics > [graphics page].
  4. Click the Default tab.
  5. The graphics page is displayed in online mode.
  6. Click the document in the Related Items tab, under Document.
  1. The assigned document opens in the Secondary pane.
