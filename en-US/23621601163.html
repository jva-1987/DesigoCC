Consistency Check Scenarios

During the Custom, Semi-automatic, or Silent installation, the Installer performs the consistency check. Depending on the situation, the Installer either aborts the installation or proceeds with the installation and generates a installation log. The installation log stores messages (informational, warning, error, and fatal) about events that occur during the installation process.

NOTE:
You may see a few messages in the installation log. However, unless a fatal (irrecoverable) error occurs, the installation program completes the installation successfully.
The installation user interface indicates the success or failure of each installation attempt, and the installation log file includes an entry for each installation process.

To help you determine the cause of a failure, it is recommended to view the log file located at the path:
[system drive]:\ProgramData\[company name]\GMS\InstallerFramework\GMS_Installer_Log.

For a Semi-automatic installation, view the installation log file that has the extension_SEMI.

For a Silent Installation, view the installation log file that has the extension _SILENT.

The digital signature validation is performed for the executable files (such as .msi, .exe, .dll, .msp, .ps1, etc.) present in EM, GMS, Prerequisites folder, PostInstallation, and Languages folder which are either not digitally signed or are signed with certificates that has any of the below issues:

  • Certificate from the broken certificate chain
  • Certificate in the Certificate Revocation List (CRL)
  • Expired certificate
  • Self-signed certificate

For Errors mode: For the non-mandatory EM executable files which are not digitally signed or has any of the issues mentioned in the scenario are excluded from installation and an error message appears on the Welcome to the Desigo CC Install Wizard dialog box. However, Desigo CC installation will continue.
For mandatory EM executable files, an error message will appear. However, Desigo CC installation will not continue.

For Warnings mode: A warning message appears on the Welcome to the Desigo CC Install Wizard dialog box for the executable files of the package which are not digitally signed or are signed with the certificates that have issues mentioned in the scenario. However, Desigo CC installation continues.
