Integrating Advantronic AD300

Scenario

You want to configure Desigo CC for connecting to the Advantronic AD300 fire panel. For background information, see the reference section.

Prerequisites

  • The Advantronic AD300 extension is installed and included in the active project.
  • You prepared the configuration (CSV) file for the Advantronic fire panel you want to integrate. See Advantronic AD300 Configuration (CSV) File.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
