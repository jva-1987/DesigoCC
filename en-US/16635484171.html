Automatically Assign Missing Functions

Scenario: You want to assign missing functions automatically based on one or more reference plants. The object type and object name must be the same in order to perform automatic assignment using the Function Checker.

 

Flowchart:

 

Prerequisites:

  • System Manager is in Engineering mode.
  • Functions in the reference plant are fully assigned.

 

Steps:

1 – Apply Reference Plant for Assignment

Functions can be automatically assigned by assigning one or more functions defined as an object on a reference plant.

  1. In System Browser, select Logical view.
  2. Select Logical > [Hierarchy] >[Plant].
  3. Drag the reference plant to the Reference plants expander.

 

2 – Automatically Assign Missing Functions

  1. Click Automatically assign functions based on reference plants.
  2. The number of assigned functions are displayed.
  3. Click OK.
  4. Automatically assigned objects have state green with a asterisk.

 

3 – Display Assigned Functions in the Browser

  1. Click Display assigned function in the tree.
  2. The functions assigned to an object are displayed behind the =>.
  3. Check the assigned functions.
  4. Click Save .
  1. Edited objects change the state from green to black.
  2. The plant can now be used as a reference plant for editing.
