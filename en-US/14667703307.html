Reset Zoom Ratio to 100 Percent

In order to keep the aspect-ratio during a zoom operation so that you do not need to use scrollbars to view the dashboard, the dashboard page must be zoomed to 100% prior to resizing.

For resetting the zoom factor to 100% and then resizing the dashboard in an external web browser, use the provided Zoom menu options for that browser.

  1. You have a dashboard page in the primary pane.
  • To reset the dashboard zoom ratio to 100% - From the Application Viewer toolbar, click Reset Zoom .
