Customize Fonts

  1. Select a label or a table in a report definition.
  2. Click the Layout tab.
  3. The Font group box displays.
  4. Do one of the following to apply a font, font style, or font size.
    • Click the drop-down arrow in the Font group box.
    • Right-click the report element, and select Font.
  5. The Font dialog box displays.
  6. Select the font type, font size, and font style as desired.
  7. (Optional) Select the Underline check box to apply it to the label. The Underline option is unavailable when you select a table in the report definition.
  8. Click OK.
