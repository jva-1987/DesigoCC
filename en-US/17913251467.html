Adjusting the SORIS Driver Settings

  1. System Manager is in Engineering mode.
  2. System Browser is in Management View.
  1. Select Project > Management System > Servers > Main Server > Drivers > [SORIS driver].
  2. Select the SORIS tab.
  3. In the Polling threads field, specify the value for the polling volume.
