Station Configuration Parameters

In Engineering mode, when you select a station object in System Browser (Desigo CC server (Main Server)/client/FEP/web server or Flex Client anonymous/identified web client), you can configure its settings in the System Management tab.

 

Host name

Label indicating the station address. You can either enter a host name or select it from the network (Browse) or obtain the host name of the current station (Current Station). This setting is required.
The host name for the Main Server is set during system installation (field is read-only).
This setting is not available for the Default Web Client, which connects anonymously.

 

Type

Indicates the type of station:

  • Desigo CC server, client, FEP, or web server
  • Web client hosting the Flex Client application (anonymous or identified web client).

This setting is available only when configuring a new client (client, web server, or web client hosting the Flex Client application). Once you save your configuration, you cannot change this setting.

 

Client profile

From the drop-down list, you can select the profile that will be applied to Desigo CC stations.
For instructions, see Set Profile for an Installed Station. For background information, see Standard Client Profiles.

 

Flex client profile

From the drop-down list, you can select the profile that will be applied to the anonymous or identified web client hosting the Flex Client application.
For instructions, see Set the Client Profile for a Flex Web Station. For background information, see Flex Client Profiles.

 

Account to use after logoff

By default, no default user is assigned to the identified web client hosting the Flex Client application, but this drop-down list allows you to select one of the available software account users for which the option default user was enabled.
You can set the default software account for an identified web client so that the Flex Client application will be displayed and active even when no operator is logged on.
For instructions, see Setting the Personal Use of Flex Client and Configuring a Flex Station With Default User.

 

Certificate

This field displays only when you configure an identified web client hosting the Flex Client application, and this is a required setting. It sets the host certificate that the web client will use to identify itself to the Desigo CC server.
You can select from a store location (Select certificate) a certificate that was imported into the server computer or you can remove the current certificate (Clear selection) and select another certificate.
For instructions, see Specify Host Name and Host Certificate on Station Object.

NOTE: You cannot save the changes if control station was enabled but a certificate is not yet selected, or the selected certificate was already assigned to another web client.

 

Personally used

It modifies the behavior when OIDC users log off. If the Flex Client is not Personally used, OIDC users are also logged off from the Identity Provider when logging off from the Flex Client. Instead, if Personally used is set, OIDC users are not logged off from the Identity Provider when logging off from the Flex Client.
This option is available only for the identified web clients, and enables only after a host certificate is assigned to an identified Flex Client.
For instructions, see Setting the Personal Use of Flex Client.

 

Control station

It sets whether the current Desigo CC station or the identified web client hosting the Flex Client application is able to control fire panels. For instructions, see Enabling Control Station in Danger Management.

This setting is not available (the check box is read-only) for the Desigo CC and Flex Client anonymous web clients.
This setting is not applicable to Building Automation Client Profiles.

 

Closed mode

Sets whether the current Desigo CC station operates in a special, locked-down mode where the user cannot shut down the system or access the operating system functions. Default is Closed mode check box cleared. For instructions, see Setting Up a Closed Mode Installed Station. For background information, see the reference section.

This setting is not available (the check box is read-only) for the Flex Client web app. To allow closed mode in Flex Client, see Installing Flex Client as Desktop App and Setting Up a Closed Mode Flex Station.
