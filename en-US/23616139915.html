Message Template

This section describes additional message template procedures.

Update or Delete a Message Template

  1. At least one message template is configured.
  1. Select Applications > Notification > Notification Templates.
  2. Select the message template to be updated or deleted.
  3. The Message Editor tab displays.
  4. Perform any of the following steps:
    • To update a message template, update the parameters in the Message Editor or Content Editor tabs. For more information, see Configure the Message in Creating and Configuring Notification Template.
    • To delete a message template, click Delete . The feature is disabled for Windows App clients.
  5. Click Save .
  1. The selected message template is updated or deleted.

Delete an Audio/Other File Content from a Message Template

  1. The message template with audio or other file content for the respective language is configured.
  1. Select Applications > Notification > Notification Templates.
  2. Select the message template whose audio or multimedia file you want to delete.
  3. Open the Content Editor tab.
  4. To remove a file from a particular language, select that particular language from the Language drop-down list and click Delete.
  5. To remove a file from ALL languages, select ALL languages from the Language drop-down list and click Delete.
  6. Click Save .
  1. The audio or other file content is deleted from the message template.

Configure Audio Content for a Message

  1. The message template to which you want to add the audio content is created.
  1. Select Applications > Notification > Notification Templates and select the message template to which you want to add audio content.
  2. Click the Content Editor tab.
  3. Open the Audio Content (refer Configure Audio Content for a Message section) expander.
  4. Select the Enabled check box.
  5. From the Language drop-down list, select the required language.
  6. To add audio content, do one of the following:
    • To upload a pre-recorded audio file, click Browse, navigate to the required file and click Open.
    • To record a message, click . In the Select Input dialog box, select the method of audio input and press to start the recording. To stop recording, click . Click Exit once done. The audio file displays in the Name field.
    • The audio file displays in the Name field.
      NOTE:
      Windows Media Player must be installed to play the audio files after launching the audio messages.
      For Windows App client, audio files can only be added to the message template from the Windows Media Library using drag-and-drop.
  1. Audio content is added to the message template.

Configure Other File Content for a Message

  1. The message template to which you want to add the other file content is created.
  1. Select Applications > Notification > Notification Templates and select the message template to which you want to add audio content.
  2. Click the Content Editor tab.
  3. Open the Other File Content (Open the Audio Content (refer Configure Other File Content for a Message section) expander.
  4. Select the Enabled check box.
  5. ) expander.
  6. Select the Enabled check box.
  7. (Optional) Select Include Dynamic Graphics check box to enable the loading of dynamic multimedia content in a message template.
  8. In the Language field, select the required language from the drop-down list. Text content can be localized. If no localized content is available for a language, the All Languages content will be sent.
  9. To add other file content to the message template, do one of the following:
    • To upload a other file content from your local file system, click Browse, navigate to the required file and click Open.
      NOTE: A PowerPoint presentation must be converted into a video file using PowerPoint 2010 before it can be uploaded. For information on supported file formats, see the Other File Content.
    • (Optional) Click View to view the uploaded file.
  10. Click Save .
  1. Other file content is added to the message template.
