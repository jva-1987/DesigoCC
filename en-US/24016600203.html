Connect to the Schindler Server

The workflow can be performed independent of Desigo CC engineering, but must take place prior to creating the Schindler export file.

We recommend consulting with a Schindler service technician on the installation.

NOTE:
The OPC tunneller must be installed when the Schindler server is not on the same computer as the Desigo CC server.

  1. The OPC installation and OPC configuration is completed on the Schindler server.
  2. The Schindler server is not on the same computer as the Desigo CC server.
  3. The following information must be available for the OPC client installation:
    −Schindler server IP address with port number, for example, 139.16.48.54:56766
    − Data source name, for example, Schindler_Tunnel_ServerDA
  1. In Windows explorer, select the file dataFEEDOPCSuite_V_xxx.exe on your installation media.
  2. Right-click and select Run as Administrator.
  3. Select the language.
  4. Click Next to continue.
  5. Accept the terms of the License Agreement and click Next.
  6. Click Next and then Install.
  7. Click Finish.
  8. The OPC suite is installed and the license manager opens.
  9. The tunnel symbol displays on the status bar.
  10. Select the Schindler client license.
  11. Close the license manager.
  12. Right-click the tunnel symbol on the status bar and select Open Softing.
  13. The configuration window opens.
  14. Click Start.
  15. The tunnel symbol changes on the toolbar.
  16. Click Start Configurator.
  17. The installation wizard opens.
  18. In the Data Source pane, select Tunnel Server.
    a. Click Add a new data source.
    b. Click Next.
  19. The Settings window opens.
  20. Do the following:
    a. In the Data source name field, enter the default name Schindler_Tunnel_ServerDA.
    b. Select DA Tunnel Server.
    c. Click Next.
  21. Do the following:
    a. In the Address field, enter the IP address and port, for example, 139.16.48.54:56766.
    b. Click Next.
  22. Do the following:
    a. Click Connection test.
    b. If properly connected, click OK.

    c. Click Next.
  23. Click Next, Next and then Finish.
  24. The OPC connection is installed.
  25. In the Data Destination pane, select OPC Client (but not the OPC UA Client).
    a. Select Activate OPC.
    b. Enter a name in the Program ID field.
  1. The OPC installation is completed. Now create the Schindler export file.
