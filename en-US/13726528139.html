Analyzing Log Data

Scenario: You can analyze the collected Log data by using a number of filters.

  1. In System Browser, select Log Viewer.
  2. Click Log Viewer .
  3. The Log Viewer opens.
  4. Continue with the further workflows as needed.

 

Printing Log Grid Contents

  1. Click Print .
  2. In the Print dialog box, select the desired printer and specify the printing configurations.
  3. Click Print.

NOTE:
The size of the font may vary depending on the number of columns in the log view grid.

 

Exporting a Log View Definition

  1. Click Export .
  2. The Browse for Folder dialog box displays.
  3. Browse for the desired location and click OK.
  1. A confirmation message displays. The Log View Definition is exported and saved.
