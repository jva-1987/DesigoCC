Troubleshooting Operator Tasks Templates

Error/Issue

Situation

Solution

The template file has been tampered. Please choose a valid configuration file.

You selected a template, and the Configuration file field in the General Settings expander shows the template name in red. This means that this template might have been tampered, and it will be no longer be available in the Add list, in the Tasks tab in Operating mode.

  • If you are sure that the template configuration file was not tampered because you edited the changes in it, click Save, and then click Yes to validate the template.
  • If you are not sure about the origin of the configuration file, do one of the following:
    • In the General Settings expander, click Browse and select a valid JSON configuration file.
    • Customize the template at the allowed customization level, and then in the General Settings expander, click Browse and select a valid JSON configuration file.
