[Default] Deletion After Custom Retention Period

Scenario
You want to permanently delete unneeded data entries from the Short Term Storage.

  1. System Manager is in Engineering mode.
  2. System Browser is in Management View.
  1. Select Project > Management System > Servers > Main Server > History Database > Archive Groups > [data group].
  2. Click the Operation tab.
  3. Min Retention Span defines the minimum length of the time period and Min Retention Unit defines the associated unit.
  4. Click the Extended Operation tab.
  5. The properties for the corresponding data group display.
  6. Type data into the following fields and confirm with Set:
    • Delete Schedule Start defines the first time to delete data.
    • Delete Schedule Period defines the length of the time interval and Delete Schedule Unit defines the associated unit.
      NOTE: The greater the interval, the more data is saved during the maximum time span.
    • Max Retention Span defines the maximum length of the time period and Max Retention Unit defines the associated unit.
  7. Select Delete Scheduled and click Enable to enable the process.
  1. Deletion is defined for the applicable data group and starts as per the defined date and time setting.
