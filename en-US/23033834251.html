(Optional) Create Additional Views

The discovery function imports data points into Management View. In configurations that require the logical or a user view the imported data points must be manually linked to be available also to these views.

Build up the View Hierarchical Structure of Levels

  1. The logical or user view is available.
  1. Select the desired view from the System Browser drop-down list.
  2. In the tree structure, select the root node of this view.
  3. Click Add Aggregator and select the appropriate aggregator.
  4. In the New Object dialog box, do the following:
    a. Enter a name and a description.
    b. Click OK.
  5. Repeat the preceding steps to add more levels to create the required hierarchy structure.
  6. Click Save .

Populate the Levels with Links to System Objects

  1. Select the desired view from the System Browser drop-down list.
  2. In the tree structure, select the root node of the view.
  3. The view builder workspace displays in the Views tab.
  4. In System Browser, select the Manual navigation check box.
  5. Select Management View from the System Browser drop-down list.
  6. The workspace is fixed. The view does not change when selecting the data points.
  7. The System Browser is in Management View and the Views tab displays the selected view.
  8. Select the object that you want to link and drag-and-drop it to the desired level in the Views tab.
    NOTE: If you want to link a subtree (object plus all its children), in the Views tab select the Link children check box. If you want to link only an individual object, in the Views tab clear the Link children check box.
  9. Repeat the preceding steps to link more objects or subtrees.
  10. Click Save .
  1. The objects are now available in System Browser, linked to the desired view.
