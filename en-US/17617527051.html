Additional Procedures

  1. The Configure Room Management are fulfilled.
  1. Select Project > [Network name] > [Building] > [Floor].
  2. Select the room or rooms.
  3. Drag the room to the room information list.

Expand default naming

  1. There are device names in the room manager editor's room information list that deviate from default naming. Unknown device names are indicated in blue font.
  1. Click Extended settings .
  2. Click Add.
  3. Enter the device names.
  4. Click OK.
  5. Click Rescan nodes .
  1. The devices can be edited as usual.

Automatically assign room objects

Scenario: There are no or only partial assignments of the room objects to room.

  1. Highlight the room or rooms.
  2. Click Configure selected rooms.
  3. The dialog box Automatically configure all selected rooms displays.
  4. Do the following to assign only the desired configuration:
    • Select the Lighting check box.
    • Select the Shading check box.
    • Select check box Brightness/presence/collision detectors.
    • Selection option By room or By segment.
      - By room: All switches are assigned to each segment (so that each switch is able to switch on or off).
      - By segment: All inputs for a segment are assigned to the same segment.
  5. Click OK.
  1. The assignments are made to the room.

 

Apply room filter

Scenario: You moved multiple rooms to room information list for editing. You want to limit the view of rooms to improve the overview.

  1. Enter the filter term, for example, Room 1 in field Filter rooms. The filter applies to the description and automatically uses a wildcard after the term. For example, the term "Room 1" also displays the results for rooms 10, 11, 100.
  2. Click Search .
  1. Only rooms matching the term display.

 

Display lighting or shading references

Scenario: You want to view the input references in the room information list. The description refers to lighting inputs, but can be used on any type of input.

  1. In the assignment list, select the Lighting inputs tab.
  2. Double-click the appropriate lighting input.
  1. The references used are displayed in the room information list.

 

Hide rooms and segment type

Scenario: You want to hide or show the room or segment type to improve the overview.

  1. Select check box Show room/segment types.
  2. Displays additional information.
  3. Clear check box Show room/segment types.
  1. Hides the additional information.

 

Display edited equipment

Scenario: You want to hide or show device information to improve the overview during editing.

  1. Select Show edited equipment.
  2. Displays devices within a segment.
  3. Select Show edited equipment.
  1. Hides the devices.

 

Check the changes to the assignments and display

Scenario: You want the check assignment changes.

  1. In the room information list, highlight the corresponding switch.
  2. Select the Detail log tab.
  3. Click Select columns .
  4. Select the Action details check box.
  1. The changes are displayed in the Detail log tab.

 

Create change report for the last 24 hours

Scenario: You must document the changes. The standard report generates a change report for the last 24 hours.

  1. In System Browser, select Application View.
  2. Select Applications > Reports > Configuration.
  3. Select report HQ Room Editor Changes.
  4. Select the Reports tab.
  5. The select report HQ Room Editor Changes displays.
  6. Enter a localized text (1, 2 and 3) for the standard report.
  7. Check the settings in the name filter Elements Added and Elements Removed:
    a. Right-click format template Elements Added (Date/Time) and select Filter > Name Filter.

    b. In the Name Filter dialog box, check the predefined path for validity and change as needed. In Valid column must show OK.

    c. Also check the settings in the format template Elements Removed.
  8. Click Save as . Enter a new name for the report for future use.
  9. Select Run .
  10. The report is generated for the last 24 hours.
  11. Click Generate and display PDF .
  1. The reports can be saved or printed.

You can change the time period in the standard report, for example, to generate a report for longer periods.

 

Create a backup of the current configuration

Scenario: You need to change an existing configuration. You want to create a backup of the existing configuration to restore it in the event of a mistake. A backup can be made on each level of an object (building, floor, or room).

  1. Click Export configuration .
  2. Enter a name for the file in the dialog box Export configuration. The select object level is proposed as the name.
  3. Click Save.
  4. A confirmation is displayed.
  5. Click OK.
  1. The XML file is saved to folder [Installation drive]:\[Installation folder]\GMSProjects\[Project name]\devices\FRM_Configurations.

 

Restore the last configuration

Scenario: Undo the change to the configuration.

  1. Click Import configuration .
  2. Select the file in the dialog box Import configuration.
  3. Click Open.
  4. A confirmation is displayed.
  5. Click OK.
  6. The configuration from the backup is updated on the project, but not yet loaded to the automation stations.
  7. Click Save room configuration on device.
  1. The configuration from the backup is loaded to the automation stations and can be tested.
