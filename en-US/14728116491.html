Installing the Application Host Base

  1. On the computer where you copied the Desigo CC software, navigate to …\InstallFiles folder, right-click the Gms.InstallerSetup.exe file, and then select Run as administrator.
  2. From the Install Wizard window, click Next until you arrive at the Setup Type Selection screen.
  3. Under Feature Selection, select Modify.
  4. Open the Connectivity expander and select the Web Services check box.
  5. Click OK to return to the Setup Type dialog box.
  6. Click Next until you reach the Ready to Install the Program dialog box.
  7. Click Install.
  8. When the installation program is complete, the Installation Complete dialog box displays.
  9. Click Close.
