Integrating IEC 61850 Generic Device

Scenario

You want to configure the IEC 61850 Generic device with the management platform.

For the background information, see the reference section.

 

Prerequisites

  • Powermanager IEC 61850 and Extension Module is installed and added to the project. Refer Install and Update an Extension.
  • You have a license for IEC 61850 and activated it.
  • System Manager is in Engineering mode.
  • In System Browser, select Application View.
