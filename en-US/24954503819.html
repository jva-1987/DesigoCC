Configure the EN 50160 Report

  1. To configure EN 50160 Report, ensure you have added IEC61850, and PowerQuality extension modules to project.
  2. You have a valid IEC 61850, and PowerQuality license, and activated it.
  3. You have integrated at least one IEC 61850 device, refer Integrating the IEC 61850 Devices.
  1. You have selected the EN 50160 report type for configuration.
  2. The Report Definition configuration page displays.
  3. In the Source expander, select devices for which the report needs to be generated.
    NOTE 1: Only PowerQuality devices with IEC61850 protocol are available for EN 50160 report.
    NOTE 2: You can configure up to 10 PowerQuality devices in one EN 50160 report.
  4. Click Save .
  5. The Save Object As dialog box displays.
  6. Enter a name and description and click OK.
  1. The EN 50160 report definition is configured.

 

For devices with both Star and Delta configurations, only Delta configuration will be displayed in EN 50160 report.
