(Optional) Uninstall the Windows App Client

Once you have uninstalled the management station, next, if you have installed a Windows App Client, you should uninstall it.

If you have installed a Windows App Client, the system creates a shortcut in the Start menu and on the desktop from where you are accessing it.

Uninstalling Desigo CC by custom or silent mode does not uninstall the Windows App Client.

 

 

You must manually uninstall the Windows App Client by selecting Control Panel > Programs and Features on the system from where you are accessing it.

  1. In the Program and Features window, select Windows App Client and right-click and select Uninstall.
  2. Click Yes.
  3. The uninstallation program completes without displaying any verification message.
