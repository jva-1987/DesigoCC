Enabling the Scalance Events

You can enable one or more Scalance events in Desigo CC.

Prerequisites

  • System Manager is in Engineering mode.
  • System Browser is in Management View.
  • Enable Events at Object Model Level for All the Scalance Devices
  • Enable Events for Individual Scalance Devices
