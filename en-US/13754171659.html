Customizing a D3 Visualization Dashboard

The dashboard configuration file is the pre-defined output file provided by the management station. This is a customizable file that can be modified to create the desired dashboard. You can modify this file to achieve the layout and controls you want to display by deleting unwanted controls or copying and pasting a control to add more controls. This file is also used while creating a web rule and fixed links to display the dashboard application.

Several other files containing the default styles for each widget type as well as a sample language file can be found in the following directory:

[installed drive]/[installed folder]]/websites/[website name folder]/[dashboard app folder]/
