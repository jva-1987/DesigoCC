Configuring Commands for URVE Room Booking

  1. You have installed, configured and integrated the URVE Room Booking Adapter, see Integrating an URVE Room Booking Adapter.
  1. Select Project > Field Networks > [SORIS network] > [URVE Room Booking adapter] > [Room].
  2. In the Operation tab, configure the following properties:
    • Comfort: Start Before Meeting: Enter how long before (in minutes) to start the comfort command for the scheduled room occupancy. You can use the arrows in the graphic to set the value. Then click Write.
    • Comfort: Release After Current Meeting: Select from the drop-down list whether to release the comfort at the beginning (Starts) or at the end (Ends) of the current meeting. Then click Write.
    • Comfort: Device Name: Enter the Desigo CC device node name to which the point you want to command belongs. Then click Write.
    • Comfort: Trigger Point Name: Enter the name of the point you want to command to activate the room comfort. Then click Write.
    • Comfort: Trigger Property Name: Enter the name of the property of the trigger point you want to command. Then click Write.
    • Comfort: Trigger Cmd Name: Enter the name of the command of the trigger property. Then click Write.
      NOTE: To get the command name, proceed as follows:
      - In the Summary bar, select Menu > Applications > Start new System Manager.
      - In System Browser, expand the device network and select the trigger point.
      - In the Object Configurator tab, double-click the Object Model label to select the object model.
      - Select the Models & Function tab.
      - In the Property expander, select the trigger property.
      - In the Command Configuration expander, on the right-hand side of the pane, locate the trigger command and copy the Name string.
      - Click Close in the title bar of the additional System Manager window to close it.
    • Comfort: Trigger Parameters: Depending on the command, enter one of the following:
      - A numeric value for single-parameter commands
      - Two numeric values separated by semicolon (for example 5;13) for dual-parameter commands
      Then click Write.
  1. Once you have properly completed all the fields, the command will be activated at the scheduled time.
  2. If you want to stop sending out the command, just modify the four properties and enter a blank string.
