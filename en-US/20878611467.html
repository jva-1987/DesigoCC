Setting Up a Closed Mode Installed Station

Scenario

You want to configure a Desigo CC installed client station to run in closed mode, which means the Desigo CC software runs at all times and no other Windows applications (Task Manager, Taskbar, or desktop icons) can be accessed by the logged on operator. For background information, see the reference section.

Prerequisites

Prevent access to system folders in closed-mode stations
Due to security reasons, in closed-mode stations it is recommended to hide critical system folders on local drive for the closed-mode user (GMSDefaultUser).
