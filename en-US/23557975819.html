Switch between Automatic and Stop Mode

Automatic mode normally is used to analyze trend data (continuous scrolling of the graphic curves).
The latest data is automatically retrieved from the system.
You can change to Manual mode for a detailed analysis (scrolling off).
In this case, the data is no longer updated automatically.

  1. Click Stop .
  2. This stops automatic data updates and suppresses the symbol to update Trend View.
  3. Define the desired date range using the slider or time bar.
  4. Click Refresh when the symbol is available and you want to upload the latest data from the History Database.
  5. Click Run to update data on a continuous basis.
