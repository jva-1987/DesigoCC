Creating and Auto-Populating the Physical View

Scenario

You want to create the physical view that presents the subsystems physical structure—for example, board, line, and sensor. To do this, you must create the view and import subsystem data that will build up its structure. For more details about views, see Overview of Views.

Prerequisites

  • System Manager is in Engineering mode.
  • System Browser is in Management View.
