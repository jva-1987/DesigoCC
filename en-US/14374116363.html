Exporting the OPC Configuration

  1. System Manager is in Engineering mode.
  2. The OPC servers and OPC items have been discovered on the specified computer (see Discovering OPC Servers and OPC Items).
  1. In System Browser, select Management View.
  2. Select Project > Field Networks > [OPC network].
  3. In the Discovery tab, open the Discovery Settings expander.
  4. Select the Export to CSV file check box.
  5. Open the Device Discovery expander.
  6. Click Import.
  1. The Discovery status section displays the status of the operation. In particular, the Final Result column indicates: Export Completed. The file is created in [Installation Drive]:\[Installation Folder]\[Project Name]\data\OPCDiscovery.
