Replace the Nested Symbol

  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L4-Project and navigate to the library you just saved the modified Symbol to.
  3. Right-click on the Symbols folder, and from the context menu, select Send to the Secondary Pane.
  4. In the Graphics Editor, click on the parent Symbol tab that is still displayed.
  5. In the Element Tree view, click on the Symbol Instance that referenced the Symbol you modified in the previous step.
  6. From the Secondary Pane, drag the customized Symbol, and drop it over the Symbol Instance Properties > Symbol Instance > Symbol Reference field.
  7. The Symbol Reference Symbol has been replaced with the customized project Symbol.
  8. Click Save As and save the modified Symbol to the L4 Project based library.
  9. Select Project > System Settings > Libraries > L4-Project and navigate to the library you just saved the modified parent Symbol to.
  10. Right-click on the Symbols folder, and from the context menu, select Send to the Secondary Pane.
