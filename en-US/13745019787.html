Triggers and Filters

This section provides reference and background information on triggers and filters used in some of the Desigo CC configuration features.

Desigo CC can be configured with triggers (in some cases called filters) which are used to determine:

  • When (Time and Organization Mode conditions)
  • For what kinds of alarms (Events conditions)
  • In what field situations (Values and States conditions)

a certain action or function should happen.

For example, you can set up triggers to specify when and for what alarms a certain guided procedure (or individual procedure step) should be made available to the operator.
