Configuring SNMP Monitoring of an Individual Property

You can configure Desigo CC to monitor an individual property (Analog, Binary, Multi-State, String, or Time Value) of a connected SNMP-capable device. For background information, see SNMP Device Monitoring.

 

  1. You configured the SNMP driver and SNMP network and the driver is running. See Configuring the SNMP Connectivity.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > Field Networks > [SNMP network].
  3. The System Management tab displays.
  4. Click New and select the individual property you want to monitor from the following:
    • New Analog Value
    • New Binary Value
    • New MultiState Value
    • New String Value
    • New Time Value
  5. In the New Object dialog box, enter a name and description.
  6. Click OK.
  7. The newly-created object is automatically selected in System Browser.
  8. Do one of the following:
    • In the System Management tab, complete the fields in the General Settings and Properties expanders. See SNMP Device Configuration Workspace for details.
    • If you previously exported an SNMP configuration file for this device, you can import it instead of manually completing these fields. See Importing an SNMP Device Configuration.
  9. Click Save .
