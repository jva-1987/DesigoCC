Create Area Summary Configuration

  1. Select Applications > Powermanager > Area.
  2. Select Enable configuration, to enable system summary in Operating mode.
  3. Select the required measurement points from the Measurement Point dropdown.
  4. Select the Calculation checkbox to enable calculations.
  5. Select the Archive checkbox, to archive the data.
  6. Select the Alarm checkbox, to enable alarms.
  7. Configure alarms in the Alarm expander.
  8. Select the check box under the Include column of the Include Devices expander to monitor the respective device.
  9. In Power Interval Settings, select the Interval dropdown for which the power values are to be calculated.
  10. Select the device to be displayed in the Favorites tile under the Devices column.
  11. Select the measurement points to be displayed in the Favorites tile under the Measurement Point column.
  12. Select the parameters to be displayed in the Favorites tile under the Parameter column.
  13. Click Save .
