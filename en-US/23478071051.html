Start the Project

  1. At least one active project is available under Projects node in SMC tree.
  2. The project is stopped and has unique port numbers.
  3. (Only applicable for Projects on FEP)
    - You have copied and imported the same Windows key file, as that on the Server computer, on the disk of the FEP computer.
    - You have re-aligned the Client/FEP Project with the updated project on the Server.
  1. In the SMC tree, select Projects > [project].
  2. Click the Project Settings tab.
  3. Project Settings displays.
  4. Click Start Project .
  1. The Pmon service in the Windows services starts which in turn starts the project.
