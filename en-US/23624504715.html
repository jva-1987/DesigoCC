Change the HDB on the Changed Server

If the MS SQL Server for HDB runs on the same system as the Desigo CC server (whose name is changed) and its reference contains the system name, you must complete the following steps:

  1. In History Databases, from the SQL Servers expander, click Scan Local/Scan Network and select the SQL Server for HDB from the system whose name is changed.
  2. Click Link to link the selected HDB.
  3. Link the HDB to the project.
