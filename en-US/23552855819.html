Climatix Heating System Functions

The functions provided with the library are used to represent the Climatix District heating unit in Desigo CC and their properties provide the most important information about the device. The function is also the entity where the graphic template is associated. Therefore, configuring a function for a Climatix controller automatically results in the graphic template linked to this function associated to the DH controller. For an overview of the graphic templates and the associated functions, see Climatix Heating System Graphic Templates.
