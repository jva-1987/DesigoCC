Set the Logical View for Maintenance

If the logical view is already present in the system, skip to step 7 in this procedure.

  1. From the System Browser drop-down list, select Management View.
  2. Select Project > System Settings > Views.
  3. The views workspace displays in the Views tab.
  4. In the View Name and View Description fields, enter a name and description for the logical view.
  5. In the Root node name and Root node description fields, enter a name and description for the root node of the logical view.
  6. From the Type drop-down list, select Logical.
  7. Click Save .
  8. The logical view object is added to System Browser under Views. Its name becomes available in the System Browser drop-down list.
  9. Display the [logical view] in the view builder workspace of the Views tab as follows:
    a. From the System Browser drop-down list, select the [logical view].
    b. In the System Browser tree, select the [logical view] root node.
  10. In the view builder workspace, populate the logical view with maintenance objects as follows:
    a. select the [logical view] level.
    b.     Drag the maintenance objects (such as Electrical, HVAC and so on) that you want to link from System Browser to the appropriate level or sublevel under the [logical view] level.
    c. Drag the Documents folder to the [logical view] level.
  11. Click Save .
