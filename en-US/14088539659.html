Troubleshooting OPC DA Server User Account

Error/Issue

Situation

Solution

The user for OPC is not configured or does not exist. Verify the Settings.

The OPC DA server user account configuration is missing or not set.

  • If the OPC user account configuration is missing, create the OPC user account (see Create a Software Account User for OPC).
  • If the OPC user account configuration is available, set this user account for the OPC DA server (see Configure the OPC DA Server).

An error was encountered while using COM services.

When you try to start a third-party OPC client, and the tracing records that the OPC user account is missing.

  • Check the OPC user account and the OPC DA server configuration.

The OPC DA server stops. The tracing tool records the following log: Changed user.

The third-party OPC client was connected but in the meantime the OPC user account was changed.

  • Reconnect the third-party OPC client.
