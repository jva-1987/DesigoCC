Configure a Background Layer and Depth

  1. System Manager is in Engineering mode.
  2. You have a graphic open and are in Design or Test mode.
  1. Make sure the Element Tree view is visible; from the bottom left side of the view, click Show all Elements .
  2. The Element Tree now displays all elements and objects that are placed on the graphic.
  3. With Layer (Layer1) selected, do the following:
    a. Select the Use check box to enable it.
    b. Select the Visible check box to enable it.
  4. The layer is now used and visible in Runtime mode and in the Graphics Viewer.
  5. Navigate to the Properties view (Layer Properties), expand the General section, and in the Description field, type Background.
  6. (Optional) Expand the Layer section, and from the Discipline drop–down menu select an associated discipline. In the future, when the graphic displays in the Graphics Viewer, you can sort the layers by discipline.
  7. Navigate to the Depths view, and click the current Depth Name and then type the new name for it.
    • For this example, type Background.
  8. The depth is renamed.
  9. Click Save.
  1. You are ready to import the background (floor plan) image or AutoCAD file.
