Modify the Language Settings of a Project

  1. On the Server you have installed the required languages that you want to configure in the project.
  1. Do one of the following:
    • Edit the StartSMC.bat file present at [Installation Drive]:\[Installation Folder]\GMSMainProject\bin by adding a /librarian switch.
    • Start the SMC by clicking the StartSMCEngineering.bat file that already contains the /librarian switch.
  2. Save the file.
  3. Launch the SMC.
  4. In the SMC tree, select and stop the project for which you want to edit the language settings.
  5. Click Edit .
  6. From the Server Information expander, select the desired language other than en-US from the Languages drop-down list.
    NOTE: You cannot edit the default set en-US language.
  7. Click Save Project .
  1. The language is edited.
