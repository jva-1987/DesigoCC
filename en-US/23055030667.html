Set Read Property and Polling Rate

  1. Under [BACnet network] > Hardware, select the device.
  2. The BACnet tab displays.
  3. Open the Other Settings expander, and select the following check boxes:
    • Use only read property
      NOTE: This setting is required for third-party devices that do not support RPM.
    • Disallow Subscriptions
      NOTE:       This setting is required to use only the Read property service for all subscriptions.
  4. Open the COV and Polling Info expander, and in the Polling rate field, decrease the value to the minimum (that is, 5 seconds).
  5. Click Save .
