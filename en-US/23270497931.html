Create and Configure Report Definition

  1. Select Applications > Powermanager > Report Defintion.
  2. Click Create and select Create Report Definition.
  3. Select the desired report type.
  4. Perform the required configuration depending on the type of report.
    • Standard Report
    • Cost Center Report
    • Load Duration Report
    • Load Variance Report
    • Top 10 Energy Report
    • Absolute Energy Report
    • Total Energy Report
    • Energy Analysis Report
    • Power Peak Report
    • Delimitation Report
    • Sankey Report
  5. Click Save .
  1. The report definition is created and displays under the Report Definition folder in System Browser.
