URVE Room Booking Commands

The following describes the commands of URVE Room Booking objects.

For details about the SORIS adapter properties and commands, in SORIS, see Properties and Commands of SORIS thingAdapter Objects.

For configuration instructions, see Configuring Commands for URVE Room Booking.

Commands available on URVE Room objects

Property

Description

Commands

Comfort: Start Before Meeting

Lead time (in minutes) of the room comfort control command before an occupancy event.

  • Write

Comfort: Release After Current Meeting

Release when meeting starts or ends.

  • Write

Comfort: Device Name

Name of the device to which the point you want to command belongs.

  • Write

Comfort: Trigger Point Name

Name of the point you want to command to activate the room comfort.

  • Write

Comfort: Trigger Property Name

Name of the property of the trigger point you want to command to activate the room comfort.

  • Write

Comfort: Trigger Cmd Name

Command Name, as defined in the library for the point you want to command.

  • Write

Comfort: Trigger Parameters

Depending on the command, one or two numeric parameters (for example 5 or 5;13)

  • Write
