TJM CMMS Troubleshooting

Error/Issue

Situation

Do the following...

CMMS Connection Status Fault

The Event Manager service cannot communicate with the TJM CMMS system.

  • Check network connections.
  • Check configuration (see Configure TJM CMMS Event Manager Parameters).
  • Check TJM CMMS status.

Service Connection Status Fault

The Event Manager service cannot communicate with the Desigo CC server.

  • Check network connections.
  • Check service status.
  • Check configuration (see Configure TJM CMMS Event Manager Parameters).
