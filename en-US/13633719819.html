Creating and Auto-Populating the Logical View

Scenario

You want to create the logical view that presents the hierarchical structure of imported devices—for example, area, section, zone, sensor. To do this, you must create the view and import subsystem data that will build up its structure. For more details about views, see Overview of Views.

Prerequisites

  • System Manager is in Engineering mode.
  • System Browser is in Management View.
