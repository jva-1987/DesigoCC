Associate the SPC Driver to a Panel

This procedure is part of the workflow for integrating an SPC intrusion system.

 

Prerequisites

  • You created an SPC network and configured an SPC driver with settings that match those in the panel configuration file.

 

Add an SPC Panel Under the Network

  1. Select Project > Field Networks > [SPC network].
  2. Click New , and select SPC Panel.
  3. In the New Object dialog box, enter a name and description.
  4. Click OK.
  5. The newly created panel is added to System Browser.

 

Associate a Driver to the SPC Panel

  1. Select Project > Field Networks > [SPC network] > [SPC panel].
  2. In the SPC tab, open the Network Setting expander.
  3. In the Monitored by driver drop-down list, select the SPC driver.
    NOTE: You can associate more than one panel to the same driver. The Receiver Id and Receiver Port values of the driver (set in Configure the SPC Driver Settings,) must match those of the panel configuration files (set in Preparing the SPC Configuration File). The Monitored by driver field becomes read-only after you import the configuration file.
  4. If your network connection uses a Network Address Translation (NAT) device, open the NAT expander, and enter the NAT address of the device.
  5. Click Save .
