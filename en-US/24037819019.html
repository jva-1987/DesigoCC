Check the List of Selected Maps and Run the Import

  1. You are on page 5 of 5: Run Import.
  2. The list of the CAD drawings to import displays.
  1. Check the lists of maps you are about to import.
  2. (Optional) If you want to modify any item of the import list, click Previous to go back. After the required modifications, press Next to reach this final page again.
  3. Click Start .
  4. A confirmation message displays.
  5. Click Yes.
  6. The import procedure starts.
  7. In System Browser, graphics are created/updated in the Graphics subtree of the Application View.
  8. When the import procedure completes, the Import Summary dialog box displays and shows the number of items successfully imported as well as the number of warnings and errors.
  9. Click Summary to display the summary log file: [drive]:\Users\[username]\AppData\Local\Temp\AdvancedCadImport_mm-dd-yyyy_nn.txt.
    The file provides a report that indicates errors and warnings.
  10. Click Details to display the detailed log file: [drive]:\Users\[username]\AppData\Local\Temp\AdvancedCadImport_Results yyymmdd hhmmss.txt. The CSV file contains a record of all actions performed.
    • For examples of log files, see Advanced CAD Importer Log Files.
  11. (Optional) In case of errors, you can click Previous , correct the settings as necessary, and try again.
    See Advanced CAD Importer Troubleshooting and Advanced CAD Importer Compatibility and Limits.
