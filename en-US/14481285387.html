Configure the DF8000 I/O via I2C

You can configure DF8000 I/O modules locally connected to the NK823x unit over the I2C link.
For background information, see DF8000 Reference.

Add the DF8000 Unit

  1. In System Browser, select Project > Field Networks > [BACnet network].
  2. Select the Object Configurator tab.
  3. Click New and select New GMS_DF8000.
  4. In the New Object dialog box:
    a. (Optional) Edit the precompiled name and description (DF8000 System via I2c Bus).
    b. Click OK.
  5. The new DF8000 System via I2c Bus object is added and automatically selected in System Browser.
  6. Select the DF8000 tab.
  7. In the Configuration expander, set the Instance number (the BACnet Device Identifier) of the DF8000.
  8. Click Save .

 

Add the Input / Output Modules

Repeat this procedure for all the input / output modules included in the hardware configuration. Any combination of modules is possible, for up to a total of 4 modules.

  1. In System Browser, select Project > Field Networks > [BACnet network] > [DF800 system via I2c bus]
  2. Click New and select the input/output module type you want to configure:
    • DF8040 Digital Input Module
    • DF8020 Relay Output Module
    • DF8045 Balanced NC Input Module
    • DF8046 Balanced NO Input Module
  3. In the New Object dialog box:
    a. (Optional) Edit the precompiled name and description.
    b. Click OK.
  4. The new module is added to System Browser, and to the Modules expander of DF800 System via I2c Bus.
  5. In the Modules expander, select the newly added module.
  6. Enter an Address that matches the setting on the hardware module.
  7. Set the I/O Logic to Normally Open or Normally Closed.
  8. The selected logic is applied to all the inputs or outputs of the module.
  9. Click Save .
  10. (Optional) You can customize the logic of individual inputs / outputs as follows:
    a. In System Browser, select Project > Field Networks > [BACnet network] > [DF800 system via I2c bus] > [module] > [input or output].
    b.     In the DF8000 tab, set the desired I/O logic for this input (Normally open or Normally closed).
    c.     Click Save .
  11. After the customization, if the module I/Os do not all apply the same logic, then its I/O Logic status in the Modules expander is indicated as Custom.
  12. (Optional) You can customize the Pulse Duration (default =16 sec) of outputs as follows:
    a. In System Browser, select Project > Field Networks > [BACnet network] > [DF800 system via I2c bus] > [module] > [output].
    b. In the DF8000 tab, set the required Pulse Duration.
    d.     Click Save .
  13. This setting will affect the Pulse On/Pulse Off commands for the selected output only.

 

(Optional) Add a Supervised Power Supply Module

Perform this step if the hardware configuration includes a DF8090 module.
NOTE: There can be up to 4 modules in total, but only one DF8090.

  1. In System Browser, select Project > Field Networks > [BACnet network] > [DF800 system via I2c bus].
  2. Click New and select DF8090.
  3. In the New Object dialog box:
    a. (Optional) Edit the precompiled name and description (DF8090 Supervised Power Supply Module).
    b. Click OK.
  4. The new DF8090 object is added to System Browser, and to the Modules expander of DF8000 System via I2c Bus.
    The DF8090 module has the fixed address 8.

 

Connect the DF8000 to the NK823x I2C Port

  1. In System Browser, select Project > Field Networks > [BACnet Network] > [NK823x] > [I2C].
  2. Select the NK8000 tab.
  3. In the Configuration expander, from the DF8000 System drop-down list, select the DF8000 System via I2c Bus configured in the preceding steps.
  4. Click Save .
