Delete Graphic Pages

  1. Rooms or segments are available and visible in the workspace.
  2. Settings File levels and Number of folder levels must have the same settings as used to generate the graphic pages.
  1. Select the File level to be considered for deletion.
  2. Select the Number of folder levels that was used for the graphic structure.
  3. Select the check box on room mapping pages to delete.
  4. Click Delete Graphic Pages .
  5. The Delete Graphic Pages dialog box displays.
  6. (Optional) Select Also delete empty folders check box.
  7. Click OK to start deleting.
  8. The Overview of generated graphic pages dialog box displays.
  9. Click OK.
  1. The delete process is completed for the graphic pages.
