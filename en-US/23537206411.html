Create a New Advanced Reporting Templates Folder

  1. You have customized the library to which you want to add the reporting templates as per your customization levels.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries.
  3. Select the respective library to which you want to add the report templates and add a new library block.
    When adding the library block, ensure that you select the New Advanced Reporting Templates option from the list that displays when you click New .
    You can only add one Advanced Reporting templates folder below a libraries folder.
  4. Enter values into the following fields: Version, Revision, Date, Notes.
  5. Click Save .
  6. The Create New Object dialog box displays.
  7. Enter a name and description for the folder and click OK.
  1. The Advanced Reporting Templates folder is added to the library.
