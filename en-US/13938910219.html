Configuring an Operating Procedure with Access Control

You want to configure an operating procedure that includes an access control step, for example, to display the details of a cardholder who is denied access during assisted treatment. For background information, see the reference section. For troubleshooting information about the access control step, see SiPass Operation Problems.

 

Prerequisites

  • Desigo CC is configured to integrate a SiPass access control system and also connected to the website of the SiPass system.
  • You have the license and application rights to configure procedures for assisted treatment.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.
