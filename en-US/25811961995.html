Link a PowerQuality Database to the Project

  1. In SMC tree, select the Database Infrastructure > [SQL Server name].
  2. Select the Databases tab.
  3. Navigate to Databases expander > Type, select Powermanager PowerQuality.
  4. Click Scan.
  5. Select the [PowerQualityDatabase name], and click Link .
  6. Navigate to Projects > [project], select PowerQuality tab.
  7. Select created database for Linked PowerQuality DB dropdown menu.
    NOTE: For remote PowerQuality database, encryption is not supported.
  8. For remote database, enter database system credentials.
  9. Click Save.
  1. PowerQuality Database is linked to project.

NOTE: To unlink the PowerQuality Web App or the PowerQuality Database from the project, remove the Linked PowerQuality Web App and the Linked PowerQuality DB field values. If both the values are not removed, then the web app and database cannot be unlinked from the project and displays an error message.
