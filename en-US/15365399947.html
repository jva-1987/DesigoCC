Associating Cameras to System Objects

Scenario: Cameras can be assigned to system objects so that selecting an object makes its associated video available in Related Items. You can do this manually, or using the coverage area feature of the graphics application. For background information see Cameras Associated to System Objects.

Select the appropriate procedure from the following.

Manually Associating an Object to a Camera

  1. System Manager is in Operating mode
  1. In System Browser, select the Application View.
  2. Select Applications > Video > Cameras > [camera].
  3. Click Edit .
  4. The Associated Objects list displays.
  5. In System Browser, select the Manual navigation check box.
  6. Navigate to and select the object you want to associate to the camera.
    NOTE: You will typically want to select objects under the Field Networks folder of the Management View.
  7. Drag-and-drop the object into the Associated Objects > Manually Assigned list.
  8. The object is added to the list.
    NOTE: To remove an association, select the object in the Manually Assigned list and click Delete.
  9. Click Save .
  10. Deselect Manual navigation.

Associating Objects to a Camera Using the Coverage Area

  1. You are authorized to configure graphics.
  2. You want to add a camera with coverage area to a graphic. This will automatically associate any objects within the coverage area to the camera.
  3. System Manager is in Operating mode.
  1. In System Browser, select Application View.
  2. Select Applications > Graphics > [graphic].
  3. Click Edit .
  4. In System Browser, select the Manual navigation check box.
  5. Select Applications > Video > Cameras > [camera].
  6. Drag-and-drop the camera onto the graphic
  7. The default camera symbol appears on the graphic. The symbol includes a partially-transparent shape that represents the camera coverage area.
  8. Move the symbol to the appropriate position in the graphic.
  9. Adjust the coverage area shape by selecting and moving the segment points.
  10. The symbols of system objects enclosed within the coverage area are highlighted with a colored frame. Make sure all the objects you want to associate to the camera are inside the area.
  11. Click Save .
  12. The system creates an automatic camera association for all the objects configured inside the configured coverage area boundaries.
    NOTE: If you do not want the camera symbol to display the coverage area, right-click the camera symbol, click Symbol Instance > Replace, and select the appropriate symbol. Use the Reset size command in the right-click menu to adjust the size of the new symbol.
