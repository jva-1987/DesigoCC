Configure TJM CMMS Event Manager Parameters

  1. After the event manager installation, the EventManagerConfigurator.exe application is available in the installation folder.
  1. Run EventManagerConfigurator.exe.
  2. The Event Manager Configurator window displays the File page, which includes three sets of configurable fields:
    - In the upper part, the Desigo CC NORIS web site access parameters.
    - In the middle, the TJM CMMS access parameters.
    - In the lower part, the event category selectors.
  4. Configure the Desigo CC NORIS web site access parameters:
    - In NORIS URL, enter the web service application URL that you configured in SMC (URL field from the Web Application Details).
    NOTE: It is recommended to use HTTPS for secure communications.
    - In Username and Password, enter the credentials of a Desigo CC user that has scope visibility on the events to transmit, and on the TJM CMMS System object (see TJM CMMS Events).
    NOTE: Use the test pushbuttons to verify the connection and the login. To run the test, click the button, and a success or failure response appears on the button itself.
  5. Configure the TJM CMMS web site access parameters:
    - In CMMS System URL, enter the TJM CMMS web site URL.
    NOTE: It is recommended to use HTTPS for secure communications.
    - In Username and Password, enter the credentials of a suitable TJM CMMS user.
  6. Configure the event categories. Select the check boxes that correspond with the event categories that you want to be transmitted, and deselect the others.
  7. In the menu at the top of the window, select the Advanced page and configure the event manager network settings for the TJM CMMS data queries:
    • In Hostname or IP Address, enter the event manager hostname/IP address.
    • In Listening port, enter the event manager TCP port.
    • In System ID, enter the Desigo CC ID, as specified.
  8. Select the File page and then Save Configuration.
  1. The ServiceConfig.json file is saved in the event manager folder. The file contains the configuration parameters for the event manager service to use.
    NOTE: Use the Import Configuration button to select a valid ServiceConfig.json file and get the parameter values from it.
