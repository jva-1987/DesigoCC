Configuring SICLIMAT X Objects and Properties

  1. Select System Settings > Libraries > L1-Headquarter > BA > Device > Siclimat X > Import Rules.
  2. In the Import Rules tab, open the Objects and Properties expander.
  3. The objects and properties (refer SICLIMAT X Library section) display and may contain the existing configuration.
  4. For each PIM, select or enter a specific Object Model.
  5. To configure a copy of an object:
    a. Select an object and click Copy Object.
    b. Repeat Step 3.
    NOTE: To remove a copy of an object, select the object and click Remove Object.
  6. Click Save .
  7. Click OK.
  1. The object and properties settings configured are activated and will be applied when importing the XML data.
