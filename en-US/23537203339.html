Configure Reporting Templates

Perform the following to add new or configured report templates back to the respective library.

  1. You have customized the respective library to which you want to add the report templates as per your customization levels.
  2. (Optional and required only when you want to add the templates to a different library) You have created an Advanced Reporting templates folder in that library. For more information, see Create a New Advanced Reporting Templates Folder.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries.
  3. Proceed as follows:
    • To add the templates to the Advanced Reporting or Energy Reporting libraries: Select the Advanced Reporting Design folder in the Advanced Reporting or Energy Reporting libraries, if you want to add the templates to the Advanced Reporting or Energy Reporting libraries.
    • To add the templates to a different library: Select the Advanced Reporting templates folder in the respective library.
  4. In the Template Configuration tab, open the Import expander.
  5. Click Browse.
  6. In the Browse for Folder dialog box, select the folder where the zipped files of the Advanced Reporting templates are located.
  7. The Advanced Reporting templates from the selected folder are listed in the Import expander.
  8. In the Import expander, select the files to be imported and click Import.
  9. On import, the imported files are listed in the Templates expander.
  1. The zipped files of the report templates are listed in the library.

NOTE:
If an XLS file is used in report templates, it must be equipped with sufficient access rights to prevent unauthorized access.
