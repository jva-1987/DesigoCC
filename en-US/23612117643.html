Start the Command Prompt as an Administrator

  • Select Windows Start menu > All Programs > Accessories and select the Command Prompt menu, right-click it and then select Run as administrator.
    • A User Account Control message may display if your UAC settings are set to Default or Always Notify, asking you if you want to allow this installation program to make changes to the computer. Click Yes. (See Return User Account Control Settings to Default or Always Notify)
  • The Command Prompt displays.
