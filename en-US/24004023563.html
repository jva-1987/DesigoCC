Consistency Check takes longer time before Installation

Problem

  • Consistency Check takes longer time before Installation begin.
  • When performing software installation on new machine you observe the delay for consistency checks before installation begins.
  • Installer consistency check used to take only few seconds to complete and start installation, but now it is taking more than 10 minutes before you are able to do anything with installation.

Cause
If you are performing a new installation on a new operating system, then consistency check may take more time due to one of the following:
- Real-time scanning from Defender Antivirus
- Application checker from Defender Smartscreen

Solution

  1. Navigate to Start > Windows Security > Virus & Threat Protection Settings.
  2. Disable Real-time protection option.
    • If you are not allowed to perform above action then contact your system administrator.
  1. The consistency check is back to normal and done within a few seconds before installation.
