Integrating CS1115/FC330

Scenario

You want to configure Desigo CC to operate on the CS1115 FC330 fire control panels. For background information, see CS1115/FC330 Integration.

Prerequisites

  • The CS1115 FC330 extension is installed and included in the active project. The following extensions are also selected automatically:
    • Fire Domain Libraries Common
    • Fire Domain Libraries EN
    • SORIS Driver
  • The CS1115 SORIS Adapter is running and connected to the CS1115/FC330 fire control panels. For instructions, see Installing and Starting the CS1115 SORIS Adapter.
  • The CS1115 adapter configuration is set. For instructions, see Setting CS1115/FC330 Configuration.
  • System Manager is in Engineering mode.
  • System Browser is in Management View.

Licensing

This extension is covered by license. Check that the required license is available; otherwise, you may not be able to configure and work with this extension.
