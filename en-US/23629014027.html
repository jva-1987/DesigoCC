Automatic Backup of Project Data

Scenario
You want to automate a backup of historical data using the using the Macro and Reaction Editor functions.

For specific example, see Creating a Reaction to Automatically Back Up the Project and History Database on Sunday Nights.

  1. A backup folder is available.
  2. You have enabled the Operational Status from Macro Backup History to run automatic backups. For more information, see Manual HDB Backup Using a Macro.
  3. System Browser is in Application View.
  1. Select Applications > Logics > Reactions.
  2. The Reactions Editor tab displays.
  3. Select Triggers > Time & Organization Mode expander and click Add.
  4. Click anywhere on the row.
  5. Additional fields to set the execution time display.
  6. Set the dates, days and time to execute the macro.
  7. From Output expander, open the Action expander.
  8. In System Browser, select the Manual navigation check box.
  9. Select Applications > Logics > Macros > Backups.
  10. Drag the Backup Online macro to the Action expander.
  11. Click Save As.
  12. Type the name and description in their respective fields.
  13. Click OK.

NOTE:
Since the older version is overwritten when creating the backup file, you must rename the backup file after creating it in order to retain multiple backup versions.
