TOA IP Intercom System Symbols

When integrating the TOA IP Intercom System extension, the following symbols are available in graphics:

I/O Symbols

DYN_3D_TOA_IO_None_None_Horizontal_001_150

  • Default symbol for 3D_EU of TOA IP Intercom I/O.
  • Symbol background changes color according to the event category color.

 

DYN_3D_TOA_IO_None_None_Horizontal_002_150

  • Secondary symbol for style 3D_EU of TOA IP Intercom I/O.
  • Symbol background changes color according to the event category color.

 

Station Symbols

DYN_3D_TOA_Station_None_None_Horizontal_001_150

  • Secondary symbol for style 3D_EU of TOA IP Intercom station.
  • Symbol background changes color according to the event category color.

 

DYN_3D_TOA_Station_None_None_Horizontal_002_150

  • Secondary symbol for style 3D_EU of TOA IP Intercom station.
  • Symbol background changes color according to the event category color.

 

DYN_3D_TOA_Station_None_None_Horizontal_003_150

  • Default symbol for style 3D_EU of TOA IP Intercom station.
  • Symbol background changes color according to the event category color.

 

Subsystem Symbol

DYN_3D_TOA_Subsystem_None_None_Horizontal_001_150

  • Default symbol for style 3D_EU of TOA IP Intercom subsystem (N-8000RS).
  • Symbol background changes color according to the event category color.
