(Optional) Verify Hotfixes Folder

  • Hotfixes are a group of patches for extensions, platform, and language packs that need to be delivered as a quick fix for critical defects.
  • To install Hotfixes, the Hotfixes folder is present in the Installation media at ...\InstallerDVD\InstallFiles in parallel to the GmsInstallerSetup.exe file.
  • The [EM]PatchConfig.xml file for the patch to be installed must have the UpdateLabel attribute. For Platform patches, the GMSPatchConfig.xml file is used.
  • If a same prerequisite of Platform or extension is present in the M300 package as well as the Hotfixes folder, then Installer gives priority to the prerequisite from the Hotfixes folder.
  • You can have both the Quality Updates and Hotfixes folder in the Installation media at ...\InstallerDVD\InstallFiles in parallel to the GmsInstallerSetup.exe file. This enables the Installer to select the latest patch for the extension or Platform from Quality Updates or Hotfixes folder during installation.
