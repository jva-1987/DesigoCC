Create the Geographical View

  1. Select Project > System Settings > Views.
  2. The views workspace displays in the Views tab.
  3. In the View Name and View Description fields, enter a name and description for this view.
  4. In the Root node name and Root node description fields, enter a name and description for the root node of this view.
  5. From the Type drop-down list, select User-Defined.
  6. (Optional) Select the Enable propagation check box, to activate the state propagation from the child nodes to the parent nodes in this view.
  7. (Optional) In the Notes field, enter any additional information.
  8. Click Save .
  1. The geographical view object is added to System Browser. Its name becomes available in the System Browser drop-down list.
