Create a Group Title

  1. The plant operator tableau graphic page is open.
  1. Select the View tab.
  2. Select the Properties check box.
  3. Select the symbol for the plant operator tableau.
  4. The Properties dialog box displays.
  5. In Row 00 Title, enter the name of the group, for example Plant control.
    NOTE: You must enter additional titles in Row xx Alternative Title for rows different from the Row 00 Title.
  6. Also, for rows different from the Row 00 Title, enter the value 3 for a dark background in Row xx Type.
  1. The group title displays in the plant operator tableau.
