Notifier Onyx Compatibility and System Limits

Compatibility

  • Desigo CC V3.0 SR3 or higher

Size Limits

Since no specific limits are identified for this integration, refer to Desigo CC system limits defined in the System Description document (A6V10415500).

Configuration Limits

  • Symbols in the graphic never blink. This means that the operator cannot determine acked and unacked conditions from the graphics.

Other Limitations

  • Because of UL standards regulations, the NOTIFIER® Modbus gateway provides read-only information. Consequently, no Desigo CC command capability is available for this integration.
  • In Event List and in the Operation tab, if Acked is not expressly specified, the event must be considered Unacked. (For example, the unack other side for Fire Alarm Acked is not Fire Alarm Unacked but only Fire Alarm).
  • Any registers defined in the import rules are valid only if no register customization has been configured on the communication interface.
