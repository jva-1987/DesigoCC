View Menu

Field/Button

Description

Options

Allows you to select the object properties you want displayed by default. Also allows you to enable or disable the automatic device discovery feature. When automatic device discovery is enabled, the Browser will automatically add any I-am received as a device to the tree list.

Enrollment Summary Filter

Allows you to edit the filters used by the Event Enrollment Summary view.

Refresh

Re-reads data from the selected device, and then refreshes the data in the data list.

Stop

Cancels all pending requests to read data. This is useful if you select a device containing hundreds of objects that have to be processed by the Browser. Selecting Stop not only cancels the request, it also purges the request to read values from the system.

Re-read Object List

Re-reads the object list for the selected device and then refreshes the tree.

Remove Device

Removes a selected device from the Browser window, though not from the BACnet network. The device can return to the Browser window if the system receives an I-Am request.

Previous

Allows you to navigate back through the previously selected views.

Next

Allows you to navigate forward through the previously selected views.

Find Next

Allows you to search the tree for a match of the criteria you enter.

Status Bar

Displays or hides the status bar.
