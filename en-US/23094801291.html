Historizing Managed Meter Information in Bulk

  1. System Manager is in Engineering mode.
  2. You have created the managed meters and they display in the Managed Meters folder in System Browser.
  1. In System Browser, select Application View.
  2. Select Applications > Energy and Power > Managed Meters.
  3. Perform one of the following steps:
    • If you want to specify the same date for all the meters in the Enter Valid From dialog box, select the Managed Meters folder and click Apply Historization .
    • If you want you specify different dates to the meters in the Enter Valid From dialog box, then select the meters for which you want to specify identical dates and click Apply Historization .
  1. The information of the multi-selected meters is historized.

NOTE:
The layout of the Enter Valid From dialog box in the web client might be different due to technical limitations.
