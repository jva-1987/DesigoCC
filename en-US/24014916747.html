Import Library Template

  1. You are in the management station.
  2. System Manager is in Engineering mode.
  3. System Browser is in Management View.
  1. In System Browser, select Project > System Settings > Libraries.
  2. In the Library Configurator tab, click Import.
  3. In the Import Libraries dialog box, select Global_D3VisualizationApplication_HQ_1.gms.
  4. Click Open. When the import is complete, the following display in System Browser:
  1. Management View - The D3 Visualization library template is added to the L1 Headquarter > Global folder
  2. Application View – A D3 Visualization node is created.
