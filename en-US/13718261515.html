Mobile App Login Errors

If you cannot sign into the mobile app, one of the following error messages will display:

  • Login unauthorized: Your credentials (user name and/or password) are incorrect, or were not accepted by the server. See Full Sign-In Credentials.
  • Server not available: The app was unable to connect to the server because it was not reachable on the network. See Server Not Available.
  • Login error: The server refused the connection for some other reason.
  • Invalid license or certificates improperly configured: This can mean that:
    • One or more of the licenses required for this app to connect are missing or invalid on the management platform. See Mobile App System Requirements and Limits.
    • There is an error with the security certificates. See Check the Certificate Used by the Web Service Application and make sure the correct corresponding certificate (if required) is installed on the mobile device.

You attempted to sign in using an unsecured connection (http://). See Obtain the Server URL and check that you entered the correct URL (beginning with https://).
