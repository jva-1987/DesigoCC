Flex Client Profiles

The Flex client profile customizes certain aspects of the Flex Client web app and desktop app. It is a separate setting from the "standard" client profile, which instead customizes the installed Desigo CC application.

Client profiles can be applied to Desigo CC stations and/or to users. For instructions see Configuring Client Profiles.

The sections below provide details about the client profiles currently installed on your system. For instructions on installing client profiles, see Install or Update an Extension in Additional Installer Procedures.
