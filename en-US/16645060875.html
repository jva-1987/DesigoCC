OPC DA Server Limits

  • Desigo CC OPC DA server supports a maximum of five third-party OPC clients.
  • The OPC items visible to third-party OPC clients correspond to some of the properties of Desigo CC objects. For more details, see OPC DA Server Library. Furthermore, only the objects configured for the OPC user group will expose their properties. The OPC user group configuration also determines the OPC items access rights–R (READABLE) or RW (READABLE/WRITABLE). See Link Scopes to the OPC User Group.
  • You can configure the scope rights for the OPC user group as for any other groups. Anyway, the following restrictions apply (Configure the Scopes for OPC Connectivity):
    • Only the objects belonging to Management View or Application View are considered.
    • The access rights on property groups are used to define the OPC items access rights.
