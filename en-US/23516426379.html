Option 1 – Enable Events for All IEE Occupancy Tool Zones

Use this method to modify an alarm configuration for all IEE Occupancy zones.

  1. You have the rights to access and modify the Desigo CC libraries.
  1. In the Headquarter IEE Occupancy Tool library, select the zone object model:
    • […] > Libraries > L1-Headquarter > Access > Device > IEE Occupancy Tool > Object Model > IEE Occupancy Tool Zone
  2. In the Models & Functions tab, click Customize .
    NOTE: If the icon is dimmed it means that the selected object model is already customized. Skip to step 4.
  3. Click OK.
  4. The customized object model is added to the customized IEE Occupancy Tool library.
  5. Select the customized object model. For example: [...] > Libraries > L4-Project > Access > Device > IEE Occupancy Tool > Object Model > IEE Occupancy Tool Zone.
  6. In the Models & Functions tab, close the Main expander and open the Properties expander.
  7. In the list on the left, select the Property for which you want to enable the event. For example: CurrentOccupancy, NumberOfEntries, NumberOfExits.
  8. The Details, Alarm Configuration and other expanders on the right update to show the settings of the selected property.
  9. In the Alarm Configuration expander, select the Alarm config.activated check box.
  10. Click Save .
  1. The modified alarm configuration will apply to all the IEE Occupancy Tool zones that are based on this object model.
