Signify Compatibility and System Limits

Compatibility

  • Interact API as documented in https://www.developer.interact-lighting.com/api-docs (version December 2020).
  • Desigo CC V4.2 or higher

Size Limits

  • In SORIS, see System Limits.

Signify Integration Limits

  • No fault can be provided for the Signify sensors as the fault information is not available in the Signify interface.
