Deleting OPC UA Client Server Data

  1. System Manager is in Engineering mode.
  2. System Browser is in Management View.
  3. The OPC UA Client Adapter web client displays on the screen. For instructions, see Access the OPC UA Web Client.

  1. To prevent dirty configuration issues, do not remove the adapter from System Browser prior to deleting server data.

  2. In the OPC UA Client Configuration section, click Connection Settings.
  3. In the Servers Discovery area, for each OPC UA third-party server you want to remove, do the following:
    a. Click Recently Used.
    b. Select the item to remove from the list.
    c. Click Delete server configuration , and then click Yes.
  4. In the Extended Operation tab, next to the URL property, click Discover.
  1. The System Browser tree refreshes. Any deleted servers are no longer available under the adapter.

If the OPC UA third-party server you removed was the last one in the Recently Used list, its data remains visible in System Browser tree even after a new configuration discovery. To clean up the server from the System Browser tree, delete the adapter manually as follows:
a. Select Project > Field Networks > [SORIS network] > [OPC UA adapter].
b. In the SORIS tab, click Delete .
