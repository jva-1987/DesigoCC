Viewing Event Details using Investigative Treatment

Scenario: You want to create an event details report that displays the details of a particular event using Investigative Treatment.

  1. Operating procedure templates (if present) are disabled.
  2. Ensure that a report containing the Event Details table is available and the Show in Related Items check box is selected
    for this report or the HQ_EventDetailsLog report is imported.
  1. Double-click the event in the Event bar.
  2. The event details display in the Related Items tab.
  4. Perform the required steps to treat the event.
  5. Select the report containing the Event Details table from the Related Items tab.
  6. The report executes in the Secondary pane and the information related to the event time, category, cause, ID,
    object description, and designation displays in the report.
  8. Click before the event entry.
  9. Information related to the event treatment such as Time, Action, User Name, Management Station, Attachment,
    Value, and Previous Value display as child records.
