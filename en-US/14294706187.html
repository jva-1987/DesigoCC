Additional Domain Groups Procedures

Prerequisites

  • System Browser is in Engineering mode.
  • In System Browser, select Management View.

 

Deleting a User Group for Domain Groups

  1. Select Project > System Settings > Security.
  2. Select the Security tab.
  3. In the Groups list, select the user group you want to delete.
  4. In the User Members list, select all members.
  5. Click Remove.
  6. The User Members list is empty.
  7. The users are moved to the Configured Users expander.
  8. Click Save .
  9. Click Delete .
  1. The user group is removed from Groups list.
  2. The user group is removed from Group Mapping expander in the LDAP tab.

 

Manual Synchronization of Domain Groups

Scenario
If a user needs to be added before the automatic synchronization takes place, you need to trigger a manual synchronization.

  1. A connection to the domain server is established.
  2. Synchronization is enabled.
  1. Select Project > System Settings > Security.
  2. Select the LDAP tab and open the Group Mapping expander.
  3. Select the Synchronization check box for all domain groups you want to synchronize.
    NOTE: Cleared Synchronization check boxes are not synchronized in that case.
  4. Click Save .
  5. Click Synchronize .
  1. The Status changes to Succeeded.
  2. New domain users are added to the Desigo CC user group. Their settings are not configured yet.
  3. More information is available in the Detail Log tab.
