Enabling the IEE Occupancy Tool Events

You can enable one or more IEE Occupancy Tool events in Desigo CC. For background information, see IEE Occupancy Tool Commands, Events, and Symbols.

Prerequisites

  • The IEEOccupancyTool_Adapter is installed and the Windows service is properly running.
  • Option 1 – Enable Events for All IEE Occupancy Tool Zones
  • Option 2 – Enable Events for Individual IEE Occupancy Tool Zones
