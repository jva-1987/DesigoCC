Working with the System Browser Library Browser

The Graphics Library Browser in System Browser, allows you toggle between a view that displays all the available symbols and graphic template objects in your project libraries.

Select the topic related to your task:

Access the Library Browser

You can access the Library Browser by navigating to a Symbol or Graphic Template folder in any of the Libraries folders.

  1. In System Browser, select Management View.
  2. Select Project > Management System > Libraries.
  3. Click either the Symbols folder or the Graphics Template folder.
  1. The Library Browser displays.

 

Create a Customized HQ Symbol

You can customize any existing HQ symbols so that they display a "Regional" or customized version of that symbol in your Project graphics.

  1. In System Browser, select Management View.
  2. Select Project.
  3. Navigate to System Settings, and then expand the Libraries folder.
  4. Expand the L1-Headquarter library folder, and navigate to the Symbols library that contains one or more symbols you would like to create a customized symbol for.
  5. Select the Graphics tab.
  6. The symbols in the symbol libraries display.
  7. Click the symbols you want to create a customized symbol for, and then click Customize.
  8. Click OK.
  9. The symbols are added to the Symbols folder now displayed under the L2-Region library folder and display in the Graphics Editor.
  10. Modify the symbol as required for your region. When finished, click Save .
  1. The updated symbol is now saved in the Regional Symbol folder, and the original symbol remains in the original library folder with Localized Symbol displayed in the upper left-hand corner of the symbol thumbnail image.

 

Create a Symbol Folder in Your Library

  1. You want to add a symbol folder, (a library block) to an existing library (library object) in your project that does not already have a symbol folder. Once you add the folder, you can import symbols into the folder.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries.
  3. Navigate to and click the library block where you want to add a symbol folder.
  4. From the Library Configurator tab, click Add new object , and then select New Library Object.
  5. The New library – GMS_Library_Object displays.
  6. Complete the fields in the Library Configurator General Settings section. You must complete the Discipline, System, Type and Version fields.
  7. Click Save data .
  8. The Save Object As dialog box displays.
    • From the drop-down list box, select the View you want to save the object to.
    • In the Name field, type a name for the new node or keep it as is.
    • In the Description, type a description for the object or keep the text as is. Click the flag icon, and repeat for the available languages. Click Ok when you are done.
  9. Click OK.
  1. The new child graphic node is created and displays in the designated library in System Browser. You can now Import symbols into the Symbol folder you just created.

 

Create a Symbol from the Library Browser

  1. You are in the symbols Library Browser and want to create a new symbol.
  2. System Manager is in Engineering mode.
  1. Click Edit .
  2. The Graphics Editor displays.
  3. Click Create New , and select New Symbol.
  4. An untitled and tabbed Symbol template displays in the Graphics Editor work area.
  5. Draw, design, and assign evaluations and mappings to the Symbol as necessary.
  6. Click Save .
  7. The Save dialog box displays.
  8. From the Project folder, navigate to the appropriate library and click the associated Symbols folder.
  9. Enter a name in the Name field, and click Save.
  1. The symbol is saved in the designated Symbols folder in two file formats: PNG and .CCS. The symbol can be previewed in the Graphics Library Browser.

 

Delete a Customized Symbol

  1. You have a customized symbol you want to delete. Deleting the customized symbol does not affect the original symbol.
  2. System Manager is in Engineering mode.
  1. Navigate to the appropriate Library where the customized symbol is located.
  2. If you place the mouse over the original symbol, the tooltip lists the locations of the customized symbols, or, if you use the Library Browser's List View to display the symbols, the Customized column, displays the location of any customized symbols.
  3. Click Delete .
  4. Click Yes.
  1. The customized Symbol is deleted from the library, and the original Symbol no longer displays Customized .

 

Edit a Symbol from the Library Browser

You can initiate the editing of a symbol or Graphic from the System Browser's Library Browser.

  1. From the Library Browser, click the symbol that you want to edit.
  2. The symbol background color is now shaded in a light orange.
  3. Right-click the selected symbol, and from the menu that displays on right-click, select Edit.
  4. The Graphics Editor is spawned, and the symbol is displayed on the active canvas.
  5. Make your edits to the Symbol. When you are finished, click one of the following:
    • Save to save the Symbol with the same name and to the same location.
    • Save As to rename the Symbol and\or save it to another library location. Enter the new Name for your Symbol and navigate to the new location for the Symbol, and then click Save.
  6. Select Home , to return to the Library Browser.
