Create a Customized CS11 Device Library

If not already done, any library customization requires the creation of the customized library structure.

  1. Select System Settings > Libraries > L1-Headquarter > Fire > Device > CS11.
  2. Select the Library Configurator tab.
  3. Click Customize the entire library to a lower level .
  4. A confirmation message box displays.
  5. Click OK.
  6. The Digital Zone library is duplicated under the allowed customization level. For example, System Settings > Libraries > L4-Project > Fire > Device > CS11.
    You can now create a custom alarm table. See Configure a New Alarm Table in a Customized CS11 Device Library
