Setting CS1115/FC330 Configuration

Scenario

You want to use the CS1115 SORIS Adapter online tool to configure the fire system for the Desigo CC management station. For general background information, see CS1115/FC330.

Prerequisites

  • The CS1115 SORIS Adapter is running. See Installing and Starting the CS1115 SORIS Adapter.
  • The Adapter integration settings have been completed. See Integrating CS1115/FC330.
