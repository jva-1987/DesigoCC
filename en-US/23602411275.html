Import the Key File

You have Client/FEP connected to the server and you want to import the same Windows key as on server.

  1. The same Windows key file as that on the server is available on the disk of the Client, FEP, or server.
  2. The system account is configured.
  1. In the Security expander, select Import key.
  2. Click Browse.
  3. In the Open dialog box, do the following:
    a. Locate and select the source directory path where the key file is located on the disk.
    b. Click Open.
  4. Type in the password of the key.
    NOTE: You must enter the same password that was entered while exporting the key file on the server. By providing the correct password, the key file is decrypted and imported into the Windows key store.
  5. Click Save System Settings.
  6. A confirmation message displays.
  7. Click OK.
  1. The key file is imported in the Windows store on the computer.
    On the server, the Windows key file that was automatically created in the Windows key store on SMC startup, is overwritten by the newly imported Windows key file.
