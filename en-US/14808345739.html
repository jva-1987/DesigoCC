Removing Mass Notification Control from Fire Operators Rights

You want to prevent one or more fire operators from accessing the mass notification control commands in management stations.

  1. System Manager is in Engineering mode.
  2. A security group has been defined in Security that excludes access to Mass Notification commands.
    See Creating a Security Group to Remove Mass Notification Control.
  3. One or more users have been created for fire operators.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Security.
  3. Select the Security tab.
  4. In the Groups list, select the group that excludes access to Mass Notification commands.
  5. In the Group Configurator expander, in Configured User, select the user or use the CTRL key to select multiple users.
  6. Click Add.
  7. The selected users are now part of the Operators – Full Control without MNS group.
  8. Click Save .

NOTE: Make sure that the same users are not part of other groups, or remove the users from such groups. For more information, see the User Administration section.
