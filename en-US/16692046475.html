Manually Configuring UPS RFC1628 Device Properties

Once you create a new device, it is recommended to import the configuration template file. However, if you want to manually configure the general settings and properties of a device from scratch, or if you want to modify some of the Object IDs after the import, do the following steps.

  1. In System Browser, select Project > Field Networks > [RFC1628 network] > [device].
  2. Open the General Settings, complete the fields, and click Save.
  3. Open the Properties expander. For each property, specify the Object ID and Type, and click Save.

For reference about how to configure general parameters, see General Settings of an SNMP Device in SNMP Device Configuration Workspace.

For reference about how to properly configure Object IDs and Types, see UPS RFC1628 Properties Configuration.
