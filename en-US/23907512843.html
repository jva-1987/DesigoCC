Print a Report

  1. You are viewing a report as PDF or Excel.
  1. Proceed as follows:
    • For a PDF file, click Print .
    • For a file that displays in Excel format, you have to print from the Microsoft menu.
  2. The Print dialog box displays.
  3. Select the configured printer and click Print .
  1. The report is printed.
