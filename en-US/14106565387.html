SNMP Library

The Headquarter SNMP library contains the default set of Data Type definitions and related object models supported by the current version of Desigo CC and is available at the following path: System Settings > Libraries > L1-Headquarter > Global > SNMP.

 

Only Headquarter experts and Customer Support are authorized to modify the SNMP library at L1-Headquarter level.
Depending on their allowed customization level, authorized experts can configure an OPC library at L2-Region, L3-Country, or L4-Project level.

 

The Headquarter SNMP libraries can be used as-is to configure SNMP monitoring of predefined devices.

However, defining a custom device requires creating a new SNMP library that will contain the device’s object models and any other data relevant to it (icons, text groups, symbols, and so on). For instructions see Defining a Custom SNMP Device and Managing the SNMP Libraries. For general background information about system libraries, see the reference section.
