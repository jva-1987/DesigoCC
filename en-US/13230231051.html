Configuring Event Categories in a Library

You can configure an event category for a customized library at Region, Country, or Project level. To do this you must customize the text group and configure the full and short name of the category, and the Summary Propagation Status.

 

  1. You are trained and authorized to work with system libraries at the target customization level. For example, L4-Project.
  2. System Manager is in Engineering mode.
  1. In System Browser, select Management View.
  2. Select Project > System Settings > Libraries > L1-Headquarter > Global > Events > Texts > Category.
  3. In the Text Group Editor tab, click Customize .
  4. Click OK.
  5. The selected text group is cloned under the allowed customization level (for example, L4-Project > Global > Events > Texts).
  6. To modify an existing category, do the following:
    a. In the language column (for example, en-US), click the field that contains the category text and modify the text.
    b. Click outside the field to activate the Save icon.
    c. Click Save .
  7. To add a new category, do the following:
    a. Add new row .
    b. In the Value field, enter a number.
    c. In the language column (for example, en-US), enter the category text.
    d. Repeat step 6.c for each language in the system.
  8. Select Project > System Settings > Libraries > L1-Headquarter > Global > Events > Texts > Category Short.
  9. Repeat steps 3 to 6.
  10. Select Project > System Settings > Libraries > L1-Headquarter > Global > Events > Texts > Propagation Summary Status.
  11. Repeat steps 3 to 6.

 

If the text group customization as described here is not possible (for example, because different extension modules require changes to text groups at the same library level), do the following workflow instead: Configuring Text Group Extensions for Categories.
