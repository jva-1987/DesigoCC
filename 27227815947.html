Project reports

The Project report folder includes a set of project-specific reports.

 

  • Basic Information report
  • Alarming report
  • Data points report
  • Project backup report
  • Databases report
  • Service texts report


    The Service Text report is visible under the Project report folder only if the system contains the relevant data.
